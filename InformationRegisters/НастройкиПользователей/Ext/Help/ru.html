Регистр «Настройки Пользователей»

Предназначен для настройки удобного режима открытия некоторых справочников, документов, печати отчетов. Содержит предопределенные элементы:

  • Контроль корреспонденции счетов - установка флага влечет появление в открытом плане счетов только тех счетов, которые корреспондируются (внесены в справочник корреспонденций) с указанным счетом;
  • Основная дата начала отчетов - при открытии период отчетов будет начинаться с этой даты;
  • Основной каталог файлов пользователя - при сохранении внешних файлов по умолчанию будет предлагаться этот каталог
  • Печать документов без предварительного просмотра
  • Учет по всем организациям - простановка этого флага дает пользователю доступ для создания и просмотра документов и их движений всех организаций, если же этот флаг не установлен, то пользователь сможет создавать и просматривать движения только той организации, которая установлена в его настройках как основная;
  • Основная валюта взаиморасчетов
  • Основная единица по классификатору - при создании новых элементов номенклатуры эта единица будет подставляться автоматически;
  • Основная организация - при создании новых документов эта организация будет подставляться автоматически;
  • Основная ставка НДС - при создании новых элементов номенклатуры эта единица будет подставляться автоматически;
  • Основной ответственный  - при создании новых документов этот элемент будет подставляться автоматически;
  • Основной склад  - при создании новых документов этот элемент будет подставляться автоматически;
  • Основной тип цен продажи   - при создании новых документов этот элемент будет подставляться автоматически.
