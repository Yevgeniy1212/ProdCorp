Регистр сведений "Порядок округления результатов расчета зарплаты"

Предназначен для хранения настройки правил округления результатов расчета по заработной плате.

Измерения регистра:

  • Вид расчета - вид начисления, удержаний, налога, сбора или отчисления, для которого задается правило округления. Если конкретный вид расчета не выбран (выбрано значение "Все виды расчета"), то указанное правило округления будет применятся для всех видов расчета, для которых не заданы собственные правила округления.

 

Ресурсы регистра:

  • Порядок округления - в графе указывается число, до которого производится округление. Например, если указано 1, то результат расчета всегда будет представлять целое число, если выбран порядок округления 0.01, то результат расчет будет содержать два знака после запятой;
  • Метод округления - определяет какое правило округления будет использовано:
    • По арифметическим правилам - значение 1.42 округляется до 1, значение 1.52 - до 2 для порядка округления 1;
    • Всегда в большую сторону - оба значения 1.42 и 1.52 будут округлены до 2 для порядка округления 1;
    • Всегда в меньшую сторону - оба значения 1.42 и 1.52 будут округлены до 1 для порядка округления 1.
