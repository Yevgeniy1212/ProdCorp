Регистр сведений

"Лимиты заявок на расходование денежных средств"

Регистр предназначен для хранения лимитов на суммы заявок на расходование денежных средств, при превышении которых заявка должна быть отправлена на утверждение на вышестоящую инстанцию.  

Для каждой записи указываются следующие данные:

  • Период - дата, начиная с которой действует установленный лимит;
  • Пользователь - пользователь, который оформляет заявку на расходование денежных средств;
  • Валюта - валюта оформляемого документа;
  • Лимит - максимальная сумма, при которой заявка допускается к проведению без утверждения вышестоящими лицами.

 

Внимание!!! Данный регистр не предназначен для самостоятельного использования. Чтобы введенные ограничения вступили в силу, необходимо использовать данные регистра в подсистеме электронного документооборота.
