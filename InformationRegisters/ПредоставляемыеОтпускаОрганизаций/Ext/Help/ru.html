Регистр сведений "Предоставляемые отпуска организаций"

В регистре содержатся сведения о размерах предоставляемых (плановых) трудовых отпусков для каждого вида отпуска. Информация указывается на определенную дату. Сведения о предоставляемых отпусках можно указать:

  • для организации в целом - такой отпуск будет предоставляться всем работникам организации. Форма ввода отпусков, предоставляемых в целом всем работникам организации, вызывается из справочника Организации (по кнопке Перейти) либо из пункта меню "Предоставляемые отпуска организаций".
  • для определенной штатной единицы - такие отпуска будут предоставляться только тем работникам, которые занимают(ли) в периоде действия вида отпуска указанную штатную единицу. Ввод отпусков, предоставляемых штатной единице, производится в форме редактирования штатного расписания организации.
  • для определенного работника в организации (персональный отпуск) - такие отпуска будут предоставляться только указанному работнику в указанной организации. В случае, если работник принят в организацию по внутреннему совместительству, то данный отпуск будет предоставляться вместе с отпусками по основному месту работы. Форма ввода персональных отпусков вызывается из меню Перейти в форме редактирования данных физического лица.

Если один и тот же вид отпуска назначен сразу несколькими способами, то общим размер предоставляемого отпуска данного вида будет определяться как сумма размеров, указанных каждым из способов.

Детальный порядок расчета предоставляемых отпусков можно получить в отчете по отпускам работника организации.

Внимание!  Сведения о предоставляемых отпусках следует вводить, начиная с календарного года, за который остались неиспользованные отпуска у работников организации на момент начала эксплуатации конфигурации "Зарплата и Управление Персоналом для Казахстана". Например, если эксплуатация конфигурации началась с 1 января 2006, и на эту дату в организации имелись работники, которым были еще не предоставлены трудовые отпуска за 2004 год, то в качестве первой даты предоставляемых отпусков следует указать 1 января 2004 года. Погасить отпуска, предоставленные за период с 1 января 2004 по 31 декабря 2005, можно с помощью документа "Компенсация за неиспользованные отпуска работникам организаций" от 31 декабря 2005, заполнив его неиспользованными отпусками на дату документа (включая 31 декабря 2005) и скорректировав на разницу между рассчитанным количеством дней неиспользованного отпуска и фактическим остатком отпусков на 1 января 2006.
