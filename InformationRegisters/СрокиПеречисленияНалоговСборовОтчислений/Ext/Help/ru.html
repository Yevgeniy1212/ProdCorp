Регистр сведений "Сроки перечисления налогов, сборов, отчислений"

В регистре содержатся сведения о сроках перечисления налогов, сборов и отчислений.

  • Месяц - первое число месяца, за который была исчислена сумма налога, сбора, отчисления, либо в котором был выплачен доход, являющий базой обложения соответствующим налогом, сбором, отчислением. Трактовка этого поля зависит от значения, указанного в ресурсе "Порядок определения срока перечисления".
  • Организация - ссылка на головную организацию, для которой следует применять указанный срок. Если значение не заполнено, то соответствующий срок будет применяться для всех организаций, для которых явно не заданы строки в регистре;
  • Вид налога - вид налога, сбора, отчисления из справочника "Налоги, сборы, отчисления";
  • Порядок определения срока перечисления - значение перечисления "Порядок определения срока перечисления", определяющее начиная с какого месяца начинается течение срока:
    • По месяцу налогового периода - срок задается относительно месяца, за который был исчислен соответствующий налог, сбор, отчисление;
    • По месяцу выплаты доходов - срок задается относительно месяца, в котором был выплачен доход, являющийся базой обложения соответствующего налога, сбора, отчисления.
  • Срок перечисления - последняя допустимая дата перечисления суммы налога, сбора, отчисления. Начиная со следующей после указанной даты, платеж считается просроченным и возникает необходимость уплаты пени за несвоевременное перечисление.

Заполнение регистра необходимо производить за каждый календарный месяц. Для упрощения расчета сроков платежей предназначена специальная форма заполнения, вызываемая по нажатию кнопки "Автозаполнение" на верхней командной панели формы.

 В форме автозаполнения необходимо указать:

  • Вид налога, сбора, отчисления, для которого производится расчет сроков перечисления;
  • Порядок определения срока перечисления;
  • Отметить организации, для которых осуществляется расчет сроков. Если отмечены все организации, то в результате заполнения будут сформированы записи регистра с пустым значением измерения Организации (т.е. относящиеся ко всем организациям);
  • Задать период заполнения;
  • Выбрать один из возможных способов расчета сроков.

После нажатия на кнопку "Заполнить" форма автозаполнения будет закрыта и в регистр будут внесены записи о сроках перечисления платежей за соответствующий период.
