Регистр сведений "Актуальные маршруты согласования документов"

Содержит список используемых на данный момент маршрутов согласования и утверждения. Если используется автоматический режим постановки на маршруты, тогда при проведении документа, который на данный момент не проведен и не маршрутизирован в системе, система назначает ему маршрут согласования и утверждения. Если используется ручной режим постановки на маршруты, тогда маршрут можно выбрать вручную из списка в форме работы с согласованием документа. При этом, из всего перечня маршрутов, заданных для вида документов, в назначении маршрута используются только актуальные маршруты – то есть маршруты, находящиеся в списке данного регистра. Наличие или отсутствие маршрута в данном списке не влияет на его структуру и не влияет на его использование для тех документов, для которых он уже назначен. Настройки данного списка используются только при присвоении маршрутов новым документам. В данный список для одного и того же вида документов может быть внесен только один маршрут с признаком «Использовать по умолчанию». Все остальные маршруты списка должны использоваться по условию.
