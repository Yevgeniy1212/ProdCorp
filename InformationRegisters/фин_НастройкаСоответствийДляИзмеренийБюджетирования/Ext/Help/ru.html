Регистр сведений "Настройка соответствий для измерений бюджетирования"

Регистр сведений «Настройка соответствий для измерений бюджетирования» представляет собой универсальное хранилище соответствий первичных и бюджетных данных.
 Заполнение колонок:

  • «Период» - дата, на которую действительно соответствие;
  • «Разрез бюджетирования» - аналитический разрез бюджетирования, для которого настраивается строка соответствий;
  • «Исходное значение» - фиксированное исходное значение, для которого настраивается соответствие данных бюджетирования. Для облегчения ввода значений можно использовать группу реквизитов «Ограничения ввода исходных значений», чтобы указать источник, из которого берутся вводимые в данный момент исходные значения;
  • «Соответствие» - фиксированное значение аналитического разреза бюджетирования, назначаемое в соответствие исходному значению.

Группа элементов «Отбор настроек» позволяет фильтровать и упрощать ввод новых соответствий по значениям отбора.

Настроенные соответствия предназначены для использования при расчете вариантов соответствий первичным данным.
