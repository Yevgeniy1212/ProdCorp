Регистр сведений "Календарное планирование бюджетного процесса"

Для описания сроков выполнения различных задач бюджетного процесса, применяемых для каждого бюджетного периода, используется регистр сведений «Календарное планирование бюджетного процесса».

В поле «Вид задачи» указывается один из видов задач:

·         Планирование – задача создания или участия в редактировании нового бюджета;

·         Контроль планов – задача проверки исполнения задач на планирование;

·         Доработка бюджета – задача корректировки бюджета;

·         Анализ исполнения бюджета – задача выполнения анализа бюджета при помощи отчетов и прочих средств анализа, требуемых организационным регламентом;

·         Уведомления о превышении лимитов при учете фактических данных – задача, формирующаяся при достижении лимита при проведении фактических документов.

Для каждой записи можно ограничить область действия при помощи полей «Сценарий» и «Организация». Также в том случае, если работу над одним и тем же действием необходимо распределить на несколько этапов, выполняемых различными исполнителями, можно задействовать поле «Этап бюджетного процесса», в котором можно указать этап, к которому будет относиться задача, из списка самостоятельно сформированных этапов.

Для записей регистра, помимо записей с видом задачи «Уведомления о превышении лимитов при учете фактических данных», можно также указать следующие поля:

·         «Бюджет» - для указания конкретного бюджета, к которому относится запись;

·         «Вид операции» - для указания вида операции документа «Бюджет», к которому относится запись;

·         «Управленческое подразделение», «Проект» - доступны для заполнения, если бюджет является бюджетом подразделений или проектов соответственно;

·         «Дата исполнения» - определяет срок начала выполнения указанной задачи. Для этого указываются:

o    «Время отсчета», «Единица отсчета» - для указания количества отсчитываемых дней/часов и т.д.

o    «Вид отсчета» (до/после), «Дата отсчета» (Начало бюджетного периода/Окончание бюджетного периода) – порядок определения даты и направления отсчета заданного количества дней/часов

·         «Срок исполнения» - определяет срок, предоставляемый на исполнение задачи с момента даты исполнения.
