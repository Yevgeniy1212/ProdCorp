Регистр сведений «Настройки регулярного контроля финансовых показателей»

Система может автоматически определять значения целевых показателей и лимитов для каждого периода, если по финансовому показателю настроены правила в регистре сведений «Настройки регулярного контроля финансовых показателей» (доступен в меню «Контроль» - «Настройки регулярного контроля финансовых показателей»).

Настройку правил для отдельного показателя можно производить по команде «Регулярный контроль показателя» верхней командной панели формы элемента справочника «Финансовые показатели».

Для каждой настройки контроля указывается период, с которого она действует, а также признак «Применять контроль», при помощи которого можно назначить или отменить контроль по указанному показателю и сценарию.

В поле «Вид контроля» указывается одно из значений:

·         Ограничивающий – жесткий контроль. Означает, что при превышении накопленного за период значения показателя над заданным настройкой проведение превышающего документа будет запрещено;

·         Целевой -  мягкий контроль. При превышении не запрещается проведение документа. Используется для мониторинга исполнения/неисполнения средствами отчетов.

В поле «Использование контролируемых значений» указывается одно из значений:

1.       При планировании – в этом случае созданная настройка будет применяться к плановым данным, вводимым по сценарию, указанному в поле «Сценарий». Размер целевого значения / лимита указывается фиксированной величиной в поле «Значение ограничения».

2.       При исполнении – в этом случае созданная настройка будет применяться к отражению фактических данных. Размер целевого значения будет браться автоматически из вводимых плановых данных по бюджетам по сценарию, указанному в поле «Сценарий». В поле «Вид допустимого отклонения» указывается вид допустимого отклонения плана от факта – либо относительное (в этом случае в поле «Отклонение» указывается процент от плана), либо абсолютное (в этом случае в поле «Отклонение» указывается фиксированный размер допустимого отклонения). В группе «Разрезы контроля» отражаются разрезы планирования, допустимые для показателя, и можно отметить те разрезы, по которым следует контролировать фактические данные. Если разрезы не отмечены, то общая сумма исполнения будет сравниваться с общей суммой плана по показателю, независимо от того, по каким значениям разрезов вводился план.
