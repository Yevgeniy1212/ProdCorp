Регистр сведений "Реестр заметок"

Система позволяет хранить в регистре сведений «Реестр заметок» произвольные текстовые заметки по любому документу/справочнику. Для каждой заметки автор может указать признак ее использования: если заметка личная, то никто кроме него не сможет ее просматривать. Если публичная, то она будет доступна всем пользователям для просмотра. Открытие заметок по объекту осуществляется при помощи кнопки «Заметки» формы списка или документа/элемента справочника.В поле «Дополнительно» можно прикрепить относящийся к заметке дополнительный объект информационной базы.

Заметки автоматически выводятся в формах работы с согласованием документа.
