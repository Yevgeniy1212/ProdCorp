Регистр сведений "Настройка дополнительных разрезов бюджетирования"

Помимо предопределенных в системе разрезов бюджетирования, имеется возможность определить до десяти дополнительных разрезов, ссылающихся на выбранную пользователем справочную информацию. Количество и тип дополнительных разрезов настраиваются в регистре сведений «Настройка дополнительных разрезов бюджетирования».

В данном регистре определяется перечень разрезов (до десяти). В колонке «Измерение» указывается системное значение, определяющее порядковый номер описываемого разреза. В колонке «Разрез» указывается элемент плана видов характеристик «Виды субконто бюджетирования», определяющий как тип, так и представление создаваемого разреза.

После того, как разрез был использован в статьях бюджета, его невозможно удалить из списка дополнительных разрезов. Также если по этому разрезу было введено хотя бы одно значение, нельзя будет изменить тип закрепленного за ним вида субконто.

После создания дополнительных разрезов их можно использовать в статьях бюджетов.

 Информация для технических специалистов: имеет ли значение количество созданных дополнительных разрезов?

Обработка каждого дополнительного разреза увеличивает количество операций, выполняемых системой при вводе документов планирования бюджетов и регистрации фактических данных, следовательно влияет на производительность системы. В связи с этим необходимо обдуманно добавлять дополнительные разрезы бюджетирования, и делать это только при обоснованной необходимости, по возможности используя статьи бюджета для классификации бюджетных данных, либо аналитику «Проект» (если она не используется на предприятии по назначению).

Влияет ли на производительность системы тип созданных разрезов?

Да, если тип разреза составной. При этом нагрузка на систему увеличивается с увеличением количества различных типов, назначенных одному и тому же разрезу. Поэтому следует использовать составной тип разреза только в случае, если нет другого выхода. Лучше добавить еще один разрез несоставного типа (если это не будет противоречить логике предметной области). Например, необходимо планировать затраты на обслуживание или амортизацию внеоборотных активов, перечень которых хранится в двух справочниках – «Основные средства» и «Нематериальные активы». В этом случае логично создать один разрез составного типа, так как оба справочника по логике представляют одну и ту же сущность с точки зрения планирования по данной статье, и могут быть использованы в данной статье одновременно.

Что делать, если нужный справочник отсутствует в списке доступных типов или необходимо использовать другой объект информационной базы?

В этом случае необходимо в конфигураторе расширить тип плана видов характеристик «Виды субконто бюджетирования». Не рекомендуется включать в тип субконто документы. Крайне не рекомендуется использовать числовой, строковый тип или тип «Дата».

 

Действия, выполняемые при включении или расширении состава универсальной аналитики

 

Для включения данной возможности, особенно если в системе уже велось планирование бюджетов, необходимо выполнить следующие шаги:

1. В настройках параметров учета на закладке "Бюджетирование - Основное" установить признак "Использовать универсальную аналитику". Этот признак невозможно будет отключить после их  добавления;

2. Добавить требуемые разрезы в регистре сведений "Настройка дополнительных разрезов бюджетирования" (доступен через меню "Бюджетирование" - "Баланс") или в панели навигации в разделе "Бюджетирование - Структура". В регистре добавляется разрез и соответствующий ему вид субконто бюджетирования из плана видов характеристик "Виды субконто бюджетирования", который будет определять тип и представление разреза. После того, как разрез будет использован в статьях бюджетов, будет невозможно отключить этот разрез или изменить тип соответствующего ему субконто. В дальнейшем эти субконто можно назначать счетам управленческого плана счетов, чтобы получать информацию по счетам бюджета в нужных разрезах;

3. В справочнике "Финансовые показатели" в поле "Используемые дополнительные разрезы" в качестве дополнительных разрезов (помимо контрагентов и номенклатуры) становятся доступными для выбора добавленные разрезы из регистра  "Настройка дополнительных разрезов бюджетирования". Здесь необходимо указать разрезы, по которым будет вестись планирование и учет исполнения для данного финансового показателя, соответствующего статьям бюджетов. После внесения плановых данных по выбранным разрезам, станет невозможным удалить отключить такие разрезы для финансового показателя;

4. Выполнить служебную обработку "Реструктуризация данных бюджетирования при изменении разрезов планирования" (доступна через меню "Бюджетирование" - "Баланс") или в панели навигации в разделе "Бюджетирование - Структура" для предложенного ряда документов. Для этого на каждой закладке необходимо нажать кнопку "Выполнить". Это действие необходимо для корректной работы механизмов бюджетирования в дальнейшем.

Внимание!!! Данную обработку обязательно следует запускать каждый раз после того, как в финансовых показателях начинает использоваться вновь добавленный разрез бюджетирования.
