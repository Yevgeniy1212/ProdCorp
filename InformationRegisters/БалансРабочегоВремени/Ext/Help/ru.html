Баланс рабочего времени

В регистре содержатся сведения о норме рабочего времени для сменных графиков с суммированным учетом рабочего времени. Информация о норме рабочего времени хранится по месяцам года.

Предусмотрена возможность автоматического заполнения норм рабочего времени для графиков работы с помощью кнопки «Первоначальное заполнение».

По заполненным данным можно получить печатную форму баланса рабочего времени.
