Регистр сведений "Сведения о ставках ИПН"

В регистре содержатся сведения о ставках индивидуального подоходного налога, устанавливаемых вне зависимости от облагаемой базы. Для налогов, взносов и отчислений, исчисляемых по шкале ставок в зависимости от размера облагаемой базы, предназначены специализированные регистры сведений. Ставки указываются для резидентов и нерезидентов в разрезе видов доходов. Ставки могут быть внесены в целом для всех организаций информационной базы, в таком случае графа Организация не заполняется. Если размер ставки различается для разных организаций, то следует вносить ставки по каждой организации в отдельности. Приоритет определения ставок, используемых для исчисления сумм налогов, сборов и отчислений, следующий:

  • если найдена строка с заполненной графой Организация, то используется значение ставки из этой строки;
  • иначе используется значение ставки из строки регистра сведений, в которой графа Организация не заполнена.
