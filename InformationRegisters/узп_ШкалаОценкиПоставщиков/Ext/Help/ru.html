Регистр сведений "Значения шкал оценки поставщиков"

Регистр сведений "Значение шкал оценки поставщиков" хранит информацию о значениях, установленных ответственным лицом для каждой шкалы в соответствии с выбранным типом значения шкалы.

Чем лучше считается значение показателя, тем выше устанавливается для него оценка. Максимальное значение оценки может быть равным 5, минимальное – 1.

Стоит обратить внимание, что изменение количества значений оценок возможно в том случае, если данная шкала не использовалась. Если шкала уже используется для оценки поставщиков, то в ней возможно изменять только наименование самой шкалы и наименования оценок.

При создании шкалы необходимо контролировать результаты. Для нормального функционирования шкалы, не должно быть оценок с одинаковыми результатам, а также оценок с нулевыми результатами. Также, при вводе результатов нежелательно разрывать натуральный ряд чисел.

Например:

1) Для шкалы "Качество продукции" установлен тип значения "Значения критериев оценки поставщиков". В соответствии с данным типом значения шкалы заполняется регистр сведений "Значение шкал оценки поставщиков":

·         значение "Отличное" - устанавливается результат 5;

·         значение "Хорошее" - устанавливается результат 4;

·         значение "Удовлетворительное" - устанавливается результат 3.

2)     Для шкалы "Срок поставки товаров" (в днях) установлен тип значения "Число". В соответствии с данным типом значения шкалы заполняется регистр сведений "Значение шкал оценки поставщиков":

·         значение "от 1 до 5" - устанавливается результат 5;

·         значение "от 6 до 10" - устанавливается результат 4;

·         значение "от 11 до 15" - устанавливается результат 3;

·         значение "от 16 до 20" - устанавливается результат 2;

·         значение "от 21 до 25" - устанавливается результат 1.
