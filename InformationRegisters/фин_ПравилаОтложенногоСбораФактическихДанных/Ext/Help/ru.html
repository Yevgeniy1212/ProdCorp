Регистр сведений "Правила отложенного сбора фактических данных"

Для возможности автоматического сбора фактических данных в отложенном режиме для финансовых показателей (по кнопке «Отложенный сбор факта» верхней командной панели) необходимо настроить правила сбора. Они хранятся в регистре сведений «Правила отложенного сбора фактических данных».

Для каждой записи регистра настраиваются следующие параметры.

Указывается сам показатель, дата ввода настройки в действие (т.к. они могут быть периодическими), а также источников данных для факта из справочника «Источники данных расчетов бюджетирования».

В зависимости от показателей учета, установленных для финансового показателя, будет необходимо для сумм и/или количества указать:

·       Показатель источника – поле источника, содержащее нужный ресурс;

·       Коэффициент пересчета – коэффициент, на который будет умножаться поле источника. Если необходимо взять его в исходной величине, то коэффициент должен быть равен единице;

·       Способ определения валюты – каким образом определяется валюта операции.
