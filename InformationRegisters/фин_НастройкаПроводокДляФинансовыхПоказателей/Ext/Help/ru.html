Регистр сведений "Настройка проводок для финансовых показателей"

Для того, чтобы при регистрации плановых и фактических данных бюджетирования автоматически формировать проводки по управленческому плану счетов, для каждого финансового показателя производится настройка применяемых шаблонов проводок. Закрепленные за финансовыми показателями шаблоны хранятся в регистре сведений «Настройка проводок для финансовых показателей». Редактирование доступно также из формы финансового показателя, по кнопке «Проводки по счетам бюджета» верхней командной панели.

Каждому финансовому показателю может быть назначено один или несколько шаблонов. Если ему шаблоны не назначены, то его данные не будут отображаться на управленческом плане счетов. Шаблоны выбираются из справочника «Шаблоны проводок по счетам бюджетирования».
