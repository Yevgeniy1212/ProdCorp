Обновление кодов строк декларации на 2012 год.

 

Для обновления регистра сведений необходимо нажать на кнопку «Проверить устаревшие записи».

 

В таблице «Существующие коды строк декларации по налогу на прибыль» отображается список записей, которые устарели и возможны для обновления. Также в этой таблице в колонке «код строки» существует возможность просмотра наименования кода строки. Для этого необходимо нажать на «многоточие» в ячейке.

 

В таблице «Новые коды строк декларации по налогу на прибыль за 2012 год» отображается список записей, которые будут добавлены и для них установлен новый код строки. Также в этой таблице в колонке «Код строки» существует возможность просмотра наименования кода строки. Для этого необходимо нажать на «многоточие» в ячейке.

 

После того, как в форме будут отображены существующие записи регистра сведений и записи для добавления в регистр, необходимо внимательно проверить данные в таблице "Новые коды строк декларации по налогу на прибыль за 2012 год". При необходимости, можно удалить либо отредактировать строки в данной таблице. Затем нажать кнопку "Добавить". Записи из таблицы «Новые коды строк декларации по налогу на прибыль за 2012 год» будут добавлены в регистр сведений «Коды строк налоговой декларации по налогу на прибыль».
