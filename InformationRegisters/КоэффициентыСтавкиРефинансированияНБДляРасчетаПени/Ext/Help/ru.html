Регистр сведений "Коэффициенты ставки рефинансирования НБ для расчета пени/коменсации"

В регистре содержатся сведения о коэффициентах, приняемых к ставке рефинансирования Национального Банка Республики Казахстан при расчете пени за несвоевременное перечисление налогов, сборов, отчислений, заработной платы.

  • Организация - ссылка на головную организацию, для которой следует применять указанный коэффициент. Если значение не заполнено, то соответствующий коэффициент будет применяться для всех организаций, для которых явно не заданы строки в регистре;
  • Вид - вид налога, сбора, отчисления из справочника "Налоги, сборы, отчисления", коменсация за задержку выплаты зарплаты;
  • Коэффициент - значение коэффициента.
