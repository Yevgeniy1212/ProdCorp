Отчет "Декларация по социальному налогу, форма 600.00 (XML)" за 2008 год

Отчет "Декларация по социальному налогу, форма 600.00 (XML)" предназначен для составления декларация по социальному налогу. Форма адаптирована к версии ЭФНО 1.6.1.41 от 13.02.2009г.

Если в настройках параметров учета (меню Предприятие\Настройка параметров учета) установлен признак поддержки работы со структурными подразделениями, то при открытии отчета необходимо указать налогоплательщика и налоговый комитет, в который будет предоставлен отчет. Если работа со структурными подразделениями в налоговом учете не поддерживается, то при открытии отчета требуется выбрать только налогоплательщика, налоговый комитет будет определен по реквизиту "Налоговый комитет" из справочника "Организации".

Для проверки правильности составления декларации предназначены отчет "Справка-расчет "Индивидуальный подоходный налог", в котором в качестве анализируемого показателя следует выбрать "Социальный налог", и отчет "Анализ начислений работником организаций", позволяющий проверить правильность указания кодов строк Декларации для разных видов начислений.

СОДЕРЖАНИЕ

1. Декларация по социальному налогу

2. Перечень выплат физическим лицам, облагаемых и не облагаемых социальным налогом (Приложение № 1 к Декларации)

 

Декларация по социальному налогу

- Заполнение строки 600.00.001 (социальный налог, исчисленный с применением ставок, установленных п.1 ст. 317 Налогового кодекса)

В строке выводятся суммы социального налога, исчисленного за работников, не являющихся иностранными специалистам, согласно данным регистра накопления "СН исчисленный".

- Заполнение строки 600.00.007 (всего исчислено налога)

В строке выводятся общие суммы социального налога, исчисленного в отчетном периоде, определяемые по формуле:

600.00.007 = 600.00.001 + 600.00.002 - 600.00.005

Внимание! Автоматическое заполнение строк 600.00.002, 600.00.003, 600.00.004 и 600.00.005E не поддерживается, поэтому в случае наличия данных, отражаемых в перечисленных строках, их следует заполнить вручную.

 

Перечень выплат физическим лицам, облагаемых и не облагаемых социальным налогом (Приложение № 1 к Декларации)

Для указания того, в какой строке Декларации по социальному налогу следует показывать суммы начислений по разным видам, в форме вида расчета начисления предназначен реквизит "Код строки декларации по СН". Строка 600.01.017 (обязательные пенсионные взносы) заполняется суммами исчисленных обязательных пенсионных взносов за отчетных период по данным регистра накопления "ОПВ расчеты с фондами".
