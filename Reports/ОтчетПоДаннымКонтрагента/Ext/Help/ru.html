Отчет "Отчет по данным контрагента"

Данный отчет показывает сводные данные, зарегистрированные для контрагента, включая информацию о его контактных лицах. Отчет может быть так же сформирован по группе контрагентов или по всем контрагентам(если в поле "контрагент" ничего не выбрано).

Отчет вызывается по кнопке "Отчет" из формы контрагента. Так же отчет можно вызвать из меню "Отчет" формы списка контрагентов, в данном меню можно выбрать вариант формирования отчета: по выбранному контрагенту или по всем.
