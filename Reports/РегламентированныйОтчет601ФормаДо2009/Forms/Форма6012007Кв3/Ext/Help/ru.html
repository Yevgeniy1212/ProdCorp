Отчет "Расчет суммы социального налога, подлежащего уплате по структурному подразделению, форма 601.00 (XML)" за 2007-2008 год.

Отчет предназначен для составления расчета суммы социального налога, подлежащей уплате по структурному подразделению, форма 601.00 с возможностью последующей выгрузки в формат XML. Форма адаптирована к версии ЭФНО 1.6.1.41 от 13.02.2009г.

Составление расчета поддерживается в случае, если в настройках параметров учета (меню Предприятие\Настройка параметров учета) установлен признак поддержки работы со структурными подразделениями. При открытии отчета необходимо указать налогоплательщика (головную организацию) и налоговый комитет, в который будет предоставлен отчет.

Для проверки правильности составления Расчета предназначен отчет "Справка-расчет "Индивидуальный подоходный налог", в котором в качестве анализируемого показателя выбран "Социальный налог".

 

СОДЕРЖАНИЕ

1. Расчет суммы социального налога, подлежащего уплате по структурному подразделению

2. Дополнительная форма к строке 601.00.003

 

Расчет суммы социального налога, подлежащего уплате по структурному подразделению

- Заполнение строки 601.00.001

В строке выводятся суммы исчисленного социального налога в целом по налогоплательщику на основании данных регистра "СН исчисленный".

- Заполнение строки 601.00.002

В строке выводятся суммы исчисленного социального налога за головную организацию (самого налогоплательщика) на основании данных регистра "СН исчисленный" по условию "Структурная единица = Налогоплательщик".

- Заполнение строки 601.00.003

В строке выводятся суммы исчисленного социального налога по филиалам/представительствам на основании данных регистра "СН исчисленный" по условию "Структурная единица Не равно Налогоплательщик".

 

Дополнительная форма к строке 601.00.003

- Заполнение графы A (номер строки)

В графе выводится номер строки по порядку в дополнительной форме.

- Заполнение графы B (РНН филиала/представительства)

В графе выводится РНН филиала/представительства. Для структурной единицы, описываемой элементом справочника "Подразделения организаций", РНН определяется из реквизита РНН соответствующей организации-владельца подразделения. Для структурной единицы, описываемой элементом справочника "Организации", РНН подставляется непосредственно из реквизита РНН.

- Заполнение графы C (сумма социального налога по филиалу/представительству)

В графе выводится сумма социального налога по соответствующей структурной единице из регистра накопления "СН исчисленный".

- Заполнение графы D (код налогового органа по месту уплаты налога)

В графе выводится код налогового органа, в котором зарегистрирована структурная единица (филиал), определяемый из графы "Налогоплательщик" регистра сведений "Исчисление налогов структурных единиц". В качестве кода налогового органа используются первые 4 цифры номера РНН налогового комитета.
