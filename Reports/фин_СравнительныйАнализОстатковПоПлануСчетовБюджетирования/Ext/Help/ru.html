Отчет "Сравнительный анализ остатков по плану счетов бюджетирования"

Отчет позволяет получить отчет произвольной настраиваемой формы для сравнительного анализа остатков по счетам бюджетирования. При этом между собой могут сравниваться остатки разных сценариев, разных периодов, плановые и фактические.

На закладке «Данные» помимо общих настроек указываются следующие настройки:

·         Статья баланса – счет или группа счетов, данные по которым необходимо вывести в отчет. Если не указана, выводятся данные по всему плану счетов.

·         В поле «Данные для анализа» вводятся строки (минимум две), определяющие сравниваемые данные. За эталон принимаются данные, задаваемые первой строкой. В каждой строке указываются данные для сравнения (т.е. сценарий планирования, если же нужны фактические данные, то сценарий не указывается), а также дата остатков.

На закладке «Структура» помимо общих настроек заполняется поле «Виды аналитических данных», определяющее виды аналитических данных, выводимых в отчете:

·         Эталон – абсолютное значение эталонных данных (данных первой строки поля «Данные для анализа»);

·         Исполнение, абс. – абсолютное значение сравниваемых данных;

·         Исполнение, % - процентное отношение сравниваемых данных к эталонным;

·         Отклонение, абс. – абсолютная разница между сравниваемыми и эталонными данными;

·         Отклонение, % – разница между сравниваемыми и эталонными данными в процентном выражении по отношению к эталонным данным;

Указанные виды аналитических данных рассчитываются для каждого из выбранных показателей отчета.

На закладке «Дополнительно» помимо общих настроек указываются следующие настройки:

·         Заголовок отчета – текстовый заголовок, выводимый в отчете;

·         Аналит. поля в одной колонке – определяет вывод аналитических полей по показателю либо в нескольких колонках, либо в одной друг под другом.
