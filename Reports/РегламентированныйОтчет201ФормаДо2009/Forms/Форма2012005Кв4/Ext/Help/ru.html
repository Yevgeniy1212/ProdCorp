Отчет "Расчет по индивидуальному подоходному налогу, форма 201.00 (XML)" за 2005 год.

Отчет "Расчет по индивидуальному подоходному налогу, форма 201.00 (XML)" предназначен для составления расчета по ИПН по выплаченным доходам, облагаемым у источника выплаты. Форма адаптирована к версии ИСИД 1.6.1.25 от 30.09.2008г.

Если в настройках параметров учета (меню Предприятие\Настройка параметров учета) установлен признак поддержки работы со структурными подразделениями, то при открытии отчета необходимо указать налогоплательщика и налоговый комитет, в который будет предоставлен отчет. Если работы со структурными подразделениями в налоговом учете не поддерживается, то при открытии отчета требуется выбрать только налогоплательщика, налоговый комитет будет определен по реквизиту "Налоговый комитет" из справочника "Организации".

В отчет, кроме данных о работниках организации, также могут быть включены сведений о выплаченных доходах получателей по исполнительным листам. Для этого предназначен флажок "Включать в отчет получателей алиментов" на закладке "Настройка расчета". При установленном флажке данные о получателях по исполнительным листам будут включены в отчет.

Если необходимо исключить из отчета данные по физическим лицам - нерезидентам РК на закладке "Настройка расчета" установить флажок "Исключать физлиц-нерезидентов (в настройках гражданства которых, установлен признак "Не является налоговым резидентом РК")".

Для проверки правильности составления Расчета предназначен отчет "Справка-расчет "Индивидуальный подоходный налог", позволяющий в произвольных разрезах проанализировать корректность формирования налогового отчета.

 

СОДЕРЖАНИЕ

1. Расчет по индивидуальному подоходному налогу

2. Дополнительная форма к форме 201.00

 

Расчет по индивидуальному подоходному налогу

- Заполнение строки 201.00.001

В строке выводятся данные о начисленных доходах физическим лицам, зарегистрированные в регистр накопления "ИПН сведения о доходах" за соответствующий месяц налогового периода. Если установлен признак включения в отчет данных по получателям алиментов, то к этой сумме также будут добавлены удержанные за соответствующие месяцы суммы по исполнительным листам, определяемые как приход по регистру накопления "Взаиморасчеты организаций с получателями по ИЛ".

- Заполнение строки 201.00.002

В строке выводятся данные о доходах, облагаемых индивидуальным подоходным налогом, определяемых из данных дополнительной форме к форме 201.00 по формуле:

201.00.002 = графа F (начислено доходов) - G (налоговые вычеты) - H (обязательные пенсионные взносы) - I (добровольные пенсионные взносы) - J (суммы, направленные на погашение вознаграждения по жилищным займам) - K (страховые премии) - L (доходы, не подлежащие налогообложению)

- Заполнение строки 201.00.003

В строке выводятся данные о суммах индивидуального подоходного налога, исчисленного за соответствующие месяцы налогового периода, определяемого как приход по регистру накопления "ИПН расчеты с бюджетом" для вида строк "Исчисление".

- Заполнение строки 201.00.004

В строке выводятся данные о сумме задолженности по доходам, невыплаченным физическим лицам на конец последнего месяца периода составления Расчета, определяемой как конечный остаток по следующим регистрам накопления:

    • Взаиморасчеты с работниками организаций - задолженность по текущим расчетам с работниками (соответствует кредитовому остатку по счету 3350 в бухгалтерском учете);
    • Взаиморасчеты с депонентами организаций - задолженность по депонированной заработной плате (соответствует кредитовому остатку по счету 3384 в бухгалтерском учете);
    • Взаиморасчеты организаций с контрагентами-физлицами - задолженность по расчетам с контрагентами-физлицами, у которых приобретались активы и/или услуги (оформляется с помощью документа "Расчет налогов при поступлении активов и услуг");
    • Взаиморасчеты организаций с получателями по ИЛ - задолженность, связанная с расчетами по исполнительным листам. Если в учетной политике по налоговому учету не установлен признак "Засчитывать в выплаченные доходы удержания по исполнительным листам", то включается остаток по неперечисленным сумма удержаний по исполнительным листам и суммам почтового/комиссионного сбора как задолженность физлицу-работнику организации. Если при формировании отчета на закладке "Настройка расчета" включен флажок "Включить в отчет получателей алиментов", то в невыплаченную задолженность также включаются неперечисленные суммы удержаний по исполнительным листам физлицам-получателям алиментов.

- Заполнение строки 201.00.005

В строке выводятся данные о выплаченных физлицам доходах по данным регистра накопления "Выплаченные доходы работникам организаций по налоговому учету". Если при формировании отчета установлен флажок "Включить в отчет получателей алиментов", то в выплаченные доходы будут включены фактически перечисленный суммы удержаний по исполнительным листам получателям алиментов.

- Заполнение строки 201.00.006

В строке выводятся данные о суммах индивидуального подоходного налога, подлежащего перечислению в бюджет. Информация собирается на основании записей-расходов по регистру накопления "ИПН расчеты с бюджетом" с видом строки "Удержание". В зависимости от настроек учетной политики по налоговому учету "При начислении ИПН и ОПВ принимать исчисленные суммы к учету как удержанные" эта сумма равняется фактически исчисленному налогу (если настройка "При начислении ИПН и ОПВ принимать исчисленные суммы к учету как удержанные" включена), либо определяется в процессе расчета документа "Удержание ИПН и ОПВ по налоговому учету" (настройка "При начислении ИПН и ОПВ принимать исчисленные суммы к учету как удержанные" отключена).

- Заполнение строки 201.00.007

В строке выводятся данные о доходах, с которых удерживаются (начисляются) обязательные пенсионные взносы. Данная строка определяется как фактически исчисленная сумма обязательных пенсионных взносов (графа H дополнительной формы), умноженная на 10.

- Заполнение строки 201.00.008

В строке выводятся данные о суммах обязательных пенсионных взносов, подлежащих перечислению в пенсионные фонды, которые определяются на основании данных о приходах в регистре накопления "ОПВ подлежит перечислению в фонды" с видом строки "Удержание". На расчет суммы взносов, подлежащий перечислению, влияет настройка в учетной политике по налоговому учету "При начислении ИПН и ОПВ принимать исчисленные суммы к учету как удержанные" (см. порядок заполнения строки 201.00.006).

- Заполнение строки 201.00.009

В строке выводятся данные о доходах физических лиц, с которых исчисляются социальные отчисления, определяемых как значение строки 201.00.010, умноженное на 100/процент социальных отчислений за соответствующий месяц налогового периода.

- Заполнение строки 201.00.010

В строке выводятся данные о суммах социальных отчислений, исчисленных за соответствующий месяц налогового периода, определяемые как приходы по регистру накопления "СО расчеты с фондами" с видом строки "Исчисление".

 

Дополнительная форма к форме 201.00

В форме, кроме данных о работниках организации, доступен также вывод сведений о начисленных и выплаченных доходов индивидуальным предпринимателям, адвокатам и частным нотариусам, зарегистрированных документом "Регистрация прочих доходов в целях налогообложения". Для этого предназначен флажок "Включить индивид. предпринимателей и частных нотариусов" в дополнительной форме отчета.

- Заполнение графы A (Номер строки)

В графе выводится номер строки по порядку в дополнительной форме.

- Заполнение графы B (ФИО физического лица)

В графе выводится фамилия, имя, отчество физического лица. Если строка отчета содержит сведения о доходах и налогах по физлицу-работнику организации, то ФИО физлица определяется из регистра сведений "Фамилия, имя, отчество физического лица", в случае отсутствия записи о полной фамилии, имени и отчестве физлица в регистре, в отчет выводится наименование из справочника "Физические лица".

- Заполнение графы B (РНН физического лица)

В графе выводится РНН физического лица из справочника "Физические лица" или справочника "Контрагенты", если строка относится к индивидуальному предпринимателю, адвокату или частному нотариусу.

- Заполнение графы D (Гражданство физического лица)

Графа заполняется, если физлицо не является налоговым резидентом Республики Казахстан - в регистре сведений "Гражданство физлиц" на дату конца периода формирования отчета для него стоит соответствующий признак.

- Заполнение графы E (Статус налогоплательщика)

Графа заполняется символом "1", если строка отчета относится к индивидуальному предпринимателю, адвокату или частному нотариусу (см. описание настройки отчета "Включить индивид. предпринимателей и частных нотариусов").

Графа заполняется символом "2", если физлицо претендует на получение льготы в соответствии с подпунктом 12) статьи 144 Налогового кодекса (является инвалидом I, II или III гр. согласно данным регистра сведений "Сведения об инвалидности физлиц", либо лицом, приравненным к участникам ВОВ по данным регистра сведений "Сведения о физлицах участниках ВОВ и приравненных к ним").

- Заполнение графы F (Начислено доходов)

Графа заполняется суммой начисленного физлицу дохода, зарегистрированного в регистре накопления "ИПН сведения о доходах" за месяцы налогового периода, входящие в период формирования отчета. Если в основной форме отчета отмечена настройка "Включать в отчет получателей алиментов", то для строк, относящихся к получателям по исполнительным листам, будут рассчитаны суммы удержаний за период отчета (см. порядок заполнения строки 201.00.001).

- Заполнение графы G (Налоговые вычеты)

Графа заполняется суммой налоговых вычетов, предоставленных физлицу в соответствии с подпунктом 1) пункта 1 статьи 152 Налогового кодекса при расчете индивидуального подоходного налога документом "Расчет ИПН, ОПВ и удержаний".

- Заполнение графы Н (Обязательные пенсионные взносы)

Графа заполняется суммой обязательных пенсионных взносов, исчисленных с доходов физлица в соответствии с пенсионным законодательством Республики Казахстан и относимые на вычеты в соответствии с подпунктом 3) пункта 1 статьи 152 Налогового кодекса при расчете документа "Расчет ИПН, ОПВ и удержаний" (на основании данных регистра "ОПВ расчеты с фондами")

- Заполнение графы I (Добровольные пенсионные взносы)

Автоматизированное заполнение данной графы не поддерживается.

- Заполнение графы J (Суммы, направленные на погашение вознаграждения по жилищным займам)

Графа заполняется суммами, направленными на погашение вознаграждения по жилищным займам и промежуточным жилищным займам, полученным физическим лицом-резидентом Республики Казахстан в жилищных строительных сберегательных банках на проведение мероприятий по улучшению жилищных условий на территории Республики Казахстан, в соответствии с законодательством Республики Казахстан о жилищных строительных сбережениях, и относимые на вычеты, согласно подпункту 7) пункта 1 статьи 152 Налогового кодекса (см. документ "Заявление на предоставление вычета по ИПН").

- Заполнение графы K (Страховые премии)

Графа заполняется суммами страховых премий, вносимых в свою пользу физическим лицом по договорам накопительного страхования и относимых на вычеты согласно подпункту 6) пункта 1 статьи 152 Налогового кодекса (см. документ "Заявление на предоставление вычета по ИПН").

- Заполнение графы L (Доходы, не подлежащие налогообложению)

Графа заполняется суммами доходов, не подлежащими в соответствии со статьей 144 Налогового кодекса. В графе указываются суммы начислений, для которых в форме настройки вида расчета установлен порядок обложения ИПН "Не облагается целиком", а также льготы, предоставленные физлицу-инвалиду либо участнику ВОВ, при расчете индивидуального подоходного налога документом "Расчет ИПН, ОПВ и удержаний".

- Заполнение графы M (Индивидуальный подоходный налог)

Графа заполняется суммами исчисленного индивидуального подоходного налога по записям-приходам по регистру накопления "ИПН расчеты с бюджетом" с видом строки "Исчисление" (см. также порядок заполнения строки 201.00.003).

- Заполнение графы N (Выплачено доходов)

Графа заполняется суммами выплаченных физлицу доходов. Порядок выборки данных для заполнения графы аналогичен строке 201.00.005.

- Заполнение графы O (Выплаты, не облагаемые социальным налогом, за исключением ОПВ)

Графа заполняется суммам доходов, начисленных физлицу, которые не облагаются социальным налогом (в форме настройки вида расчета установлен порядок обложения для СН "Не облагается целиком"). Если при формировании отчета установлен признак "Включать в отчет получателей алиментов", то эта графа также будет включать суммы удержаний по исполнительным листам для физлиц-получателей алиментов.

- Заполнение графы P (Выплаты, облагаемые социальным налогом)

Графа рассчитывается по формуле:

Графа P = Графа F (начислено доходов) - Графа H (Обязательные пенсионные взносы) - Графа O (Выплаты, не облагаемые социальным налогом, за исключением ОПВ)

- Заполнение графы Q (Социальный налог)

Графа заполняется суммами социального налога, исчисленного по физическому лицу, за месяцы налогового периода, входящие в отчетный период, согласно данным регистра накопления "СН исчисленный".

- Заполнение графы R (Расходы работодателя, с которых исчисляются социальные отчисления)

Графа заполняется аналогично строке 201.00.009.

- Заполнение графы S (Сумма социальных отчислений)

Графа заполняется аналогично строке 201.00.010.
