Отчет "Ведомость амортизации ОС"

ОС ® Ведомость амортизации ОС

Отчет предназначен для анализа данных по основным средствам (ОС).

В отчете можно проанализировать данные по бухгалтерскому и управленческому учету, установить отбор по материально-ответственному лицу (МОЛ), вывести дату принятия к учету и др.

Счета бухгалтерского учета, по которым собираются данные, определяются из регистра сведений Счета учета ОС: выбираются все возможные значения реквизитов Счет учета (БУ) и Счет начисления амортизации (БУ) в разрезе организаций (структурных подразделений).

Дополнительные реквизиты Счет учета БУ и Счет начисления амортизации БУ определяются из регистра сведений Счета учета ОС на конец отчетного периода в разрезе организаций.

Если в конфигурации на счетах учета ОС установлены субконто Подразделения и (или) Работники организаций (МОЛ), то по умолчанию в состав группировок добавляются, соответственно, группировки Подразделение (БУ) и МОЛ(БУ).

Если аналитический учет ОС по Подразделениям и МОЛ не ведется, то по умолчанию добавляются группировки Подразделение (упр.) и МОЛ (упр.). Данные для этих группировок определяются из регистра сведений Местонахождение ОС (бухгалтерский учет).

Показатель Первоначальная стоимость получается из регистра сведений Первоначальные сведения ОС (бухгалтерский учет).

Показатели Ликвидационная стоимость и Стоимость для вычисления амортизации определяются из регистра сведений Параметры амортизации ОС (бухгалтерский учет).

Показатель Остаточная стоимость рассчитывается как разность показателей Стоимость на конец периода - Амортизация на конец периода.

Данные из регистров сведений получаются на конец отчетного периода.

Настройка Исключать внутренние обороты отключает вывод в отчет данных о проводках, в которых Счет = Кор. счету и ОС Дт = ОС Кт.
