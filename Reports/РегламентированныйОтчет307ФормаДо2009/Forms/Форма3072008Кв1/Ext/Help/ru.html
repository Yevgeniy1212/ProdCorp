Отчет «Реестр счетов-фактур по товарам (работам, услугам), приобретенным в течение налогового периода, форма 307.00 (XML)» за 2008 год.

Отчет «Реестр счетов-фактур по товарам (работам, услугам), приобретенным в течение налогового периода» предназначен для составления отчетности по полученным счетам-фактурам.

Форма адаптирована к версииНДС 1.6.1.21 от 03.02.2009г.

Если в настройках параметров учета (меню Предприятие\Настройка параметров учета) установлен признак поддержки работы со структурными подразделениями, то при открытии отчета необходимо указать налогоплательщика и налоговый комитет, в который будет предоставлен отчет. Если работы со структурными подразделениями в налоговом учете не поддерживается, то при открытии отчета требуется выбрать только налогоплательщика, налоговый комитет будет определен по реквизиту "Налоговый комитет" из справочника "Организации".

Отчет может быть сформирован по налогоплательщику в целом, либо по выбранным структурным подразделениям налогоплательщика, не являющихся самостоятельными плательщиками НДС. Указать список структурных подразделений для формирования отчета можно в поле "Структурные единицы" основной формы 307.00.

В данном отчете существует возможность группировки счетов-фактур, если дата выписки счета-фактуры, входящий номер счета-фактуры, признак Дополнительная и Контрагент документа совпадают. При этом при расшифровке граф D, E и F можно увидеть список сгруппированных документов. Вывод документов осуществляется в соответствии с настройками отбора. Сортировка осуществляется по дате выписки счетов-фактур.
