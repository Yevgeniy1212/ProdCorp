Регистр налогового учета по индивидуальному подоходному налогу и социальному налогу

Отчет предоставляет возможность получить подробную расшифровку расчета по налогам, взносам и отчислениям, исчисляемых с заработной платы работников, за указанный в отчете период в следующих разрезах:

  • Налогоплательщик;
  • Месяц налогового периода;
  • Налоговый комитет;
  • Структурная единица (подразделение) налогоплательщика;
  • Работник.

В зависимости от выбранных настроек в отчете предоставляется детальная расшифровка по документам-регистраторам (в случае выбора группировки "Детализация: документ движения (регистратор)") и периодам образования задолженности для расчета удельного веса выплаченных доходов.

Отчет может быть использован для проверки правильности формирования регламентированных налоговых отчетов.

В правой панели отчета доступны отборы по структурной единице и по работнику.

В окне настройки отчета доступны следующие параметры формирования отчета:

  • Включить ИП и прочие физ. лица - если установлено значение "да", то в отчет будут включены данные по доходу и выплатам индивидуальным предпринимателям и прочим физическим лицам зарегистрированные с помощью документов "Регистрация прочих доходов в целях налогообложения" и "Расчет налогов при поступлении активов и услуг".
  • Включить получателей алиментов - если установлено значение "да", то в отчет будут включены сведения об удержанных суммах по исполнительным листам в пользу получателей этих сумм, а также о выплатах, начальной и конечной задолженности по расчетам по исполнительным листам;
  • Выводить только контрагентов - если установлено значение "да", то отчет будет сформирован по данным, зарегистрированных на контрагентов.
  • Удельный вес - если установлено значение "да", в отчет выводятся дополнительные разделы, расшифровывающие расчет удельного веса выплаченных доходов и суммы ИПН и ОПВ, подлежащих перечислению, в случае применения соответствующей настройки в учетной политике по налоговому учет для организации.
