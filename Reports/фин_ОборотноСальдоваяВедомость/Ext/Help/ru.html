Отчет "Оборотно-сальдовая ведомость (упр.)"

Отчет предназначен для получения данных по дебетовым и кредитовым начальным и конечным остаткам, а также оборотам по счетам за выбранный период.

На закладке «Данные» указываются следующие настройки:

·         Период формирования отчета;

·         Сценарий – если указан, то выводятся плановые данные по выбранному сценарию. Иначе выводятся фактические данные по счетам;

·         Периодичность отчета – периодичность группировки данных, в которой будут выводиться остатки и обороты отчета. Если не указана, группировка по вложенным периодам не производится;

·         По субсчетам – требуется ли детализировать данные разделов плана счетов по подчиненным счетам;

·         Выводить забалансовые счета – требуется ли включать в отчет данные по забалансовым счетам;

·         Детализация счетов – в данном списке следует указывать те разделы плана счетов, по которым детализация должна отличаться от общей, заданной параметром «По субсчетам». Для выбранного счета/раздела плана счетов можно указать необходимость детализации как по субсчетам, так и по субконто.

Для данных отчета доступна расшифровка следующими отчетами:

·         Оборотно-сальдовая ведомость по счету;

·         Карточка счета;

·         Анализ счета;

·         Обороты счета (по месяцам или по дням);
