Отчет «Декларация по корпоративному подоходному налогу, форма 100.00 (XML)» за 2022 год

Отчет «Декларация по корпоративному подоходному налогу» предназначен для составления отчетности по КПН.

 

Работа с расшифровкой строк регламентированного отчета.

Расшифровка строк возможна для тех ячеек, которые выделены голубым цветом.

 

СОДЕРЖАНИЕ

1) Заполнение приложения 100.01 «Расходы по приобретенным товарам, работам, услугам»

2) Заполнение приложения 100.02 «Вычеты по фиксированным активам»

При заполнении формы 100.00 учитываются данные не только из заполненных приложений, но и непосредственно данные, отнесенные на статьи затрат или доходы, для которых определен код строки Декларации в регистре сведений «Коды строк декларации по налогу на прибыль», начинающийся с «100.00». При заполнении строк декларации следует исходить из того, что если в конфигурации не автоматизировано заполнение нужного приложения, то необходимо для статьи затрат или дохода в регистре «Коды строк декларации по налогу на прибыль» проставить код строки, начинающийся с «100.00», а не код приложения, заполнение которого не предусмотрено. Если необходимого кода строки нет в классификаторе, то его нужно установить «вручную».

При формировании отчета по структурным единицам необходимо настроить регистр сведений «Исчисление налогов структурных единиц» по разделу КПН таким образом, чтобы для всех структурных единиц организации в качестве плательщика КПН была указана головная организация и ее налоговый комитет. При указании кодов строк декларации по КПН нужно указывать коды строк непосредственно для организации плательщика КПН (головной организации), либо не указывать организацию вовсе.

    ЗАПОЛНЕНИЕ ПРИЛОЖЕНИЯ 100.01.

    Примечание!!! Приложение заполняется данными когда головная организация не являлась плательщиком налога на добавленную стоимость за период формирования налогового отчета.

    Раздел. Расходы

    Для правильного заполнения приложения 100.01 «Расходы по реализованным товарам, выполненным работам, оказанным услугам» и расшифровок к нему необходимо использовать коды головной формы:

    1) 100.00.019IIIB - Финансовые услуги
    2) 100.00.019IIIC - Рекламные услуги
    3) 100.00.019IIID - Консультационные услуги
    4) 100.00.019IIIE - Маркетинговые услуги
    5) 100.00.019IIIF - Дизайнерские услуги
    6) 100.00.019IIIG - Инжиниринговые услуги
    7) 100.00.019IIIA и 100.00.019IIIH - Прочие (запасы и прочие работы и услуги)

    Включаются корреспонденции счетов налогового учета по виду учета НУ и ПР:

    Дт счетов раздела 70Н «Расходы», 80Н «Производство» Кт счетов раздела 3310Н «Расчеты с контрагентами и прочие взаиморасчеты», Дт счетов раздела 1600Н «Прочие активы» Кт счетов раздела 3310Н «Расчеты с контрагентами и прочие взаиморасчеты»,  имеющих аналитику «Контрагенты», в разрезе статей затрат, имеющих соответствующий код строки декларации в регистре сведений «Коды строк декларации по налогу на прибыль».

           

    Внимание!!! Буквы в кодах строк Декларации - латинские.

    Примечание!!! Если установлен флаг «Отображать затраты, у которых не установлен код строки Декларации» то в окне сообщений будут выводиться статьи затрат по которым есть оборот, но для статьи затрат не установлен код строки декларации. Данные суммы не будут попадать в приложение.

    ЗАПОЛНЕНИЕ ПРИЛОЖЕНИЯ 100.02.

    В данной форме отражаются данные только по тем фиксированным активам, по которым и в регистре сведений «Фиксированные активы организации» существуют записи на период формирования отчета. Кроме этого для этих записей параметр «Группа налогового учета» должен иметь вид налогового учета в группе ФА «по объектам» - для I группы и «по группам» - для II, III и IV групп (см. Справочник «Группы налогового учета ФА»).

    - Заполнение строки 100.02.001 (Стоимостный баланс групп (подгрупп) на начало налогового периода)

    В строку переносится СНД счета 2100Н «ВА, включенные в стоимостный баланс» с видом учета НУ;

    В строку переносится разница кредитового и дебетового оборота счета 2100Н «ВА, включенные в стоимостный баланс» с видом движения стоимости ФА «Выбытие» с видом учета НУ;

    В строку переносится дебетовый оборот счета 2100Н «ВА, включенные в стоимостный баланс» с видом движения стоимости «Увеличение стоимости за счет ремонта» и видом учета НУ;
