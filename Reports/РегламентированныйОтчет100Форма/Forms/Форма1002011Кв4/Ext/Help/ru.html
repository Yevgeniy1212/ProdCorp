Отчет «Декларация по корпоративному подоходному налогу, форма 100.00 (XML)» за 2011 год

Отчет «Декларация по корпоративному подоходному налогу» предназначен для составления отчетности по КПН.

 

Работа с расшифровкой строк регламентированного отчета.

Расшифровка строк возможна для тех ячеек, которые выделены голубым цветом.

 

СОДЕРЖАНИЕ

1) Заполнение приложения 100.04 «Расходы по реализованным товарам, выполненным работам, оказанным услугам»

2) Заполнение приложения 100.07 «Вычеты по фиксированным активам»

3) Заполнение приложения 100.08 «Инвестиционные налоговые преференции»

При заполнении формы 100.00 учитываются данные не только из заполненных приложений, но и непосредственно данные, отнесенные на статьи затрат или доходы, для которых определен код строки Декларации в регистре сведений «Коды строк декларации по налогу на прибыль», начинающийся с «100.00». При заполнении строк декларации следует исходить из того, что если у нас не заполняется нужное приложение, то необходимо для статьи затрат или дохода в регистре «Коды строк декларации по налогу на прибыль» проставить код строки, начинающийся с «100.00», а не код приложения, заполнение которого не предусмотрено. Если необходимого кода строки нет в классификаторе, то его нужно установить «вручную».

Например, если необходимо отразить в декларации доход по полученным вознаграждениям, то необходимо в регистре сведений «Коды строк декларации по налогу на прибыль» для вида дохода «Дивиденды» указывать не код строки «100.04.005», а код строки декларации «100.00.014».

При формировании отчета по структурным единицам необходимо настроить регистр сведений «Исчисление налогов структурных единиц» по разделу КПН таким образом, чтобы для всех структурных единиц организации в качестве плательщика КПН была указана головная организация и ее налоговый комитет. При указании кодов строк декларации по КПН нужно указывать коды строк непосредственно для организации плательщика КПН (головной организации), либо не указывать организацию вовсе.

В строке не отражаются расходы по заработной плате:

  • включаемые в первоначальную стоимость активов, не подлежащих амортизации, в соответствии со статьей 87 Налогового кодекса(корреспонденции со счетами раздела 20Н «Внеоборотные активы»);

    Строка 100.00.030.V включаются корреспонденции счетов налогового учета с видом учета НУ:

    Последующие расходы – это расходы на ремонт по фиксированным активам и арендуемым основным средствам, налоговый учет которых производится в соответствии со статьей 122 Налогового Кодекса. В строке отражаются расходы, учтенные на счете 7900Н «Расходы на ремонт ВА» с видом «текущий» и «капитальный», как с видом НУ, так и с видом ПР.

    В строке100.00.030.VII отражаются все расходы на незавершенное строительство по счету налогового учета 2400Н (НУ+ПР), а также списание ТМЗ и РБП на счета 2500Н и 2400Н.

    Списание РБП включается в строку при условии, что код строки Декларации для статьи затрат РБП содержит код «100.00.030».

    - Заполнение строки 100.00.030.VIII (Стоимость работ и услуг, себестоимость ТМЗ, не относимые на вычеты на основании статьи 115 НК)

    В строке 100.00.030.VIII отражаются все расходы с видом учета ПР, которые были отражены в 100.00.030.III, за исключением незавершенного строительства (2400Н), ремонтов (7900Н ),  а так же списания ТМЗ и РБП на внеоборотные активы, которые были ранее отражены в строках 100.00.030.VI и 100.00.030.VII.

    Также в строку включаются затраты по списанию ТМЗ на расходы, не относящиеся к предпринимательской деятельности.

    В операциях по приобретению ТМЗ, не в целях предпринимательской деятельности (с видом учета ПР) возможно два режима отражения операций в форме:

    - исключение из вычетов в периоде приобретения таких ТМЗ.

    - исключение из вычетов в периоде отнесения таких ТМЗ на затраты.

    В случае, если в периоде приобретения таких ТМЗ, уменьшать вычет отчетного года не требуется, то  в настройке формы следует установить признак "Включать сведения о ПР операциях по ТМЗ в строку 100.00.030.VIII в периоде отнесения ТМЗ на затраты".

    - Заполнение строки 100.00.030.IX ( Стоимость работ и услуг, себестоимость ТМЗ, признаваемые расходами будущих периодов и подлежащие отнесению на вычеты в последующие налоговые периоды)

    Строка 100.00.030.IX включаются корреспонденции счетов налогового учета с видом учета НУ:

    Раздел. Расходы

    Для правильного заполнения приложения 100.04 «Расходы по реализованным товарам, выполненным работам, оказанным услугам» и расшифровок к нему необходимо использовать коды головной формы:

    5) 100.00.030IIIF - Дизайнерские услуги
    6) 100.00.029IIIG - Инжиниринговые услуги

    Внимание!!! Буквы в кодах строк Декларации - латинские.

    Примечание!!! Если установлен флаг «Отображать затраты, у которых не установлен код строки Декларации» то в окне сообщений будут выводиться статьи затрат по которым есть оборот, но для статьи затрат не установлен код строки декларации. Данные суммы не будут попадать в приложение.

    ЗАПОЛНЕНИЕ ПРИЛОЖЕНИЯ 100.07.

    В данной форме отражаются данные только по тем фиксированным активам, по которым и в регистре сведений «Фиксированные активы организации» существуют записи на период формирования отчета. Кроме этого для этих записей параметр «Группа налогового учета» должен иметь вид налогового учета в группе ФА «по объектам» - для I группы и «по группам» - для II, III и IV групп (см. Справочник «Группы налогового учета ФА»).

     

    Раздел. Вычеты по фиксированным активам

    - Заполнение строки 100.07.001 (Стоимостный баланс групп (подгрупп) на начало налогового периода)

    В строку переносится СНД счета 2400Н «ВА, включенные в стоимостный баланс» с видом учета НУ;

    В строку переносится разница дебетового и кредитового оборота счета 2400Н «ВА, включенные в стоимостный баланс» с видом движения стоимости «Поступление» с видом учета НУ;

    - Заполнение строки 100.07.003 (Стоимость выбывших фиксированных активов)

    В строку переносится разница кредитового и дебетового оборота счета 2400Н «ВА, включенные в стоимостный баланс» с видом движения стоимости ФА «Выбытие» с видом учета НУ;

    - Заполнение строки 100.07.004 (Последующие расходы по фиксированным активам, относимые на увеличение стоимостных балансов групп (подгрупп) в соответствии с пунктом 3 статьи 122 Налогового кодекса)

    В строку переносится дебетовый оборот счета 2400Н «ВА, включенные в стоимостный баланс» с видом движения стоимости «Увеличение стоимости за счет ремонта» и видом учета НУ;

    - Заполнение строки 100.07.005 (Стоимостный баланс групп (подгрупп) на конец налогового периода)

    Стоимостный баланс подгруппы, группы на конец налогового периода определяется по формуле

    100.07.005 = 100.07.001 + 100.07.002 – 100.07.003 + 100.07.004;

    - Заполнение строки 100.07.006 (Амортизационные отчисления по фиксированным активам)

    В строку переносятся начисленная амортизация по фиксированным активам, т.е кредитовый оборот счета 2400Н «ВА, включенные в стоимостный баланс» с видом движения стоимости ФА «Амортизация» и видом учета НУ;

    - Заполнение строки 100.07.007 (Двойная норма амортизации в соответствии с пунктом 6 статьи 120 НК)

    В строку переносятся начисленная амортизация по фиксированным активам которые в регистре сведений «Фиксированные активы, учитываемые отдельно» на конец текущего налогового периода еще не включены в состав группы, т.е кредитовые обороты счетов раздела Н300 «Внеоборотные активы» с видом движения стоимости ФА «Амортизация» и видом учета НУ;

    - Заполнение строки 100.07.008 (Величина стоимостного баланса группы и признаваемый убыток по подгруппам (I группы) при выбытии всех фиксированных активов)

    В строку переносится кредитовый оборот счета 2400Н «ВА, включенные в стоимостный баланс» с видом движения стоимости «Списание при выбытии всех активов группы» и видом учета НУ;

    - Заполнение строки 100.07.009 (Величина стоимостного баланса группы (подгруппы) менее 300 месячных расчетных показателей, относимая на вычеты)

    В строку переносится кредитовый оборот счета 2400Н «ВА, включенные в стоимостный баланс» с видом движения стоимости «Списание стоимостного баланса группы менее минимума» и видом учета НУ;

    - Заполнение строки 100.07.010 (Последующие расходы по фиксированным активам, относимые на вычеты в соответствии с пунктом 2 статьи 122 Налогового кодекса)

    В строку переносится дебетовый оборот счета 7900Н «Расходы на ремонт ВА» по виду ремонта ОС «Текущий» с видом учета НУ;

    ЗАПОЛНЕНИЕ ПРИЛОЖЕНИЯ 100.08.
