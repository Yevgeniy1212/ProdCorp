Отчет "Исполнение бюджета"

Отчет «Исполнение бюджета» предназначен для выполнения сравнительного анализа между плановыми и фактическими данными бюджетного периода, между данными различных сценариев или периодов. Основным его назначением является план-фактный анализ как результат  сравнения плановых и фактических данных периода.

Помимо общих для всех отчетов настроек необходимо настроить следующие параметры отчета.

На закладке «Данные» помимо общих настроек указываются следующие:

·         Бюджет– бюджет, по статьям которого необходимо получить данные;

·         Данные для анализа – перечень сравниваемых между собой наборов данных по бюджету. За эталон принимаются данные, задаваемые первой строкой. При план-фактном анализе в первой строке настраиваются данные плана, а во второй – фактические. В каждой строке указываются данные для сравнения (т.е. сценарий планирования, если же нужны фактические данные, то сценарий не указывается), а также границы периода сбора данных. Доступны команды «Период строки» (позволяет выбрать период для указанной строки) и «Период отчета» (позволяет выбрать единый период для всех строк отчета).

На закладке «Структура» помимо общих настроек заполняется поле «Виды аналитических данных», определяющее виды аналитических данных, выводимых в отчете:

·         Эталон – абсолютное значение эталонных данных (данных первой строки поля «Данные для анализа»);

·         Исполнение, абс. – абсолютное значение сравниваемых данных;

·         Исполнение, % - процентное отношение сравниваемых данных к эталонным;

·         Отклонение, абс. – абсолютная разница между сравниваемыми и эталонными данными;

·         Отклонение, % – разница между сравниваемыми и эталонными данными в процентном выражении по отношению к эталонным данным;

Указанные виды аналитических данных рассчитываются для каждого из выбранных показателей отчета.

На закладке «Дополнительно» помимо общих настроек указываются следующие настройки:

·         Заголовок отчета – текстовый заголовок, выводимый в отчете;

·         Аналит. поля в одной колонке – определяет вывод аналитических полей по показателю либо в нескольких колонках, либо в одной друг под другом.
