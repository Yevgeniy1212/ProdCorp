Отчет "Отчет о результатах ФХД (XML)" за 2009 год и последующие периоды

Правила заполнения отчета

Заполнение строк за отчетный период:

- 010 Доход от реализации продукции и оказания услуг: заполняется как разница кредитового(КО) и дебетового оборота(ДО) по счету 5610 «Итоговая прибыль итоговый убыток» в корреспонденции со счетами раздела 6000 «Доход от реализации продукции и оказания услуг». 

- 020 Себестоимость реализованной продукции и оказанных услуг: заполняется как дебетовый оборот(ДО) по счету 5610 «Итоговая прибыль итоговый убыток» в корреспонденции со счетами раздела 7000 «Себестоимость реализованной продукции и оказанных услуг». 

- 030 Валовая прибыль: 
                    строка 030 =  строка 010 - строка 020. 

- 040 Доходы от финансирования: заполняется как кредитовый оборот(КО) по счету 5610 «Итоговая прибыль итоговый убыток» в корреспонденции со счетами раздела 6100 «Доходы от финансирования». 

- 050 Прочие доходы: заполняется как кредитовый оборот(КО) по счету 5610 «Итоговая прибыль итоговый убыток» в корреспонденции со счетами раздела 6200 «Прочие доходы». 

- 060 Расходы на реализацию продукции и оказание услуг: заполняется как дебетовый оборот(ДО) по счету 5610 «Итоговая прибыль итоговый убыток» в корреспонденции со счетами раздела 7100 «Расходы по реализации продукции и оказанию услуг». 

- 070 Административные расходы: заполняется как дебетовый оборот(ДО) по счету 5610 «Итоговая прибыль итоговый убыток» в корреспонденции со счетами раздела 7200 «Административные расходы». 

- 080 Расходы на финансирование: заполняется как дебетовый оборот(ДО) по счету 5610 «Итоговая прибыль итоговый убыток» в корреспонденции со счетами раздела 7300 «Расходы на финансирование». 

- 090 Прочие расходы: заполняется как дебетовый оборот(ДО) по счету 5610 «Итоговая прибыль итоговый убыток» в корреспонденции со счетами раздела 7400 «Прочие расходы». 

- 100 Доля прибыли/убытка организаций, учитываемых по методу долевого участия: заполняется как разница кредитового(КО) оборота по счету 5610 «Итоговая прибыль итоговый убыток» в корреспонденции со счетами раздела 6400 «Доля прибыли организаций учитываемых по методу долевого участия» и дебетового оборота (ДО) по счету 5610 «Итоговая прибыль итоговый убыток» в корреспонденции со счетами раздела 7600 «Доля в убытке организаций учитываемых методом долевого участия».

- 110 Прибыль (убыток) за период от продолжаемой деятельности: 
                    строка 110 =  строка 030 + строка 040 + строка 050 - строка 060 - строка 070 - строка 080 - строка 090 + строка 100. 

- 120 Прибыль/убыток от прекращенной деятельности: заполняется как разница кредитового(КО) оборота по счету 5610 «Итоговая прибыль итоговый убыток» в корреспонденции со счетами раздела 6300 «Доходы, связанные с прекращаемой деятельностью» и дебетового оборота (ДО) по счету 5610 «Итоговая прибыль итоговый убыток» в корреспонденции со счетами раздела 7500 «Расходы связанные с прекращаемой деятельностью».

- 130 Прибыль (убыток) от налогообложения: 
                    строка 130 =  строка 110 + строка 120. 

- 140 Расходы по корпоративному подоходному налогу: заполняется как дебетовый оборот(ДО) по счету 5610 «Итоговая прибыль итоговый убыток» в корреспонденции со счетами раздела 7700 «Расходы по корпоративному подоходному налогу». 

- 150 Чистая прибыль (убыток) за период до вычета доли меньшинства: 
                    строка 150 =  строка 130 - строка 140.

- 160 Доля меньшинства: автоматически не заполняется.

- 170 Итоговая прибыль (итоговый убыток) за период: 
                    строка 170 = строка 150 - строка 160.

- 180 Прибыль на акцию: автоматически не заполняется.

Заполнение строк за предыдущий период:

Заполнение строк отчета за предыдущий период осуществляется аналогично заполнению строк на отчетный период.

Работа с отчетом

 

Для работы необходимо установить значение следующих реквизитов в форме отчета:

 

«Вид формы»  -  вид результирующей формы - «Основная» или «Корректировочная», значение выбирается из выпадающего списка

 

«Код налоговой инспекции»   -   код налоговой инспекции Организации

  

В диалоговом окне формы отчета предусмотрен набор кнопок, соответствующий списку

возможных действий:

 

Кнопка «Выгрузить в XML» -  при помощи этой кнопки осуществляется выгрузка информации в формате XML для клиентского приложения ИС СОНО.

 

Кнопка «Выгрузить в Excel»  при помощи этой кнопки осуществляется выгрузка информации в формате MS Excel для клиентского приложения ИС СОНО.

 

Кнопка «Заполнить»  -  по данной кнопке происходит автоматическое заполнение отчета данными из бухгалтерского учета. При желании пользователь может отредактировать или заполнить данные самостоятельно.

 

Кнопка «Очистить» -  очищает бланк отчета 

 

Кнопка «Печать» - позволяет выполнить предварительный просмотр, либо сразу распечатать отчет

Кнопка вызова справки – пользователь может просмотреть текущую справку, либо перейти к общей информации о формах.

  

Кроме этого, в процессе работы с отчетом пользователь может самостоятельно редактировать данные, представленные в бланке отчета. По двойному щелчку на ячейке открывается окно для прямого редактирования значения.

 

Все основные механизмы работы с отчетом (печать, форматирование данных, выгрузка в

XML-формате и т.д.) заимствованы из  регламентированных отчетов типовой конфигурации
