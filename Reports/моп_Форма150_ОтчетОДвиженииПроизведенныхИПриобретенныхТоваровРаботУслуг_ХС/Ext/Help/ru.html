Отчет "Отчет о движении произведенных и приобретенных товаров работ услуг (XML)"

Отчет "Отчет о движении произведенных и приобретенных товаров работ услуг" предназначен для составления отчетности по приобретенным, произведенным, списанным и реализованным ТМЗ.

Отчет формируется после проведения регламентной операции "Расчет и корректировка себестоимости продукции (услуг)".

В отчет попадают остатки и обороты счетов учета ТМЗ (счета 1310, 1320, 1330, 1350) за указанный период. В зависимости от корсчета,  суммы попадают в те или иные колонки.

Перечень счетов учета ТМЗ и корсчетов можно изменить на форме настроек, которая открывается по нажатию на кнопку "Настройка" командной панели. Также, на форме настроек, с помощью флажка "Исключить товары, по которым не было оборотов", возможно отключить вывод информации о товарах, по которым были остатки на начало отчетного периода, но не было оборотов за отчетный период.

 

Расшифровка некоторых колонок отчета:

 

·         Код ТН ВЭД – данные из регистра «Объекты мониторинга». Двойной щелчок откроет карточку товара, откуда можно перейти к регистру «Объекты мониторинга»;

·         Остаток на начало месяца (количество, сумма) – сальдо начальное дебетовое за указанный период;

·         Количество, себестоимость произведенного товара – выпуск (дт обороты в корреспонденции со счетами 8100, 8200, 8300, 8400 подразделов);

·         Количество, сумма прочего поступления – весь дебетовый оборот за вычетом выпуска, возврата от покупателя (дт обороты в корреспонденции со счетами 7000 подраздела), возврата поставщику (кт обороты в корреспонденции со счетами 5000, 3300, 1200 подразделов), внутреннего перемещения (кт обороты в корреспонденции со счетами учета ТМЗ);

·         Количество, себестоимость реализованного товара – реализация (разница дт и кт оборотов в корреспонденции со счетами 7000 подраздела);

·         Количество, сумма прочего выбытия – весь кредитовый оборот за вычетом возврата от покупателя (кт обороты в корреспонденции со счетами 7000 подраздела), возврата поставщику (кт обороты в корреспонденции со счетами 5000, 3300, 1200 подразделов), внутреннего перемещения (кт обороты в корреспонденции со счетами учета ТМЗ);

·         Остаток на конец месяца (количество, сумма) – сальдо конечное дебетовое за указанный период.

 

 

Общая информация о работе с отчетами

 

Отчет имеет две формы.

В первой, основной форме пользователю нужно установить Организацию и период отчета.

 

После нажатия «ОК» открывается форма отчета, верхняя часть которой используется для управления формированием, а в нижней части расположен сам бланк отчета.

 

Работа с отчетом

 

Для работы необходимо установить значение следующих реквизитов в форме отчета:

 

«Вид формы»   -   вид результирующей формы, значение выбирается из выпадающего списка

 

«Код налоговой инспекции»   -   код налоговой инспекции Организации

  

В диалоговом окне формы отчета предусмотрен набор кнопок, соответствующий списку

возможных действий:

 

Кнопка «Выгрузить в XML» -  при помощи этой кнопки осуществляется выгрузка информации в формате XML для клиентского приложения ИС СОНО.

                                                                         

Кнопка «Заполнить»  -  по данной кнопке происходит автоматическое заполнение отчета данными из бухгалтерского учета. При желании пользователь может отредактировать или заполнить данные самостоятельно.

 

Кнопка «Очистить» -  очищает бланк отчета.

 

Кнопки добавления, удаления строк – пользователь может добавить новую строку (и заполнить ее самостоятельно) или удалить любую выделенную

 

Кнопки последовательного перехода по страницам – пользователь может перейти на следующую / предыдущую страницу отчета; текущая страница и полное количество страниц отображаются между этими кнопками

 

Кнопка «Настройка» - открывает форму настроек.  

 

Кнопка «Печать» - позволяет выполнить предварительный просмотр, либо сразу распечатать отчет.

 

Кнопка произвольного перехода к странице – пользователь может перейти на любую страницу отчета.

 

Кнопка «Настройка...»  -  открывает форму с дополнительными настройками, где можно задать следующие параметры:

  •  счета учета ТМЗ, обороты и остатки по которым, попадут в отчет (по умолчанию заданы счета 1310, 1320, 1330, 1350);
  •  счета, корреспондирующие со счетами учета ТМЗ, обороты по которым будут исключены (по умолчанию заданы счета 1240 и 3340);

 

Примечание: обороты между счетами учета ТМЗ и счетами 1240, 3340, исключаются по умолчанию, в связи с необходимостью исключить внутренние обороты между филиалами и структурными подразделениями;

 

Кнопка вызова справки – пользователь может просмотреть текущую справку, либо перейти к общей информации о формах.

  

Кроме этого, в процессе работы с отчетом пользователь может самостоятельно редактировать данные, представленные в бланке отчета. По двойному щелчку на ячейке открывается элемент (карточка счета, контрагента или товара, и т.п.), после редактирования которого обновляются все связанные поля (адрес контрагента, код ТН ВЭД товара, и т.п.). При двойном щелчке на ячейке с обычным текстом или числом (например, в поле "Сумма НДС"), открывается окно для прямого редактирования значения.

 

Все основные механизмы работы с отчетом (печать, форматирование данных, выгрузка в

XML-формате и т.д.) заимствованы из  регламентированных отчетов типовой конфигурации
