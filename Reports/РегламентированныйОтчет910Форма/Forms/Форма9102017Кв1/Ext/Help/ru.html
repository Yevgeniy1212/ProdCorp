Отчет «Упрощенная декларация для субъектов малого бизнеса, форма 910.00 (XML)» за 2017 год.

Отчет предназначен для составления упрощенной декларации субъектами малого бизнеса (форма 910.00)

Правила составления:

- Правила составления налоговой отчетности (упрощенной декларации) для субъектов малого бизнеса (Форма 910.00) за 2017 год.

Специальный налоговый режим на основе упрощенной декларации применяют субъекты малого бизнеса, определенные ст. 433 Налогового Кодекса РК, соответствующие установленным ограничениям:

  • Индивидуальные предприниматели - в справочнике "Организации" для них реквизит "Юр. / физ. лицо" должен быть установлен в значение "Физ. лицо";
  • Юридические лица - в справочнике "Организации" для них реквизит "Юр. / физ. лицо" должен быть установлен в значение "Юр. лицо".

Раздел. Исчисление налогов, социальных отчислений и обязательных пенсионных взносов

- Заполнение строки 910.00.001

В строке выводится сумма налогооблагаемого дохода, полученного за налоговый период, определяемая как разность кредитового и дебетового оборотов по счетам разделов 6000, 6100, 6200, 6300 и 6400, при этом проводка Дт 6ххх - Кт 5610 (Итоговая прибыль итоговый убыток), формируемая при реформации баланса в момент закрытия месяца, не учитывается. В качестве облагаемых доходов учитываются только те виды, для которых в справочнике "Доходы" установлен признак "Является объектом налогообложения" на закладке "Отражение в упрощенной декларации".

В конфигурации "Зарплата и Управление Персоналом для Казахстана" данная строка автоматически не заполняется.

- Заполнение строки 910.00.002

В строке указывается доход, определяемый в соответствии с Законом Республики Казахстан от 5 июля 2008 года "О трансфертном ценообразовании". Заполнение строки производится вручную.

- Заполнение строки 910.00.003

В строке выводятся данные о среднесписочном количестве работников за налоговый период. Количество работников для ИП определяется как :

Численность1 + Численность2 + Численность3 + Численность4 + Численность5 + Численность6

910.00.003 = -------------------------------------------------------------------------------------------------------------------------------- + 1, где 1 - сам ИП.

6

Количество работников для ТОО:

Численность1 + Численность2 + Численность3 + Численность4 + Численность5 + Численность6

910.00.003 = --------------------------------------------------------------------------------------------------------------------------------

6

где Численность - списочная численность работников за каждый месяц отчетного полугодия. При подсчете численности учитывается количество дней, в течение которых действовал трудовой договор работника в месяце, в отношении к общему числу дней в месяце.

- Заполнение строки 910.00.004

В строке указывается среднемесячная заработная плата одного наемного работника, определяемая по формуле:

ФондЗП1 ФондЗП2 ФондЗП3 ФондЗП4 ФондЗП5 ФондЗП6

-------------------- + ------------------- + ----------------- + ----------------- + ----------------- + -----------------

Численность1 Численность2 Численность3 Численность4 Численность5 Численность6

910.00.004 = ------------------------------------------------------------------------------------------------------------------------------------

6

где ФондЗП1, ФондЗП2, ФондЗП3, ФондЗП4, ФондЗП5 и ФондЗП6 - фонд заработной платы наемных работников соответственно за 1, 2, 3, 4, 5 и 6 месяцы. Сведения о фонде заработной платы работников приводятся в отчете "Анализ начислений сотрудникам организаций" (показатель - фонд заработной платы).

- Заполнение строки 910.00.005

В строке выводится сумма исчисленных налогов, определяемая согласно п. 1 ст. 436 Налогового Кодекса РК по формуле:

910.00.005 = 910.00.001 * 3%

- Заполнение строки 910.00.006

В строке указывается корректировка суммы налогов в соответствии с п.2 статьи 436 НК. Сумма корректировки составляет 1,5 процента от суммы налога за каждого работника, исходя из среднесписочной численности работников. Корректировка производится в случае если среднемесячная заработная плата работников (строка 910.00.004) по итогам отчетного периода составила у индивидуальных предпринимателей - не менее 2-кратного, у юридических лиц - не менее 2,5-кратного минимального размера заработной платы.

- Заполнение строки 910.00.007

Строка рассчитывается арифметически:

910.00.007 = 910.00.005 - 910.00.006

- Заполнение строки 910.00.008

В строке указывается сумма индивидуального (корпоративного) подоходного налога, подлежащая уплате в бюджет, определяемая как:

910.00.008 = 910.00.007 * 0,5

- Заполнение строки 910.00.009

В строке указывается сумма социального налога, подлежащая уплате в бюджет, определяемая как:

910.00.009 = 910.00.007 * 0,5 - 910.00.011 - 910.00.021

- Заполнение строки 910.00.010

В строке указывается доход для исчисления социальных отчислений за индивидуального предпринимателя. Сумма определяется исходя из величины фактически полученного дохода с учетом ограничений на минимальный и максимальный размер дохода ИП, применяемого для исчисления СО (пункт 4 "Правил исчисления и перечисления социальных отчислений"). Сумма может быть скорректирована вручную.

- Заполнение строки 910.00.011

В строке рассчитывается сумма социальных отчислений за индивидуального предпринимателя, подлежащая к уплате. Строка рассчитывается как:

910.00.011 = 910.00.010 * 5% (процент социальных отчислений)

В форме 2015 года реализована настройка "Исчислять социальные отчисления за ИП без ограничения на сумму исчисленного социального налога", при установке которой сумма СО рассчитывается без учета величины исчисленного СН.

- Заполнение строки 910.00.012

В строке указывается доход для исчисления обязательных пенсионных взносов за индивидуального предпринимателя. Строка заполняется произведением минимальной заработной платы на количество месяцев отчетного периода. Сумма может быть скорректирована вручную.

- Заполнение строки 910.00.013

В строке рассчитывается сумма обязательных пенсионных взносов за индивидуального предпринимателя, подлежащих к уплате. Строка рассчитывается как:

910.00.013 = 910.00.012 * 10% (ставка обязательных пенсионных взносов)

- Заполнение строки 910.00.014 (Сумма индивидуального подоходного налога, подлежащего перечислению в бюджет с доходов граждан РК)

В строке выводятся данные о суммах индивидуального подоходного налога, подлежащего перечислению в бюджет. Информация собирается на основании записей-расходов по регистру накопления "ИПН расчеты с бюджетом" с видом строки "Удержание". В зависимости от настроек учетной политики по налоговому учету "При начислении ИПН, ОПВ и ВОСМС принимать исчисленные суммы к учету как удержанные" эта сумма равняется фактически исчисленному налогу (если настройка "При начислении ИПН, ОПВ и ВОСМС принимать исчисленные суммы к учету как удержанные" включена), либо определяется в процессе расчета документа "Удержание ИПН, ОПВ и ВОСМС по налоговому учету" (настройка "При начислении ИПН, ОПВ и ВОСМС принимать исчисленные суммы к учету как удержанные" отключена).

В строку включаются сведения только по тем сотрудникам, которые являются гражданами РК.

- Заполнение строки 910.00.015 (Сумма индивидуального подоходного налога, подлежащего перечислению в бюджет с доходов иностранцев и лиц без гражданства)

В строке выводятся данные о суммах индивидуального подоходного налога, подлежащего перечислению в бюджет. Информация собирается на основании записей-расходов по регистру накопления "ИПН расчеты с бюджетом" с видом строки "Удержание". В зависимости от настроек учетной политики по налоговому учету "При начислении ИПН, ОПВ и ВОСМС принимать исчисленные суммы к учету как удержанные" эта сумма равняется фактически исчисленному налогу (если настройка "При начислении ИПН, ОПВ и ВОСМС принимать исчисленные суммы к учету как удержанные" включена), либо определяется в процессе расчета документа "Удержание ИПН, ОПВ и ВОСМС по налоговому учету" (настройка "При начислении ИПН, ОПВ и ВОСМС принимать исчисленные суммы к учету как удержанные" отключена).

В строку включаются сведения только по тем сотрудникам, которые не являются гражданами РК.

- Заполнение строки 910.00.016 (Начисленные доходы, с которых удерживаются (начисляются) обязательные пенсионные взносы)

В строке выводятся данные о доходах работников, с которых удерживаются (начисляются) обязательные пенсионные взносы. Данная строка определяется по данным регистра "ОПВ сведения о доходах".

- Заполнение строки 910.00.017 (Сумма обязательных пенсионных взносов, к уплате)

В строке выводятся данные о суммах обязательных пенсионных взносов, подлежащих перечислению в пенсионные фонды, которые определяются на основании данных о приходах в регистре накопления "ОПВ подлежит перечислению в фонды" с видом строки "Удержание". На расчет суммы взносов, подлежащий перечислению, влияет настройка в учетной политике по налоговому учету "При начислении ИПН, ОПВ и ВОСМС принимать исчисленные суммы к учету как удержанные".

- Заполнение строки 910.00.018 (Доходы работников, принимаемые для исчисления обязательных профессиональных пенсионных взносов)

В строке выводятся данные о доходах работников, с которых удерживаются (начисляются) обязательные профессиональные пенсионные взносы. Строка заполняется по данным регистра "ОППВ сведения о доходах".

- Заполнение строки 910.00.019 (Сумма обязательных профессиональных пенсионных взносов, к уплате)

В строке выводятся данные о суммах обязательных профессиональных пенсионных взносов, подлежащих перечислению по данным регистра "ОППВ расчеты с фондами" с видом строки "Исчисление".

- Заполнение строки 910.00.020 (Доходы физических лиц, с которых исчисляются социальные отчислений)

В строке выводятся данные о расходах работодателя, облагаемых социальными отчислениями, указанными в регистре "СО сведения о доходах" за отчетный период.

- Заполнение строки 910.00.021 (Сумма социальных отчислений, к уплате)

В строке выводятся данные о суммах социальных отчислений, исчисленных за соответствующий месяц налогового периода, определяемые как приходы по регистру накопления "СО расчеты с фондами" с видом строки "Исчисление".
