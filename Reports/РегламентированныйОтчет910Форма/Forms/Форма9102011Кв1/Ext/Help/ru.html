Отчет "Упрощенная декларация для субъектов малого бизнеса, форма 910.00 (XML)" за 2011 год.

Отчет предназначен для составления упрощенной декларации субъектами малого бизнеса (форма 910.00)

Правила составления:

- Правила составления налоговой отчетности (упрощенной декларации) для субъектов малого бизнеса (Форма 910.00) за 2011 год.

Специальный налоговый режим на основе упрощенной декларации применяют субъекты малого бизнеса, определенные ст. 433 Налогового Кодекса РК, соответствующие установленным ограничениям:

  • Индивидуальные предприниматели - в справочнике "Организации" для них реквизит "Юр. / физ. лицо" должен быть установлен в значение "Физ. лицо";
  • Юридические лица - в справочнике "Организации" для них реквизит "Юр. / физ. лицо" должен быть установлен в значение "Юр. лицо".

 

Раздел. Исчисление налогов, социальных отчислений и обязательных пенсионных взносов

- Заполнение строки 910.00.001

В строке выводится сумма налогооблагаемого дохода, полученного за налоговый период, определяемая как разность кредитового и дебетового оборотов по счетам разделов 6000, 6100, 6200, 6300 и 6400, при этом проводка Дт 6ххх - Кт 5610 (Итоговая прибыль итоговый убыток), формируемая при реформации баланса в момент закрытия месяца, не учитывается. В качестве облагаемых доходов учитываются только те виды, для которых в справочнике "Доходы" установлен признак "Является объектом налогообложения" на закладке "Отражение в упрощенной декларации".

В конфигурации "Зарплата и Управление Персоналом для Казахстана" данная строка автоматически не заполняется.

- Заполнение строки 910.00.002

Для строки невозможно автоматическое заполнение.

- Заполнение строки 910.00.003

В строке выводятся данные о количестве работников, без учета самого индивидуального предпринимателя. Количество работников за один месяц определяется как списочная численность работников согласно Инструкции по заполнению статистической отчетности по труду, утвержденной Приказом Председателя Агентства Республики Казахстан по статистике от 12 июля 2002 года N 38-Г. При подсчете численности учитывается количество дней, в течение которых действовал трудовой договор работника в месяце, в отношении к общему числу дней в месяце.

- Заполнение строки 910.00.004

В строке выводится среднесписочная численность работников в целом за налоговый период, определяемая по формуле:

                        910.00.003.I + 910.00.003.II + 910.00.003.III

910.00.004 =   -------------------------------------------------------

                                                          3

- Заполнение строки 910.00.005

В строке выводится среднесписочная численность работников включая индивидуального предпринимателя.

- Заполнение строки 910.00.006

В строке выводится среднесписочная численность, превышающая предельную численность. Согласно ст. 433 Налогового Кодекса РК для индивидуальных предпринимателей предельная среднесписочная численность работников за налоговый период составляет 25 человек, для юридических лиц - 50 человек.

- Заполнение строки 910.00.007

В строке указывается среднемесячная заработная плата одного наемного работника, определяемая по формуле:

                         ФондЗП1                              ФондЗП2                    ФондЗП3

                     --------------------     +     -------------------   +    -----------------

                      910.00.003.I                   910.00.003.II         910.00.003.III 

910.00.007 =  ----------------------------------------------------------------------- 

                                                                  3

где ФондЗП1, ФондЗП2 и ФондЗП3 - фонд заработной платы наемных работников соответственно за 1, 2 и 3 месяцы. Сведения о фонде заработной платы работников приводятся в отчете "Анализ начислений работникам организаций" (показатель - фонд заработной платы).

- Заполнение строки 910.00.008

В строке выводится сумма исчисленных налогов, определяемая согласно п. 1 ст. 436 Налогового Кодекса РК по формуле:

910.00.008 = 910.00.001 * 3%

- Заполнение строки 910.00.009

В строке выводится сумма налогов с дохода, превышающего предельную сумму дохода. Согласно ст. 433 предельных доход индивидуального предпринимателя за налоговый период составляет 10 млн. тенге, для юридического лица - 25 млн. тенге. Налог с дохода, превышающего предельную сумму доход, определяется по формуле:

910.00.009 = (910.00.001 - 10 000 000) * 3% - для индивидуального предпринимателя

910.00.009 = (910.00.001 - 25 000 000) * 3% - для юридического лица.

-  Заполнение строки 910.00.010

В строке выводится уменьшение суммы налога в зависимости от среднесписочной численности работников, в случае если среднемесячная заработная плата наемных работников (910.00.007) составила у индивидуальных предпринимателей не менее 2-кратного, у юридических лиц не менее 2,5-кратного минимального размера месячной заработной платы, установленной законодательным актом РК. Расчет значения строки 910.00.010 осуществляется по формуле:

910.00.010 = (910.00.008 - 910.00.009) * (910.00.005 - 910.00.006) * 0,015

-  Заполнение строки 910.00.011

В строке указывается сумма налогов после корректировки, рассчитываемая по формуле:

910.00.011 = 910.00.008 - 910.00.010

- Заполнение строки 910.00.012

В строке указывается сумма индивидуального (корпоративного) подоходного налога, подлежащая уплате в бюджет, определяемая как:

910.00.012 = 910.00.011 * 0,5

- Заполнение строки 910.00.013

В строке указывается сумма социального налога, подлежащая уплате в бюджет, определяемая как:

910.00.013 = 910.00.011 * 0,5 - 910.00.022 - 910.00.015

- Заполнение строки 910.00.014

В строке указывается доход для исчисления социальных отчислений за индивидуального предпринимателя. Сумма вносится вручную.

- Заполнение строки 910.00.015

В строке выводятся социальные отчисления за индивидуального предпринимателя, рассчитываемые как:

910.00.015 = 910.00.014 * 5% (процент социальных отчислений)

 - Заполнение строки 910.00.016

В строке указывается доход для исчисления обязательных пенсионных взносов за индивидуального предпринимателя. Сумма вносится вручную.

- Заполнение строки 910.00.017

В строке выводятся обязательные пенсионные взносы за индивидуального предпринимателя, рассчитываемые как:

910.00.017 = 910.00.016 * 10% (процент обязательных пенсионных взносов)

 

Форма 910.01. Раздел. Исчисление индивидуального подоходного налога с доходов граждан Республики Казахстан

- Заполнение строки 910.01.001 (Начисленные доходы)

В строке выводятся данные о начисленных доходах физическим лицам, зарегистрированные в регистр накопления "ИПН сведения о доходах" за соответствующий месяц налогового периода. Если установлен признак включения в отчет данных по получателям алиментов, то к этой сумме также будут добавлены удержанные за соответствующие месяцы суммы по исполнительным листам, определяемые как приход по регистру накопления "Взаиморасчеты организаций с получателями по ИЛ".

В строку включаются сведения только по работникам, являющимся гражданами РК.

- Заполнение строки 910.01.002 (Сумма индивидуального подоходного налога, подлежащего перечислению в бюджет)

В строке выводятся данные о суммах индивидуального подоходного налога, подлежащего перечислению в бюджет. Информация собирается на основании записей-расходов по регистру накопления "ИПН расчеты с бюджетом" с видом строки "Удержание". В зависимости от настроек учетной политики по налоговому учету "При начислении ИПН, ОПВ и ВОСМС принимать исчисленные суммы к учету как удержанные" эта сумма равняется фактически исчисленному налогу (если настройка "При начислении ИПН, ОПВ и ВОСМС принимать исчисленные суммы к учету как удержанные" включена), либо определяется в процессе расчета документа "Удержание ИПН, ОПВ и ВОСМС по налоговому учету" (настройка "При начислении ИПН, ОПВ и ВОСМС принимать исчисленные суммы к учету как удержанные" отключена).

В строку включаются сведения только по работникам, являющимся гражданами РК.

Раздел. Исчисление социальных отчислений и обязательных пенсионных взносов по выплаченным доходам, в том числе по доходам нерезидентов Республики Казахстан

- Заполнение строки 910.01.010 (Начисленные доходы, с которых удерживаются (начисляются) обязательные пенсионные взносы)

В строке выводятся данные о доходах, с которых удерживаются (начисляются) обязательные пенсионные взносы. Данная строка определяется как фактически исчисленная сумма обязательных пенсионных взносов, умноженная на 10.

Примечание!!! Строка 910.01.010 рассчитывается только по работникам.

- Заполнение строки 910.01.011 (Сумма обязательных пенсионных взносов, подлежащих перечислению)

В строке выводятся данные о суммах обязательных пенсионных взносов, подлежащих перечислению в пенсионные фонды, которые определяются на основании данных о приходах в регистре накопления "ОПВ подлежит перечислению в фонды" с видом строки "Удержание". На расчет суммы взносов, подлежащий перечислению, влияет настройка в учетной политике по налоговому учету "При начислении ИПН, ОПВ и ВОСМС принимать исчисленные суммы к учету как удержанные" (см. порядок заполнения строки 910.00.019).

Примечание!!! Строка 910.01.011 рассчитывается только по работникам.

- Заполнение строки 910.01.012 (Доходы физических лиц, с которых исчисляются социальные отчислений)

В строке выводятся данные о расходах работодателя, выплачиваемых физическим лицам в виде доходов, определяемых как значение строки 910.01.013, умноженное на 100/процент социальных отчислений за соответствующий месяц налогового периода.

- Заполнение строки 910.01.013 (Сумма социальных отчислений)

В строке выводятся данные о суммах социальных отчислений, исчисленных за соответствующий месяц налогового периода, определяемые как приходы по регистру накопления "СО расчеты с фондами" с видом строки "Исчисление".
