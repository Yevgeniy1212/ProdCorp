Отчет "Упрощенная декларация для субъектов малого бизнеса, форма 910.00 (XML)" за 2007-2008 год.

Отчет предназначен для составления упрощенной декларации субъектами малого бизнеса (форма 910.00) с возможностью последующей выгрузки в формат XML. Форма адаптирована к версии ЭФНО 1.6.1.41 от 13.02.2009г.

Специальный налоговый режим на основе упрощенной декларации применяют субъекты малого бизнеса, определенные п. 2 ст. 376 Налогового Кодекса РК, соответствующие установленным ограничениям:

  • Индивидуальные предприниматели - в справочнике "Организации" для них реквизит "Юр. / физ. лицо" должен быть установлен в значение "Физ. лицо";
  • Юридические лица - в справочнике "Организации" для них реквизит "Юр. / физ. лицо" должен быть установлен в значение "Юр. лицо".

 

- Заполнение строки 910.00.001

В строке выводится сумма налогооблагаемого дохода, полученного за налоговый период, определяемая как разность кредитового и дебетового оборотов по счетам разделов 6000, 6100, 6200, 6300 и 6400, при этом проводка Дт 6ххх - Кт 5610 (Итоговая прибыль итоговый убыток), формируемая при реформации баланса в момент закрытия месяца, не учитывается. В качестве облагаемых доходов учитываются только те виды, для которых в справочнике "Доходы" установлен признак "Является объектом налогообложения" на закладке "Отражение в упрощенной декларации".

- Заполнение строки 910.00.002

В строке выводятся данные о количестве работников, включая самого индивидуального предпринимателя. Количество работников за один месяц определяется как списочная численность работников согласно Инструкции по заполнению статистической отчетности по труду, утвержденной Приказом Председателя Агентства Республики Казахстан по статистике от 12 июля 2002 года N 38-Г. При подсчете численности учитывается количество дней, в течение которых действовал трудовой договор работника в месяце, в отношении к общему числу дней в месяце.

- Заполнение строки 910.00.003

В строке выводится среднесписочная численность работников в целом за налоговый период, определяемая по формуле:

                        910.00.002A + 910.00.002B + 910.00.002С

910.00.003 =   -------------------------------------------------------

                                                          3

- Заполнение строки 910.00.004

В строке выводится среднесписочная численность, превышающая предельную численность. Согласно п. 2 ст. 376 Налогового Кодекса РК для индивидуальных предпринимателей предельная среднесписочная численность работников за налоговый период составляет 25 человек, для юридических лиц - 50 человек.

- Заполнение строки 910.00.005

В строке выводится сумма исчисленных налогов, определяемая согласно п. 1 ст. 377 Налогового Кодекса РК по формуле:

910.00.005 = 910.00.001 * 3%

- Заполнение строки 910.00.006

В строке выводится сумма налогов с дохода, превышающего предельную сумму дохода. Согласно п.2 ст. 376 предельных доход индивидуального предпринимателя за налоговый период составляет 10 млн. тенге, для юридического лица - 25 млн. тенге. Налог с дохода, превышающего предельную сумму доход, определяется по формуле:

910.00.006 = (910.00.001 - 10 000 000) * 3% - для индивидуального предпринимателя

910.00.006 = (910.00.001 - 25 000 000) * 3% - для юридического лица.

- Заполнение строки 910.00.007

В строке указывается среднемесячная заработная плата одного наемного работника, определяемая по формуле:

                         ФондЗП1                              ФондЗП2                    ФондЗП3

                     --------------------     +     -------------------   +    -----------------

                      910.00.002A - 1                   910.00.002B - 1         910.00.002C - 1

910.00.007 =  ----------------------------------------------------------------------- - для индивидуального предпринимателя

                                                                    3

                         ФондЗП1                              ФондЗП2                    ФондЗП3

                     --------------------     +     -------------------   +    -----------------

                      910.00.002A                        910.00.002B                 910.00.002C

910.00.007 =  ----------------------------------------------------------------------- - для юридического лица

                                                                    3

где ФондЗП1, ФондЗП2 и ФондЗП3 - фонд заработной платы наемных работников соответственно за 1, 2 и 3 месяцы. Сведения о фонде заработной платы работников приводятся в отчете "Анализ начислений работникам организаций" (показатель - фонд заработной платы).

-  Заполнение строки 910.00.008

В строке выводится уменьшение суммы налога в зависимости от среднесписочной численности работников, в случае если среднемесячная заработная плата наемных работников (910.00.007) составила у индивидуальных предпринимателей не менее 2-кратного, у юридических лиц не менее 2,5-кратного минимального размера месячной заработной платы, установленной законодательным актом РК. Расчет значения строки 910.00.008 осуществляется по формуле:

910.00.008 = (910.00.005 - 910.00.006) * (910.00.003 - 910.00.004) * 0,015

-  Заполнение строки 910.00.009

В строке указывается сумма налогов после корректировки, рассчитываемая по формуле:

910.00.009 = 910.00.005 - 910.00.008

- Заполнение строки 910.00.010

В строке указывается сумма индивидуального (корпоративного) подоходного налога, подлежащая уплате в бюджет, определяемая как:

910.00.010 = 910.00.009 * 0,5

- Заполнение строки 910.00.011

В строке указывается сумма социального налога, подлежащая уплате в бюджет, определяемая как:

910.00.011 = 910.00.009 * 0,5 - 910.00.012 - 910.00.014

- Заполнение строки 910.00.012

В строке указывается сумма социальных отчислений за наемных работников (переносится из строки 201.00.010D формы 201.00). Сведения о социальных отчисления за работников приводятся в отчете "Справка-расчет "Индивидуальный подоходный налог"" (показатель - социальные отчисления).

- Заполнение строки 910.00.013

В строке указывается доход для исчисления социальных отчислений за индивидуального предпринимателя. Сумма определяется как фактически исчисленные социальные отчисления за индивидуального предпринимателя по регистру накопления "СО расчеты с фондами" (если расчет отчислений производился с помощью документа "Расчет налогов, взносов и отчислений сотрудников организаций"), деленные на 3%,  либо заносится вручную.

- Заполнение строки 910.00.014

В строке выводятся социальные отчисления за индивидуального предпринимателя, рассчитываемые как:

910.00.014 = 910.00.013 * 3% (процент социальных отчислений)

 - Заполнение строки 910.00.015

В строке указывается доход для исчисления обязательных пенсионных взносов за индивидуального предпринимателя. Сумма определяется как фактически исчисленные ОПВ за индивидуального предпринимателя по регистру накопления "ОПВ расчеты с фондами" (если расчет отчислений производился с помощью документа "Расчет удержаний сотрудников организаций"), деленные на 10%,  либо заносится вручную.

- Заполнение строки 910.00.016

В строке выводятся обязательные пенсионные взносы за индивидуального предпринимателя, рассчитываемые как:

910.00.016 = 910.00.015 * 10% (процент обязательных пенсионных взносов)
