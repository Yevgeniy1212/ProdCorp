Отчет «Средняя численность сотрудников организаций»

Отчет выводит среднесписочную численность и фактическую численность сотрудников организации за заданный в отчете период. Поддерживается расшифровка показателей отчета в разрезе месяцев периода, дней месяца, должностей, подразделений и сотрудников. По умолчанию учитываются работники, работающие по основному месту работы, совместительству и договору ГПХ.

Виды отчета:

  • «Численность работников в среднем за период» - данные фактической и списочной численности выводятся полностью за весь период.
  • «Численность по месяцам» - данные фактической и списочной численности выводятся в разрезе месяцев за выбранный период.

    Два способа вывода отчета:

  • «Для проверки общих статистических отчетов» - используется для расшифровки статистических отчетов «1-Т (годовая)», «1-Т (условия труда) годовая», «1-Т (ЗП)», «1-Т квартальная», «1-Т (месячная)», «№ 1-Т (месячная). Приложение 1».

    В списочную численность включаются работники, принятые по трудовому договору, независимо от срока его заключения, кроме лиц, выполняющих работу по договорам гражданско-правового характера, а также принятых на работу по совместительству.

    Для получения фактической численности из численности работников списочного состава исключаются следующие категории работников:

    - находящиеся в отпуске по беременности и родам;

    - отпуске по уходу за ребенком;
    - учебном отпуске;
    - неоплачиваемом учебном отпуске;
    - отпуск без сохранения зарплаты;
    - отпуске без содержания (в связи с простоем производства).
  • «Для проверки статистического отчета "2-Т (ПРОФ)"» - используется для расшифровки статистического отчета «2-Т (ПРОФ)».
    В списочную численность включаются работники, принятые по трудовому договору, независимо от срока его заключения, кроме лиц, выполняющих работу по договорам гражданско-правового характера, а также принятых на работу по совместительству.

    Для получения фактической численности из численности работников списочного состава исключаются следующие категории работников:

    - принятые на работу после 1 числа месяца отчетного периода;

    - уволенные до 1 числа месяца, следующего за месяцем отчетного периода;

    - принятые на неполный рабочий день или неполную рабочую неделю;

    - находящиеся в отпуске по беременности и родам;
    - отпуске по уходу за ребенком;
    - учебном отпуске;
    - неоплачиваемом учебном отпуске;
    - отпуск без сохранения зарплаты;
    - отпуске без содержания (в связи с простоем производства);
    - ежегодном отпуске, отсутствовавшие по невыясненной причине (в т.ч. прогул);
    - имевшие листки временной нетрудоспособности.
