Отчет "Отчет по проводкам (упр.)

Отчет используется в основном в качестве механизма расшифровки к другим отчетам по управленческому плану счетов. Предназначен для получения данных по конечным хозяйственным операциям за указанный период по выбранному сценарию либо по фактическим данным. Для формирования отчета обязательно должен быть настроен отбор по счету. Также можно настроить отбор по другим параметрам операций плана счетов.

Также см. «Общие настройки отчетов»
