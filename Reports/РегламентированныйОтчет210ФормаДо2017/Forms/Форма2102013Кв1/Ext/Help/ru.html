Отчет «Декларация по индивидуальному подоходному налогу и социальному налогу по иностранцам и лицам без гражданства, форма 210.00 (XML)» за 2013 год.

Отчет «Декларация по индивидуальному подоходному налогу и социальному налогу, форма 210.00 (XML)» предназначен для исчисления индивидуального подоходного налога, социального налога, а также для исчисления, удержания (начисления) и перечисления сумм обязательных пенсионных взносов в накопительный пенсионный фонд (далее – обязательные пенсионные взносы), начисления и перечисления сумм социальных отчислений в Государственный фонд социального страхования (далее – социальные отчисления) для иностранных граждан и лиц без гражданства.

Если в настройках параметров учета (меню Предприятие \ Настройка параметров учета) установлен признак поддержки работы со структурными подразделениями, то при открытии отчета необходимо указать налогоплательщика и налоговый комитет, в который будет предоставлен отчет. Если работы со структурными подразделениями в налоговом учете не поддерживается, то при открытии отчета требуется выбрать только налогоплательщика, налоговый комитет будет определен по реквизиту «Налоговый комитет» из справочника «Организации».

В отчет, кроме данных о работниках организации, также могут быть включены сведения о выплаченных доходах получателей по исполнительным листам. Для этого предназначен флажок «Включать в отчет получателей алиментов» на закладке «Настройка расчета». При установленном флажке данные о получателях по исполнительным листам будут включены в отчет. В форме отображаются данные только о тех получателях алиментов, которые являются физическими лицами - иностранными гражданами.

Для проверки правильности составления Декларации предназначен отчет «Регистр налогового учета по индивидуальному подоходному налогу и социальному налогу», позволяющий в произвольных разрезах проанализировать корректность формирования налогового отчета.

Внимание! Автоматическое заполнение предусмотрено только для строк, перечисленных ниже по тексту, все прочие строки формы в случае наличия данных, отражаемых в них, необходимо заполнить вручную.

Декларация по индивидуальному подоходному налогу и социальному налогу (форма 210.00)

 

Раздел. Исчисление индивидуального подоходного налога с доходов резидентов

- Заполнение строки 210.00.001 (Начисленные доходы)

В строке выводятся данные о начисленных доходах физическим лицам - иностранцам, зарегистрированные в регистр накопления «ИПН сведения о доходах» за соответствующий месяц налогового периода. Если установлен признак включения в отчет данных по получателям алиментов, то к этой сумме также будут добавлены удержанные за соответствующие месяцы суммы по исполнительным листам, определяемые как приход по регистру накопления «Взаиморасчеты организаций с получателями по ИЛ».

- Заполнение строки 210.00.002 (Сумма исчисленного индивидуального подоходного налога)

В строке выводятся данные о суммах индивидуального подоходного налога, исчисленного за соответствующие месяцы налогового периода, определяемого как приход по регистру накопления «ИПН расчеты с бюджетом» для вида строк «Исчисление».

- Заполнение строки 210.00.003 (Задолженность по доходам, не выплаченным иностранцам и лицам без гражданства)

В строке выводятся данные о сумме задолженности по доходам, невыплаченным иностранцам и лицам без гражданства на конец последнего месяца периода составления Декларации, определяемой как конечный остаток по следующим регистрам накопления:

    • Взаиморасчеты с работниками организаций - задолженность по текущим расчетам с работниками (соответствует кредитовому остатку по счету 3350 в бухгалтерском учете в случае, если заработная плата была отражена на счетах учета БУ);
    • Взаиморасчеты с депонентами организаций - задолженность по депонированной заработной плате (соответствует кредитовому остатку по счету 3384 в бухгалтерском учете в случае, если заработная плата была отражена на счетах учета БУ);
    • Взаиморасчеты организаций с контрагентами-физлицами - задолженность по расчетам с контрагентами-физлицами, у которых приобретались активы и/или услуги (оформляется с помощью документа «Расчет налогов при поступлении активов и услуг»);
    • Взаиморасчеты организаций с получателями по ИЛ - задолженность, связанная с расчетами по исполнительным листам. Если в учетной политике по налоговому учету не установлен признак «Засчитывать в выплаченные доходы удержания по исполнительным листам», то включается остаток по не перечисленным суммам удержаний по исполнительным листам и суммам почтового/комиссионного сбора как задолженность физлицу-работнику организации. Если при формировании отчета на закладке «Настройка расчета» включен флажок «Включить в отчет получателей алиментов», то в невыплаченную задолженность также включаются не перечисленные суммы удержаний по исполнительным листам физлицам-получателям алиментов.

- Заполнение строки 210.00.004 (Выплачено доходов)

В строке выводятся данные о выплаченных физлицам - иностранцам доходах по данным регистра накопления «Выплаченные доходы работникам организаций по налоговому учету». Если в учетной политике по налоговому учету не установлен признак «Засчитывать в выплаченные доходы удержания по исполнительным листам», то в выплаченные доходы будут включены фактически перечисленные суммы удержаний по исполнительным листам и суммы почтового/комиссионного сбора по физлицу-работнику организации. Если при формировании отчета установлен флажок «Включить в отчет получателей алиментов», то в выплаченные доходы будут дополнительно включены фактически перечисленные суммы удержаний по исполнительным листам получателям алиментов.

- Заполнение строки 210.00.005 (Сумма индивидуального подоходного налога, подлежащего перечислению в бюджет)

В строке выводятся данные о суммах индивидуального подоходного налога, подлежащего перечислению в бюджет. Информация собирается на основании записей-расходов по регистру накопления «ИПН расчеты с бюджетом» с видом строки «Удержание». В зависимости от настроек учетной политики по налоговому учету «При начислении ИПН и ОПВ принимать исчисленные суммы к учету как удержанные» эта сумма равняется фактически исчисленному налогу (если настройка «При начислении ИПН и ОПВ принимать исчисленные суммы к учету как удержанные» включена), либо определяется в процессе расчета документа «Удержание ИПН и ОПВ по налоговому учету» (настройка «При начислении ИПН и ОПВ принимать исчисленные суммы к учету как удержанные» отключена).

- Заполнение строки 210.00.006 (Сумма налога, подлежащего перечислению за головную организацию)

В строке выводятся данные об индивидуальном подоходном налоге, подлежащем перечислению в бюджет, с доходов работников, относящихся к головной организации. Данные собираются по расходу регистра «ИПН расчеты с бюджетом» с видом строки «Удержание» для тех строк, у которых графы «Структурная единица» и «Налогоплательщик» совпадают. Данная строка заполняется в случае, если в настройках параметров учета включена поддержка работы со структурными подразделениями.

- Заполнение строки 210.00.007 (Сумма налога, подлежащего перечислению по филиалам/представительствам)

В строке выводятся данные об индивидуальном подоходном налоге, подлежащем перечислению в бюджет, с доходов работников, относящихся к структурным подразделениям организации. Данные собираются по расходу регистра «ИПН расчеты с бюджетом» с видом строки «Удержание» для тех строк, у которых графы «Структурная единица» и «Налогоплательщик» различаются. Данная строка заполняется в случае, если в настройках параметров учета включена поддержка работы со структурными подразделениями.

Исчисление индивидуального подоходного налога с доходов иностранцев и лиц без гражданства (форма 210.01)

Приложение 210.01 автоматически заполняется данными по исчислению индивидуального подоходного налога с доходов иностранцев и лиц без гражданства.

В форме, кроме данных о работниках-иностранцах организации, доступен также вывод сведений о начисленных и выплаченных доходах индивидуальным предпринимателям, адвокатам и частным нотариусам, не являющихся гражданами РК, зарегистрированных документом "Регистрация прочих доходов в целях налогообложения" или "Расчет налогов при поступлении активов и услуг". Для этого предназначен флажок "Включить индивид. предпринимателей и частных нотариусов" в дополнительной форме отчета.

- Заполнение графы A (Номер строки)

В графе выводится номер строки по порядку в дополнительной форме.

- Заполнение графы B (ФИО иностранца или лица без гражданства)

В графе выводится фамилия, имя, отчество физического лица. Если строка отчета содержит сведения о доходах и налогах по физлицу-работнику организации, то ФИО физлица определяется из регистра сведений "Фамилия, имя, отчество физического лица", в случае отсутствия записи о полной фамилии, имени и отчестве физлица в регистре, в отчет выводится наименование из справочника "Физические лица".

- Заполнение графы С (РНН иностранца или лица без гражданства)

В графе выводится РНН физического лица из справочника "Физические лица" или справочника "Контрагенты", если строка относится к индивидуальному предпринимателю, адвокату или частному нотариусу.

- Заполнение графы D (ИИН иностранца или лица без гражданства)

В графе выводится ИИН физического лица из справочника "Физические лица" или справочника "Контрагенты", если строка относится к индивидуальному предпринимателю, адвокату или частному нотариусу.

- Заполнение графы Е (признак резиденства)

В графе указывается значение "1" в случае, если физическое лицо является резидентом РК и "0", если нерезидентом.

В случае необходимости в настройке формы 210.01 можно указать собственную систему отметки этой графы .

- Заполнение графы F (Код страны резиденства)

В графе указывается числовой код страны резиденства, указанной для иностранца или лица без гражданства в регистре сведений "Гражданство физлиц". Страна резиденства определяется как последняя иностранная страна по иностранцу на дату конца периода формирования отчета.

- Заполнение графы G (Номер налоговой регистрации иностранца или лица без гражданства в стране резиденства)

В графе указывается Номер налоговой регистрации, указанный для иностранца или лица без гражданства в регистре сведений "Гражданство физлиц" на дату конца периода формирования отчета.

- Заполнение графы H (Вид документа, удостоверяющего личность иностранца или лица без гражданства)

Графа заполняется значениями:
       01 – паспорт иностранного гражданина;
       02 – удостоверение личности иностранного гражданина;
       03 – паспорт моряка;
       04 - вид на жительство.
Данные значения устанавливаются для вида документа физического лица в справочнике «Документы, удостоверяющие личность». Для корректного заполнения данной графы необходимо заполнить у имеющихся видов документов реквизит «Код вида документа в налоговой отчетности» в справочнике «Документы, удостоверяющие личность».

- Заполнение графы I (Код вида дохода)

В графе указывается код вида дохода по следующей схеме: В случае если налогоплательщик является иностранной организацией (установлен флажок «Иностранная организация» в справочнике «Организации»), в данной графе устанавливается значение «1181» - доходы физического лица-нерезидента от деятельности в Республике Казахстан по трудовому договору (контракту), заключенному с нерезидентом, являющимся работодателем, в ином случае (если организация является резидентом) - «1180» - доходы физического лица-нерезидента от деятельности в Республике Казахстан по трудовому договору (контракту), заключенному с резидентом, являющимся работодателем. При необходимости код вида дохода можно изменить в форме, выбрав значение из классификатора видов доходов.

- Заполнение графы J, K, L (сведения о международном договоре).

Автоматическое заполнение граф не предусмотрено, сведения заполняются вручную.

- Заполнение графы M (Ставка налога)

Для автоматически заполняемых данных из подсистемы учета заработной платы ставка устанавливается в размере 10%.

- Заполнение графы N (Начислено доходов)

Графа заполняется суммой начисленного физлицу дохода, зарегистрированного в регистре накопления "ИПН сведения о доходах" за месяцы налогового периода, входящие в период формирования отчета. Если в основной форме отчета отмечена настройка "Включать в отчет получателей алиментов", то для строк, относящихся к получателям по исполнительным листам, будут рассчитаны суммы полученного дохода за период отчета.

- Заполнение графы O (Налоговые вычеты)

Графа заполняется суммой налоговых вычетов, предоставленных физлицу в соответствии с подпунктом 1) пункта 1 статьи 166 Налогового кодекса при расчете индивидуального подоходного налога документом "Расчет ИПН, ОПВ и удержаний".

- Заполнение графы P (Обязательные пенсионные взносы)

Графа заполняется суммой обязательных пенсионных взносов, исчисленных с доходов физлица в соответствии с пенсионным законодательством Республики Казахстан и относимые на вычеты в соответствии с подпунктом 2) пункта 1 статьи 166 Налогового кодекса при расчете документа "Расчет ИПН, ОПВ и удержаний" (на основании данных регистра "ОПВ расчеты с фондами")

- Заполнение графы Q (Добровольные пенсионные взносы)

Автоматизированное заполнение данной графы не поддерживается.

- Заполнение графы R (Страховые премии)

Графа заполняется суммами страховых премий, вносимых в свою пользу физическим лицом по договорам накопительного страхования и относимых на вычеты согласно подпункту 4) пункта 1 статьи 166 Налогового кодекса (см. документ "Заявление на предоставление вычета по ИПН").

- Заполнение графы S (Суммы, направленные на погашение вознаграждения по жилищным займам)

Графа заполняется суммами, направленными на погашение вознаграждения по жилищным займам и промежуточным жилищным займам, полученным физическим лицом-резидентом Республики Казахстан в жилищных строительных сберегательных банках на проведение мероприятий по улучшению жилищных условий на территории Республики Казахстан, в соответствии с законодательством Республики Казахстан о жилищных строительных сбережениях, и относимые на вычеты, согласно подпункту 5) пункта 1 статьи 166 Налогового кодекса (см. документ "Заявление на предоставление вычета по ИПН").

- Заполнение графы T (Оплата медицинских услуг (кроме косметологических))

Автоматизированное заполнение данной графы не поддерживается.

- Заполнение графы V (Доходы, не подлежащие налогообложению)

Графа заполняется суммами доходов, не подлежащими обложению ИПН в соответствии со статьей 156 Налогового кодекса. В графе указываются суммы начислений, для которых в форме настройки вида расчета установлен порядок обложения ИПН "Не облагается целиком", а также льготы, предоставленные физлицу-инвалиду, либо участнику ВОВ, либо другой категории лиц, приравненных к ним, при расчете индивидуального подоходного налога документом "Расчет ИПН, ОПВ и удержаний".

- Заполнение графы U (Индивидуальный подоходный налог)

Графа заполняется суммами исчисленного индивидуального подоходного налога по записям с видом движения "Приход" по регистру накопления "ИПН расчеты с бюджетом" с видом строки "Исчисление".

- Заполнение графы W (Выплачено доходов)

Графа заполняется суммами выплаченных физлицу доходов. Порядок выборки данных для заполнения графы аналогичен строке 210.00.005.

- Заполнение графы X (Выплаты, не облагаемые социальным налогом, за исключением ОПВ)

Графа заполняется суммам доходов, начисленных физлицу, которые не облагаются социальным налогом (в форме настройки вида расчета установлен порядок обложения для СН "Не облагается целиком"). Если при формировании отчета установлен признак "Включать в отчет получателей алиментов", то эта графа также будет включать суммы удержаний по исполнительным листам для физлиц-получателей алиментов.

- Заполнение графы Y (Выплаты, облагаемые социальным налогом)

Графа рассчитывается по формуле:

Графа Y = Графа N (начислено доходов) - Графа P (Обязательные пенсионные взносы) - Графа X (Выплаты, не облагаемые социальным налогом, за исключением ОПВ)

- Заполнение графы Z (Социальный налог)

Графа заполняется суммами социального налога, исчисленного по физическому лицу, за месяцы налогового периода, входящие в отчетный период, согласно данным регистра накопления "СН исчисленный".

- Заполнение графы AA (Расходы работодателя, с которых исчисляются социальные отчисления)

Графа заполняется аналогично строке 210.00.021.

- Заполнение графы AB (Сумма социальных отчислений)

Графа заполняется аналогично строке 210.00.022.

Исчисление суммы индивидуального подоходного налога и социального налога по структурному подразделению (форма 210.02)

- Заполнение строки 210.02.001 (Сумма индивидуального подоходного налога, подлежащего перечислению)

В строке выводятся данные о суммах индивидуального подоходного налога, подлежащего перечислению в бюджет по соответствующей структурной единице. Информация собирается на основании записей с видом движения "Расход" по регистру накопления «ИПН расчеты с бюджетом» с видом строки «Удержание». В зависимости от настроек учетной политики по налоговому учету «При начислении ИПН и ОПВ принимать исчисленные суммы к учету как удержанные» эта сумма равняется фактически исчисленному налогу (если настройка «При начислении ИПН и ОПВ принимать исчисленные суммы к учету как удержанные» включена), либо определяется в процессе расчета документа «Удержание ИПН и ОПВ по налоговому учету» (настройка «При начислении ИПН и ОПВ принимать исчисленные суммы к учету как удержанные» отключена).

- Заполнение строки 210.02.002 (Сумма обязательных пенсионных взносов, подлежащих перечислению)

В строке выводятся данные о суммах обязательных пенсионных взносов, подлежащих перечислению в пенсионные фонды, которые определяются на основании записей с видом движения "Приход" в регистре накопления «ОПВ подлежит перечислению в фонды» с видом строки «Удержание» по соответствующей структурной единице. На расчет суммы взносов, подлежащий перечислению, влияет настройка в учетной политике по налоговому учету «При начислении ИПН и ОПВ принимать исчисленные суммы к учету как удержанные».

- Заполнение строки 210.02.003 (Сумма исчисленного социального налога)

В строке выводятся данные о суммах исчисленного социального налога, определяемых по регистру накопления «СН исчисленный» по соответствующей структурной единице.

- Заполнение строки 210.02.004 (Сумма социальных отчислений)

В строке выводятся данные о суммах социальных отчислений, исчисленных за соответствующий месяц налогового периода, определяемые по виду движения "приход" по регистру накопления «СО расчеты с фондами» с видом строки «Исчисление» по соответствующей структурной единице.
