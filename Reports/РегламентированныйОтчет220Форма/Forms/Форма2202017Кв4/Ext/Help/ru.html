Отчет «Декларация по индивидуальному подоходному налогу, форма 220.00 (XML)» за 2017 год

Отчет «Декларация по индивидуальному подоходному налогу» предназначен для составления отчетности по ИПН физическими лицами – индивидуальными предпринимателями, осуществляющими исчисление и уплату налогов в общеустановленном порядке,  а также физическими лицами-нерезидентами.

 

Работа с расшифровкой строк регламентированного отчета.

Расшифровка строк возможна для тех ячеек, которые выделены голубым цветом.

 

СОДЕРЖАНИЕ

1) Заполнение приложения 220.01 «Расходы налогоплательщиков, не являющихся плательщиками НДС, по реализованным товарам, выполненным работам, оказанным услугам»

2) Заполнение приложения 220.04 «Вычеты по фиксированным активам»

При заполнении формы 220.00 учитываются данные не только из заполненных приложений, но и непосредственно данные, отнесенные на статьи затрат или доходы, для которых определен код строки Декларации в регистре сведений «Коды строк декларации по налогу на прибыль», начинающийся с «220.00». При заполнении строк декларации следует исходить из того, что если в конфигурации не автоматизировано заполнение нужного приложения, то необходимо для статьи затрат или дохода в регистре «Коды строк декларации по налогу на прибыль» проставить код строки, начинающийся с «220.00», а не код приложения, заполнение которого не предусмотрено. Если необходимого кода строки нет в классификаторе, то его нужно установить «вручную».

- Ф.И.О. налогоплательщика

    4) 220.00.009IIIE - Маркетинговые услуги
    5) 220.00.009IIIF - Дизайнерские услуги

    Дт счетов раздела 70Н «Расходы», 80Н «Производство» Кт счетов раздела 3310Н «Расчеты с контрагентами и прочие взаиморасчеты», Дт счетов раздела 1600Н «Прочие активы» Кт счетов раздела 3310Н «Расчеты с контрагентами и прочие взаиморасчеты»,  имеющих аналитику «Контрагенты», в разрезе статей затрат, имеющих соответствующий код строки декларации в регистре сведений «Коды строк декларации по налогу на прибыль».

           

    Внимание!!! Буквы в кодах строк Декларации - латинские.

    Примечание!!! Если установлен флаг «Отображать затраты, у которых не установлен код строки Декларации» то в окне сообщений будут выводиться статьи затрат по которым есть оборот, но для статьи затрат не установлен код строки декларации. Данные суммы не будут попадать в приложение.

    ЗАПОЛНЕНИЕ ПРИЛОЖЕНИЯ 220.04.

    В данной форме отражаются данные только по тем фиксированным активам, по которым и в регистре сведений «Фиксированные активы организации» существуют записи на период формирования отчета. Кроме этого для этих записей параметр «Группа налогового учета» должен иметь вид налогового учета в группе ФА «по объектам» - для I группы и «по группам» - для II, III и IV групп (см. Справочник «Группы налогового учета ФА»).

     

    Раздел. Вычеты по фиксированным активам

    - Заполнение строки 220.04.001 (Стоимостный баланс групп (подгрупп) на начало налогового периода)

    В строку переносится СНД счета 2100Н «ВА, включенные в стоимостный баланс» с видом учета НУ;

    - Заполнение строки 220.04.002 (Стоимость поступивших фиксированных активов)

    В строку переносится разница дебетового и кредитового оборота счета 2100Н «ВА, включенные в стоимостный баланс» с видом движения стоимости «Поступление» с видом учета НУ;

    - Заполнение строки 220.04.003 (Стоимость выбывших фиксированных активов)

    В строку переносится разница кредитового и дебетового оборота счета 2100Н «ВА, включенные в стоимостный баланс» с видом движения стоимости ФА «Выбытие» с видом учета НУ;

    - Заполнение строки 220.04.004 (Последующие расходы по фиксированным активам, относимые на увеличение стоимостных балансов групп (подгрупп) в соответствии с пунктом 3 статьи 122 Налогового кодекса)

    В строку переносится дебетовый оборот счета 2100Н «ВА, включенные в стоимостный баланс» с видом движения стоимости «Увеличение стоимости за счет ремонта» и видом учета НУ;

    - Заполнение строки 220.04.005 (Стоимостный баланс групп (подгрупп) на конец налогового периода)

    Стоимостный баланс подгруппы, группы на конец налогового периода определяется по формуле:

    220.04.005 = 220.04.001 + 220.04.002 – 220.04.003 + 220.04.004;

    - Заполнение строки 220.04.006 (Амортизационные отчисления по фиксированным активам)

    В строку переносится начисленная амортизация по фиксированным активам, т.е кредитовый оборот счета 2100Н «ВА, включенные в стоимостный баланс» с видом движения стоимости ФА «Амортизация» и видом учета НУ;

    - Заполнение строки 220.04.007 (Двойная норма амортизации в соответствии с пунктом 6 статьи 120 НК)

    В строку переносится начисленная амортизация по фиксированным активам которые в регистре сведений «Фиксированные активы, учитываемые отдельно» на конец текущего налогового периода еще не включены в состав группы, т.е кредитовые обороты счетов раздела 20Н «Внеоборотные активы» с видом движения стоимости ФА «Амортизация» и видом учета НУ;

    - Заполнение строки 220.04.008 (Величина стоимостного баланса группы и признаваемый убыток по подгруппам (I группы) при выбытии всех фиксированных активов)

    В строку переносится кредитовый оборот счета 2100Н «ВА, включенные в стоимостный баланс» с видом движения стоимости «Списание при выбытии всех активов группы» и видом учета НУ;

    - Заполнение строки 220.04.009 (Величина стоимостного баланса группы (подгруппы) менее 300 месячных расчетных показателей, относимая на вычеты)

    В строку переносится кредитовый оборот счета 2100Н «ВА, включенные в стоимостный баланс» с видом движения стоимости «Списание стоимостного баланса группы менее минимума» и видом учета НУ;

    - Заполнение строки 220.04.010 (Последующие расходы по фиксированным активам, относимые на вычеты в соответствии с пунктом 2 статьи 122 Налогового кодекса)

    В строку переносится дебетовый оборот счета 7900Н «Расходы на ремонт ВА» по виду ремонта ОС «Текущий» с видом учета НУ;

    - Заполнение строки 220.04.011 ( Итого вычетов по фиксированным активам)

    Строка 220.04.011 определяется как сумма строк с 220.04.011I по 220.04.011IV;

    Строка 220.04.011I определяется как сумма строк 220.04.006I, 220.04.007I, 220.04.009I, 220.04.010I (220.04.006I + 220.04.007I + 220.04.009I + 220.04.010I);

    Строки 220.04.011II - 220.04.011IV расчитываются как суммы соответствующих строк 220.04.006, 220.04.007, 220.04.008, 220.04.009, 220.04.010 (220.04.006 + 220.04.007 + 220.04.008 + 220.04.009 + 220.04.010);

    Для корректного заполнения приложения 220.04 необходимо, чтобы были выполнены операции раздела «Регламентные операции налогового учета» в помощнике «Закрытия месяца» за отчетный год.
