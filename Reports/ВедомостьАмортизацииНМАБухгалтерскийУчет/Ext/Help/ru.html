Отчет "Ведомость амортизации НМА"

Отчет предназначен для анализа данных по нематериальным активам (НМА).

В отчете можно проанализировать данные по бухгалтерскому и управленческому учету, установить отбор по счетам учета или реквизитам нематериального актива, вывести дату установки текущего состояния НМА и др.

Счета бухгалтерского учета, по которым собираются данные, определяются из регистра сведений Счета учета НМА: выбираются все возможные значения реквизитов Счет учета (БУ) и Счет начисления амортизации (БУ) в разрезе организаций (структурных подразделений).

Показатели Первоначальная стоимость и Стоимость для вычисления амортизации определяются из регистра сведений Первоначальные сведения о НМА на конец отчетного периода.

Показатель Остаточная стоимость рассчитывается как разность показателей Стоимость (на конец периода) - Амортизация (на конец периода).

Данные из регистров сведений получаются на конец отчетного периода.


В отчете можно установить следующие настройки:

Показатели
Группировка
Отбор
Сортировка
Оформление
Дополнительные данные
