Отчет "ЭЛЕВАТОР: Журнал регистрации приказов на отгрузку"

Конфигурация "1С-Рейтинг: Элеватор".

Отчет отражает данные количества зерна, фактически отгруженного зерна и остаток зерна по приказу. Предусмотрена возможность отражения данных по установленному отбору.

Настройка отчета

По кнопке "Сформировать" выполняется построение отчета в соответствии с текущей настройкой.

Кнопки сохранения и восстановления значений позволяют сохранять пользовательские настройки отчета, чтобы использовать их в дальнейшем.

В форме отчета предусмотрены несколько видов отбора данных:

  • по организации;
  • по контрагенту;
  • по номенклатуре.
