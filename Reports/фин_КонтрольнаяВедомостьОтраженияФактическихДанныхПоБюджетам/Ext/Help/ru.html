Отчет "Контрольная ведомость отражения фактических данных по бюджетам"

Отчет предназначен для сверки данных информационной базы с их отражением в фактических данных по бюджетам. Это позволяет отслеживать отраженные и неотраженные в бюджетах хозяйственные операции, проводить сверку и анализ настроек и ввода первичных данных.

Настройки формирования отчета отображаются при его открытии, либо при нажатии на кнопку «Показать настройки». В списке «Источник данных» приводятся все источники (таблицы информационной базы), для которых настроено отражение фактических данных и доступна сверка. Источником могут быть: шапка какого-либо документа, его табличные части, его движения по какому-либо регистру, либо движениях всех документов по какому-либо регистру. При этом шапка документа и каждая из его табличных частей, наборов движений являются отдельными источниками данных, т.е. по каждой табличной части и/или набору движений документа сверка производится раздельно.

В группе настроек «Структура и показатели» настраивается детализация расшифровки отражения в бюджетах.

Переключатель «Отображение разрезов» позволяет настроить отображение разрезов расшифровки в колонках (в виде шахматной ведомости по первичным данным и их отражению по разным статьям бюджетов) или в строках отчета (расшифровка регистрации факта будет располагаться под каждой строкой первичных данных). Если требуется выводить детальные поля источников первичных данных (см. далее о настройке «Выводить данные источника»), тогда отображение разрезов возможно только в колонках.

В поле «Разрезы бюджетных данных» настраивается расшифровка отраженных фактических данных по финансовым показателям, статьям бюджетов и бюджетам, по которым прошла регистрация. При необходимости флажок «Выводить детализацию по бюджетным данным с иерархией» позволяет группировать данные по иерархии стаей бюджетов и/или других бюджетных справочников.

Настройка «Детализация по шаблонам регистрации» позволяет детализировать данные регистрации факта по шаблонам регистрации и отдельным их правилам, в согласии с которыми были сформированы фактические проводки по бюджетам.

Флажок «Выводить данные источника» позволяет вывести в отчет не только документы и номера строк, но и значения полей документов, строк табличных частей и записей регистров. По-умолчанию выводятся все поля в автоматическом порядке. При необходимости настроить удобный вариант расшифровки данных источников, следует создать настройку для этого источника в справочнике «Шаблоны расшифровки данных источников регистрации факта».

В группе настроек «Показатели отчета» выбираются показатели бюджетирования, величина которых будет анализироваться в отчете. Если ни один из показателей не выбран, будет анализироваться только наличие/отсутствие регистрации.

В группе настроек «Отборы» имеется возможность настроить ограничения выводимых данных по следующим параметрам:

  • Период отчета – в графическом поле выбора периода настраивается временной интервал, за который производится сверка;
  • Отбор по признаку отображения в бюджетах – позволяет определить, какие исходные данные следует выводить в отчет – отраженные в бюджетах, не отраженные, либо все;
  • По указанным бюджетам – позволяет анализировать наличие/отсутствие регистрации только по статьям выбранных бюджетов;
  • По указанным шаблонам регистрации – позволяет анализировать наличие/отсутствие регистрации только по указанным шаблонам регистрации и/или отдельным правилам этих шаблонов;
  • По указанным финансовым показателям – позволяет анализировать наличие/отсутствие регистрации только по выбранным финансовым показателям.
