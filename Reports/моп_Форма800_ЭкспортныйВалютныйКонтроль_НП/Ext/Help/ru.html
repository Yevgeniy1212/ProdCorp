Отчет "Экспортный валютный контроль (XML)"

Отчет "Экспортный валютный контроль" предназначен для составления отчетности по проданным за пределами Казахстана ТМЗ, ОС и НМА.

В отчет попадают все строки регистра накопления «Мониторинг экспорта» который заполняется документом "Движения мониторинга" по операциям, проведенным в валюте, отличающейся от валюты регламентированного учета, установленной в константе "Валюта регламентированного учета".  

Документ "Сведения по экспорту"

Некоторые колонки заполняются по данным документа Сведения по экспорту: Условия поставки, Место отгрузки товаров (работ, услуг), Дата паспорта сделки, № паспорта сделки,  Дата инвойса, № инвойса, Дата ГТД, Номер ГТД, Срок оплаты, Таможенные пошлины и сборы. Для того, чтобы данные документа Сведения по экспорту были использованы при заполнении отчета, необходимо в документе Сведения по экспорту в реквизите "Документ партии" табличной части "ТМЗ" указать документ реализации. Таким образом создается связь между документами. Рекомендуется документ Сведения по экспорту создавать с использованием механизма ввода на основании, в этом случае часть реквизитов будет заполнена автоматически, что упрощает ввод данных.

Расшифровка некоторых колонок отчета:

 

  • Номер строки: двойной щелчок левой кнопкой мыши откроет документ реализации;
  • Код ТН ВЭД - код ТНВЭД, заданный, для соответствующего объекта мониторинга, в регистре «Объекты мониторинга» в конфигурации "Бухгалтерия для Казахстана" или в регистре "Соответствия кодов ТНВЭД" в конфигурации "Управление производственным предприятием для Казахстана". Двойной щелчок левой кнопкой мыши откроет форму элемента объекта мониторинга, откуда можно перейти к регистру «Объекты мониторинга» или к регистру "Соответствия кодов ТНВЭД";
  • Пин-код - персональный идентификационный номер–код бензина (за исключением авиационного), дизельного топлива и мазута, в конфигурации  "Бухгалтерия для Казахстана", заполняется из регистра "Объекты мониторинга", если установлено дополнение "1С-Рейтинг: Нефтебаза", заполняется из документов отгрузки ГСМ (Горюче-смазочных материалов);
  • Место отгрузки товаров (работ, услуг) – код страны поставщика;
  • Пункт доставки товара (работ, услуг), страна - заполняется из поля "Адрес доставки" документа реализации;
  • Цена, Сумма (без налогов) – суммы в валюте из табличной части документа реализации;
  • Количество - количество из табличной части документа реализации;
  • Валюта контракта – валюта операций;
  • Курс тенге – курс валюты на дату документа реализации.

 

Общая информация о работе с отчетами

 

Отчет имеет две формы.

В первой, основной форме пользователю нужно установить Организацию и период отчета.

 

После нажатия «ОК» открывается форма отчета, верхняя часть которой используется для управления формированием, а в нижней части расположен сам бланк отчета.

 

Работа с отчетом

 

Для работы необходимо установить значение следующих реквизитов в форме отчета:

 

«Вид формы»   -   вид результирующей формы - «Основная» или «Корректировочная», значение выбирается из выпадающего списка

 

«Код налоговой инспекции»   -   код налоговой инспекции Организации

  

В диалоговом окне формы отчета предусмотрен набор кнопок, соответствующий списку

возможных действий:

                                                                          

Кнопка «Заполнить»  -  по данной кнопке происходит автоматическое заполнение отчета данными из бухгалтерского учета. При желании пользователь может отредактировать или заполнить данные самостоятельно.

 

Кнопка «Выгрузить в XML» -  при помощи этой кнопки осуществляется выгрузка информации в формате XML для клиентского приложения ИС МОП.

 

Кнопка «Очистить» -  очищает бланк отчета

 

Кнопки добавления, удаления строк – пользователь может добавить новую строку (и заполнить ее самостоятельно) или удалить любую выделенную

 

Кнопки последовательного перехода по страницам – пользователь может перейти на следующую / предыдущую страницу отчета; текущая страница и полное количество страниц отображаются между этими кнопками

 

Кнопка «Печать» - позволяет выполнить предварительный просмотр, либо сразу распечатать отчет

 

Кнопка произвольного перехода к странице – пользователь может перейти на любую страницу отчета.

 

Кнопка вызова справки – пользователь может просмотреть текущую справку, либо перейти к общей информации о формах.

  

Кроме этого, в процессе работы с отчетом пользователь может самостоятельно редактировать данные, представленные в бланке отчета. По двойному щелчку на ячейке открывается элемент (карточка счета, контрагента или товара, и т.п.), после редактирования которого обновляются все связанные поля (адрес контрагента, код ТН ВЭД товара, и т.п.). При двойном щелчке на ячейке с обычным текстом или числом (например, в поле "Таможенные пошлины и сборы"), открывается окно для прямого редактирования значения.

 

Все основные механизмы работы с отчетом (печать, форматирование данных, выгрузка в

XML-формате и т.д.) заимствованы из  регламентированных отчетов типовой конфигурации

"Бухгалтерия для Казахстана"
