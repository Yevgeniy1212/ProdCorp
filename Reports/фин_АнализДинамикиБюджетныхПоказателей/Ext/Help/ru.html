Отчет "Анализ динамики бюджетных показателей"

Отчет предназначен для получения статистических показателей динамики выбранного показателя за требуемый период.

Рассчитывает следующие данные для каждого из анализируемых периодов:

·         Абсолютный прирост базисный;

·         Абсолютный прирост цепной;

·         Темп роста базисный;

·         Темп роста цепной;

·         Темп прироста базисный;

·         Темп прироста цепной.

По значениям одного из этих показателей или по абсолютным значениям периодов может быть построена диаграмма.

Также позволяет получить следующие средние показатели динамики:

·         Средний уровень;                           

·         Средний абсолютный прирост;

·         Темп роста за период, %;

·         Средний темп роста, %;

·         Темп прироста за период, %;

·         Средний темп прироста, %;

·         Абсолютное значение 1% прироста;

·         Коэффициент вариации, %.

Имеется возможность рассчитать прогноз динамики показателя на выбранное количество периодов на основе среднего темпа роста.

Для формирования отчета заполняются следующие настройки отчета:

·         Финансовый показатель – показатель, динамика которого анализируется;

·         Данные – указывается сценарий плановых данных, либо не заполняется в случае анализа фактических данных;

·         Периодичность – периодичность группировки данных показателя;

·         Анализируемый ресурс – ресурс показателя (количество, сумма упр. учета и т.д.), значение которого будет анализироваться;

·         Анализируемый период – период сбора данных;

·         Базисный период – один из периодов анализируемого периода, выступающий в качестве базисного для расчета статистических показателей динамики;

·         Диаграмма по ресурсу – указывается необходимость формирования диаграммы и статистический показатель, значение которого будет использовано для построения диаграммы. Строится в разрезе анализируемых периодов;

·         Количество прогнозируемых периодов – количество периодов, на которые необходимо рассчитать прогноз на основании среднего темпа роста;

·         Отбор – фильтр на анализируемые данные по разрезам планирования.
