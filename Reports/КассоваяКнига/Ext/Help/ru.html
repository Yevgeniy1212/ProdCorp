Отчет «Кассовая книга»

Отчет «Кассовая книга» формирует регламентную форму кассовой книги. Данный отчет формируется по организации  и кассе предприятия, выбранных в настройках отчета за указанный период. При формировании кассовой книги номер сформированного листа запоминается и записывается в регистре сведений "Номера листов кассовой книги" с соответствующей датой.

В настройках также можно указать ряд параметров:

  • Разбивать листы по дням. Если признак установлен, то данные за каждый день, в котором были какие-либо движения по кассе, будут выводится на отдельном листе. Если признак не установлен - данные будут выводится единым списком, упорядоченном по дате выписки первичных документов.
  • Сортировать документы по виду.  В переделах дня документы будут дополнительно упорядочены по виду.
  • Выводить название кассы.  При установленном признаке в заголовке отчета будет выводится наименование кассы.
  • Выводить основания. При установке этого флажка в графе кассовой книги "От кого получено или кому выдано" дополнительно будет выводиться содержание полей "Основание" приходных и расходных кассовых ордеров.
  • Разбивать листы по дням. Если признак установлен, то данные за каждый день, в котором были какие-либо движения по кассе, будут выводится на отдельном листе. Если признак не установлен - данные будут выводится единым списком, упорядоченном по дате выписки первичных документов.
  • Печатать вкладной лист. При установленом признаке отчет содержит раздел "Вкладной лист кассовой книги".
  • Печатать отчет кассира. При установленом признаке отчет содержит раздел "Отчет кассира".
  • Печатать титульный лист. Позволяет получить печатную форму титульного листа кассовой книги по итогам года.
  • Пересчитать номера листов с начала года. Данный признак имеет смысл устанавливать при формировании отчета в целом за год. При этом, будет выполнена перенумерация листов кассовой книги в регистре сведений "Номера листов кассовой книги".
