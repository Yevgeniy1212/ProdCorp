Заполнение строки 200 "Сальдо конец предыдущего периода (стр. 160-стр. 170+стр. 180-стр.190)".

Графа 3 "Уставный капитал" =  по графе 3: Значение строки 160 - Значение строки 170 + Значение строки 180 - Значение строки 190.
Графа 4 "Резервный капитал" =  по графе 4: Значение строки 160 - Значение строки 170 + Значение строки 180 - Значение строки 190.
Графа 5 "Нераспределенная прибыль" =  по графе 5: Значение строки 160 - Значение строки 170 + Значение строки 180 - Значение строки 190.
Графа 7 "Доля меньшинства" =  по графе 7: Значение строки 160 - Значение строки 170 + Значение строки 180 - Значение строки 190.

Заполнение графы 6 "Всего" и графы 8 "Итого капитал".

Графа 6 "Всего" по всем строкам = Графа 3 "Уставный капитал" + Графа 4 "Резервный капитал" + Графа 5 "Нераспределенная прибыль".

Графа 8 "Итого капитал" по всем строкам = Графа 6 "Всего" + Графа 7 "Доля меньшинства".
