Отчет "Отчет по бюджету"

Отчет предназначен для вывода плановых данных выбранного сценария по бюджету, либо для вывода его фактических данных. Может быть использован в качестве механизма формирования печатной формы бюджета.  Позволяет просматривать бюджет по состоянию на нужную дату, устанавливаемую в параметре «Граница актуальности».

Помимо общих для всех отчетов параметров, на закладке «Данные» указываются:

·         Бюджет – бюджет, по статьям которого необходимо получить данные;

·         Вид данных – определяет, какие данные следует вывести в отчет:

o    Плановые – выводятся данные плана по сценарию, указанному в поле «Сценарий»;

o    Фактические – выводятся фактические данные с периодичностью, указанной в поле «Периодичность»;

·         Период отчета – период, который используется для получения данных.
