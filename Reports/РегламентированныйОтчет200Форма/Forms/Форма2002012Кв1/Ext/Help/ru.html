Отчет «Декларация по индивидуальному подоходному налогу и социальному налогу, форма 200.00 (XML)» за 2012 год.

Правила составления:

- Правила составления налоговой отчетности (декларации) по индивидуальному подоходному налогу и социальному налогу (Форма 200.00) на 2012 год.

Отчет «Декларация по индивидуальному подоходному налогу и социальному налогу, форма 200.00 (XML)» предназначен для исчисления индивидуального подоходного налога, социального налога, а также для исчисления, удержания (начисления) и перечисления сумм обязательных пенсионных взносов в накопительный пенсионный фонд (далее – обязательные пенсионные взносы), начисления и перечисления сумм социальных отчислений в Государственный фонд социального страхования (далее – социальные отчисления).

Если в настройках параметров учета (меню Предприятие\Настройка параметров учета) установлен признак поддержки работы со структурными подразделениями, то при открытии отчета необходимо указать налогоплательщика и налоговый комитет, в который будет предоставлен отчет. Если работы со структурными подразделениями в налоговом учете не поддерживается, то при открытии отчета требуется выбрать только налогоплательщика, налоговый комитет будет определен по реквизиту "Налоговый комитет" из справочника "Организации".

В отчет, кроме данных о работниках организации, также могут быть включены сведений о выплаченных доходах получателей по исполнительным листам. Для этого предназначен флажок "Включать в отчет получателей алиментов" на закладке "Настройка расчета". При установленном флажке данные о получателях по исполнительным листам будут включены в отчет.

Если необходимо исключить из отчета данные по физическим лицам - нерезидентам РК на закладке "Настройка расчета" установить флажок "Исключать физлиц-нерезидентов (в настройках гражданства которых, установлен признак "Не является налоговым резидентом РК")".

Для проверки правильности составления Декларации предназначен отчет "Регистр налогового учета по индивидуальному подоходному налогу и социальному налогу", позволяющий в произвольных разрезах проанализировать корректность формирования налогового отчета.

Внимание! Автоматическое заполнение предусмотрено только для строк, перечисленных ниже по тексту, все прочие строки формы в случае наличия данных, отражаемых в них, необходимо заполнить вручную.

Декларация по индивидуальному подоходному налогу и социальному налогу (форма 200.00)

 

Раздел. Исчисление индивидуального подоходного налога

- Заполнение строки 200.00.001 (Начисленные доходы)

В строке выводятся данные о начисленных доходах физическим лицам, зарегистрированные в регистр накопления "ИПН сведения о доходах" за соответствующий месяц налогового периода. Если установлен признак включения в отчет данных по получателям алиментов, то к этой сумме также будут добавлены удержанные за соответствующие месяцы суммы по исполнительным листам, определяемые как приход по регистру накопления "Взаиморасчеты организаций с получателями по ИЛ".

Для заполнения сведений о видах доходов: 200.00.001.A "Доходы работников" и 200.00.001.B "Дивиденды, вознаграждения и выигрыши"  в  форме вида расчета начисления необходимо заполнить реквизит "Код строки декларации по ИПН и СН", значением "200.00.001.A" или "200.00.001.B".

- Заполнение строки 200.00.002 (Сумма исчисленного индивидуального подоходного налога)

В строке выводятся данные о суммах индивидуального подоходного налога, исчисленного за соответствующие месяцы налогового периода, определяемого как приход по регистру накопления "ИПН расчеты с бюджетом" для вида строк "Исчисление".

- Заполнение строки 200.00.003 (Задолженность по доходам, не выплаченным физическим лицам)

В строке выводятся данные о сумме задолженности по доходам, невыплаченным физическим лицам на конец последнего месяца периода составления Декларации, определяемой как конечный остаток по следующим регистрам накопления:

    • Взаиморасчеты с работниками организаций - задолженность по текущим расчетам с работниками (соответствует кредитовому остатку по счету 3350 в бухгалтерском учете);
    • Взаиморасчеты с депонентами организаций - задолженность по депонированной заработной плате (соответствует кредитовому остатку по счету 3384 в бухгалтерском учете);
    • Взаиморасчеты организаций с контрагентами-физлицами - задолженность по расчетам с контрагентами-физлицами, у которых приобретались активы и/или услуги (оформляется с помощью документа "Расчет налогов при поступлении активов и услуг");
    • Взаиморасчеты организаций с получателями по ИЛ - задолженность, связанная с расчетами по исполнительным листам. Если в учетной политике по налоговому учету не установлен признак "Засчитывать в выплаченные доходы удержания по исполнительным листам", то включается остаток по неперечисленным суммам удержаний по исполнительным листам и суммам почтового/комиссионного сбора как задолженность физлицу-работнику организации. Если при формировании отчета на закладке "Настройка расчета" включен флажок "Включить в отчет получателей алиментов", то в невыплаченную задолженность также включаются неперечисленные суммы удержаний по исполнительным листам физлицам-получателям алиментов.

- Заполнение строки 200.00.004 (Выплачено доходов)

В строке выводятся данные о выплаченных физлицам доходах по данным регистра накопления "Выплаченные доходы работникам организаций по налоговому учету". Если при формировании отчета установлен флажок "Включить в отчет получателей алиментов", то в выплаченные доходы будут включены фактически перечисленный суммы удержаний по исполнительным листам получателям алиментов.

- Заполнение строки 200.00.005 (Сумма индивидуального подоходного налога, подлежащего перечислению в бюджет)

В строке выводятся данные о суммах индивидуального подоходного налога, подлежащего перечислению в бюджет. Информация собирается на основании записей-расходов по регистру накопления "ИПН расчеты с бюджетом" с видом строки "Удержание". В зависимости от настроек учетной политики по налоговому учету "При начислении ИПН и ОПВ принимать исчисленные суммы к учету как удержанные" эта сумма равняется фактически исчисленному налогу (если настройка "При начислении ИПН и ОПВ принимать исчисленные суммы к учету как удержанные" включена), либо определяется в процессе расчета документа "Удержание ИПН и ОПВ по налоговому учету" (настройка "При начислении ИПН и ОПВ принимать исчисленные суммы к учету как удержанные" отключена).

- Заполнение строки 200.00.006 (Сумма налога, подлежащего перечислению за головную организацию)

В строке выводятся данные о индивидудальном подоходном налоге, подлежащем перечислению в бюджет, с доходов работников, относящихся к головной организации. Данные собираются по расходу регистра "ИПН расчеты с бюджетом" с видом строки "Удержание" для тех строк, у которых графы "Структурная единица" и "Налогоплательщик" совпадают.

- Заполнение строки 200.00.007 (Сумма налога, подлежащего перечислению по филиалам/представительствам)

В строке выводятся данные о индивидудальном подоходном налоге, подлежащем перечислению в бюджет, с доходов работников, относящихся к головной организации. Данные собираются по расходу регистра "ИПН расчеты с бюджетом" с видом строки "Удержание" для тех строк, у которых графы "Структурная единица" и "Налогоплательщик" различаются.

Раздел. Исчисление обязательных пенсионных взносов

- Заполнение строки 200.00.008 (Начисленные доходы, с которых удерживаются (начисляются) обязательные пенсионные взносы)

В строке выводятся данные о доходах, с которых удерживаются (начисляются) обязательные пенсионные взносы. Данная строка определяется как фактически исчисленная сумма обязательных пенсионных взносов, умноженная на 10.

- Заполнение строки 200.00.009 (Сумма обязательных пенсионных взносов, подлежащих перечислению)

В строке выводятся данные о суммах обязательных пенсионных взносов, подлежащих перечислению в пенсионные фонды, которые определяются на основании данных о приходах в регистре накопления "ОПВ подлежит перечислению в фонды" с видом строки "Удержание". На расчет суммы взносов, подлежащий перечислению, влияет настройка в учетной политике по налоговому учету "При начислении ИПН и ОПВ принимать исчисленные суммы к учету как удержанные" (см. порядок заполнения строки 200.00.005).

- Заполнение строки 200.00.010 (Сумма обязательных пенсионных взносов, подлежащих перечислению за головную организацию)

В строке выводятся данные о суммах обязательных пенсионных взносов, подлежащих перечислению в пенсионные фонды, которые определяются на основании данных о приходах в регистре накопления "ОПВ подлежит перечислению в фонды" с видом строки "Удержание" для тех строк, у которых графы "Структурная единица" и "Налогоплательщик" совпадают.

- Заполнение строки 200.00.011 (Сумма обязательных пенсионных взносов, подлежащих перечислению по филиалам/представительствам)

В строке выводятся данные о суммах обязательных пенсионных взносов, подлежащих перечислению в пенсионные фонды, которые определяются на основании данных о приходах в регистре накопления "ОПВ подлежит перечислению в фонды" с видом строки "Удержание" для тех строк, у которых графы "Структурная единица" и "Налогоплательщик" различаются.

Исчисление суммы индивидуального подоходного налога и социального налога по структурному подразделению (форма 200.02)

- Заполнение строки 200.02.001 (Сумма индивидуального подоходного налога, подлежащего перечислению)

В строке выводятся данные о суммах индивидуального подоходного налога, подлежащего перечислению в бюджет по соответствующей структурной единице. Информация собирается на основании записей-расходов по регистру накопления "ИПН расчеты с бюджетом" с видом строки "Удержание".
