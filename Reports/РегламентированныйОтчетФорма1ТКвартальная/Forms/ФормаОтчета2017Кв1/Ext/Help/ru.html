1. Титульный лист

Код БИН - определяется по данным из Карточки организации (юр. лица) из табличной части "Реквизиты" поля "ИИН/БИН", по которой заполняется отчет.

2. Раздел 1

Ячейка "Фактическое место расположения юридического лица"- определяется  из карточки организации (юр. лица), табличной части "Контактная  информация", по  которой заполняется отчет.

Ячейка "Код КАТО"- определяется по данным регистра сведений "Адресный классификатор", страны, в которой зарегистрирована Организация (юр. лицо).

Ячейка "Наименование экономической  деятельности" - определяется по данным карточки Организации (юр. лица), табличной части "Коды" - поле "Вид деятельности", "Код ОКЭД".

3. Раздел 2

Строка 1 "Списочная численность  работников в среднем за отчетный период - всего, человек" - определяется согласно "Инструкции по заполнению статистической отчетности по труду".

Списочная численность работников - численность лиц, принятых по  трудовому договору, независимо от срока его заключения, кроме лиц выполняющих работы по совместительству.

Среднесписочная численность (далее С/С) работников за квартал определяется путем суммирования среднесписочной численности работников за все месяцы отчетного периода и деления полученной суммы на 3. С/С за месяц рассчитывается по формуле:

С/С численность за месяц = ∑С дн / К дн, где ∑С дн – cумма С/С работников за каждый календарный день месяца.

К дн – число календарных дней месяца.

С/С за квартал рассчитывается по формуле:

С/С численность за квартал = (∑С/С мес.) / 3, где ∑С/С мес – сумма С/С работников за все месяцы отчетного года.

Строка 2 "Фактическая численность работников (принимаемая для исчисления средней заработной платы, человек"- определяется согласно "Инструкции по заполнению статистической  отчетности по труду".

Фактическая численность работников (принимаемая для исчисления средней заработной платы) – численность работников списочного состава за вычетом отдельных категорий работников.

Порядок исчисления фактической численности, в среднем за период (принимаемой для исчисления средней заработной платы), по организациям, работавшим неполный месяц (квартал, год), аналогичен порядку исчисления списочной численности в среднем  за  период.

Строка 3 "Фонд заработной платы работников - всего, тысяч тенге"- определяется согласно "Инструкции по заполнению статистической  отчетности по труду".

К фонду заработной платы (оплаты труда) относятся начисленные суммарные денежные средства организации для оплаты труда работников (должностные оклады (тарифные ставки), доплаты, надбавки, премии и иные выплаты стимулирующего и компенсирующего характера), с учетом налогов и других удержаний (подоходный налог, обязательные пенсионные взносы) в соответствии с нормативными правовыми актами Республики Казахстан и независимо от источника их финансирования и срока их фактических выплат. В фонде заработной платы учитываются все выплаты как в денежной, так и в натуральной форме, переведенные в денежную единицу.

Для автоматического заполнения строки 3 "Фонд заработной платы работников - всего, тысяч тенге" необходимо в карточке вида расчета указать фонд выплат денежных средств, из которого осуществляется выплата. Поле "Фонд выплат"  денежных средств заполняется в карточке видов расчета на закладке "Статистический учет".

Данные о фонде заработной платы собираются и заполняются автоматически только для тех видов расчета, для которых заполнен реквизит "Фонд выплат". Если он не заполнен, а начисление по такому виду расчета выполнено, данные не попадут в фонд начисленной заработной платы.

Строка 4 "Среднемесячная номинальная заработная плата одного работника, тенге" - определяется путем деления суммы начисленного фонда заработной платы на фактическую численность работников.

СНЗП (Среднемесячная номинальная заработная плата) = ФЗП (Фонд заработной платы работников) / ФЧР (Фактическая численность работников) / N месяцев

Строки 1.1,2.1, 3.1, 4.1  заполняются в разрезе вида экономической деятельности.

Строка 5 "Число отработанных человеко-часов всеми работниками, тысяч человеко-часов" - определяется согласно "Инструкции по заполнению статистической отчетности по труду". При заполнении данных учитывается фактически отработанное всеми работниками предприятия время, в соответствующих видах расчетах которых установлен признак "Засчитывается как отработанное время" на закладке "Время".

4. Раздел 3

Заполняется данными о движении рабочей чилы, численности работающих неполное время и наличии вакансий.

Строка 1 "Списочная численность  работников на начало отчетного периода" - определяется согласно "Инструкции по заполнению статистической отчетности по труду". Указывается численность работников предприятия на начало отчетного периода на основании документов "Прием на работу сотрудников организации".

Строка 2 "Принято работников за отчетный период" - определяется согласно "Инструкции по заполнению статистической отчетности по труду". Указывается численность работников предприятия на основании документов "Прием на работу сотрудников организации" за отчетный период.

Строка 3 "Выбыло работников за отчетный период, всего" - определяется согласно "Инструкции по заполнению статистической отчетности по труду". Строка заполняется как сумма строк 3.1 - 3.7.

Строка 3.1-3.7 - заполняются по данным документов "Увольнение из организации" за отчетный период с указанием статьи ТК РК и причины увольнения.

Строка 4 "Списочная численность  работников на конец отчетного периода" - определяется согласно "Инструкции по заполнению статистической отчетности по труду". Указывается численность работников предприятия на конец отчетного периода.

Строка 5 "Число вакантных рабочих мест (требуемых работников) на конец отчетного периода" - определяется согласно "Инструкции по заполнению статистической отчетности по труду".

Строка 6 "Численность работающих неполный рабочий день или неполную рабочую неделю" - заполняется в случае, если сотрудники работают по графикам работы с сокращенным рабочем временем.

Строка 7 "Численность работников, временно неработающих в связи с простоем производства"- заполняется в случае, если сотрудники временно не работают в связи с простоем производства.
