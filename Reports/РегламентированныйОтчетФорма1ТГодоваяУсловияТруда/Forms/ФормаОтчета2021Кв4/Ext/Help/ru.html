1. Титульный лист

Код БИН - определяется по данным из Карточки организации (юр. лица) из табличной части "Реквизиты" поля "ИИН/БИН", по которой заполняется отчет.

2. Раздел 1

Ячейка "Фактическое место расположения юридического лица"- определяется  из карточки организации (юр. лица), табличной части "Контактная  информация", по  которой заполняется отчет.

Ячейка "Код КАТО"- определяется по данным регистра сведений "Адресный классификатор", страны, в которой зарегистрирована Организация (юр. лицо).

Ячейка "Наименование экономической  деятельности" - определяется по данным карточки Организации (юр. лица), табличной части "Коды" - поле "Вид деятельности", "Код ОКЭД".

3. Раздел 2

Столбец 1 "Всего"- указывается численность работников списочного состава по каждой строке раздела в среднем за отчетный год.

Столбец 2 "Из него женщин"- указывается численность женщин из общей численности списочного состава работников по каждой строке раздела в среднем за отчетный год.

Столбец 3 "Лица до 18 лет"- указывается численность работников, которые являются лицами до 18 лет, из общей численности списочного состава работников по каждой строке раздела в среднем за отчетный год.

Строка 1 "Списочная численность работников (без находящихся в дополнительном отпуске по уходу за ребенком)"- определяется согласно "Инструкции по заполнению статистической  отчетности по труду".

При заполнении данных по списочной численности работников учитывается численность работников списочного состава за вычетом отдельных категорий работников, имеющих формальное прикрепление к работе (лица, находящиеся в отпусках по беременности и родам, по уходу за ребенком и другие).

Строка 1.1 "из нее рабочих"- определяются по основным группам занятий сотрудников. Для автоматического заполнения строк необходимо заполнить поле "Категория должности" в справочнике "Должности организаций".

Строка 1.1.1 "из нее занятых в ночную смену"- определяются по основным графикам работы сотрудников. Строки заполняются автоматически количеством сотрудников, для которых в графике работы установлен признак "Вести учет ночных часов".

Строка 2 "Фактическая численность работников, человек"- определяется согласно "Инструкции по заполнению статистической  отчетности по труду".

Фактическая численность работников (принимаемая для исчисления средней заработной платы) – численность работников списочного состава за вычетом отдельных категорий работников, имеющих формальное прикрепление к работе (лица, находящиеся в отпусках по беременности и родам, по уходу за ребенком и другие).

Строка 2.1-2.9 - определяются по основным вредным производственным факторам в соответствии с санитарно-гигиеническими требованиями. Для автоматического заполнения строк в регистре сведений "Штатное расписание" для штатной единицы необходимо заполнить поле "Условия труда" (в котором указано соответствующее значение строки "Основной вредный производственный фактор").

Строка 3 - определяется согласно тяжелым условиям труда. Строки заполняются автоматически количеством работников, для которых в условиях труда установлен признак "Тяжелый физический труд".

Строка 4 - определяется согласно "Инструкции по заполнению статистической  отчетности по труду". Строки заполняются автоматически количеством работников, для которых в условиях труда установлен признак "Работа на оборудовании, не отвечающем требованиям безопасности".

4. Раздел 3

Заполняется данными по всем категориям работников, получающих льготы и компенсации за работу во вредных условиях труда в среднем за отчетный год.

Строка 1.1-1.5 - определяется согласно основным компенсациям, назначенным сотрудникам во вредных и других неблагоприятных условиях труда. Для автоматического заполнения строк в соответствующих видах расчета, назначенных работникам в качестве льгот и компенсаций за вредные условия труда, необходимо заполнить поле "Вид компенсации" на закладке "Статистический учет".

Строка 1.6 - автоматическое заполнение строки не предусмотрено.

Строка 2 - определяется согласно "Инструкции по заполнению статистической  отчетности по труду". Показывается среднесписочная численность работников, имеющих за работу во вредных и других неблагоприятных условиях труда хотя бы одну из льгот и компенсаций, указанных в строках 1.1-1.6.

5. Раздел 4

Указывается информация о затратах предприятия на льготы и компенсации за работу во вредных и неблагоприятных условиях труда работников, указанных в разделе 2.
