Отчет «Декларация по акцизу, форма 400.00» за 2007 год.

Отчет «Декларация по акцизу» предназначен для составления отчетности по акцизу. Форма адаптирована к версии ЭФНО 1.6.1.29 от 07.08.2008г.

 

СОДЕРЖАНИЕ

1) Заполнение приложения 400.01 «Облагаемые операции по спирту»»

2) Заполнение приложения 400.02 «Облагаемые операции по алкогольной продукции»

3) Заполнение приложения 400.03 «Облагаемые операции по табачным изделиям»

4) Заполнение приложения 400.04 «Облагаемые операции по сырой нефти, газовому конденсату»

5) Заполнение приложения 400.05 «Облагаемые операции по бензину (за исключением авиационного), дизельному топливу»

 

 

ЗАПОЛНЕНИЕ ПРИЛОЖЕНИЯ 400.01. «Облагаемые операции по спирту»

 

Для правильного заполнения приложения 400.01 необходимо, чтобы в справочнике «Номенклатура» для элементов справочника был заполнен реквизит «Вид подакцизного ТМЗ».

В форме отражаются обороты по регистру накопления «Акциз», причем в форму попадают только те данные, у которых вид операции в регистре сведений «Коды строк декларации по акцизу» имеет код строки, начинающийся с «400.01», и вид ТМЗ равный значению «Спирт». В форму попадают данные из регистра накопления «Акциз» по номенклатуре, реализованной или возвращенной от покупателя, у которой в справочнике «Номенклатура» реквизит «Вид подакцизного ТМЗ» содержит значение «Спирт».

- Заполнение граф:

 

       1. В графе А отражается количество ТМЗ из регистра накопления «Акциз» по видам операции на соответствующий код строки приложения 400.01.

       2. Графа В. При заполнении формы графа В «Ставка акциза» рассчитывается как отношение суммы акциза к налоговой базе (литр).

       3. В графе С отражается сумма акциза из регистра накопления «Акциз» по видам операции на соответствующий код строки приложения 400.01.

Строка 400.01.012 «Акциз, уплаченный поставщиком спирта при приобретении для производства алкогольной продукции» в рамках конфигурации не заполняется. 

В строку 400.01.013А попадает оборот суммы акциза из регистра накопления «Акциз» за первую декаду отчетного налогового периода. 

В строку 400.01.013В попадает оборот суммы акциза из регистра накопления «Акциз» за вторую декаду отчетного налогового периода. 

В строку 400.01.013С попадает оборот суммы акциза из регистра накопления «Акциз» за третью декаду отчетного налогового периода.

Примечание!!! В форме отражаются данные по регистру накопления «Акциз», где для вида операции в регистре сведений «Коды строк декларации по акцизу» определен код строки, начинающийся с «400.01» и равный «400.01.001», «400.01.002», «400.01.003», «400.01.004», «400.01.005», «400.01.006», «400.01.007», «400.01.008», «400.01.009», «400.01.010», «400.01.011».

 

 

ЗАПОЛНЕНИЕ ПРИЛОЖЕНИЯ 400.02. «Облагаемые операции по алкогольной продукции»

 

Для правильного заполнения приложения 400.02 необходимо, чтобы в справочнике «Номенклатура» для элементов справочника был заполнен реквизит «Вид подакцизного ТМЗ».

В форме отражаются обороты по регистру накопления «Акциз», причем в форму попадают только те данные, у которых вид операции в регистре сведений «Коды строк декларации по акцизу» имеет код строки, начинающийся с «400.02», и вид ТМЗ равный значению «Алкогольная продукция». В форму попадают данные из регистра накопления «Акциз» по номенклатуре, реализованной или возвращенной от покупателя, у которой в справочнике «Номенклатура» реквизит «Вид подакцизного ТМЗ» содержит значение «Алкогольная продукция».

Заполнение строк:

– В строку 400.02.001А записывается укрупнено «Алкогольная продукция».

– В строке 400.02.002 отражаются обороты по количеству ТМЗ из регистра накопления «Акциз» при условии, что виду операции регистра накопления «Акциз» соответствует код строки «400.02.002» и вид ТМЗ «Алкогольная продукция» в регистре сведений «Коды строк декларации по акцизу», а также для данного ТМЗ в справочнике «Номенклатура» реквизит «Вид подакцизного ТМЗ» содержит значение «Алкогольная продукция».

– Строки 400.02.003 – 400.02.008 заполняются на основании данных, непосредственно отнесенных на данные коды строк, т.е. аналогично заполнению строки 400.02.002.

– Строка 400.02.009 «Порча, утрата учетно-контрольных марок» в рамках конфигурации не заполняется.

– Строка 400.02.010 при заполнении из базы рассчитывается как сумма строк с 400.02.002 по 400.02.008.

– Значение строки 400.02.011 «Ставка акциза» рассчитывается как отношение суммы акциза (значение в строке 400.02.012) к налоговой базе (значение в строке 400.02.010).

– В строке 400.02.012 отражается общая сумма акциза по ТМЗ, у которых в справочнике «Номенклатура» реквизит «Вид подакцизного ТМЗ» содержит значение «Алкогольная продукция», и видам операции, для которых в регистре сведений «Коды строк декларации по акцизу» определен код строки декларации, начинающийся с «400.02».

– В строку 400.02.012А попадает оборот суммы акциза из регистра накопления «Акциз» за первую декаду отчетного налогового периода.

– В строку 400.02.012В попадает оборот суммы акциза из регистра накопления «Акциз» за вторую декаду отчетного налогового периода.

– В строку 400.02.012С попадает оборот суммы акциза из регистра накопления «Акциз» за третью декаду отчетного налогового периода.

– Строка 400.02.012D «Исчислено акциза до или в день получения учетно-контрольных марок» в рамках конфигурации не заполняется.

Примечание!!! В форме отражаются данные по регистру накопления «Акциз», где для вида операции в регистре сведений «Коды строк декларации по акцизу» определен код строки, начинающийся с «400.02» и равный «400.02.002», «400.02.003», «400.02.004», «400.02.005», «400.02.006», «400.02.007», «400.02.008».

ЗАПОЛНЕНИЕ ПРИЛОЖЕНИЯ 400.04. «Облагаемые операции по сырой нефти, газовому конденсату»

 

Для правильного заполнения приложения 400.04 необходимо, что в справочнике «Номенклатура» для элементов справочника был заполнен реквизит «Вид подакцизного ТМЗ».


В форме отражаются обороты по регистру накопления «Акциз», причем в форму попадают только те данные, у которых вид операции в регистре сведений «Коды строк декларации по акцизу» имеет код строки, начинающийся с «400.04», и вид ТМЗ равный значению «Нефть и газовый конденсат». В форму попадают данные из регистра накопления «Акциз» по номенклатуре, реализованной или возвращенной от покупателя, у которой в справочнике «Номенклатура» реквизит «Вид подакцизного ТМЗ» содержит значение «Нефть и газовый конденсат».

Заполнение строк:

– Строка 400.04.001. В строке 400.04.001 отражаются обороты по количеству ТМЗ (налоговая база (тонна)) из регистра накопления «Акциз» при условии, что виду операции регистра накопления «Акциз» соответствует код строки «400.04.001» и вид ТМЗ «Нефть и газовый конденсат» в регистре сведений «Коды строк декларации по акцизу», а также для данного ТМЗ в справочнике «Номенклатура» реквизит «Вид подакцизного ТМЗ» содержит значение «Нефть и газовый конденсат».

– Строки 400.04.002 – 400.04.009 заполняются на основании данных, непосредственно отнесенных на данные коды строк, т.е. аналогично заполнению строки 400.04.001.

– Строка 400.04.010 рассчитывается как сумма строк с 400.04.001 по 400.04.009.

– Строка 400.04.011. Значение строки 400.04.011 «Ставка акциза» рассчитывается как отношение суммы акциза (значение в строке 400.04.012) к налоговой базе (значение в строке 400.04.010).

– Строка 400.04.012. В строке 400.04.012 отражается общая сумма акциза по ТМЗ, у которых в справочнике «Номенклатура» реквизит «Вид подакцизного ТМЗ» содержит значение «Нефть и газовый конденсат», и видам операции, для которых в регистре сведений «Коды строк декларации по акцизу» определен код строки декларации, начинающийся с «400.04».

– В строку 400.04.012А попадает оборот суммы акциза из регистра накопления «Акциз» за первую декаду отчетного налогового периода.

– В строку 400.04.012В попадает оборот суммы акциза из регистра накопления «Акциз» за вторую декаду отчетного налогового периода.

– В строку 400.04.012С попадает оборот суммы акциза из регистра накопления «Акциз» за третью декаду отчетного налогового периода.
