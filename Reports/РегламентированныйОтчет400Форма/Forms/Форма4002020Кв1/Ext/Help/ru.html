Отчет «Декларация по акцизу, форма 400.00» за 2019 год

Отчет «Декларация по акцизу» предназначен для составления отчетности по акцизу.

Правила составления:

- Правила Составления налоговой отчетности (декларации) по акцизу (Форма 400.00) на 2017 год.

 

СОДЕРЖАНИЕ

1) Заполнение основного приложения 400.00 «Декларация по акцизу»

2) Заполнение приложения 400.01 «Облагаемые операции по спирту и(или) виноматериалу»

3) Заполнение приложения 400.02 «Облагаемые операции по алкогольной продукции, а также спиртосодержащей продукции медицинского назначения, зарегистрированной в соответствии с законодательством Республики Казахстан в качестве лекарственного средства»

4) Заполнение приложения 400.03 «Облагаемые операции по табачным изделиям»

5) Заполнение приложения 400.04 «Облагаемые операции по сырой нефти, газовому конденсату»

6) Заполнение приложения 400.05 «Облагаемые операции по бензину (за исключением авиационного), дизельному топливу»

7) Заполнение приложения 400.08 «Облагаемые операции по подакцизным товарам, предусмотренным подпунктом 6) статьи 462 Налогового кодекса»

 

Внимание! В рамках конфигурации приложения 400.06 «Вычет из налога», 400.07 «Подакцизные товары, не подлежащие обложению акцизом» автоматически не заполняются.

 

ЗАПОЛНЕНИЕ ОСНОВНОГО ПРИЛОЖЕНИЯ 400.00. «Декларация по акцизу»

 

Приложение 400.00 заполняется на основании данных, полученных из приложений 400.01, 400.02, 400.03, 400.04, 400.05, 400.06, 400.07 и 400.08, а также данными, введенными вручную.

РАЗДЕЛ. Общая информация о налогоплательщике


- Строка 1 «ИИН (БИН)» - реквизит «БИН / ИИН» текущего налогоплательщика(организации) из справочника «Организации». 
- Строка 2 «ФИО или наименование налогоплательщика» - полное наименование текущего налогоплательщика(организации) из справочника «Организации».
- Строка 3 «Налоговый период» - налоговый период, выбранный в основной форме отчета. 
- Строка 4 «Вид декларации» -  заполняется вручную. По умолчанию устнавливается Вид декларации «Очередная».
- Строка 5 «Номер и дата уведомления» - заполняется вручную, возможность заполнения номера и даты уведомления предоставляется в случае выбора вида декларации «По уведомлению».
- Строка 6 «Категория налогоплательщика» - заполняется вручную.
- Строка 7 «Код валюты» - заполняется автоматически и равно «KZT».
- Строка 8 «Представленные приложения» - заполняется автоматически, в зависимости от того заполнены ли приложения.
- Строка 9 «Представленные расчеты за структурные подразделения или объекты, связанные с налогообложением» - заполняется вручную.

- Поле «ФИО руководителя» - указывается  ФИО руководителя организации из регистра сведений «Ответственные лица организаций».

- Поле «Входящий номер документа» - поле не заполняется и не редактируется.
- Поле «Код органа государственных доходов» - заполняется вручную.
- Поле «Дата почтового штемпеля» - поле не заполняется и не редактируется.

 

ЗАПОЛНЕНИЕ ПРИЛОЖЕНИЯ 400.01. «Облагаемые операции по спирту и (или) виноматериалу»

 

Для правильного заполнения приложения 400.01 необходимо, чтобы в справочнике «Номенклатура» для элементов справочника был заполнен реквизит «Вид подакцизного ТМЗ».

В форме отражаются обороты по регистру накопления «Акциз», причем в форму попадают только те данные, у которых вид операции в регистре сведений «Коды строк декларации по акцизу» имеет код строки, начинающийся с «400.01», и вид ТМЗ равный значению «Спирт» или «Виноматериал». В форму попадают данные из регистра накопления «Акциз» по номенклатуре, реализованной или возвращенной от покупателя, у которой в справочнике «Номенклатура» реквизит «Вид подакцизного ТМЗ» содержит значение «Спирт» или «Виноматериал». Для каждого вида подакцизного ТМЗ вводится отдельный лист приложения.

Заполнение строк:

Строка 400.01.001 - в графе А "Вид продукции" указывается вид подакцизного ТМЗ, в графе B - код бюджетной классификации по акцизу

В строках 400.01.002 - 400.01.013: 

    - в графе А «Налоговая база(литр)» отражается количество отгруженных ТМЗ, определяемое по данным регистра накопления «Акциз». 
    - в графе В «Ставка акциза»  указывается значение ставки акциза по операции отгрузки, определяемое по данным регистра накопления «Акциз». 
    - в графе С «Сумма акциза» отражается сумма акциза из регистра накопления «Акциз» по соответствующему виду операции.


Строка 400.01.014 - Акциз, уплаченный поставщикам продукции при приобретении для производства алкогольной продукции. 

Строка 400.01.015 - Итого исчислено акциза» рассчитывается по формуле.

Примечание!!! В форме отражаются данные по регистру накопления «Акциз», где для вида операции в регистре сведений «Коды строк декларации по акцизу» определен код строки, начинающийся с «400.01» и равный «400.01.001», «400.01.002», «400.01.003», «400.01.004», «400.01.005», «400.01.006», «400.01.007», «400.01.008», «400.01.009», «400.01.010», «400.01.011», «400.01.012»,  «400.01.013».

ЗАПОЛНЕНИЕ ПРИЛОЖЕНИЯ 400.02. «Облагаемые операции по алкогольной продукции, а также спиртосодержащей продукции медицинского назначения, зарегистрированной в соответствии с законодательством Республики Казахстан в качестве лекарственного средства»

 

Для правильного заполнения приложения 400.02 необходимо, чтобы в справочнике «Номенклатура» для элементов справочника был заполнен реквизит «Вид подакцизного ТМЗ».

В форме отражаются обороты по регистру накопления «Акциз», причем в форму попадают только те данные, у которых вид операции в регистре сведений «Коды строк декларации по акцизу» имеет код строки, начинающийся с «400.02», и вид ТМЗ равный значению «Алкогольная продукция». В форму попадают данные из регистра накопления «Акциз» по номенклатуре, реализованной или возвращенной от покупателя, у которой в справочнике «Номенклатура» реквизит «Вид подакцизного ТМЗ» содержит значение «Алкогольная продукция».

Заполнение строк:

– В строку 400.02.001А записывается укрупненно «Алкогольная продукция».

– Строка 400.02.001В заполняется вручную.

– В строке 400.02.002 отражаются обороты по количеству ТМЗ из регистра накопления «Акциз» при условии, что виду операции регистра накопления «Акциз» соответствует код строки «400.02.002» и вид ТМЗ «Алкогольная продукция» в регистре сведений «Коды строк декларации по акцизу», а также для данного ТМЗ в справочнике «Номенклатура» реквизит «Вид подакцизного ТМЗ» содержит значение «Алкогольная продукция».

Примечание!!! В форме отражаются данные по регистру накопления «Акциз», где для вида операции в регистре сведений «Коды строк декларации по акцизу» определен код строки, начинающийся с «400.02» и равный «400.02.002», «400.02.003», «400.02.004», «400.02.005», «400.02.006», «400.02.007», «400.02.008», «400.02.009».

 

ЗАПОЛНЕНИЕ ПРИЛОЖЕНИЯ 400.03. «Облагаемые операции по табачным изделиям»


Для правильного заполнения приложения 400.03 необходимо, что в справочнике «Номенклатура» для элементов справочника был заполнен реквизит «Вид подакцизного ТМЗ».

В форме отражаются обороты по регистру накопления «Акциз», причем в форму попадают только те данные, у которых вид операции в регистре сведений «Коды строк декларации по акцизу» имеет код строки, начинающийся с «400.03», и вид ТМЗ равный значению «Сигареты с фильтром», «Сигареты без фильтра», «Табак», «Сигары» или «Сигариллы». В форму попадают данные из регистра накопления «Акциз» по номенклатуре, реализованной или возвращенной от покупателя, у которой в справочнике «Номенклатура» реквизит «Вид подакцизного ТМЗ» соответственно содержит значение «Сигареты с фильтром», «Сигареты без фильтра», «Табак», «Сигары» или «Сигариллы».

Для каждого вида подакцизного ТМЗ заполняется отдельный лист Декларации.


- Строка 400.03.001 - в графе А "Вид табачных изделий" указывается вид подакцизного ТМЗ, в графе B - код бюджетной классификации по акцизу.

– Строки 400.03.002 – 400.03.010 заполняются на основании данных регистра накопления «Акциз» о количестве отгруженных подакцизных ТМЗ, исходя из кода строки Декларации заданного для операции отгрузки.

– Графы А и В строки 400.03.011 I «Порча, утрата акцизных марок» заполняются вручную.

– Графы А и В строки 400.03.011 II «Порча, утрата акцизных марок принятых налоговыми органами в соответствии с п.3 ст.469 Налогового кодекса » заполняются вручную.

– Строка 400.03.012 рассчитывается по формуле: (сумма строк с 400.03.002 по 400.03.011 I).

– Строка 400.03.013. Значение строки 400.03.013 «Ставка акциза» рассчитывается как отношение суммы акциза (значение в строке 400.03.014) к налоговой базе (значение в строке 400.03.012).

– Строка 400.03.014. В строке 400.03.014 отражается общая сумма акциза по ТМЗ.

– Строка 400.03.015. «Сумма акциза, по испорченным акцизным маркам принятых налоговыми органами в соответствии с п.3 ст. 469 Налогового кодекса» заполняется вручную.

– Строка 400.03.016. рассчитывается по формуле: как разность величин, указанных в строках 400.03.014 и 400.03.015 (рассчитывается автоматически).

ЗАПОЛНЕНИЕ ПРИЛОЖЕНИЯ 400.05. «Облагаемые операции по бензину (за исключением авиационного), дизельному топливу»

 

Для правильного заполнения приложения 400.05 необходимо, что в справочнике «Номенклатура» для элементов справочника был заполнен реквизит «Вид подакцизного ТМЗ».

В форме отражаются обороты по регистру накопления «Акциз», причем в форму попадают только те данные, у которых вид операции в регистре сведений «Коды строк декларации по акцизу» имеет код строки, начинающийся с «400.05», и вид ТМЗ равный значению «Бензин» или «Дизельное топливо». Заполнение граф:

ЗАПОЛНЕНИЕ ПРИЛОЖЕНИЯ 400.08. «Облагаемые операции по подакцизным товарам, предусмотренным подпунктом 6) статьи 462 Налогового кодекса»


Для правильного заполнения приложения 400.08 необходимо, что в справочнике «Номенклатура» для элементов справочника был заполнен реквизит «Вид подакцизного ТМЗ».

В форме отражаются обороты по регистру накопления «Акциз», причем в форму попадают только те данные, у которых вид операции в регистре сведений «Коды строк декларации по акцизу» имеет код строки, начинающийся с «400.08», и вид ТМЗ равный значению «Легковые автомобили». В форму попадают данные из регистра накопления «Акциз» по номенклатуре, реализованной или возвращенной от покупателя, у которой в справочнике «Номенклатура» реквизит «Вид подакцизного ТМЗ» содержит значение «Легковые автомобили».

Заполнение строк:

– Строка 400.08.001. В строке 400.08.001 отражаются обороты по количеству ТМЗ (налоговая база (куб.см)) из регистра накопления «Акциз» при условии, что виду операции регистра накопления «Акциз» соответствует код строки «400.08.001» и вид ТМЗ «Легковые автомобили» в регистре сведений «Коды строк декларации по акцизу», а также для данного ТМЗ в справочнике «Номенклатура» реквизит «Вид подакцизного ТМЗ» содержит значение «Легковые автомобили».

– Строки 400.08.002 – 400.08.008 заполняются на основании данных, непосредственно отнесенных на данные коды строк, т.е. аналогично заполнению строки 400.08.001.

– Строка 400.08.009 рассчитывается как сумма строк с 400.08.001 по 400.08.008.

– Строка 400.08.010. Значение строки 400.08.010 «Ставка акциза» рассчитывается как отношение суммы акциза (значение в строке 400.08.011) к налоговой базе (значение в строке 400.08.009).

– Строка 400.08.011. В строке 400.08.011 отражается общая сумма акциза по ТМЗ, у которых в справочнике «Номенклатура» реквизит «Вид подакцизного ТМЗ» содержит значение «Легковые автомобили», и видам операции, для которых в регистре сведений «Коды строк декларации по акцизу» определен код строки декларации, начинающийся с «400.08».
