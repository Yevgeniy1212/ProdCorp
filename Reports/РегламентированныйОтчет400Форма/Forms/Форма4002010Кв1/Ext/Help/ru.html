Отчет «Декларация по акцизу, форма 400.00» за 2009 год

Отчет «Декларация по акцизу» предназначен для составления отчетности по акцизу. Форма адаптирована к версии ЭФНО 1.6.1.49 от 26.06.2009г.

Правила составления:

- Правила Составления налоговой отчетности (декларации) по акцизу (Форма 400.00) на 2009 год.

 

СОДЕРЖАНИЕ

1) Заполнение основного приложения 400.00 «Декларация по акцизу»

2) Заполнение приложения 400.01 «Облагаемые операции по спирту и(или) виноматериалу»

3) Заполнение приложения 400.02 «Облагаемые операции по алкогольной продукции и (или) пиву с объемным содержанием этилового спирта не более 0,5 процента»

4) Заполнение приложения 400.03 «Облагаемые операции по табачным изделиям»

5) Заполнение приложения 400.04 «Облагаемые операции по сырой нефти, газовому конденсату»

6) Заполнение приложения 400.05 «Облагаемые операции по бензину (за исключением авиационного), дизельному топливу»

7) Заполнение приложения 400.08 «Облагаемые операции по легковым автомобилям»

 

Внимание! В рамках конфигурации приложения 400.06 «Вычет из налога», 400.07 «Подакцизные товары, не подлежащие обложению акцизом» автоматически не заполняются.

 

ЗАПОЛНЕНИЕ ОСНОВНОГО ПРИЛОЖЕНИЯ 400.00. «Декларация по акцизу»

 

Приложение 400.00 заполняется на основании данных, полученных из приложений 400.01, 400.02, 400.03, 400.04, 400.05, 400.06, 400.07 и 400.08, а также данными, введенными вручную.

РАЗДЕЛ. Общая информация о налогоплательщике


- Строка 1 «РНН» - реквизит «РНН» текущего налогоплательщика(организации) из справочника «Организации». 
- Строка 2 «ИИН (БИН)» - реквизит «БИН / ИИН» текущего налогоплательщика(организации) из справочника «Организации». 
- Строка 3 «ФИО или наименование налогоплательщика» - полное наименование текущего налогоплательщика(организации) из справочника «Организации».
- Строка 4 «Налоговый период» - налоговый период, выбранный в основной форме отчета. 
- Строка 5 «Вид декларации» -  заполняется вручную. По умолчанию устнавливается Вид декларации «Очередная».
- Строка 6 «Номер и дата уведомления» - заполняется вручную, возможность заполнения номера и даты уведомления предоставляется в случае выбора вида декларации «По уведомлению».
- Строка 7 «Категория налогоплательщика» - заполняется вручную.
- Строка 8 «Код валюты» - заполняется автоматически и равно «KZT».
- Строка 9 «Представленные приложения» - заполняется автоматически, в зависимости от того заполнены ли приложения.
- Строка 10 «Представленные расчеты за структурные подразделения или объекты, связанные с налогообложением» - заполняется вручную.

РАЗДЕЛ. Ответственность налогоплательщика

- Поле «Дата подачи декларации» - указывается дата, которая может быть отредактирована в поле ввода «Дата подписи». 
- Поле «ФИО руководителя (налогоплательщика)» - указывается  ФИО руководителя организации из регистра сведений «Ответственные лица организаций». 
- Поле «ФИО должностного лица, принявшего Декларацию» - поле не заполняется и не редактируется.
- Поле «Дата приема Декларации в налоговом органе» - поле не заполняется и не редактируется.
- Поле «Входящий номер документа» - поле не заполняется и не редактируется.
- Поле «Код налогового органа» - заполняется автоматически как первые 4 символа РНН Налогового комитета, выбранного в основной форме отчета. Данное поле может редактироваться вручную.
- Поле «Дата почтового штемпеля» - поле не заполняется и не редактируется.

 

ЗАПОЛНЕНИЕ ПРИЛОЖЕНИЯ 400.01. «Облагаемые операции по спирту и (или) виноматериалу»

 

Для правильного заполнения приложения 400.01 необходимо, чтобы в справочнике «Номенклатура» для элементов справочника был заполнен реквизит «Вид подакцизного ТМЗ».

В форме отражаются обороты по регистру накопления «Акциз», причем в форму попадают только те данные, у которых вид операции в регистре сведений «Коды строк декларации по акцизу» имеет код строки, начинающийся с «400.01», и вид ТМЗ равный значению «Спирт» или «Виноматериал». В форму попадают данные из регистра накопления «Акциз» по номенклатуре, реализованной или возвращенной от покупателя, у которой в справочнике «Номенклатура» реквизит «Вид подакцизного ТМЗ» содержит значение «Спирт» или «Виноматериал».

Заполнение строк:

Строка 400.01.001 «Спирт, реализуемый для производства алкогольной продукции» заполняется по следующему принципу: 

       - Графа А «Налоговая база(литр)». В ней отражается количество ТМЗ, для которых определен вид ТМЗ равный значению «Спирт», из регистра накопления «Акциз» по видам операции на код строки «400.01.001». 
       - Графа В «Ставка акциза». При заполнении графа В «Ставка акциза» рассчитывается как отношение значения графы С «Сумма акциза» строки «400.01.001» к значению графы А «Налоговая база(литр)» строки «400.01.001». 
       - В графе С «Сумма акциза» отражается сумма акциза из регистра накопления «Акциз» по видам операции, ссылающимся на код строки «400.01.001».

Строки 400.01.002 - 400.01.012 заполняются аналогично строке 400.01.001. 

Строка 400.01.013 «Акциз, уплаченный поставщиком спирта при приобретении для производства алкогольной продукции» в рамках конфигурации не заполняется. 

Строка 400.01.014 «Итого исчислено акциза» рассчитывается по формуле.

Строка 400.01.015 «Виноматериал, реализуемый для производства алкогольной продукции» заполняется по следующему принципу: 

       - Графа А «Налоговая база(литр)». В ней отражается количество ТМЗ, для которых определен вид ТМЗ равный значению «Виноматериал», из регистра накопления «Акциз» по видам операции на код строки «400.01.015». 
       - Графа В «Ставка акциза». При заполнении графа В «Ставка акциза» рассчитывается как отношение значения графы С «Сумма акциза» строки «400.01.015» к значению графы А «Налоговая база(литр)» строки «400.01.015». 
       - В графе С «Сумма акциза» отражается сумма акциза из регистра накопления «Акциз» по видам операции, ссылающимся на код строки «400.01.015».

Строки 400.01.016 - 400.01.026 заполняются аналогично строке 400.01.015. 

Строка 400.01.027 «Акциз, уплаченный поставщиком виноматериала при приобретении для производства алкогольной продукции» в рамках конфигурации не заполняется. 

Строка 400.01.028 «Итого исчислено акциза» рассчитывается по формуле.

Примечание!!! В форме отражаются данные по регистру накопления «Акциз», где для вида операции в регистре сведений «Коды строк декларации по акцизу» определен код строки, начинающийся с «400.01» и равный «400.01.001», «400.01.002», «400.01.003», «400.01.004», «400.01.005», «400.01.006», «400.01.007», «400.01.008», «400.01.009», «400.01.010», «400.01.011», «400.01.012» для спирта, и «400.01.015», «400.01.016», «400.01.017», «400.01.018», «400.01.019», «400.01.020», «400.01.021», «400.01.022», «400.01.023», «400.01.024», «400.01.025», «400.01.026» для виноматериала.

 

 

ЗАПОЛНЕНИЕ ПРИЛОЖЕНИЯ 400.02. «Облагаемые операции по алкогольной продукции и (или) пиву с объемным содержанием этилового спирта не более 0,5 процента»

 

Для правильного заполнения приложения 400.02 необходимо, чтобы в справочнике «Номенклатура» для элементов справочника был заполнен реквизит «Вид подакцизного ТМЗ».

В форме отражаются обороты по регистру накопления «Акциз», причем в форму попадают только те данные, у которых вид операции в регистре сведений «Коды строк декларации по акцизу» имеет код строки, начинающийся с «400.02», и вид ТМЗ равный значению «Алкогольная продукция». В форму попадают данные из регистра накопления «Акциз» по номенклатуре, реализованной или возвращенной от покупателя, у которой в справочнике «Номенклатура» реквизит «Вид подакцизного ТМЗ» содержит значение «Алкогольная продукция».

Заполнение строк:

– В строку 400.02.001А записывается укрупненно «Алкогольная продукция».

– Строка 400.02.001В заполняется вручную.

– В строке 400.02.002 отражаются обороты по количеству ТМЗ из регистра накопления «Акциз» при условии, что виду операции регистра накопления «Акциз» соответствует код строки «400.02.002» и вид ТМЗ «Алкогольная продукция» в регистре сведений «Коды строк декларации по акцизу», а также для данного ТМЗ в справочнике «Номенклатура» реквизит «Вид подакцизного ТМЗ» содержит значение «Алкогольная продукция».

– Строки 400.02.003 – 400.02.009 заполняются на основании данных, непосредственно отнесенных на данные коды строк, т.е. аналогично заполнению строки 400.02.002.

– Строка 400.02.010 «Порча, утрата учетно-контрольных марок» в рамках конфигурации не заполняется автоматически.

– Строка 400.02.011 при заполнении из базы рассчитывается как сумма строк с 400.02.002 по 400.02.009.

– Значение строки 400.02.012 «Ставка акциза» рассчитывается как отношение суммы акциза (значение в строке 400.02.013) к налоговой базе (значение в строке 400.02.011).

– В строке 400.02.013 отражается общая сумма акциза по ТМЗ, у которых в справочнике «Номенклатура» реквизит «Вид подакцизного ТМЗ» содержит значение «Алкогольная продукция», и видам операции, для которых в регистре сведений «Коды строк декларации по акцизу» определен код строки декларации, начинающийся с «400.02».

Примечание!!! В форме отражаются данные по регистру накопления «Акциз», где для вида операции в регистре сведений «Коды строк декларации по акцизу» определен код строки, начинающийся с «400.02» и равный «400.02.002», «400.02.003», «400.02.004», «400.02.005», «400.02.006», «400.02.007», «400.02.008», «400.02.009».

Для правильного заполнения приложения 400.03 необходимо, что в справочнике «Номенклатура» для элементов справочника был заполнен реквизит «Вид подакцизного ТМЗ».

В форме отражаются обороты по регистру накопления «Акциз», причем в форму попадают только те данные, у которых вид операции в регистре сведений «Коды строк декларации по акцизу» имеет код строки, начинающийся с «400.03», и вид ТМЗ равный значению «Сигареты с фильтром», «Сигареты без фильтра», «Табак», «Сигары» или «Сигариллы». РАЗДЕЛ. Облагаемые операции по сигаретам с фильтром


– Строка 400.03.001. В строке 400.03.001 отражаются обороты по количеству ТМЗ (налоговая база (штук)) из регистра накопления «Акциз» при условии, что виду операции регистра накопления «Акциз» соответствует код строки «400.03.001» и вид ТМЗ «Сигареты с фильтром» в регистре сведений «Коды строк декларации по акцизу», а также для данного ТМЗ в справочнике «Номенклатура» реквизит «Вид подакцизного ТМЗ» содержит значение «Сигареты с фильтром».

– Строки 400.03.001 – 400.03.008 заполняются на основании данных, непосредственно отнесенных на данные коды строк, т.е. аналогично заполнению строки 400.03.001.

– Строка 400.03.009 «Порча, утрата акцизных марок» в рамках конфигурации автоматически не заполняется.

– Строка 400.03.010 рассчитывается как сумма строк с 400.03.001 по 400.03.009.

– Строка 400.03.011. Значение строки 400.03.011 «Ставка акциза» рассчитывается как отношение суммы акциза (значение в строке 400.03.012) к налоговой базе (значение в строке 400.03.010).

– Строка 400.03.012. В строке 400.03.012 отражается общая сумма акциза по ТМЗ, у которых в справочнике «Номенклатура» реквизит «Вид подакцизного ТМЗ» содержит значение «Сигареты с фильтром», и видам операции, для которых в регистре сведений «Коды строк декларации по акцизу» определен код строки декларации, начинающийся с «400.03».
