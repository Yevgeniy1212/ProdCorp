Отчет "Бухгалтерский баланс"

Отчет "Бухгалтерский баланс" предназначен для составления отчетности по балансу предприятия.

Данный отчет собирает суммы из плана счетов НСФО (утвержденный приказом Министра финансов Республики Казахстан №185 от 23.05.2007 г., вступающим в действие с 01.01.2008 г.) в форме, повторяющей план счетов МСФО (разработанный в соответствии с приказом Министра Финансов Республики Казахстан от 22 декабря 2005 года № 426 «Об утверждении инструкции (основы) по
разработке рабочего плана счетов бухгалтерского учета для организаций, составляющих финансовую отчетность в соответствии с международными стандартами финансовой отчетности»). План счетов НСФО в целом совпадает с планом счетов МСФО, однако имеются некоторые отличия, в этом случае соответствия указаны в скобках, в графе "Код счета". Отчет также заполняется по данным пользовательских счетов, но только для подчиненных счетам, соответствующим строкам отчета, не являющимся итоговыми. Например, если у счета 1270 - "Краткосрочные вознаграждения к получению" добавить субсчет, то данные по остаткам и оборотам данного субсчета попадут в строку 16 - "Краткосрочные вознаграждения к получению", однако если у счета 5000 - "Уставный капитал" добавить субсчет, то поскольку счет 5000 предположительно соответствует итоговой строке 198 - "Выпущенный капитал", то данные субсчета не попадут в отчет и об этом будет выведено сообщение.

В отчете не реализовано заполнение строк, соответствующих детальной аналитике счетов, в частности строки: 55 - 91, 93, 98, 99, 122 - 124, 126, 127, 129 - 131, 135 - 137, 139 - 141, 147, 148, 150, 151, 153 - 154, 157 - 159, 164, 165, 167 - 171, 173 - 175, 178, 179, 181, 182, 189 - 191, 193 - 195, 200 - 202, 204 - 207, 212 - 214, 218 - 221, 233, 234, 236, 237, 240 - 246, 248 - 250, 253 - 255, 260 - 285, 287 - 308, 310, 311, 313 - 330, 334 - 339, 355 - 358, 364 - 389, 391 - 408, 411 - 417, 420 - 425, 437 - 440, 442 - 445, 447 - 450, 452 - 471. Данные строки заполняются в ручную.

 

Общая информация о работе с отчетами

 

Отчет имеет две формы.

В первой, основной форме пользователю нужно установить Организацию и период отчета.

 

После нажатия «ОК» открывается форма отчета, верхняя часть которой используется для управления формированием, а в нижней части расположен сам бланк отчета.

 

Работа с отчетом

 

Для работы необходимо установить значение следующих реквизитов в форме отчета:

 

«Вид формы»  -  вид результирующей формы, значение выбирается из выпадающего списка

 

«Код налоговой инспекции»   -   код налоговой инспекции Организации

  

В диалоговом окне формы отчета предусмотрен набор кнопок, соответствующий списку

возможных действий:

 

Кнопка «Выгрузить в XML» -  при помощи этой кнопки осуществляется выгрузка информации в формате XML для клиентского приложения ИС СОНО.

 

Кнопка «Выгрузить в Excel»  при помощи этой кнопки осуществляется выгрузка информации в формате MS Excel для клиентского приложения ИС СОНО.

 

Кнопка «Заполнить»  -  по данной кнопке происходит автоматическое заполнение отчета данными из бухгалтерского учета. При желании пользователь может отредактировать или заполнить данные самостоятельно.

 

Кнопка «Очистить» -  очищает бланк отчета 

 

Кнопка «Печать» - позволяет выполнить предварительный просмотр, либо сразу распечатать отчет

Кнопка вызова справки – пользователь может просмотреть текущую справку, либо перейти к общей информации о формах.

  

Кроме этого, в процессе работы с отчетом пользователь может самостоятельно редактировать данные, представленные в бланке отчета. По двойному щелчку на ячейке открывается окно для прямого редактирования значения.

 

Все основные механизмы работы с отчетом (печать, форматирование данных, выгрузка в

XML-формате и т.д.) заимствованы из  регламентированных отчетов типовой конфигурации

"Бухгалтерия для Казахстана"
