Отчет "Анализ счета (упр.)"

Отчет предназначен для получения данных по дебетовым и кредитовым начальным и конечным остаткам, а также оборотам по указанному счету (группе счетов) за выбранный период, с детализацией по субсчетам, субконто, корсчетам и субконто корсчетов.

На закладке «Данные» указываются следующие настройки:

·         Период формирования отчета;

·         Счет – счет или группа счетов, данные по которым необходимо получить;

·         Сценарий – если указан, то выводятся плановые данные по выбранному сценарию. Иначе выводятся фактические данные по счетам;

·         Периодичность отчета – периодичность группировки данных, в которой будут выводиться остатки и обороты отчета. Если не указана, группировка по вложенным периодам не производится;

·         По субсчетам – требуется ли детализировать данные выбранной группы счетов по подчиненным счетам;

·         По кор. счетам – требуется ли детализировать данные по оборотам счетам в разрезе корреспондирующих счетов;

·         По субконто кор. счетов – требуется ли детализировать данные по оборотам счетам в разрезе субконто корреспондирующих счетов;

Если необходимо детализировать данные отчета по субконто выбранного счета, то такие субконто настраиваются в группировках отчета.

Для данных отчета доступна расшифровка отчетом «Карточка счета». При расшифровке данных по корсчетам или кор. субконто расшифровка выполняется отчетом «Отчет по проводкам».
