Обработка "Регламентное назначение соответствий бюджетированию"

Записывает в регистр сведений «Правила регламентного назначения соответствий бюджетирования» правила для задания соответствий бюджетированию, а в регистр сведений «Настройка соответствий для измерений бюджетирования» заданные значения соответствий.  

В  обработке в обязательном порядке указывается:

  • дата установки соответствий;
  • «Разрез» - аналитический разрез бюджетирования, для которого настраивается строка соответствий;
  • «Имя справочника» - имя справочника для которого задаются соответствия.
     Заполнение колонок:
  • «Исходное значение» - фиксированное исходное значение, для которого настраивается соответствие данных бюджетирования. Источником данных является справочник выбранный в реквизите «Имя справочника»;
  • «Соответствие» - фиксированное значение аналитического разреза бюджетирования, устанавливаемое в соответствие исходному значению.

При нажатии на кнопку Заполнить табличная часть заполняется значениями из указанного справочника, для которых еще нет заданных соответствий в регистре сведений «Настройка соответствий для измерений бюджетирования».

По кнопке Сохранить указанные соответствия из табличной части записываются в регистр сведений «Настройка соответствий для измерений бюджетирования». А «Имя справочника» и «Разрез» в регистр сведений «Правила регламентного назначения соответствий бюджетирования».

Если у элемента «Исходное значение» включен флаг, то в справочнике соответствия будет создан новый элемент с наименованием элемента «Исходное значение».

Настроенные соответствия предназначены для использования при расчете вариантов соответствий первичным данным.
