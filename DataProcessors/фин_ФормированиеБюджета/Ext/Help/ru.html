Обработка "Формирование бюджета"

Обработка выполняет четыре вида задач:

·         Создание бюджета на основании данных другого периода – для облегчения ввода бюджетов на последующие периоды;

·         Создание бюджета на основании данных другого сценария этого же периода – для облегчения ввода бюджетов по различным сценариям;

·         Создание бюджета на основании данных другого сценария меньшей длительности бюджетного периода за определенный период планирования формируемого сценария – для облегчения ввода укрупненных бюджетов на основании более детальных – например, формирование годового бюджета на основании поквартальных;

·          Создание бюджета определенного ЦФО или проекта на основании данных другого ЦФО/проекта.

При копировании данных возможен ввод поправочных коэффициентов, на которые будут умножаться копируемые показатели.

Шаги мастера:

·         Шаг 1 - Основные параметры – настраиваются параметры бюджета, который будет сформирован в результате работы мастера. При помощи нажатия на гиперссылки <Укажите бюджет> и <Укажите сценарий> выбираются соответствующие параметры из справочников. Признак «Заявка на формирование бюджета» предназначен для указания системе, что необходимо создать документ «Бюджет» с видом операции «Заявка». После указания сценария настраивается период планирования, на который создается бюджет. Если необходимо сформировать бюджет конкретного проекта или управленческого подразделения (центра ответственности), то в группе «Дополнительные параметры» переключатель устанавливается в соответствующее положение, после чего становится доступным выбор соответствующего разреза планирования.

Способ формирования бюджета определяет источник данных для формирования бюджета:

o    По заявкам на формирование бюджета – бюджет будет формироваться на основании документов «Бюджет» с видом операции «Заявка»;

o    По имеющимся оборотам бюджетов – бюджет будет сформирован на основании имеющихся плановых или фактических оборотов (плановых показателей) по статьям бюджетирования из регистра «Обороты бюджетов».

·         Шаг 2 – Область сбора данных – настраивается отбор копируемых данных (заявок/оборотов).

В поле «Сценарий» указывается копируемый сценарий, по которому необходимо фильтровать данные. В случае формирования бюджета на основании заявок указание копируемого сценария обязательно. В случае формирования бюджета на основании оборотов (плановых показателей) по статьям бюджетов, если сценарий не указан, тогда будут взяты фактические данные.

В поле «Период планирования» указывается копируемый интервал. Если у копируемого сценария периодичность планирования отличается от периодичности планирования по сценарию формируемого бюджета, тогда копируемый период все равно охватывает весь период, начинающийся с момента указанного в поле «Период планирования», с учетом горизонта планирования формируемого бюджета.

В группе «Отборы копируемых данных» можно настроить отбор данных по конкретному подразделению/проекту.

В случае формирования бюджета на основании заявок, доступно поле отбора по состоянию заявок.

Если значения отборов не указаны, то соответствующие отборы не применяются.

·         Шаг 3 – Дополнительные настройки – предназначен для выполнения действий, предшествующих итоговому получению результата.

Для формирования бюджета по заявкам открывается окно выбора заявок.

В списке «Заявки на формирование бюджета» находятся заявки, попадающие под условия фильтра и не вошедшие в другие бюджеты. При помощи специализированных кнопок в правой части экрана доступные заявки перемещаются в список выбранных заявок, или удаляются из него. При перемещении в список выбранных не утвержденных заявок, система предложит автоматически установить признак утверждения.

В случае формирования бюджета по оборотам на данном шаге будет предложено выбрать поправочные коэффициенты в целом для всех строк формируемого бюджета, по отношению к копируемым данным.

Поправочный коэффициент величиной в единицу означает, что изменений копируемых показателей не производится.

·         Шаг 4 – Предварительный просмотр – позволяет просмотреть получаемые данные, как они будут выглядеть в формируемом бюджете. Данные приводятся с учетом поправочных коэффициентов.

Кнопка Ввод поправок на коэффициент по статьям позволяет управлять списком поправочных коэффициентов, применяемых к отдельным статьям. При корректировке коэффициентов данные поля просмотра результата обновляются автоматически. При большом объеме данных такой процесс может занять определенное время, поэтому стоит максимально полно отражать сведения по поправочным коэффициентам за один прием редактирования.

Поправочные коэффициенты применяются только к тем статьям, которые внесены в данный список. Поправочный коэффициент задается отдельно для суммы и отдельно для количества.

Поправочные коэффициенты могут применяться, например, при планировании бюджетов на основании предыдущего периода для учета инфляции и т.д.

Внимание!!! При формировании бюджета на основании оборотов (плановых показателей) цена рассчитывается на основании заданных источников расчета цены для каждой отдельной статьи оборотов.

Результатом выполнения данной обработки является сформированный документ «Бюджет».
