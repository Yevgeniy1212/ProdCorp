Инструмент «Расчет регламентированной зарплаты»

Инструмент позволяет создавать, рассчитывать и проводить документы по расчету заработной платы за месяц. В процессе расчета обрабатываются документы следующих видов "Начисление зарплаты работникам организаций", "Расчет ИПН, ОПВ и удержания", "Расчет СН и СО", "Удержание ИПН и ОПВ по налоговому учету" и "Отражение зарплаты в регл учете". Виды автоматически рассчитываемых документов устанавливаются с помощью флажков, расположенных в форме настройки, вызываемой при нажатии на одноименную кнопку на верхней панели инструментов формы обработки. По умолчанию производится расчет всех указанных видов документов.

Для расчета необходимо выбрать месяц расчета и указать организацию, заработная плата работников которой будет рассчитываться. Расчет производится по каждому отмеченному подразделению из списка подразделений организации, т.е. для каждого отмеченного подразделения создается свой документ каждого типа, если среди уже существующих документов нет документа, сформированного по родительскому подразделению или по всей организации в целом, иначе будет перерассчитываться уже существующий документ. Для этого предназначены пункт меню "Рассчитать" на верхней панели формы.

В процессе расчета осуществляются следующие действия:

  1. Анализируются существующие в информационной базе документы перечисленных видов за указанный месяц по организации, определяются подразделения, по которым отсутствуют какие-либо из указанных документов. При необходимости недостающие документы будут созданы. Документы "Начисление зарплаты работникам организаций" и "Расчет ИПН, ОПВ и удержаний", у которых заполнен реквизит "Вид расчета" (т.е. при заполнении осуществляется отбор только по определенному виду расчета), при определении документов, подлежащих расчету, не учитываются, т.к. не содержат полный список всех плановых расчетов, поэтому в табличном поле документов не отображаются. Кроме того, в процессе анализа игнорируются документы "Начисление зарплаты работникам организаций", содержащие заполненный реквизит "Перерасчет документа", поскольку такие документы содержат данные о перерасчете за прошлые периоды.
  2. Для каждого документа последовательно вызываются функции автозаполнения документа, расчета и проведения. Документы обрабатываются в следующем порядке:
    • Начисление зарплаты работникам организаций;
    • Расчет ИПН, ОПВ и удержания;
    • Расчет СН и СО;
    • Удержание ИПН и ОПВ по налоговому учету (если в учетной политике по налоговому учету для организации не указан признак "При начислении ИПН и ОПВ принимать исчисленные суммы к учету как удержанные", иначе данный вид документов вообще не обрабатывается);
    • Отражение зарплаты в регл учете (формируется только один документ по всей организации за месяц).

Внимание!  Так как для учета возможных изменений в кадровых документах по работникам организации в процессе расчета всегда выполняется автозаполнение документов перед расчетом, следует избегать ручных корректировок документов либо отключить их автоматический расчет.

По результатам расчета возможно формирование следующих отчетов непосредственно из формы обработки:

  • Свод начислений и удержаний;
  • Расчетная ведомость организаций;
  • Расчетный листок;
  • Расчет по исчислению и удержанию индивидуального подоходного налога и исчислению социального налога;
  • Контрольная ведомость по отражению зарплаты в регламентированном учете.

Формирование отчетов осуществляется по выбранной организации за месяц проведения расчета. Возможно формировать отчеты только по отмеченным подразделениям организации - для этого следует отметить флажок "По отмеченным подразделения", либо только по указанному работнику - его следует выбрать в соответствующем поле на закладках "Расчетный листок" и "Расчет ИПН и СН".
