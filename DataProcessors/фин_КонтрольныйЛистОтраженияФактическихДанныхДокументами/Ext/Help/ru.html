Обработка "Контрольный лист отражения фактических данных документами"

Обработка предназначена для анализа отражения документов выбранного вида по шаблонам отражения фактических данных.

В форме обработки выбирается вид документов, документы которого будут проверяться, а также период, которому принадлежит проверяемый набор документов.

После нажатия на кнопку проверки на закладке «Контрольный лист» отражается список документов, в колонке «Отражен» указывается, отражен ли документ хотя бы по одному из имеющихся шаблонов. Затем идут колонки с названиями имеющихся для вида документов шаблонов с пометкой, по каким из них  документ отражен. Если нужно выполнить проверку только для конкретного шаблона, то он выбирается в поле «Ограничить шаблоном».

На закладке «Список неотраженных документов» приводится перечень тех документов, которые не отражены ни по одному шаблону.
