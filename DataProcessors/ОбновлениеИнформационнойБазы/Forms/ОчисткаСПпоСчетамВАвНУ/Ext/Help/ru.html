Очистка СП по счетам ВА в НУ

Обработка используется в контексте обновления с версии 2.0.13 на 2.0.14

Внимание! Выполнение обработки может занимать значительное время.

 

Учет внеоборотных активов по налоговому учету ведется в целом по организации. 

В связи с этим изменением производиться очистка измерения «Структурное подразделение» регистра бухгалтерии «Налоговый» по счетам внеоборотных активов.
