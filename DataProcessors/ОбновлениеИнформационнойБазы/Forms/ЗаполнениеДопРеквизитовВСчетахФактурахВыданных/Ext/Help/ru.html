Обработка выданных счетов-фактур

Обработка используется в контексте обновления с версии 2.0.17  на 2.0.18

Выполнение обработки необходимо для заполнения пустых реквизитов счетов-фактур выданных:

  •  Грузополучатель - заполняется значением реквизита Контрагент.

  •  Пункт назначения - в реквизит записывается Юридический адрес Грузополучателя.

Обработка может занимать значительное время, поэтому содержит возможность указания периода для обработки, что позволяет провести полную обработку данных в несколько этапов.

Обрабатываются только проведенные документы.

При повтором выполнении обработки модифицируются только ранее не обработанные документы.
