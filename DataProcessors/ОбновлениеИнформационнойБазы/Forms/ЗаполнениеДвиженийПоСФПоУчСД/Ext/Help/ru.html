Обработка выданных счетов-фактур

Обработка используется в контексте обновления с версии 2.0.12  на 2.0.13

Выполнение обработки необходимо для заполнения движения по документам

Обработка может занимать значенительное время, поэтому содержит возможность указания периода для обработки, что позволяет провести полную обработку данных в несколько этапов.

Для подготовки налоговой отчетности и формирования регистров налогового учета по счета-фактурам необходимо обязательное выполнение обработки счетов-фактур отчетного периода.
