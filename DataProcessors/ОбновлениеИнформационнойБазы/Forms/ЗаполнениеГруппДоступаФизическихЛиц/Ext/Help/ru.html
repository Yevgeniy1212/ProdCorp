Обработка «Заполнение групп доступа физических лиц»

Обработка используется в контексте обновления на версию 1.5.9.

В прежних версиях доступ к справочнику "Физические лица" указывался непосредственно для каждого элемента справочника. Для повышения скорости обработки данных теперь доступ задается для групп физических лиц из нового справочника "Группы доступа физических лиц" и для каждого физического лица указывается, к какой группе доступа он относится. Обработка создает группы по умолчанию в соответствии с прежними настройками доступа.
