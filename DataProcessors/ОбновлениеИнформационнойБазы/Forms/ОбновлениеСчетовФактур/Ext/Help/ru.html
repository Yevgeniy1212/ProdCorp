Обработка выданных и полученных счетов-фактур

Обработка используется в контексте обновления с версии 1.5.17  на 1.5.18

Выполнение обработки необходимо для заполнения новых реквизитов счетов-фактур:

  •  Поставщик.

  •  Покупатель.

  •  Страна назначения (только в выданных счетах-фактурах).

При повтором выполнении обработки модифицируются только ранее не обработанные документы.

Обработка может занимать значенительное время, поэтому содержит возможность указания периода для обработки, что позволяет провести полную обработку данных в несколько этапов.

Для подготовки налоговой отчетности и формирования регистров налогового учета по счета-фактурам необходимо обязательное выполнение обработки счетов-фактур отчетного периода.
