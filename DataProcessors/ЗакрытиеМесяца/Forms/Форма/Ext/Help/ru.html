Обработка "Закрытие месяца"

Обработка предназначена для создания и проведения документов "Закрытие месяца".

При открытии и после указания организации и периода на форме обработки отражается:

  • Список необходимых операций,
  • Состояние операций.

    • Список необходимых операций

    Список необходимых регламентных операций формируется для каждой операции отдельно в соответствии с:

  • периодом (например, Расчет суммы НДС, разрешенной к зачету предлагается только в конце квартала);

    • Операции распределяются на шесть групп:

    1. Формирование затрат.
    2. Закрытие производственных счетов.
    3. Зачет бухгалтерских счетов.
    4. Итоговые операции по расчету прибыли/убытка.
    5. Регламентные операции налогового учета.
    6. Расчеты налога на прибыль.

    Состояние операций

    На форме обработки "Закрытие месяца" отражается состояние необходимых для выполнения регламентных операций с помощью следующей символики:

  • Выполненные успешно операции помечаются зеленым цветом и картинкой ,
  • Выполненные успешно операции, записи которых отредактированы вручную, помечаются зеленым цветом и картинкой ,
  • Операции, при выполнении которых были обнаружены ошибки, помечаются розовым цветом и картинкой ,
  • Невыполненные операции помечаются черным цветом без картинки.

    • Функции обработки

    Обработка "Закрытие месяца" предоставляет возможность:

  • Выполнить все необходимые операции закрытия месяца. Нажатие на кнопку "Выполнить закрытие месяца" вызывает выполнение всех  регламентных операций. 
  • Частично выполнить закрытие месяца (Меню, возникающее при нажатии на выполненную регламентную операцию, пункт "Выполнить все операции до выбранной").
  • Отменить закрытие месяца, Нажатие на кнопку "Отменить закрытие месяца" вызывает отмену выполнения всех регламентных операций указанного периода.
  • Частично отменить выполнение закрытия месяца (Меню, возникающее при нажатии на невыполненную регламентную операцию, пункт "Отменить все операции после выбранной").
  • Сформировать отчеты, объясняющие расчеты и отражающие результаты выполнения регламентных операций (Кнопка "Сформировать справку-расчет"). При выборе этой команды из меню указанной выполненной операции формируются те справки-расчеты, которые соответствуют этой операции.
  • Посмотреть результаты проведения регламентной операции (Меню операции, кнопка "Показать проводки").
  • Составить отчет о выполнении регламентных операций.
  • Составить отчет о ошибках, обнаружившихся при выполнении регламентных операций.
  • Настроить список выполняемых регламентных операций.
