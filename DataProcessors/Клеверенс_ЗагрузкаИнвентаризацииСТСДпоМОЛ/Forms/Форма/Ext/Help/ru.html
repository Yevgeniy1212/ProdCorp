Данная форма предназначена для загрузки документов инвентаризации основных средств с ТСД (терминала сбора данных). Форма позволяет заполнять документы инвентаризации основных средств 1С данными, полученными с ТСД.

Также форма позволяет настроить обмен данными между 1С и ТСД.

Настройка подключения ТСД

Возможны два режима обмена данными:

1) Напрямую с ТСД. Для выбора этого режима нужно установить переключатель ТСД на форме;

2) Через папку обмена (этот вариант используется, если на компьютере, к которому подключен ТСД, работа с 1С происходит через терминальную сессию, при этом используется специальная утилита обмена данными см. http://cleverence.ru/site.aspx?page=MobileSmarts-1C-Driver-Sync). Для выбора этого режима нужно установить переключатель Папка обмена на форме и указать папку в поле выбора.

Загрузка документов инвентаризации ОС с ТСД

По нажатию кнопки Обновить список происходит заполнение списка документов ТСД. Кнопки Выбрать все и Выключить все служат для выбора и снятия выбора с документов ТСД соответственно. По нажатию кнопки Обновить список ОС по выделенным документам происходит заполнение таблицы Основные средства на основании данных из выбранных документов ТСД. Для удаления документов с ТСД предназначена кнопка Удалить выбранные.

Для выбора документа инвентаризаци ОС 1С, который будет заполнен полученными с ТСД данными, следует нажать кнопку "..." в поле Документ инвентаризации ОС и выбрать документ в появившемся окне. Чтобы данные попали в выбранный документ нужно нажать кнопку Заполнить.
