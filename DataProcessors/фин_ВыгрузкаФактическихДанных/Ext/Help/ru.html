Документ «Учет фактических данных по бюджетам»

Для отложенной регистрации фактических данных в оборотах по бюджетам по истечении определенного периода, используется документ «Учет фактических данных по бюджетам».

Ввод фактических оборотов по статьям бюджетирования возможен либо вручную, либо путем автоматического заполнения. При ручном вводе необходимо указать дату операции (колонка «Период»), финансовый показатель, значения оборотов и аналитику. При автоматизированном способе нужно настроить фильтры на значения ЦФО и финансовые показатели, указать период выборки данных в шапке документа. Если ЦФО не указан, документ заполняется данными по всем ЦФО. Если не указан финансовый показатель, тогда документ заполняется данными по всем финансовым показателям, для которых настроены источники фактических данных. В этом случае система проведет отбор всех финансовых показателей согласно настроенным фильтрам и сформирует фактические обороты. Выборка из источника данных для каждой статьи будет проводиться за указанный период. После автоматического заполнения фактических оборотов результат может быть скорректирован пользователем вручную.

Помимо этого, на плане счетов бюджетирования будут отображены проводки, согласно настройке для каждого финансового показателя. Однако в отличие от плановых данных аналитика по сценарию в проводках не будет указана.

Источники для получения фактических данных указываются при описании каждого финансового показателя по бюджетам. В качестве источника данных для автоматического формирования фактических оборотов по статье используется шаблон запроса к учетным данным. Соответственно, для получения суммового и количественного оборота по статье прописываются показатели, определенные для источника. Для заполнения документа автоматизированным способом нужно нажать кнопку Заполнить. В этом случае в табличную часть документа будут введены обороты по статьям согласно источникам фактических данных, установленным у финансовых показателей. Если из источника можно получить первичный документ, которым они были зарегистрированы, тогда в колонке «Документ движений» приводится ссылка на него.

Валюта формирования фактического оборота по статье может браться либо из валюты источника, либо из валюты, определенной по умолчанию для показателя, либо из валюты регламентированного или управленческого учета (в зависимости от настроек получения данных в элементе справочника «Финансовые показатели»).
