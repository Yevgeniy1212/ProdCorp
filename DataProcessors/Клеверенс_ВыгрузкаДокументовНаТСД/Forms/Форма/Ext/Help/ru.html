Данная форма предназначена для выгрузки документов инвентаризации ОС на ТСД (терминал сбора данных).

Также форма позволяет настроить обмен данными между 1С и ТСД и получить список имеющихся на ТСД документов и удалить их при необходимости.

Настройка подключения ТСД

Возможны два режима обмена данными:

1) Напрямую с ТСД. Для выбора этого режима нужно установить переключатель ТСД на форме;

2) Через папку обмена (этот вариант используется, если на компьютере, к которому подключен ТСД, работа с 1С происходит через терминальную сессию, при этом используется специальная утилита обмена данными см. http://cleverence.ru/site.aspx?page=MobileSmarts-1C-Driver-Sync). Для выбора этого режима нужно установить переключатель Папка обмена на форме и указать папку в поле выбора.

Выгрузка документа на ТСД

Для того, чтобы выгрузить документ инвентаризации на ТСД нужно нажать кнопку "..." в поле Документ инвентаризации ОС и выбрать документ в появившемся окне, затем нажать кнопку Выгрузить на ТСД.

Документы ТСД

В таблице отображается информация о документах, имеющихся в данный момент на ТСД.

По кнопке Обновить список информация о документах запрашивается с ТСД, кнопки Выбрать все и Выключить все служат для выбора всех документов в списке и снятия выбора соответственно. По кнопке Удалить выбранные происходит удаление выбранных документов с ТСД.
