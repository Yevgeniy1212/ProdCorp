Обработка «Настройка прав доступа»

Используется для настройки прав пользователя (заполняется регистр сведений «Настройки прав доступа пользователей»).

Обработку «Настройка прав доступа» можно вызвать из формы списка или из формы элемента справочника «Группы пользователей» с помощью кнопки «Права» (Рисунок 1).

Права доступа настраиваются только для назначенных видов объектов доступа (Рисунок 1).

 


Рисунок 1 – Вызов обработки «Настройка прав доступа».

 

Виды доступа:

  • Видимость в списке - права на доступ к списку элементов справочника без права доступа к документам (записям регистров), в реквизитах (измерениях) которых записана ссылка на соответствующий элемент справочника, т.е. без права доступа к «подчиненной» информации по элементу;
  • Просмотр данных - права на доступ к документам (записям регистров), в реквизитах (измерениях) которых записана ссылка на соответствующий элемент справочника, т.е. фактически право доступа к «подчиненной» информации по элементу;
  • Редактирование данных - права на редактирование документов (записей регистров), в реквизитах (измерениях) которых записана ссылка на соответствующий элемент справочника.

 

Для включения ограничения прав доступа на уровне записей в целом для конфигурации значение константы «Использовать ограничения прав доступа на уровне записей» необходимо установить в Истина (Рисунок 2).

 


Рисунок 2 – Использовать ограничения прав доступа на уровне записей.

 

Особенности настройки прав доступа к конкретным объектам:

  • Права доступа к физическим лицам настраиваются только для групп физических лиц (справочник «Группы доступа физических лиц»).
