Обработка "Заполнение контролируемых финансовых показателей"

В документе "Установка лимитов и целевых значений" с видами ограничения оборотов "Предварительный бюджет" и "Контролирующий сценарий (выборочно)" имеется возможность автоматического заполнения строк документа при помощи мастера. Мастер открывается из формы документа, из пункта меню "Добавить" командной панели списка ограничений. В данном меню есть два пункта: "Заполнить" (в этом случае табличная часть будет полностью перезаполнена по результатам работы мастера) и "Добавить" (в этом случае табличная часть будет дополнена по результатам работы мастера).

     В форме мастера указывается: период планирования, разрезы и финансовые показатели, по оборотам которых необходимо заполнить документ. В результате работы мастера будут получены следующие строки документа:

     - для вида ограничения оборотов "Предварительный бюджет" - в документ будут добавлены строки с запланированными по сценарию документа оборотами. Также для тех финансовых показателей, которые входят в выбранные для заполнения бюджеты и статьи бюджетов, но не имеют запланированных оборотов, будут добавлены пустые строки для каждого периода планирования;

     - для вида ограничения оборотов "Контролирующий сценарий (выборочно)" - в документ будут добавлены строки только с теми финансовыми показателями, которые выбраны в мастере, и по которым имеются запланированные обороты;

     Если указаны разрезы детализации плановых данных, тогда имеющиеся запланированные обороты будут расшифрованы по выбранным разрезам, иначе плановые данные будут взяты в документ сводно.
