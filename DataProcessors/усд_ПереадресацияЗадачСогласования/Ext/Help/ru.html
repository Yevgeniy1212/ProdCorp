Обработка "Переадресация задач согласования"

Для того, чтобы переадресовать задачи согласования на конкретного исполнителя вместо текущего или вместо групповой адресации, используется инструмент «Переадресация задач согласования».

Данная обработка предназначена для открытия только из форм списка задач согласования и формы утверждения документов. В нее попадают те документы, к которым были применены команды переадресации в указанных формах. Пользоваться данной обработкой могут пользователями с назначенными ролями «Полные права» или «Управление согласованием и утверждением документов».

В колонке «Текущий исполнитель» указывается текущий исполнитель, задачи которого переадресуются. Если задачи находятся на исполнении не у конкретного пользователя, то эта колонка остается пустой. В колонке «Новый исполнитель» указывается лицо, на которое перенаправляется задача.

В форме имеются следующие команды:

·         Установить пометки, снять пометки – для управления пометками в списке документов;

·         Установить исполнителя – установить выбранного исполнителя для всех помеченных документов;

·         Переадресовать – переадресовать все помеченные документы, для которых выбран новый исполнитель.
