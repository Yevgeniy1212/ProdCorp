Обработка "Удаление документов с маршрутов"

Для снятии документов с маршрутов и очистки истории движений используется обработка Удаление документов с маршрутов.

В форме выбирается способ выполнения удаления:

·         Помеченные на удаление документы – в этом случае для документов, помеченных на удаление, очищается список назначений маршрутов в регистре сведений Назначения маршрутов и история движений по этапам в регистрах Движение документов по маршрутам, Активные этапы рассмотрения документов;

·         Документы, не имеющие движений по маршрутам - используется для удаления назначений маршрутов в регистре сведений Назначения маршрутов, для которых не существует записей движений по этапам в регистрах Движение документов по маршрутам, Активные этапы рассмотрения документов. Это может понадобиться в случае некорректной настройки маршрута, когда маршрут был назначен, но был настроен неправильно и движения по маршруту сформированы не были;

·         Документы, не имеющие назначенных маршрутов - используется для удаления записей движений по этапам в регистрах Движение документов по маршрутам, Активные этапы рассмотрения документов, для которых нет назначений маршрутов в регистре сведений Назначения маршрутов. Может использоваться при ручном удалении назначения маршрута в списке Назначения маршрутов, для последующей очистки движений по маршрутам и снятия с маршрута;

·         Указанные вручную – используется для ручного выбора документов, находящихся на маршрутах, для которых необходимо удалить список назначений маршрутов в регистре сведений Назначения маршрутов и историю движений по этапам в регистрах Движение документов по маршрутам, Активные этапы рассмотрения документов. Ручной выбор документов производится в списке документов:

В списке документов при нажатии кнопки Поиск выводится иерархический список документов на маршрутах, с группировкой по маршрутам и видам документов. Удаляемые документы помечаются флажком слева от выбранного документа.

Выполнение выбранного действия производится при нажатии на кнопку Выполнить верхней командной панели.
