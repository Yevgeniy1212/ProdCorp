Настройка параметров аналитического учета ТМЗ

  1. Для начала работы в программе необходимо настроить параметры аналитического учета товарно -материальных запасов. Для этого выберите в меню Предприятие пункт Настройка параметров учета.
  2. В открывшейся форме Настройка параметров учета перейдите на закладку Аналитический учет ТМЗ. Установите флажки Вести складской учет и Вести суммовой учет для бухгалтерского  учета так, как показано на рисунке:

     
  3. Нажмите кнопку ОК для установки параметров аналитического учета и закрытия формы Настройка параметров учета.

ВАЖНО!
По умолчанию в программе не установлены указанные в форме настройки параметры аналитического учета ТМЗ. Если в программе предполагается вести учет товарно-материальных запасов, например, в разрезе складов, то на закладке «Аналитический учет ТМЗ» нужно установить флажок «Вести складской учет». При выполнении такой настройки на счетах группы  1300 «Запасы», предусматривающих аналитический учет по субконто «Номенклатура», будет добавлено субконто «Склады».

 

ПРИМЕЧАНИЕ
Субконто в программе «1С:Бухгалтерия 8 для Казахстана» — это объект аналитического учета («Номенклатура», «Склады», «Контрагенты» и т.д.). Под видом субконто понимается множество однотипных объектов аналитического учета, из которого выбирается объект (например, справочник «Номенклатура»).
