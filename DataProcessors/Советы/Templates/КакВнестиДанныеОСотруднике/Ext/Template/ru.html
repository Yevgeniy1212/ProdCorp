Как внести данные о сотруднике

В справочнике Сотрудники указывают данные характеризующие физическое лицо как работника (место работы, вид договора, должность, основное начисление, персональные надбавки и т.д.). Для того, чтобы внести данные по сотруднику необходимо открыть в интерфейсе кадровый учет организаций - меню Кадровый учет - Сотрудники, кнопка Добавить.

  1. В появившемся окне необходимо выбрать:
    • Создать нового сотрудника и ввести его личные данные в справочник физических лиц - если регистрируется новый сотрудник, данные о котором отсутствуют в справочнике Физические лица. В данном случае также необходимо заполнить раздел Личные данные сотрудника.
    • Создать нового сотрудника, выбрав его из справочника физических лиц - если сотрудник ранее работал на предприятии по другому договору или личные данные были предварительно внесены в справочник Физические лица, тогда необходимо выбрать соответствующий элемент справочника Физические лица.
  2. В разделе Данные сотрудника указывается организация, вид договора, вид занятости и табельный номер сотрудника.
  3. Для сотрудников с видом договора Трудовой договор, необходимо заполнить данные на закладке Трудовой договор. Информация с данной вкладки будет отображена в печатной форме трудового договора, а также использована при приеме сотрудника на работу.
  4. По кнопке ОК данные о сотруднике будут записаны.
Примечание: для каждого места работы в организации для одного и того же физического лица необходимо создавать отдельного сотрудника.
