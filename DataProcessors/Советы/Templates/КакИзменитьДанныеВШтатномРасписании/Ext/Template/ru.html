Как изменить данные в штатном расписании

Для работы со штатным расписанием используется инструмент штатное расписание (интерфейс кадровый учет организаций - меню Предприятие - Штатное расписание организаций). В появившемся окне необходимо выбрать организацию и подразделение в разделе Подразделения. После произведенных действий в разделе Штатные единицы будут отображены все штатные единицы указанной организации и подразделения.

Для добавления новой штатной единицы:

  1. Нажмите кнопку Добавить.
  2. В открывшемся окне необходимо заполнить данные о штатной единице.
  3. В поле Строка штатного расписания действует с необходимо внести дату, с которой начинает действовать штатная единица.
  4. Нажмите ОК - данные будут сохранены.

Для изменения данных о штатной единице необходимо :

  1. В разделе Штатные единицы выделите штатную единицу, данные которой необходимо изменить.
  2. Нажмите кнопку Скопировать.
  3. В открывшемся окне внесите изменения.
  4. В поле Строка штатного расписания действует с необходимо указать дату, с которой вступят изменения.
  5. Нажмите ОК - данные будут сохранены.

Если штатная единица прекращает свое действие:

  1. В разделе Штатные единицы выделите штатную единицу, действие которой необходимо прекратить.
  2. Нажмите кнопку Скопировать.
  3. В открывшемся окне в поле количество ставок укажите 0.
  4. В поле Строка штатного расписания действует с необходимо указать дату, с которой действие штатной единицы будет прекращено.
  5. Нажмите ОК - данные будут сохранены.
Примечание: с помощью кнопки По подразделению в разделе Штатные единицы можно включить/отключить отбор штатных единиц по подразделению, чтобы отобразить штатные единицы конкретного подразделения (указывается в разделе Подразделения) или всей организации.
