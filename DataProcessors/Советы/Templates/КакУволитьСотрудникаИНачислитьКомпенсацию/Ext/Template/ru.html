Как уволить сотрудника и начислить ему компенсацию

  1. Для увольнения сотрудника из организации предназначен документ Увольнение из организации (интерфейс кадровый учет организаций - меню Кадровый учет - Увольнение из организации, кнопка Добавить). Документом регистрируется увольнение сотрудников из организации, а также «прекращается» расчет для этих сотрудников всех назначенных им постоянных начислений и удержаний. В документе указывается список увольняемых сотрудников, даты и причины увольнения. При авторасчете табличная часть «Компенсируемые неиспользованные трудовые отпуска» заполняется списком неиспользованных трудовых отпусков на дату увольнения (включительно) по видам отпусков и рабочим годам.
  2. После того, как сотрудник уволен из организации необходимо рассчитать компенсацию при увольнении (интерфейс расчет зарплаты организаций - меню Расчет зарплаты - Первичные документы - Расчет при увольнении сотрудника организации, кнопка Добавить). В созданном документе указать:
  • организацию и сотрудника;
  • месяц начисления;
  • количество неиспользованных дней отпуска;
  • по кнопке Рассчитать\Рассчитать будет автоматически рассчитан результат компенсации;
  • в случае, если в информационной базе данные расчетов за предшествующие месяцы отсутствуют, тогда данные о заработке можно ввести вручную - в строках по видам заработка на закладке Расчет среднего заработка. И по кнопке Рассчитать выбрать пункт Рассчитать начисления.
