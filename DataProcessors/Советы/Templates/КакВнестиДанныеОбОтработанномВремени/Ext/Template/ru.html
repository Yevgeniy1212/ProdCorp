Как внести данные об отработанном времени

Для регистрации фактического времени явок и неявок сотрудников по видам использования рабочего времени используется документ Табель учета рабочего времени:

  1. Создайте новый документ Табель учета рабочего времени (интерфейс расчет зарплаты организаций - меню Расчет зарплаты - Табель учета рабочего времени).
  2. В поле От - установите дату документа (как правило, программа по умолчанию устанавливает последнюю дату месяца).
  3. В поле Организация укажите организацию, по которой проводится начисление.
  4. В поле Месяц укажите период, за который будут введены данные.
  5. Если регистрация отработанного времени выполняется по подразделениям, в поле Подразделение необходимо выбрать подразделение, по которому будет регистрироваться отработанное время. Если данные будут указаны в целом по организации, то поле Подразделение заполнять не нужно.
  6. Данные об отработанно времени в документе могут быть введены двумя способами:
    • По дням периода (число рабочих часов вводится по каждому дню месяца) - ввод времени в ячейке, соответствующей конкретной дате, выполняется путем ввода строки в формате: <Вид времени> <Количество часов>. Например, можно ввести строку "Я 8 С 2", что будет означать: явка - 8 часов, сверхурочно - 2 часа; "Я 8 Н 2", что будет означать: явка - 8 часов, ночные - 2 часа.
    Примечание: условные обозначения использования рабочего времени приведены в справке к документу Табель учета рабочего времени.
    • Сводно, в целом за период - данные вводятся сводно, за весь месяц.
  7. Для автоматического заполнения табличной части документа Табель учета рабочего времени, нажмите на кнопку Автозаполнение (расположенной на командной панели документа) и выберите вариант автозаполнения.
  8. Проведите документ и закройте форму – кнопка OK.
