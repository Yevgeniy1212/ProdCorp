Как начислить зарплату и выполнить удержание налогов и отчислений, таких как ИПН и ОПВ, СН и СО

Для начисления зарплаты в системе следует оформить документ Начисление зарплаты:

  1. Создайте новый документ Начисление зарплаты (в интерфейсе расчет зарплаты организаций - меню Расчет зарплаты - Начисление зарплаты сотрудникам организаций - кнопка Добавить).
  2. В поле От - установите дату начисления зарплаты (как правило это последняя дата месяца).
  3. В поле Месяц начисления - с помощью кнопки выбора из списка выберите месяц начисления.
  4. Если начисление зарплаты выполняется по подразделениям, в поле Подразделение необходимо выбрать подразделение, по которому будет оформляться начисление зарплаты. Если зарплату начисляется в целом по организации, то поле Подразделение заполнять не нужно.
  5. Для автоматического заполнения табличной части документа Начисление зарплаты, нажмите на кнопку Автозаполнение (расположенной на командной панели документа) и выберите вариант автозаполнения.
  6. Для расчета суммы начислений по каждому виду расчета нажмите на кнопку Рассчитать - Рассчитать. В поле Результат после нажатия этой кнопки должны рассчитаться суммы по каждому виду виду начисления.
  7. Проведите документ Начисление зарплаты (кнопка Действия - Провести).

Для расчета суммы индивидуального подоходного налога, обязательных пенсионных взносов и прочих удержаний необходимо оформить документ Расчет ИПН, ОПВ и удержаний.

  1. Создайте новый документ Расчет ИПН, ОПВ и удержаний (в интерфейсе расчет зарплаты организаций - меню Расчет зарплаты - Расчет ИПН, ОПВ и удержаний - кнопка Добавить).
  2. В поле От - установите дату документа (как правило, программа по умолчанию устанавливает последнюю дату месяца).
  3. В поле Месяц начисления - с помощью кнопки выбора из списка, выберите месяц, за который будут рассчитываться сумма налогов и удержаний.
  4. В поле Подразделение выберите подразделение, по которому необходимо выполнить расчет налогов, если расчет налогов и удержаний выполняется в целом по организации, то поле Подразделение заполнять не нужно.
  5. Для автоматического заполнения табличной части документа Расчет ИПН, ОПВ и удержаний, нажмите на кнопку Автозаполнение (расположенной на командной панели документа) и выберите вариант автозаполнения.
  6. Для расчета сумм ИПН, ОПВ, сумм удержаний, а так же прочих вычетов нажмите на кнопку Рассчитать - Рассчитать.
  7. Проведите документ Расчет ИПН, ОПВ и удержаний (кнопка Действия - Провести).

Для расчета суммы социального налога и социальных отчислений необходимо оформить документ Расчет СН и СО.

  1. Создайте новый документ Расчет СН и СО (в интерфейсе расчет зарплаты организаций - меню Расчет зарплаты - Расчет СН и СО - кнопка Добавить).
  2. В поле От - установите дату документа (как правило, программа по умолчанию устанавливает последнюю дату месяца).
  3. В поле Месяц начисления - с помощью кнопки выбора из списка, выберите месяц, за который будут рассчитываться сумма налогов и удержаний.
  4. В поле Подразделение выберите подразделение, по которому необходимо выполнить расчет налогов, если расчет налогов и удержаний выполняется в целом по организации, то поле Подразделение заполнять не нужно.
  5. Для автоматического заполнения табличной части документа Расчет СН и СО, нажмите на кнопку Автозаполнение (расположенной на командной панели документа) и выберите вариант автозаполнения.
  6. Для расчета суммы социального налога и суммы социальных отчислений по каждому сотруднику нажмите на кнопку Рассчитать - Рассчитать.
  7. Проведите документ Расчет СН и СО (кнопка Действия - Провести).

Документы Начисление зарплаты сотрудникам организации, Расчет ИПН, ОПВ и удержаний и Расчет СН и СО не формируют проводки по счетам бухгалтерского учета. Для отражения регламентированной зарплаты в бухгалтерском учете предназначен документ Отражение зарплаты в регламентированном учете.

  1. Создайте новый документ Отражение зарплаты в регламентированном учете (в интерфейсе расчет зарплаты организаций - меню Бухучет зарплаты - Отражение зарплаты в учете - кнопка Добавить).
  2. В поле От - установите дату документа (как правило, программа по умолчанию устанавливает последнюю дату месяца).
  3. В поле Месяц начисления - с помощью кнопки выбора из списка, выберите месяц, за который будут рассчитываться сумма налогов и удержаний.
  4. В поле Подразделение выберите подразделение, по которому необходимо выполнить расчет налогов, если расчет налогов и удержаний выполняется в целом по организации, то поле Подразделение заполнять не нужно.
  5. Для автоматического заполнения табличной части документа, нажмите на кнопку Автозаполнение (расположенной на командной панели документа).
  6. Документ Отражение зарплаты в регламентированном учете формирует проводки и разносит информацию на счета бухгалтерского учета. Для правильного заполнения табличной части необходимо предварительно заполнить справочник Способы отражения зарплаты в учете и оформить документы Ввод сведений об учете плановых начислений сотрудников организации или документ Учет основного заработка работников организации в регламентированном учете.
  7. Провести документ - кнопка ОK.

Кроме того начислять зарплату, рассчитывать налоги можно также с помощью обработки Расчет регламентированной зарплаты.
