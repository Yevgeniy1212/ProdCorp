Как заполнить график работ

  1. Перед началом работы с графиками, необходимо заполнить Производственный календарь.
  2. Затем заполняем графики работ - меню Предприятие - Рабочее время - Графики работы - кнопка Добавить. При вводе нового графика работы следует указать его название. Выбрать год. Для заполнения графика используется "Помощник заполнения графика", вызываемый по кнопке "Заполнить график".
  3. Помощник позволяет выбрать способ заполнения графика: "Заполнить по шаблону" либо "Настроить вручную".
  4. Способ заполнения "Заполнить по шаблону". Используя данный способ, нужно выбрать готовый шаблон для заполнения графика: Пятидневка (40-час.раб.неделя), Шестидневка (40-час.раб.неделя), Сутки через двое (40-час.раб.неделя), Сутки через трое (40-час.раб.неделя). При нажатии на кнопку "Заполнить" произойдет заполнение графика в соответствии с выбранным шаблоном.  
    • Пятидневка (40-час.раб.неделя) - стандартная 40-часовая пятидневная рабочая неделя, 8- часовой рабочий день, выходной: суббота, воскресенье.
    • Шестидневка (40-час.раб.неделя) - 40-часовая шестидневная рабочая неделя, 7-часовой рабочий день (пон. - пт.), 5-часовой рабочий день (сб.), выходной: воскресенье.
    • Сутки через двое (40-час.раб.неделя) - 40-часовая рабочая неделя, суммированный учет рабочего времени.
    • Сутки через трое (40-час.раб.неделя) - 40-часовая рабочая неделя, суммированный учет рабочего времени.
  5. Для шаблонов "Сутки через двое " и "Сутки через трое" имеется возможность вести учет ночных часов. При установке признака "Вести учет ночных часов" начало дневных часов будет считаться с 06-00ч., начало ночных часов с 22-00ч.
  6. Также для шаблонов "Сутки через двое " и "Сутки через трое" необходимо установить "Дату начала отсчета периодичности сменного графика". По умолчанию дата устанавливается на 01 января.
  7. Способ заполнения "Настроить вручную". Данный способ используется для просмотра настроек шаблонных графиков, а также для ручной корректировки необходимых параметров для заполнения графика.

Для просмотра настроек шаблонного графика необходимо:

  1. Нажать на гиперссылку "Изменить параметры заполнения графика", откроется помощник заполнения. 
  2. Установить способ заполнения графика "настроить вручную"
  3. Нажать кнопку "Далее"
  4. Посмотреть или изменить при необходимости "Параметры графика"
  5. Нажать кнопку "Далее"
  6. Посмотреть "Расписание работы"
  7. Нажать кнопку "Заполнить" для заполнения графика, в случае если менялись настройки графика, или кнопку "Отмена" для возврата к форме графика.

Для ручной настройки графика необходимо:

  1. Открыть помощник заполнения графика
  2. В "Способе заполнения графика" выбрать способ "Настроить вручную"
  3. Нажать кнопку "Далее"
  4. В "Параметрах графика" выбрать тип графика: "Пятидневка", "Шестидневка" или"Сменный"
  5. Указать количество часов в неделе
  6. Проставить необходимые признаки "Суммированный учет рабочего времени", "Используется совместно с план-графиком", "Вести учет ночных часов", "Учитывать праздничные дни", "Сокращенное рабочее время"
    • Признак "Суммированный учет рабочего времени". Суммированный учет рабочего времени применяется в непрерывно действующих производствах, цехах, участках и на некоторых видах работ, где по условиям производства (работы) не может быть соблюдена установленная для данной категории работников ежедневная или еженедельная продолжительность рабочего времени.
    • Признак "Используется совместно с план-графиком". Используется в ситуациях замещения одного работника другими за счет смещений производственных смен, когда должен был работать работник. В этом случае удобно указать график работы как используемый совместно с план-графиком (установить соответствующий флажок). При этом заполнять такой график следует наиболее распространенной схемой рабочего времени. Этот график будет рассматриваться как основной график работника, при необходимости ему можно будет ввести индивидуальный график за месяц с помощью документа «План-график учета рабочего времени», данные которого будут использоваться при расчетах регламентированной зарплаты.
    • Признак "Вести учет ночных часов". Устанавливается в случае, если график предусматривает работу в ночное время, и часы работы в ночное время оплачиваются работникам в повышенном размере.
    • Признак "Учитывать праздничные дни". Используется в случаях, когда необходимо, чтобы при заполнении календаря графика учитывались праздничные дни по производственному календарю: смена, выпавшая на праздничный день в этом случае будет нерабочей.
    • Признак "Сокращенное рабочее время". Сокращенное рабочее время устанавливается, если график предусматривает работу в режиме неполного рабочего времени.  Если графиком предусмотрено сокращение продолжительности ежедневной работы на определенное количество рабочих часов во все дни недели, либо сокращение рабочих дней в рабочей неделе. При использовании "Сокращенного рабочего времени" можно указать график для расчета нормы времени: текущий либо график соответствующего ему полного рабочего времени. В таком случае при расчете начислений по тарифным ставкам отработанное время будет рассчитано по назначенному сотруднику графику неполного рабочего времени, а норма времени – по тому графику, который будет задан как график полного рабочего времени. Таким образом, назначенная сотруднику тарифная ставка будет соответствующим образом уменьшена при расчете.
  7. Нажать кнопку "Далее" для перехода к "Расписанию работ"
    1. В "Расписании работ" для пяти- и шестидневных графиков необходимо указать продолжительность рабочей недели и часы рабочего времени. Для сменных графиков работ обязательно следует указать рабочие смены, при этом время начала, и время окончания работы будет указываться в параметрах смены.
    2. Сведения о сменах заполняются в справочнике Смены (меню Предприятие - Рабочее время - Смены). Для каждой смены необходимо заполнить табличную часть Периоды смены. Смена может состоять как из одного периода, так и из нескольких.
  8. Завершение работы с Помощником заполнения графика и автоматическое заполнение календаря графика по выполненным настройкам производится по кнопке «Заполнить».
  9. Просмотреть или изменить параметры заполнения графика можно через ссылку "Изменить параметры заполнения графика"

Примеры заполнения графиков работы

График пятидневка, шестидневка:

  1. Добавляем новый график работы.
  2. В поле Наименование указываем название графика.
  3. Нажимаем кнопку Заполнить график.
  4. В поле Заполнить по шаблону выбираем значение "Пятидневка (40 час.раб.неделя)" или "Шестидневка (40 час.раб.неделя)".
  5. Нажмите Заполнить - данные будут сохранены и будет произведено заполнение графика.

График неполного рабочего времени (например, график сотрудников на "полставки", когда вместо восьмичасового рабочего дня назначается четырехчасовой):

  1. Добавляем новый график работы.
  2. В поле Наименование указываем название графика.
  3. Нажимаем кнопку Заполнить график.
  4. Выбираем способ заполнения графика Настроить вручную.
  5. Нажимаем Далее
  6. В поле Тип графика выбираем значение "Пятидневка" или "Шестидневка".
  7. В поле Часов в неделе указываем количество часов по сокращенному графику (например, 20).
  8. Устанавливаем флажок в поле Сокращенное рабочее время.
  9. В поле Норма времени рассчитывается устанавливаем флажок По другому графику  и указываем график полного рабочего времени с нормой часов в неделе (у графика полного рабочего времени флажок Сокращенное рабочее время не устанавливается).
  10. Нажимаем Далее
  11. Проверяем данные в расписании работы и при необходимости корректируем количество часов
  12. Нажмите Заполнить - данные будут сохранены и будет произведено заполнение графика.

Важно! Для пятидневных и шестидневных графиков работы в случае если необходимо вводить расписание работы не укрупненно по часам, а необходимо указывать периоды работы сотрудников, с вводом перерывов на обед, либо прочих перерывов, следует при настройке графика установить флажок "Вести учет ночных часов".

Примеры заполнения сменного графика работы:

  1. Добавляем новый график.
  2. В поле Наименование указываем название графика.
  3. Нажимаем кнопку Заполнить график.
  4. Выбираем способ заполнения графика Настроить вручную.
  5. Нажимаем Далее
  6. В поле Тип графика выбираем значение "Сменный".
  7. Устанавливаем флажок в поле Суммированный учет рабочего времени, а также при необходимости Вести учет ночных часов
  8. Снимаем флажок Учитывать праздничные дни
  9. Нажимаем Далее
  10. При необходимости указываем время начала дневных и ночных часов.
  11. Заполняем поле Начать с: датой отсчета (первый день графика).
  12. В табличной части Смены необходимо указать рабочие смены на каждый день цикла (1 день смены соответствует дате отсчета). Нерабочие дни цикла также следует вносить в табличную часть, но при этом не заполнять в эти дни смены.
  13. Нажмите Заполнить - данные будут сохранены и будет произведено заполнение графика
