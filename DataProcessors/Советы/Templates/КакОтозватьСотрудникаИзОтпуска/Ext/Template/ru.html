Как отозвать сотрудника из отпуска

  1. Документ Отзывы из отпусков организации (в интерфейсе кадровый учет организаций - меню Кадровый учет - Отпуска - Отзыв из отпусков организации кнопка Добавить) предназначен для отражения досрочного отзыва сотрудника из отпуска.
  2. В графе От указывается дата составления документа. В табличной части Сотрудники указывается сотрудник, который будет отозван из отпуска.
  3. Далее указывается дата отзыва сотрудника из отпуска. Если при досрочном отзыве из трудового отпуска за неиспользованные дни отпуска предоставляется компенсация, то следует установить флажок в колонке «Предоставить компенсацию».
  4. Если компенсация предоставляется за все неиспользованные дни отпуска, то флажок необходимо поставить в строке табличного поля Сотрудники, иначе - отметить в табличном поле «Отзыв из трудовых отпусков» те строки отпуска, которые следует компенсировать. Некомпенсированные виды отпусков будут предоставлены в следующий раз.
  5. Для того, чтобы оформить начисление отпуска необходимо оформить документ Начисление отпуска сотрудникам организации.
  6. Провести документ - кнопка ОK.
