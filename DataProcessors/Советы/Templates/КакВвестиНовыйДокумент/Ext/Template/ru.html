Как ввести новый документ

Существуют следующие способы ввода новых документов:

1) Ввести новый документ.

2) Ввести документ копированием.

3) Ввести на основании.

 

1) Ввести новый документ:

  1. Выберите в главном меню нужный вид документов (например: Расчет зарплаты - Начисление зарплаты работникам организации).
  2. В открывшемся окне новый документ можно добавить одним из следующих способов:
    • Способ 1: Выберите меню Действие – Добавить.
    • Способ 2: Нажмите кнопку Добавить.
    • Способ 3: Нажмите в любой строке Списка(журнала) документов правой кнопкой мыши – Добавить.
    • Способ 4: Нажмите на клавиатуре клавишу Insert.

Откроется новая форма документа, готовая к заполнению.

3. Заполните поля в документе.

4. Нажмите конпку OK - документ будет проведен, форма документа закроется.

2) Ввести документ копированием:

  1. Выберите в главном меню нужный вид документов (например: Расчет зарплаты - Начисление зарплаты работникам организации).
  2. В открывшемся окне со списком (журналом) документов установите курсор на документ, который будет скопирован.
  3. Затем:
    • Способ 1: в меню выберите Действия – Скопировать.
    • Способ 2: нажмите кнопку Скопировать.
    • Способ 3: нажмите правой кнопкой мыши на том документе, который хотите скопировать, из появившегося меню выберите пункт Скопировать.
  4. Откроется форма документа с заполненными полями, доступными для редактирования.
  5. Заполните поля в документе.
  6. Нажмите конпку OK - документ будет проведен, форма документа закроется.

3) Ввести на основании:

  1. Выберите в главном меню нужный вид документов (например: Кадровый учет организации - Кадровый учет - Прием на работу в организацию).
  2. В открывшемся списке (журнале) документов выберите документ-основание.
  3. Нажмите правой кнопкой мыши на документе-основании и выберите пункт На основании. Откроется список всех документов, которые можно ввести на основании выбранного. Или нажмите кнопку На основании в меню открывшегося окна и выберите нужный документ..
  4. Откроется форма документа, в котором реквизиты будут заполнены информацией, скопированной из документа-основания.
  5. Заполните поля в документе.
  6. Проведите документ и закройте форму – кнопка OK.
