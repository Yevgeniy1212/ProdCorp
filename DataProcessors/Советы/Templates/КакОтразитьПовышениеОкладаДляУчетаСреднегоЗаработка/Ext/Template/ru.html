Как отразить повышение оклада для учета среднего заработка при расчете размера отпускных и больничных

Чтобы отразить коэффициент индексации, который в дальнейшем будет участвовать при расчете среднего заработка можно использовать два способа - документом Кадровое перемещение организаций (в этом случае коэффициент будет рассчитан автоматически) и документом Ввод коэффициентов индексации (для внесения коэффициента вручную). Рассмотрим оба примера:

Первый способ - создайте новый документ Кадровое перемещение организаций (в интерфейсе кадровый учет организаций - меню Кадровый учет - Кадровые перемещения организаций - кнопка Добавить).

  • Если кадровые изменения связаны с повышением заработка, которое в дальнейшем будет учитываться при расчете средней заработной платы в документе обязательно следует УСТАНОВИТЬ признак Индексация заработка с коэффициентом.
  • Если изменение величины оклада сотрудника связано, например, с замещением временно отсутствующего сотрудника и не является повышением его основного оклада, устанавливать флажок Индексация заработка с коэффициентом НЕ НУЖНО.
  • После проведения документа, коэффициент индексации посмотреть в регистре сведений Коэффициенты индексации заработка (в интерфейсе расчет зарплаты организаций - меню Расчет зарплаты - Коэффициенты индексации заработка).
  • Если необходимо выполнить операцию повышения оклада сразу нескольким сотрудникам организации на один и тот же коэффициент, значение коэффициента повышения можно указать в поле рядом с флажком Индексация заработка с коэффициентом. При заполнении табличной части документа у сотрудников, которые были добавлены на закладке Сотрудники, их текущие оклады на закладке Начисления будут автоматически увеличены на заданный размер.

    • Второй способ - когда коэффициент индексации заносится вручную с помощью документа Ввод коэффициентов индексации (в интерфейсе расчет зарплаты организаций - меню Расчет зарплаты - Ввод коэффициентов индексации заработка - кнопка Добавить).

  • В новом документе указать дату документа и заполнить поле Месяц.
  • По кнопке Автозаполнение, в документ можно занести тех сотрудников к которым нужно будет применить коэффициент индексации заработка. После того как табличная часть будет заполнена сотрудниками, в поле Коэффициент табличной части документа вручную можно будет указать нужный коэффициент индексации, который впоследствии будет применяться для расчета средней зарплаты.
