Как внести новый вид расчета

Все начисления регламентированной зарплаты подразделены на две группы: основные и дополнительные начисления.

  1. Список начислений доступен через меню - в интерфейсе расчет зарплаты организаций - меню Предприятие - Настройка расчета зарплаты - Основные начисления (Дополнительные начисления).
  2. Для того, чтобы внести новый вид расчета - необходимо нажать на кнопку Добавить. Если есть необходимость просмотреть имеющиеся виды начисления, то для этого достаточно щелкнуть мышью по одному из видов начисления.
  3. Если мы вносим новый вид расчета, то ему необходимо присвоить наименование и код расчета.
  4. На закладке Расчеты обязательно нужно указать последовательность расчета. Последовательность определяется исходя из того - будет ли данный вид расчета зависеть от результатов других начислений. Если он не зависит от результатов других начислений, то ему присваивается первичное начисление (например: вид начисления "Оклад по дням" - ему будет присвоена первичная очередность, т.к. для расчета данного вида начисления не нужен результат других начислений. Для вида начисления "Премия" будет установлена очередность - зависимое первого уровня, потому что для её расчета необходимо будет указать показатели (на закладке "Прочие" в разделе Базовые и Ведущие расчеты), от которых в дальнейшем будет зависеть результат расчета.
  5. На закладке Расчеты также необходимо указать Способ расчета, т.к. именно от способа расчета будет зависеть по какой формуле будет рассчитан данный вид начисления. Существует два варианта ввода способа расчета: Регламентированный (предопределенный в программе список формул расчета), Произвольная формула расчета (ввод формулы расчета может быть осуществлен при помощи Редактора формул расчета). Произвольные формулы расчета применяются в том случае, если не подходит ни один из регламентированных способов расчета. 
    Если установить флажок Рассчитывать результат методом "от обратного", то при расчете результат будет увеличен на сумму компенсации удержания. Компенсируемые виды удержаний (которые необходимо компенсировать расчетом "от обратного") указываются на вкладке Расчет от обратного.
  6. На закладке Время (у дополнительных начислений данная закладка отсутствует) обязательно для заполнения поле Учет времени. Заполнение этого признака будет влиять на корректное формирование табеля учета рабочего времени. Признак Засчитывается как отработанное при предоставлении вычета обязателен для основного вида начисления (например для "Оклада по дням"). На основании этого признака будет рассчитываться общий фонд рабочего времени. Признак Засчитывается в норму времени устанавливается тем видам начислений, которые будут считаться при расчете, как входящие в общую норму времени (например: "оклад по дням" и "оплата отпуска" - признак может быть установлен как для одного вида начисления, так и для другого, но общее время по всем видам начислений за месяц не должно превышать норму по графику, которую работник должен отработать). Признак Засчитывается как отработанное время при предоставлении вычета будет учитываться при подсчете отработанного времени для предоставления вычета по ИПН, согласно п. 1 ст. 152 НК РК.
  7. На закладке Бухгалтерский и налоговый учет обязательным для заполнения является группа настроек Учет налогообложения, т.е. здесь указывается какими видами налогов будет облагаться данный вид начисления. Обязательным для заполнения является также раздел Декларация по социальному налогу, т.к. от этого напрямую зависит корректное формирование 600 формы. Данный реквизит нужно заполнить по кнопке История, в появившемся окне указать период с которого начнется действие данного реквизита и соответственно код строки декларации. Поле Отражение в учете выбирается из справочника Способы отражения зарплаты в регламентированном учете и заполняется в том случае, если данная корреспонденция счетов будет применяться ко всем сотрудникам.
  8. Закладка Статистический учет предназначено для корректного формирования форм статистической отчетности.
  9. На закладке Прочее указываются Базовые и Ведущие расчеты (данные виды расчетов устанавливаются только в случае, когда нужно указать от каких показателей будет зависеть размер данного начисления), а также настраиваются вытесняющие расчеты (это те расчеты, которые при их проведении будут вытеснять данный вид начисления).
