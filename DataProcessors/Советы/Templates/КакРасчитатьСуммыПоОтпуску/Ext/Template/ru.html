Как начислить суммы отпускных работнику организации

  1. Введите новый документ Начисление зарплаты (Зарплата – Начисление зарплаты – кнопка Добавить).
  2. В поле Месяц начисления укажите месяц начисления суммы отпускных.
  3. Нажмите на кнопку Рассчитать и выберите значение Рассчитать отпуск.

В окне обработки Расчет отпуска заполните поля:

  1. В поле Физ лицо выберите работника организации, по которому будет рассчитываться сумма отпускных.
  2. В поле Период С.. По.. укажите период нахождения работника организации в отпуске.
  3. Заполните табличную часть, выберите вид отпуска (например, Основной отпуск, Отпуск за вредные условия труда и т.д.), для этого из плана вида расчетов Начисления организации выберите виды начисления, у которых в поле Способ расчета установлено значение – По среднему заработку для отпуска.
  4. Для вида расчета (вида отпуска) укажите Дату начала и Дату окончания.
    Важно!!! Количество дней в табличной части Отпуска должны совпадать с количеством в поле Кален. дни.
  5. Для расчета суммы размера отпускных и размера среднего заработка нажмите на кнопку Рассчитать\Рассчитать.
  6. Для получения печатной формы по расчету среднего заработка нажмите на кнопку Печать.
  7. Закройте форму обработки Расчет отпуска – кнопка ОК.
  8. В документе Начисление зарплаты для корректного отражения суммы отпускных по бухгалтерскому и налоговому учету заполните поле Отражение в бухучете.
  9. Проведите документ и закройте форму – кнопка OK.
Примечание: Документ Начисление зарплаты не отражает информацию на счетах бухгалтерского учета, но разносит ее по соответствующим регистрам накопления.
