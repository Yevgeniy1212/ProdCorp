Как проверить расчет налогов по сотруднику

Если есть необходимость проверить правильность расчета налогов по сотруднику, то для этого можно воспользоваться отчетом Расчет по исчислению и удержанию индивидуального подоходного налога и исчислению социального налога (меню Отчеты - Регламентированные отчеты - папка Прочие - Расчет по исчислению и удержанию ИПН и исчислению СН).

  1. В появившемся окне выбрать организацию и период составления отчета.
  2. По кнопке ОК - откроется отчет.
  3. После того, как откроется отчет, необходимо выбрать период за который будет проверяться расчет налогов, а также выбрать из справочника Физические лица проверяемого сотрудника.
  4. Для того, чтобы автоматически заполнить отчет необходимо нажать на кнопку Заполнить.

Просмотреть расчет налогов по сотруднику можно с помощью справки-расчета "Индивидуальный подоходный налог" (меню Отчеты - Справка-расчет "Индивидуальный подоходный налог").

  1. В появившемся окне указать период составления справки-расчета в поле Период с:.
  2. По кнопке Отбор можно будет сформировать справку на выбор по Налогоплательщику, налоговому комитету или же по конкретному работнику - если же не один из этих параметров не будет заполнен, то справка-расчет заполниться данными по всем работникам организации.
  3. По кнопка Настройка имеется возможность настроить выводимые на экран параметры.
  4. Заполнить справку-расчет - по кнопке Сформировать.
