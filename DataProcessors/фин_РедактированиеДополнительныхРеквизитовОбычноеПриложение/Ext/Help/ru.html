Обработка "Редактирование дополнительных реквизитов"

Предназначена для редактирования дополнительных реквизитов выбранного документа, не может быть открыта в самостоятельном режиме.

В данном окне отображается список дополнительных реквизитов. В списке выбираются нужные значения для каждого реквизита. Измененные значения передаются в документ только при нажатии кнопки «Сохранить». При помощи гиперссылки «Редактировать список дополнительных реквизитов» открывается соответствующий элемент справочника «Виды документов» для редактирования списка дополнительных реквизитов.

В группе «Отражение документа в бюджетах» указывается признак интерактивного отражения и используемая схема (способ) отражения в бюджетах с учетом версионности на дату документа.
