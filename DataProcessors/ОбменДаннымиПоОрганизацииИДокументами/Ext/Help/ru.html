Обработка "Выгрузка/загрузка документов"

Обработка предназначена для выгрузки и загрузки отдельных документов. Например, данной обработкой можно воспользоваться для ввода на основании документов в узле информационной базы, когда документы, являющиеся основанием не могут быть выгружены в этот узел из-за использования плана обмена "По организации" (с фильтрацией документов при выгрузке).

 

На странице "Выгрузка документов" выполняется выгрузка документов в файл формата XML. Для выполнения выгрузки необходимо:

  • указать файл выгрузки, период выгрузки, и узел, для которого выгружаются документы (узел выбирается из плана обмена "По организации");

     

  • в списке "Виды выгружаемых документов" указать нужные виды документов (если список видов документов пуст, воспользуйтесь кнопкой "Заполнить");

  • при необходимости выбрать значения отбора по организации и(или) по контрагенту;

     

  • заполнить список выгружаемых документов (кнопка "Заполнить");

  • в списке "Выгружаемые документы" указать нужные документы и нажать кнопку "Выгрузить".

     

    На странице "Загрузка документов" выполняется загрузка документов из файла в формате XML. Загруженные документы можно просмотреть и ввести на их основании документы, которые можно сохранить в рабочей информационной базе. Для выполнения загрузки (и в дальнейшем ввода на основании) необходимо:

  • указать файл загрузки документов;

  • нажать кнопку "Загрузить" (появившиеся в списке документы открываются для просмотра двойным щелчком мыши).

     

    Для ввода документов на основании загруженных документов, необходимо:

  • указать какие действия необходимо выполнять с создаваемыми документами;

  • отметить в списке загруженных документов нужные документы;

  • нажать кнопку "Ввести на основании".

     

    Некоторые особенности:

  • перед выгрузкой/загрузкой документов желательно выполнять обмен данными, чтобы изменения конфигурации(если они были) и изменения данных(если они были) перенеслись в узлы-получатели документов;

  • выгрузка/загрузка документов допустима в пределах одной конфигурации;

  • в типовой конфигурации ввод на основани обрабатывается только для документов "Передача ОС", "Реализация ТМЗ и услуг", "Расходный кассовый ордер" и "Платежное поручение исходящее".
