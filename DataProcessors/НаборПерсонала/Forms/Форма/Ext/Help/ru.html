Форма инструмента «Набор персонала»

Инструмент содержит три закладки:

  • на закладке «Кандидаты» содержится таблица со списком кандидатов, справочная информация по кандидату, отмеченному в таблице, и доступ к средствам для работы с кандидатами; в список по умолчанию выводятся кандидаты, по которым решение еще не принято (статус – «принят к рассмотрению»), а также те, кому уже предлагали работу (статус – «предложили работать»),
  • на закладке «Собеседования» представлен список назначенных собеседований,
  • закладка «Текущие вакансии» содержит отчет о текущем количестве вакансий.

Работа с кандидатами

На закладке два поля: слева таблица со списком кандидатов, справа справочная информация по отмеченному в списке кандидату. При запуске инструмента в список попадают кандидаты только со статусами «принят к рассмотрению» и «предложили работать».

Над списком кандидатов расположены кнопки, осуществляющие следующие действия:

  • ввести новых кандидатов  - отображает форму для ввода новых кандидатов, которая позволяет быстро вводить новых кандидатов и обладает рядом полезных функций,
  • получить справку  - формирует справку для отмеченного в списке кандидата. Справка автоматически обновляется при выделении очередного кандидата,
  • подробные данные кандидата  - отображает карточку с личными данными из справочника физических лиц для отмеченного в списке кандидата,
  • назначить собеседование  – отображает форму назначения собеседования для отмеченного в списке кандидата. Эта форма позволяет запланировать собеседование и при необходимости зарезервировать помещения и время участников, 
  • меню Изменить статус кандидата позволяет быстро изменить статус кандидата: предложить работу, отложить рассмотрение или отклонить кандидата, при этом вводится документ «Оценка кандидата»,
  • принять на работу – создает, и заполняет документ «Прием на работу» для отмеченного в списке кандидата. Если ранее в предложениях работы этому кандидату указывались конкретные рабочие места (должности и подразделения), менеджер может выбрать одно из таких предложений,
  • обновить данные о кандидатах – актуализирует таблицу со списком кандидатов,
  • отбор – отображает форму, в которой указываются статусы, по которым формируется список кандидатов основной формы инструмента.

 

В интерактивно обновляющейся справке отражается следующая информация по кандидату:

  • текущий статус кандидата; даты регистрации и последней смены статуса,
  • контактная информация,
  • сведения о собеседованиях назначенных кандидату,
  • список документов, менявших статус кандидата.

Работа со списком назначенных собеседований

Работа со списком назначенных собеседований осуществляется на закладке «Собеседования».

В списке документов «Регистрация собеседования кандидата» выведены следующие данные о собеседовании: флажок, указывающий на факт состоявшегося собеседования; дата и время собеседования, ФИО кандидата и автора документа.

Анализ плана по набору персонала

Работа с планом по набору персонала осуществляется на закладке «Текущие вакансии», которая содержит отчет по существующим вакансиям (строится нажатием кнопки «Сформировать»).

В отчете показывается количество свободных вакансий, а также  плановое и фактическое количество работников; информация в отчете группируется по должностям и подразделениям.
