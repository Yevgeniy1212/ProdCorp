Форма «Особенности текущей версии подсистемы «Регламентированная отчетность»»

Форма «Особенности текущей версии подсистемы «Регламентированная отчетность»» предназначена для отображения информации об особенностях регламентированной отчетности в текущей версии конфигурации. Формы автоматически выводится на экран при первом после обновления релиза конфигурации открытии формы «Регламентированная и финансовая отчетность».

Флажок «не открывать это окно автоматически» на нижней командной панели формы «Особенности текущей версии подсистемы «Регламентированная отчетность»» позволяет запретить автоматическое открытие этой формы при каждом открытии формы «Регламентированная и финансовая отчетность». По умолчанию флажок установлен.

Для вызова формы «Особенности текущей версии подсистемы «Регламентированная отчетность»» можно использовать кнопку «Информация для пользователей» в верхней командной панели формы «Регламентированная и финансовая отчетность».
