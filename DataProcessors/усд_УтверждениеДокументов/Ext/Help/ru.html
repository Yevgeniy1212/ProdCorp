Обработка "Утверждение документов"

Процесс согласования происходит в обработке Утверждение документа, открываемой из монитора согласования документа, из панели задач согласования или из пунктов меню.

В списке «Реквизиты, подлежащие проверке», выводится список контролируемых реквизитов (задается для этапа при настройке маршрута) с текущим их значением. Пользователь имеет возможность изменить их значение и сохранить результаты изменений в документ при нажатии кнопки Сохранить новые значения реквизитов командной панели таблицы значений.

Поле «Исполнитель» заполняется только при наличии потребности выбрать конкретного исполнителя следующего этапа, если это разрешено в настройках этапа.

Поле «Приоритет» (если включена приоритетизация в настройках системы) позволяет указать приоритет, с которым документ пойдет на следующий этап.

Работа с кнопками командной панели (состав зависит от используемых параметров учета и прав пользователя):

·         Утвердить – поставить подпись. При этом документ либо остается ждать прочих подписей, либо переходит на следующий этап (в зависимости от настроек этапа и полученных подписей);

·         Вернуть – отправить документ на этап возврата для доработки, с указанием причины во всплывающем окне;

·         Отклонить – завершить работу с документом на маршруте, указав причину во всплывающем окне;

·         Переадресовать – переадресовать документ другому исполнителю;

·         Вложения – работать со списком прикрепленных файлов;

·         Создать заметку – открыть список заметок по документу;

·         Версии – открыть форму работы с версиями документа;

·         Карта – открывает карту состояния документа на маршрутах;

·         Форма информации – открывает отчет по предыдущему, текущему, следующему этапам.

Действия по согласованию доступны пользователю только если он имеет право подписи на текущем этапе согласования.

При групповом согласовании документов открывается форма, подобная форме индивидуального согласования.

В этой форме документы отображаются в списке «Перечень документов», а их контролируемые на этапе реквизиты – в колонках этого списка. В эту форму могут попасть только документы одного вида и находящиеся на одном этапе. Установка значений контролируемых реквизитов осуществляется пакетно, при помощи команды «Заполнить реквизиты». Применяется только к отмеченным документам.

Действия по утверждению/отклонению/переадресации применяются ко всем отмеченным документам сразу.
