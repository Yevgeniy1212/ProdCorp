Данная форма предназначена для выгрузки справочника основных средств на ТСД (терминал сбора данных) и печати этикеток основных средств.

Также форма позволяет настроить обмен данными между 1С и ТСД.

Настройка подключения ТСД

Возможны два режима обмена данными:

1) Напрямую с ТСД. Для выбора этого режима нужно установить переключатель ТСД на форме;

2) Через папку обмена (этот вариант используется, если на компьютере, к которому подключен ТСД, работа с 1С происходит через терминальную сессию, при этом используется специальная утилита обмена данными см. http://cleverence.ru/site.aspx?page=MobileSmarts-1C-Driver-Sync). Для выбора этого режима нужно установить переключатель Папка обмена на форме и указать папку в поле выбора.

Выгрузка основных средств

Для отбора выгружаемых основных средств служит таблица Отбор, в которую по кнопке Добавить можно добавить условие отбора, если условия не заданы в выборку попадут все основные средства.

По нажатию кнопки Заполнить список согласно отбору заполняется таблица Основные средства.

Для выгрузки необходимо нажать кнопку Выгрузить в нижней панели формы.

Печать этикеток основных средств

Для того, чтобы распечатать этикетки основных средств нужно сначала заполнить таблицу Основные средства, для этого следует нажать кнопку Заполнить список согласно отбору, задав предварительно условие отбора (если требуется). Затем следует нажать кнопку Распечатать этикетки.

Для редактирования шаблона этикетки следует выбрать пункт Открыть редактор этикетки в меню Настройки печати. При этом откроется окно редактора этикетки, с помощью которого можно отредактировать внешний вид этикетки, задать поля данных для печати.

Пункт меню Распечатать этикетки с предпросмотром служит для печати с предварительным просмотром выводимых на печать страниц.
