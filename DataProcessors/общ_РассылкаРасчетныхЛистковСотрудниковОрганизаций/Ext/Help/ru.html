Документ «Рассылка анкет»

Документом «Рассылка анкет» предоставляет возможность упростить рассылку по электронной почте. Список документов рассылки анкет вызывается из меню «Анкетирование» или кнопкой «Рассылка» из формы редактирования анкеты.

В форме рассылки обязательно указывается электронные адреса ответов (поле «Е-mail ответов») и получателей письма (на закладке «Получатели»). После нажатия на кнопку «Отправить» будет сформировано столько писем, сколько указано получателей. Для каждого письма будет автоматически сформировано вложение, представляющее собой HTML-вариант анкеты (файл с именем anketa???.ank), после чего письма будут отправлены.

На закладке «Вложения» можно указать список дополнительных файлов, которые будут автоматически включены в рассылаемые письма.

Документ рассылки может быть двух типов: первичная рассылка и повторная рассылка. Часто бывает так, что по каким-либо причинам следует повторно разослать ту же самую анкету, но уже не всем адресатам, а только некоторым. Для решения таких задач следует воспользоваться кнопкой «Повторная рассылка». В появившемся документе повторной рассылки следует удалить тех адресатов, которые не должны получить анкету и отправить им электронные письма.

Документ рассылки предназначен также для контроля ответов на разосланную анкету. Документ можно вводить даже когда вы не собираетесь рассылать анкету по электронной почте.
