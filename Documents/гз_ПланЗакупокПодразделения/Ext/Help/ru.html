Документ «ГЗ: План закупок подразделения»

Конфигурация "1С-Рейтинг: Управление государственными закупками"

Документ предназначен для регистрации планов закупок подразделений в рамках программы государственных закупок.

Документ имеет два вида операции:

  • Планирование - план закупок подразделения на приобретение товаров, работ и услуг;
  • Корректировка - корректировка плана закупок подразделения.

Реквизиты шапки документа

  • Организация - организация, которая производит государственные закупки;
  • Подразделение - подразделение организации, подающее план закупок;
  • Место поставки - место поставки товаров, оказания услуг, выполнения работ;
  • Год - год плана закупок подразделения;
  • Состояние - состояние документа, документ может быть проведен только если документ имеет состояние "Утвержден".

Табличная часть Товары, работы и услуги используется для регистрации плана закупок подразделения на приобретение товаров, работ и услуг.

Загрузить из Excel – по нажатию кнопки открывается обработка Загрузка данных из табличного документа, предназначенная для загрузки данных в табличную часть Товары, работы и услуги.

Документ используется при формировании годового плана государственных закупок, а также при формировании документов Заказ подразделения.
