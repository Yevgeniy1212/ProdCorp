Документ «Установка долей распределения по проектам»

Для денежных средств можно задать способ распределения по проектам. Способ распределения устанавливается в справочнике «Виды распределения по проектам». В шапке документа выбирается «Вид распределения». Это элемент справочника «Виды распределения по проектам».

В табличной части указываются проекты и доля распределения от общего объема.

 

При проведении документа данные записываются в регистр сведений «Распределение по проектам». Далее при проведении документов, которыми отражаются движения денежных средств, если распределение по проектам должно быть выполнено при проведении документа, то на основании данных регистра сведений «Распределение по проектам» производится распределение денежных средств по проектам и формируются движения с указанными проектами по результатам распределения.
