Документ «Регистрация договоров»

Документ предназначен для регистрации в системе договоров и дополнительных соглашений по товарным и нетоварным операциям. На основании этого документа могут быть введены следующие виды документов: Поступление товаров и услуг, Передача ОС, Реализация товаров и услуг, Счет на оплату покупателю, Доверенность.

Вид операции, которая будет произведена в результате проведения документа, определяется непосредственно в самой форме документа с помощью кнопки "Операция". При выборе вида операции автоматически формируется и соответствующее название документа. В любой момент времени можно изменить вид операции (даже для проведенного документа) и конфигурация корректно пересчитает все данные в информационной базе.

  • Для вида операции доступны следующие значения:

    • Договор
    • Дополнительное соглашение

     

    Регистрация нового договора осуществляется документом, в котором установлен вид операции  "Договор".

    Изменение параметров договора, значений реквизитов, специфики осуществляется документом, в котором установлен вид операции  "Дополнительное соглашение".
