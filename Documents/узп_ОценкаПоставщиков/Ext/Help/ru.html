Документ "Оценка поставщиков"

Документ "Оценка поставщиков" предназначен для установки оценок поставщикам по указанным критериям и шкалам оценки поставщиков.

В табличной части на закладке "Оценка" в графе "Критерий" отразятся все введенные в систему критерии, тогда как в расчете будут участвовать только те критерии, которые были указаны в выбранной схеме расчета для лота.

В графе "Значение критерия" для каждого критерия устанавливаются значения, соответствующее данному поставщику. Бывают случаи, когда поставщик уже принимал участие в закупках такого характера, и для него были указаны значения критериев. Тогда в документе "Оценка поставщика" после выбора контрагента в поле "Поставщик", система выдаст информационное сообщение, по которому можно установить последние значения, указанные для данного поставщика, либо не устанавливать.

Исключение составляют два критерия: "Срок деятельности" и "Цена".

Значение критерия "Срок деятельности" заполняется автоматически для потенциального поставщика на основании данных из регистра сведений "Дополнительные характеристики контрагентов".

Значение критерия "Цена" не отображается в табличной части "Оценка", он учитывается при формировании результата выбора поставщиков и рассчитывается в следующем порядке:

1) Определяется максимальная и минимальная цена поставщиков для  номенклатурной позиции.

2) Поставщик с максимальной ценой получает оценку 0, поставщик с минимальной ценой получает оценку 5, остальные поставщики получают оценки от 0,1 до 4,9 в соответствии с предоставленной ими цены.

Далее баллы по всем критериям рассчитываются автоматически в соответствии с настройками критериев.
