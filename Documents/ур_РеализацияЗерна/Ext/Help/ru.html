Документ "Реализация зерна"

Документ «Реализация товаров и услуг» служит для отражения операций по реализации зерна.  На основании этого документа могут быть введены следующие виды документов: Счет-фактура выданный, Приходный кассовый ордер, Платежный ордер (поступление денежных средств), Платежное поручение (входящее), Возврат ТМЗ от покупателя 

Реализация зерна осуществляется документов, в котором установлен вид операции  "реализация".

В состав документа входят:

  • Организация
  • Источник финансирования - источник финансирования
  • Тип операции - тип операции реализации
  • Склад - склад, с которого будет списано продаваемое зерно
  • Контрагент - покупатель зерна
  • Договор контрагента - договор, по которому осуществляется продажа
  • Валюта документа - валюта, в которой оформляется реализация зерна. Всегда совпадает с валютой договора с контрагентом
  • Курс взаиморасчетов - курс валюты взаиморасчетов
  • Кратность взаиморасчетов - кратность валюты взаиморасчетов
  • Тип цен - тип цены, по которой производится реализация
  • Адрес доставки 
  • Банковский счет организации
  • Учитывать НДС
  • СуммаВключает НДС
  • Сумма документа - общая сумма документа по итогам всех табличных частей
  • Счет учета расчетов с контрагентом - счет, на который будут отражены проводки по учету взаиморасчетов
  • Счет учета расчетов по авансам - в случае, если данный реквизит заполнен, будет проводится зачет авансов выданных организацией 
  • Сделка - расчетный документ для учета взаиморасчетов
  • Ответственный
  • Комментарий
  • Табличная часть "Зерно" - служит для отражения операций по реализации зерна. Состав:
    • Номенклатура - товары и материалы, реализуемые данным документом
    • Единица измерения - единица измерения номенклатуры
    • Коэффициент - коэффициент пересчета количества номенклатуры в базовые единицы
    • Количество
    • Цена
    • Сумма
    • Ставка НДС
    • Сумма НДС
    • Счет учета (БУ) - счет учета запасов, с которого будет списана себестоимость реализуемой номенклатуры
    • Счет учета НДС по реализации (БУ) - счет учета для отражения суммы НДС по реализации для бухгалтерских проводок
    • Счет доходов (БУ) - счет учета для отражения суммы дохода от реализации
    • Субконто доходов (БУ) - аналитика счета доходов 
    • Счет списания себестоимости (БУ) - кор. счет списания себестоимости ТМЗ 
    • Субконто списания себестоимости (БУ) - аналитика счета списания себестоимости 
    • Вид операции для НДС реализации - аналитика счета НДС для регистра "НДС"
  • Печатные формы:
    • Акт выполненных работ
    • Расходная накладная

 

Бухгалтерский учет

Документ формирует проводки по бухгалтерскому учету. При проведении формируются проводки по списанию себестоимости зерна со счетов учета, указанные в табличной части "Зерно", отражению дохода от реализации товаров и услуг, формируется дебиторская задолженность на счетах учета расчетов с контрагентами и счетах учета расчетов по авансам, кроме того формируются проводки по отражению НДС по реализации.

Одновременно с проводками по бухгалтерскому учету НДС производятся записи в регистре накопления "НДС" на суммы НДС, указанные в документе.

Одновременно с проводками по бухгалтерскому учету ТМЗ производятся записи в регистре накопления "Зерно на складах" на количество, указанное в документе
