Документ «Отзыв из отпусков организаций»

Документ предназначен для отражения досрочного отзыва сотрудника из отпуска. Если сотрудник выходит после отпуска на работу в запланированную дату, то его возврат следует оформлять документом "Возврат на работу организации".

При отзыве из трудовых отпусков кроме указания даты отзыва необходимо также указать виды трудовых отпусков, которые не были использованы сотрудником, в табличном поле "Отзыв из трудовых отпусков". Для автоматического определения видов отпусков предназначена кнопка "Рассчитать".

Если при досрочном отзыве из трудового отпуска за неиспользованные дни отпуска предоставляется компенсация, то следует отметить флажок "Предоставить компенсацию". Если компенсация предоставляется за все неиспользованные дни отпуска, то флажок необходимо поставить в строке табличного поля Сотрудники, иначе - отметить в табличном поле "Отзыв из трудовых отпусков" те строки отпуска, которые следует компенсировать. Некомпенсированные виды отпусков будут предоставлены в следующий раз.

По документу можно получить две печатные формы – на каждого сотрудника или на список сотрудников из документа.

Внимание! Для сотрудников, работающих в организации в том числе и по внутреннему совместительству, отзыв из отпуска регистрируется отдельно для каждого места работы.
