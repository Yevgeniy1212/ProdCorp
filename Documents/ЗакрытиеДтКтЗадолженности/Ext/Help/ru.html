Документ «Закрытие дт/кт задолженности»

Документ предназначен для проведения перекрытия дебиторской/кредиторской задолженности по счетам взаиморасчетов.

Табличная часть «Закрытие задолженности»

В табличной части указываются контрагенты, договора (и расчетные документы), по которым существует остаток по счетам расчетов с контрагентами, указанными в шапке документа при совпадении валюты договора по счету дебета и валюты договора по счету кредита, сумма и валютная сумма перекрываемой задолженности.

Табличная часть может быть заполнена как вручную, так и автоматически (с помощью кнопки «Заполнить»). При автоматическом заполнении производится анализ остатков по счетам бухгалтерского учета, которые указываются в шапке документа, на дату документа.

Документ формирует проводки по счетам бухгалтерского учета.

Документ не имеет печатной формы.
