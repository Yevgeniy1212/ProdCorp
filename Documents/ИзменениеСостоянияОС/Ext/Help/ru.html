Документ «Изменение состояния ОС»

Документ предназначен для изменения состояния основного средства по бухгалтерскому учету.

Также данным документом можно изменить состояние начисления амортизации и признак налогообложения основных средств имущественным, земельным, транспортным налогом.

Для изменения состояния начисления амортизации или состояния налогообложения имущественным, земельным, транспортным налогом необходимо установить соответствующий признак в настройках документа, которая открывается по кнопке Настройка командной панели. Затем в табличной части для каждого объекта основных средств нужно указать изменяемое состояние (признак).

Данный документ не формирует никаких проводок, он лишь изменяет значение соответствующих ресурсов регистров сведений, хранящих информацию об основных средствах.

Для быстрого заполнения табличной части однотипными объектами основных средств, имеющими одинаковые наименования, нужно ввести в табличную часть хотя бы один такой объект. Затем следует воспользоваться кнопкой Заполнить в командной панели табличной части. При выборе в выпадающем меню пункта По наименованию табличная часть будет заполнена объектами основных средств, имеющими такое же наименование, как у введенного первоначально.

Для заполнения табличной части основными средствами, отобранными по каким-либо критериям, можно воспользоваться специализированной формой, который вызывается при помощи пункта Заполнить списком ОС или Добавить из списка ОС в меню Заполнить.

Предусмотрена возможность открытия настроек при вводе нового документа. Чтобы воспользоваться данной возможностью необходимо в настройках пользователя (Сервис - Настройки пользователя) установить признак Запрашивать настройки документа при вводе нового.
