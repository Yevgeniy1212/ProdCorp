Документ "Реализация услуг по переработке"

Документ предназначен для отражения в бухгалтерском и налоговом учете операции оказания услуг по выпуску продукции из давальческого сырья и отражения отчета об использованных при выпуске продукции давальческих материалах.

Общие реквизиты документа

  • Контрагент.  Заказчик переработки сырья. Значение реквизита выбирается из справочника "Контрагенты". Реквизит является обязательным для заполнения.

  • Договор.  Договор с заказчиком переработки сырья с видом "С покупателем".

    Закладка "Продукция (услуги по переработке)"

    На закладке указывается продукция из давальческого сырья, услуги по выпуску которой оформляются данным документом.

  • Номенклатура. Выпущенная продукция из давальческого сырья по стоимости услуг по ее производству. Значение реквизита выбирается из справочника "Номенклатура" и является обязательным для заполнения.
    • Спецификация. Спецификация продукции, если для данной продукции заведена спецификация. Значение реквизита выбирается из справочника "Спецификации номенклатуры". Такая информация позволит автоматически рассчитать состав материалов, необходимый для производства указанной продукции заполнить закладку "Материалы заказчика".

    • Содержание услуги. Дополнительная информация об оказанной услуге.  

    • Количество. Количество выпущенной продукции.
    • Цена. Цена реализации услуги по выпуску продукции.
    • Сумма. Сумма реализации услуги по выпуску продукции.
    • Ставка НДС. 
    • Сумма НДС.
    • Всего.
    • Стоимость (плановая). Плановая цена продукции, произведенной из сырья заказчика.
    • Сумма (плановая). Плановая стоимость продукции, произведенной из сырья заказчика.
    • Счет учета (БУ). Счет бухгалтерского учета производства продукции из давальческого сырья (например, 8120).
    • Счет учета (НУ). Счет учета по налоговому учету.
    • Счет доходов (БУ). Счет бухгалтерского учета доходов от оказания услуг.
    • Субконто доходов (БУ). Объекты аналитики для целей бухгалтерского учета доходов.
    • Счет доходов (НУ). Счет учета доходов от оказания услуг по налоговому учету.
    • Субконто доходов (НУ). Объекты аналитики для целей учета доходов по налоговому учету.
    • Счет списания себестоимости(БУ). Счет бухгалтерского учета расходов по оказанию услуг.
    • Субконто  списания (БУ). Объект аналитики для целей бухгалтерского учета списания себестоимости.
    • Счет списания себестоимости(НУ). Счет учета расходов по оказанию услуг  по налоговому учету.
    • Субконто  списания (БУ). Объекты аналитики для целей списания себестоимости  по налоговому учету.
    • Счет учета НДС по реализации (БУ). Счет учета для отражения суммы НДС по реализации для бухгалтерских проводок.
    • Вид операции для НДС реализации. Аналитика счета НДС для регистра "НДС".

    Предусмотрено заполнение данной табличной части по кнопке "Заполнить". Заполнение производится по отчету производства за смену, который будет указан из предложенного списка и в котором счет учета готовой продукции будет 8120 или по остаткам счета 8120.

    Закладка "Материалы заказчика"

    На закладке указываются материалы, принятые в переработку, которые были использованы при выпуске продукции.

    • Номенклатура. Материал, принятый в переработку. Значение реквизита выбирается из справочника "Номенклатура" и является обязательным для заполнения.

    • Количество. Количество материала.
    • Счет учета (БУ). Счет бухгалтерского учета материала, принятого в переработку и переданного в производство (например, 9022). Значение реквизита выбирается из плана счетов бухгалтерского учета.

    Предусмотрено заполнение данной табличной части по кнопке "Заполнить". Заполнение производится давальческими материалами, которые были переданы в производство и учтены на счете 9022 "Материалы, переданные в производство" или состав материалов можно рассчитать по спецификациям и продукции, указанным на закладке "Продукция и услуги".

    Закладка "Счета учета расчетов"

    На закладке в документе заполняются «Счет учета расчетов с контрагентом» и «Счет учета расчетов по авансам». Эти счета заполняются в соответствии с заданными в регистре сведений "Счета учета расчетов с контрагентами" значениями по умолчанию для каждого контрагента. При необходимости пользователь может изменить корреспонденцию счетов.

    Особенности проведения документа

    По данным табличной части "Продукция и услуги" отражается реализация услуг по переработке по плановым ценам и списание готовой продукции, произведенной из материалов заказчика со счета 8120 по плановым ценам.

    По данным табличной части "Материалы заказчика" отражается списание материалов заказчика со счета 9022, как использованных на производство продукции для заказчика, отчет по которым принят заказчиком.
