Документ "Реализация ТМЗ и услуг"

Документ «Реализация товаров и услуг» служит для отражения операций по реализации товаров и услуг.  На основании этого документа могут быть введены следующие виды документов: Счет-фактура выданный, Приходный кассовый ордер, Платежный ордер (поступление денежных средств), Платежное поручение (входящее), Возврат ТМЗ от покупателя 

Вид операции, которая будет произведена в результате проведения документа, определяется непосредственно в самой форме документа с помощью кнопки "Операция". При выборе вида операции автоматически формируется и соответствующее название документа и при необходимости изменяется состав граф табличной части документа. В любой момент времени можно изменить вид операции (даже для проведенного документа) и конфигурация корректно пересчитает все данные в информационной базе.

Для вида операции доступны следующие значения:

  • реализация
  • передача структурному подразделению

 

Реализация ТМЗ осуществляется документов, в котором установлен вид операции  "реализация".

Передача ТМЗ структурному подразделению осуществляется документом, в котором установлен вид операции  "передача структурному подразделению".

В состав документа входят:

  • Организация
  • Склад - склад, с которого будут списаны продаваемые ТМЗ
  • Контрагент - покупатель ТМЗ и услуг
  • Договор контрагента - договор, по которому осуществляется продажа
  • Валюта документа - валюта, в которой оформляется реализация ТМЗ и услуг. Всегда совпадает с валютой договора с контрагентом
  • Курс взаиморасчетов - курс валюты взаиморасчетов
  • Кратность взаиморасчетов - кратность валюты взаиморасчетов
  • Тип цен - тип цены, по которой производится реализация
  • Адрес доставки 
  • Банковский счет организации
  • Учитывать НДС
  • СуммаВключает НДС
  • Сумма документа - общая сумма документа по итогам всех табличных частей
  • Счет учета расчетов с контрагентом - счет, на который будут отражены проводки по учету взаиморасчетов
  • Счет учета расчетов по авансам - в случае, если данный реквизит заполнен, будет проводится зачет авансов выданных организацией 
  • Сделка - расчетный документ для учета взаиморасчетов
  • Ответственный
  • Комментарий
  • Табличная часть "ТМЗ" - служит для отражения операций по реализации ТМЗ. Состав:
    • Номенклатура - товары и материалы, реализуемые данным документом
    • Единица измерения - единица измерения номенклатуры
    • Коэффициент - коэффициент пересчета количества номенклатуры в базовые единицы
    • Количество
    • Цена
    • Сумма
    • Ставка НДС
    • Сумма НДС
    • Счет учета (БУ) - счет учета запасов, с которого будет списана себестоимость реализуемой номенклатуры
    • Счет учета НДС по реализации (БУ) - счет учета для отражения суммы НДС по реализации для бухгалтерских проводок
    • Счет доходов (БУ) - счет учета для отражения суммы дохода от реализации
    • Субконто доходов (БУ) - аналитика счета доходов 
    • Счет списания себестоимости (БУ) - кор. счет списания себестоимости ТМЗ 
    • Субконто списания себестоимости (БУ) - аналитика счета списания себестоимости 
    • Вид операции для НДС реализации - аналитика счета НДС для регистра "НДС"
  • Табличная часть "Услуги" - служит для отражения операций по оказанию сторонним организациям услуг. Состав:
    • Номенклатура - услуга
    • Содержание - содержание услуги
    • Количество
    • Цена
    • Сумма
    • Ставка НДС
    • Сумма НДС
    • Счет учета НДС по реализации (БУ) - счет учета для отражения суммы НДС по реализации для бухгалтерских проводок
    • Счет доходов (БУ) - счет учета для отражения суммы суммы дохода от реализации
    • Субконто доходов (БУ) - аналитика счета доходов 
    • Вид операции для НДС реализации - аналитика счета НДС для регистра "НДС"
  • Табличная часть "Источники происхождения" - служит для указания источников происхождения у товаров. Заполняется, если в конфигурации используется учет товаров по источникам происхождения. Состав:
    • Номенклатура - товары и материалы из табличной части "ТМЗ"

Бухгалтерский учет

Документ формирует проводки по бухгалтерскому учету. При проведении формируются проводки по списанию себестоимости ТМЗ со счетов учета, указанные в табличной части "Товары", отражению дохода от реализации товаров и услуг, формируется дебиторская задолженность на счетах учета расчетов с контрагентами и счетах учета расчетов по авансам, кроме того формируются проводки по отражению НДС по реализации.

Одновременно с проводками по бухгалтерскому учету НДС производятся записи в регистре накопления "НДС" на суммы НДС, указанные в документе.
