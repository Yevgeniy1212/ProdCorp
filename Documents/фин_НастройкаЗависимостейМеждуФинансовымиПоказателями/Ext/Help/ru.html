Документ «Настройка зависимостей между финансовыми показателями»

В системе разработан механизм, который позволяет настроить перечень финансовых показателей, зависимых от влияющего показателя.

Настройка зависимостей финансовых показателей производится документом «Настройка зависимостей между финансовыми показателями» (меню «Расчеты и зависимости» -«Настройка зависимостей между финансовыми показателями»).

Данный документ имеет два вида операции:

·         Индивидуальная установка зависимостей – предназначен для установки зависимостей между отдельными финансовыми показателями;

·         Групповая установка зависимостей – предназначен для установки зависимости между группами (агрегатами) финансовых показателей, имеющих общую схему зависимостей.

Соответственно, модель зависимости начинает действовать с момента (даты) проведения документа. В шапке документа указываются следующие данные:

·         Организация – доступна для заполнения, если ведется бюджетирование по организациям. Можно не указывать, тогда настроенная зависимость будет действовать для всех организаций;

·         Агрегат показателей / Финансовый показатель – влияющий финансовый показатель или агрегат показателей из справочника «Агрегаты финансовых показателей», при формировании оборотов по которым необходимо рассчитать зависимость;

·         Сценарий – определяет, для плановых данных какого сценария применима данная зависимость. Если сценарий не указан, то зависимость будет применима к данным всех сценариев;

·         Ограничение области действия по разрезам планирования - фильтры на значения измерений бюджетирования влияющих оборотов. Зависимые плановые показатели (обороты) будут сформированы в том случае, если в первичной бюджетной операции значения измерений, по которым заданы ограничения, совпадут с данными фильтров в модели зависимых оборотов;

·         Использование – область использования данной зависимости:

o    Использовать при вводе плановых показателей – зависимость будет использоваться только для расчета плановых показателей;

o    Использовать при вводе как плановых, так и фактических показателей – зависимость будет использоваться также и при формировании фактических данных.  В случае задания зависимости, используемой при исполнении, в документе не может быть указан сценарий, так как такая зависимость должна использоваться для всех сценариев планирования;

Механизм применения зависимостей при учете фактических данных не следует использовать для тех случаев, когда зависимые показатели являются различными по своей сути. Например, существует зависимость между суммой приобретения и суммой оплаты поставщикам, которая используется при планировании бюджета для расчета сумм оплаты в зависимости от сумм закупок. Однако применять данную зависимость и к фактическим данным будет методически некорректно, т.к. по сути приобретение и оплата являются различными показателями, подкрепляемыми отдельными (и, возможно, разнесенными по времени) фактами. Как следствие, суммы и периоды приобретения и оплаты при планировании могут совпадать, а при учете фактических данных - различаться. В результате применении данной зависимости может быть корректным только при планировании, но не при учете фактических данных. В другой ситуации, например, если существует зависимость между суммой реализации товаров и суммой общего дохода (т.е. сумма дохода всегда включает суммы реализации помимо прочих показателей, и при вводе сумм реализации необходимо увеличить и сумму дохода), то такая зависимость может быть корректно применена и при планировании, и при учете фактических данных, так как факт реализации и увеличения доходов будут наступать одновременно.

·         Состояние – гриф утверждения данного документа. Для того, чтобы провести документ, необходимо установить гриф «Утвержден»;

·         Срок действия – дата окончания действия зависимости. Желательно устанавливать дату таким образом, чтобы она соответствовала окончанию бюджетных периодов тех плановых данных, для которых она применяется. Если срок действия не указан, зависимость будет действовать без ограничений по времени.

В табличной части «Зависимости» указываются правила формирования зависимых финансовых показателей.

Таким образом, при планировании некоторых плановых показателей по статье можно автоматически формировать обороты по зависимым финансовым показателям. По такому принципу можно строить целые цепочки зависимостей финансовых показателей.

При указании зависимых статей фиксируются следующие параметры:

·         финансовый показатель / агрегат финн. показателей;

·         значения аналитических разрезов (если их значения известны заранее и отличаются от аналитики влияющей статьи); если значения аналитики не указаны явно, то они подставляются из аналитики влияющей статьи;

·         профиль изменения по периодам;

·         если для зависимого финансового показателя ведется количественный или суммовой учет, то указывается источник получения таких оборотов (или количественные обороты из влияющего показателя или суммовые обороты из влияющего показателя, или же, если для финансового показателя задан источник для расчета цены – по цене источника на основании рассчитанного количества зависимого финансового показателя);

·         для суммовых и количественных плановых показателей можно указать коэффициенты, на которые будут скорректированы соответствующие обороты из влияющего показателя.

Все суммовые плановые показатели (обороты) для зависимых финансовых показателей формируются в валюте влияющей операции. Использование количественных оборотов влияющего финансового показателя  для задания суммовых оборотов в зависимой операции может быть удобно для следующих случаев. Например, количественные обороты влияющей статьи задают нам некий плановый объем реализации. Если для зависимой статьи указать в качестве источника суммовых оборотов количественные обороты по реализации, а в качестве коэффициента — плановую себестоимость продукции, то суммовые плановые показатели (обороты) зависимой статьи будут показывать нам себестоимость реализации планового объема.

Важно!!! Если в реквизитах зависимой статьи описан источник для получения цены, то при использовании механизма зависимых статей оборотов он использоваться не будет, если это не указано напрямую. Суммовые и количественные обороты в таком случае будут браться непосредственно из механизма настройки зависимости финансовых показателей.

В обработке «Структура зависимостей финансовых показателей» можно посмотреть сформированную структуру зависимостей показателей по выбранному сценарию.

Групповая установка зависимостей

Отличие групповой установки зависимостей от индивидуальной установки состоит в том, что в данном документе зависимости задаются не для отдельных финансовых показателей, а для их групп. Такие группы задаются в справочнике «Агрегаты финансовых показателей».

В документе настройки зависимостей на закладке «Состав агрегатов статей» состав этих групп может быть уточнен, применительно к текущему документу.

На данной закладке входящие в состав групп показатели отображаются в виде матрицы:

№ /Группа

Группа А

 ...

Группа Х

1
...
n

Соответственно в системе будет существовать столько же зависимостей, выстроенных по настроенным между агрегатами правилам, сколько строк в данной таблице. Каждая строка данной таблицы указывает конкретные финансовые показатели, подставляемые в настроенные для агрегатов правила зависимости. При выборе группы как влияющей или зависимой на предыдущих закладках, в текущей таблице автоматически создается колонка для этой группы.

Справа от табличной части расположены кнопки:

·         Статью вверх – перемещает статью в выделенной ячейке в строку, выше текущей. При этом если место, на которое перемещается выделенная статья, занято другой статьей, то замещаемая статья перемещается на место текущей;

·         Статью вниз – перемещает статью строкой ниже, аналогично предыдущей команде;

·         Выбрать статью – позволяет установить/заменить статью в выбранной ячейке;

·         Добавить группу – добавляет строку соответствий в таблицу;

·         Удалить строку – удаляет текущую строку соответствий из таблицы.

 

Настройки расчета устанавливаемых зависимостей

В некоторых случаях возникает необходимость в создании однотипных настроек не только для разных финансовых показателей, для которых используется групповая настройка. Бывает необходимым создать идентичную настройку для различных разрезов планирования. Например, несколько подразделений, занимающихся реализацией продукции, получают премию от суммы продаж, но у каждого подразделения собственная ставка для расчета премии. Для обоих влияющим показателем будет «Реализации продукции», одинаковый результирующий показатель – «Премия с продаж», но разные коэффициенты, применяемые к сумме. В этом случае можно было бы создать несколько похожих документов настройки зависимостей, у каждого из которых был бы настроен фильтр по конкретному подразделению в поле «Ограничение области действия по разрезам планирования». Также на закладке «Зависимости» в каждом документе был бы установлен свой собственный процент премии.

Однако, если необходимо установить зависимость, общую для набора аналитических разрезов, но на выходе имеющую различную аналитику планирования или суммовые/количественные значения, рассчитанные по общим правилам, используется механизм настроек расчета.

Созданная настройка выбирается в поле «Настройка расчета» на закладке «Влияющие показатели». После того, как настройка выбрана, система позволяет при указании ограничения области действия по разрезам указать для разреза значение «При наличии в настройке расчета». В этом случае данная зависимость будет применяться лишь к плановым показателям с таким значением аналитического разреза, которое указано в данной настройке.

Также на закладке «Зависимости» появляются дополнительные функции.

Числовые ресурсы настройки можно выбирать:

1.       В колонке «Реквизит для сумы/количества», в этом случае сумма/количество зависимого показателя будут взяты не из влияющих оборотов, а из настройки;

2.       В колонке «Дополнительный коэффициент» (для суммы/количества), если необходимо, чтобы числовой ресурс настройки применялся в качестве множителя для суммы/количества влияющих оборотов.

Также в поле с аналитическим разрезом можно указать значение «По настройке», и аналитическое значение данного разреза будет взято не из влияющих оборотов, а из настройки.

Настройки расчета хранятся в справочнике «Настройки расчета зависимостей бюджетирования».

 

Профили изменения планов по периодам

 

При автоматическом формировании зависимых плановых показателей (оборотов) на основании других плановых показателей (оборотов) часто возникает необходимость указания временного сдвига оборотов — результатов расчета - на определенный интервал времени от исходных оборотов, причем, с разбивкой на несколько значений (например, оплата по реализации поступает в течение последующих трех месяцев от момента реализации в пропорции 50-30-20). Также бывает необходимо сформировать несколько одинаковых оборотов по бюджетам, сумма которых будет некоторой функцией от распределяемого по периодам значения и количества периодов смещения.

Определение профиля выполняется в справочнике «Профили изменения планов по периодам» (меню «Расчеты и зависимости» - «Профили изменения планов по периодам»).
