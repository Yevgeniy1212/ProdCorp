Документ "Внутреннее перемещение наличных денежных средств"

Документ предназначен для отражения в управленческом и бухгалтерском учетах внутреннего перемещения наличных денежных средств из одной кассы в другую. Перемещение  возможно только между кассами, которые имеют одну и ту же валюту хранения денежных средств.

Бухгалтерский учет

Документ формирует проводки по бухгалтерскому учету. При проведении формируются проводки по перемещению денежных средств из кредита счета учета кассы организации отправителя, в дебет счета учета кассы организации получателя.
