Документ "Движения мониторинга"

Конфигурация "1С-Рейтинг: Налоговый мониторинг".

Документ выполняет движения по регистрам мониторинга, необходимым для формирования ряда отчетов дополнения "1С-Рейтинг: Налоговый мониторинг". Для формирования движений анализируются данные документов конфигураций "Бухгалтерия для Казахстана", "Управление производственным предприятием для Казахстана" и "1С-Рейтинг: Нефтебаза" в зависимости от того, в какую конфигурацию встроено дополнение.

Документ содержит четыре флага, каждый из которых представляет собой регламентную операцию формирования движений по конкретному регистру мониторинга. Одна регламентная операция для каждой организации за один и тот же месяц должна быть произведена только один раз.

Движения по мониторингу реализации

При проведении документа с установленным флажком «Движения по мониторингу реализации» будут выполнены движения по регистру накопления Мониторинг реализации.

Движения по мониторингу покупок

При проведении документа с установленным флажком «Движения по мониторингу покупок» будут выполнены движения по регистру накопления Мониторинг покупок.

Движения по мониторингу импорта

При проведении документа с установленным флажком «Движения по мониторингу импорта» будут выполнены движения по регистру накопления Мониторинг импорта.

Движения по мониторингу экспорта

При проведении документа с установленным флажком «Движения по мониторингу экспорта» будут выполнены движения по регистру накопления Мониторинг экспорта.
