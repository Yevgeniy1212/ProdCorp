Документ «Возврат ТМЗ от покупателя»

Документ «Возврат товаров от покупателя» служит для отражения операций по возвращению покупателем приобретенного ранее товара. Стоит учитывать, что проводки документа такие же, как и у документа Реализация ТМЗ, услуг, но "в обратном направлении". На основании этого документа могут быть введены следующие виды документов: Счет-фактура выданный,

В состав документа входят:

  • Организация
  • Склад - склад, на который будут оприходованы возвращаемые ТМЗ
  • Контрагент - покупатель ТМЗ
  • Договор контрагента - договор, по которому осуществляется возврат
  • Валюта документа - валюта, в которой оформляется возврат ТМЗ. Всегда совпадает с валютой договора с контрагентом
  • Курс взаиморасчетов - курс валюты взаиморасчетов
  • Кратность взаиморасчетов - кратность валюты взаиморасчетов
  • Тип цен - тип цены, по которой производится возврат
  • Адрес доставки 
  • Учитывать НДС
  • СуммаВключает НДС
  • Сумма документа - общая сумма документа по итогам всех табличных частей
  • Счет учета расчетов с контрагентом - счет, на который будут отражены проводки по учету взаиморасчетов
  • Счет учета расчетов по возвратам - в случае, если данный реквизит заполнен, проводки по возврату себестоимости и НДС будут корреспондировать именно с этим счетом. После этого произведется зачет "неоплаченной"  части возврата на счет учета расчетов с контрагентами.
  • Сделка - расчетный документ для учета взаиморасчетов (в качестве него также может выступать документ продажи; в этом случае себестоимость возвращаемого товара будет рассчитываться из документа реализации)
  • Ответственный
  • Комментарий
  • Табличная часть "ТМЗ" - служит для отражения операций по возврату ТМЗ. Состав:
    • Номенклатура - товары и материалы, возвращаемые данным документом
    • Единица измерения - единица измерения номенклатуры
    • Коэффициент - коэффициент пересчета количества номенклатуры в базовые единицы
    • Количество
    • Себестоимость - себестоимость возвращаемого товара, которая должна заполняться в случае, если в качестве сделки не выбрано ничего или выбран документ отличный от документа Реализация ТМЗ, услуг
    • Цена
    • Сумма
    • Ставка НДС
    • Сумма НДС
    • Счет учета (БУ) - счет учета запасов, на который будет оприходована себестоимость возвращаемой номенклатуры
    • Счет учета НДС по реализации (БУ) - счет учета для отражения суммы НДС по возврату для бухгалтерских проводок
    • Счет доходов (БУ) - счет учета для отражения суммы суммы дохода от реализации
    • Субконто доходов (БУ) - аналитика счета доходов 
    • Счет списания себестоимости (БУ) - кор. счет списания себестоимости ТМЗ 
    • Субконто списания себестоимости (БУ) - аналитика счета списания себестоимости
    • Вид операции для НДС реализации - аналитика счета НДС для регистра "Корректировка НДС"

 

Бухгалтерский учет

Документ формирует проводки по бухгалтерскому учету. При проведении формируются проводки по оприходованию себестоимости ТМЗ на счетов учета, указанные в табличной части "ТМЗ", сторнированию дохода от реализации товаров, сторнируется дебиторская задолженность на счетах учета расчетов с контрагентами и счетах учета расчетов по возвратам, кроме того формируются проводки по сторнированию НДС по реализации.

Одновременно с проводками по бухгалтерскому учету НДС производятся записи в регистре накопления "Корректировка НДС" на суммы НДС, указанные в документе.
