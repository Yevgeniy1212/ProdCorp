Документ «Отражение налоговой отчетности в регламентированном учете»

Документ «Отражение налоговой отчетности в регламентированном учете» предназначен для автоматизированного формирования проводок по бухгалтерскому и налоговому учету рассчитанных сумм налога.

Документ может быть заполнен по данным налоговых отчетов: «Форма 701.01. Расчет текущих платежей по земельному налогу и налогу на имущество», «Форма 701.00. Расчет текущих платежей по налогу на транспорт». Ввод документа осуществляется при нажатии кнопки «Ввести документ отражения в учете» в форме отчета. Также отражение данных налоговой отчетности можно производить непосредственно из формы «Регламентированная и финансовая отчетность», не открывая сохраненный отчет.

При отражении данных в учете автоматически формируются проводки Дт 7210 - Кт Счета налога.

  • Для корректного заполнения Субконто «Статьи затрат» на счете 7210 необходимо заполнить реквизит «Статья затрат» для Вида налога, указанного в строке табличной части.
  • Счет налога автоматически подставляется из указанного счета в реквизите «Счет учета» для Вида налога, указанного в строке табличной части.
  • Код бюджетной классификации автоматически подставляется из одноименного реквизита для Вида налога, указанного в строке табличной части. Примечание: при заполнении по данным регламентированного отчета в качестве КБК устанавливается значение КБК, определенное для разноски данных регламентированного отчета.
Документ заполняется данными из регламентированного отчета, а именно:
  • поле «Период» - датой уплаты налога, определенной по данным регламентированного отчета;
  • поле «Вид налога» - подставляется предопределенный элемент, если он есть для данного вида налога в справочнике «Налоги, сборы и отчисления». Если нет, то производится поиск элемента данного справочника по КБК, определенного по данным регламентированного отчета;
  • поле «КБК» - заполняется по данным регламентированного отчета. При изменении поля «Вид налога» в поле «КБК» подставляется значение КБК, определенного в справочнике «Налоги, сборы и отчисления» для данного вида налога;
  • поле «Вид платежа в бюджет» - подставляется элемент «Налог (взносы): начислено / уплачено»;
  • поле «Счет затрат (БУ)» - заполняется автоматически счетом «7210 - Административные расходы»;
  • поле «Субконто 1» для счета затрат (БУ) - заполняется статьей затрат, определенной в справочнике «Налоги, сборы и отчисления» для вида налога текущей строки табличной части;
  • поле «Субконто 2» для счета затрат (БУ) - заполняется подразделением, определенным в «Учетной политике (бухгалтерский учет)» как «Подразделение для регламентных операций»;
  • поле «Счет налога (БУ)» - заполняется счетом учета, определенным в справочнике «Налоги, сборы и отчисления» для вида налога текущей строки табличной части;
  • поля «Сумма» и «Содержание» - формируются автоматически по данным регламентированного отчета;
  • поля счетов по НУ - формируются автоматически, используются данные из регистра сведений «Соответствие счетов БУ и НУ», при этом в поле Счет налога (НУ) должен указываться счет из раздела Н600 «Обязательство по налогам»;
  • поле «Структурная единица» - заполняется по данным регламентированного отчета. В случае, если отчет заполнен по одной структурной единице, то будет подставлена именно эта структурная единица, в противном случае - Организация регламентированного отчета;
  • поле «Налоговый комитет» - подставляется значение из регламентированного отчета.
