Документ "Реструктуризация ОС"

Документ предназначен для полной или частичной реструктуризации основного средства на несколько составляющих основных средств, ТМЗ или прочих активов.

Общие реквизиты документа

  • Основное средство – основное средство, подлежащее реструктуризации.
  • Частичное списание – признак, указывающий что ОС будет списано/реструктурировано частично. При включении признака частичного списания появляются дополнительные реквизиты:
    • Коэффициент частичного списания - В данном поле указывается коэффициент, пропорционально которому требуется списать остаточную стоимость ОС. При заполнении коэффициента автоматически заполняются поля "Стоимость част. списания (БУ)" и "Списываемая амортизация (БУ)" по следующему алгоритму:
      Списываемая амортизация (БУ)= (Накопленная амортизация на дату документа + Амортизация за месяц) * Коэффициент.
      где "Амортизация за месяц" = 0, если в учетной политике по бух. учету включена настройка "Применять параметры начисления амортизации в месяце их изменения".
      Стоимость част. списания (БУ) = Первоначальная стоимость ОС * Коэффициент - Списываемая амортизация (БУ)
    • Общая сумма списания первонач. стоим. - в данном поле указывается сумма, на которую необходимо уменьшить первоначальную стоимость исходного ОС. Так как списываемая сумма амортизации относится на стоимость ОС, то данный реквизит содержит общую сумму, из которой необходимо выделить часть, которая будет распределена на составляющие активы. Эта часть указывается в поле "Стоимость частичного списания (БУ)". Таким образом, суммы списания подчиняются формуле:
      Общ. сумма списания первонач. стоим. = Стоимость част. списания (БУ) + Списываемая амортизация (БУ)
    • Списываемая амортизация (БУ) - в данном поле необходимо указать сумму, на которую будет уменьшена накопленная амортизация ОС. сумы списания  и амортизации зависимы по следующей формуле:
      Списываемая стоимость (НУ) - в данном поле необходимо указывать сумму, на которую будет уменьшена стоимость ВА по НУ.

При частичном списании пересматриваются параметры амортизации ОС, поэтому необходимо заполнить следующие реквизиты, в зависимости от способа начисления амортизации:

Если способ амортизации ОС – «Линейный способ», «По сумме чисел лет использования», «Способ уменьшаемого остатка», то:

  • Новый срок использования ОС – если в результате реструктуризации срок использования был изменен,  то в данном поле необходимо вводить новый срок использования ОС, иначе нужно заполнить значением старого срока.
  • Факт. срок использования – фактический срок использования основного средства на момент выполнения реструктуризации.
  • Процент годовой амортизации – значение данного поля рассчитывается автоматически на основе нового и фактического срока использования ОС.
  • Коэффициент ускорения – данное  поле необходимо заполнять только в том случае, если способ амортизации основного средства  - «Способ уменьшаемого остатка», соответственно только тогда поле становится видимым.

Если способ амортизации ОС – «Производственный способ», то:

  • Новый объем работ – если в результате реструктуризации объем работ ОС был изменен, то в данном поле необходимо ввести новый объем работ основного средства, иначе нужно заполнить значением старого объема работ.
  • Факт. объем  работ – фактический объем работ основного средства на момент выполнения реструктуризации.

Закладка "ОС"

На данной закладке необходимо указывать основные средства, появившиеся в результате реструктуризации исходного основного средства.
По команде меню "Заполнить" -> "Распределить стоимость на список ОС (БУ)" происходит распределение стоимости исходного ОС на составляющие ОС равными частями:
- при полной реструктуризации распределяемая сумма равна остаточной стоимости ОС.
- при частичной реструктуризации распределяемая сумма равна сумме, указанной в поле "Стоимость част. списания (БУ)".
Если заполнены другие табличные части, то распределяемая сумма будет уменьшена на сумму этих табличных частей.

Закладка "ТМЗ"

На данной закладке необходимо указывать ТМЗ, появившиеся в результате реструктуризации исходного ОС. В поле "Склад" указывается склад, на который будут оприходованы ТМЗ.
По команде меню "Заполнить" -> "Распределить стоимость пропорц. кол/ву на ТМЗ (БУ)" происходит распределение стоимости исходного ОС на ТМЗ пропорционально количеству ТМЗ.
- при полной реструктуризации распределяемая сумма равна остаточной стоимости ОС.
- при частичной реструктуризации распределяемая сумма равна сумме, указанной в поле "Стоимость част. списания (БУ)".
Если заполнены другие табличные части, то распределяемая сумма будет уменьшена на сумму этих табличный частей.

Закладка "Прочие"

На данной закладке указываются произвольные счета, на которые будет списана остаточная/частичная стоимость ОС.

Особенности проведения документа

В режиме полной реструктуризации основное средство снимается с учета, соответственно, формируются движения по списанию.

В режиме частичного списания пересматриваются параметры начисления амортизации ОС, соответственно, записываются новые параметры.

Если в учетной политике по бух. учету не установлена настройка «Применять параметры начисления амортизации в месяце их изменения», то независимо от режима документа выполняется начисление амортизации за текущий месяц.

Так же при проведении выполняется проверка правильности заполнения сумм табличных частей:
- если производится частичное списание, то сумма табличных частей должна быть равна списываемой стоимости ОС, т.е. значению поля «Стоимость част. списания (БУ)».
- если производится полная реструктуризация, то сумма табличных частей должна быть равна остаточной стоимости ОС.

При проведении остаточная/частичная стоимость ОС списывается на другие основные средства, ТМЗ, или произвольные счета из соответствующих табличных частей:
- Проводки по списанию стоимости на основные средства формируются в данном документе
- Проводки по списанию стоимости на ТМЗ формируются в документе «Оприходование ТМЗ», введенном на основании документа реструктуризации
- Проводки по списанию стоимости на произвольные счета, указанные на закладке «Прочее», формируются в данном документе.
