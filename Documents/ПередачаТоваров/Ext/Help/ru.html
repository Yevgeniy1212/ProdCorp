Документ "Передача ТМЗ"

Документ предназначен для отражения в бухгалтерском и налоговом учете следующих операций:

  • Передача материалов переработчику. Передача собственных материалов стороннему переработчику для выпуска продукции по договору на переработку сырья.
  • Передача продукции заказчику. Передача продукции, выпущенной из давальческого сырья, заказчику.

Общие реквизиты документа

В шапке документа можно выбрать следующие реквизиты:

  • Склад. Место хранения, с которого списываются передаваемые материалы или продукция. Значение реквизита выбирается из справочника "Склады". Реквизит является обязательным для заполнения.

  • Контрагент.  В случае передачи материалов переработчику переработчик, которому передается сырье. В случае передачи продукции заказчику заказчик, которому передается готовая продукция, изготовленная из его сырья. Значение реквизита выбирается из справочника "Контрагенты". Реквизит является обязательным для заполнения.

Табличная часть "ТМЗ"

На закладке указывается продукция, выпущенная из давальческого сырья, которая передается заказчику или сырье, передаваемое переработчику.

  • Номенклатура. Продукция или материалы, передаваемые заказчику. Значение реквизита выбирается из справочника "Номенклатура" и является обязательным для заполнения.

  • Количество. Количество продукции или материалов.

  • Счет учета (БУ). Счет бухгалтерского учета товаров на складе. Значение реквизита выбирается из плана счетов бухгалтерского учета и заполняется автоматически при выборе продукции на основании данных регистра сведений "Счета учета номенклатуры".

  • Счет передачи (БУ). Счет бухгалтерского передачи товаров. Значение реквизита выбирается из плана счетов бухгалтерского учета и заполняется автоматически при выборе продукции на основании данных регистра сведений "Счета учета номенклатуры".

Табличная часть "Источники происхождения"

В табличной части указываются источники происхождения продукции или сырья. Заполняется, если в конфигурации используется учет товаров по источникам происхождения.

  • Номенклатура. Продукция или материалы, передаваемые заказчику. Значение заполняется из табличной части "ТМЗ".

  • Источник происхождения. Источник происхождения продукции или материалов.

  • Количество. Количество продукции или материалов.
