Документ «Ввод сведений о регл учете затрат по заработной плате по структурным подразделениям»

Документ предназначен для регистрации структурных подразделений, по которым будут отражаться расчеты и затраты по заработной плате в бухгалтерском учете. Структурное подразделение указывается отдельно для каждого сотрудника, по каждому месту работы.

Ввод сведений в данном документе не влияет на кадровый учет и расчет заработной платы, которые ведутся в целом по организации.

За каждым рабочим местом (основной или занимаемой по совмещению должности) сотрудника закрепляется структурное подразделение, по которому ведется отражение его взаиморасчетов. Если оно не задано, то по данному рабочему месту отражение идет по организации в целом, без учета структурного подразделения. Если сотрудник работает в нескольких организациях, тогда по каждой организации создается отдельный документ.

Отбор сотрудников отдельного структурного подразделения, в зависимости от выбранного в документе структурного подразделения, доступен в документах:

  1. Возврат зарплаты сотрудников организаций
  2. Зарплата к выплате организаций
  3. ИЛ перечисление получателям
  4. ОПВ Возврат взносов
  5. ОПВ Перечисление в фонды
  6. Отражение зарплаты в регл учете
  7. СО Возврат отчислений
  8. СО Перечисление в фонды

Если в параметрах учета включен признак "Поддержка работы со структурными подразделениями во взаиморасчетах с сотрудниками", в этих документах подбор и заполнение табличной части сотрудниками ведется с учетом выбранного в шапке структурного подразделения согласно следующим правилам:

  1. Налоги отражаются по структурному подразделению, заданному для основного места работы сотрудника. Таким образом, документы по ОПВ и СО при автозаполнении и расчете должны отражать суммы только по тем сотрудникам, для которых структурное подразделение, выбранное в шапке документа, является основным (т.е. задано для основного места работы);
  2.  "Зарплата к выплате" заполняется следующим образом:
    • сумма к выплате по всем местам работы сотрудника в организации отражается в документе, структурное подразделение которого является основным для физ.лица (т.е. задано по основному месту работы)
    • если работник принят по совместительству в  подразделение, выделенное на отдельный баланс (введенное в справочнике "Организации"), тогда
      • в той организации, в которой находится его основное место работы (вне зависимости от того, головная это организация или филиал), сумма к выплате по всем местам работы сотрудника в данной организации отражается в документе, структурное подразделение которого является основным для физ.лица (т.е. задано по основному месту работы), с учетом налогов по всем рабочим местам во всех организациях
      • в той организации, в которой находятся только дополнительные места работы (совмещаемые), сумма к выплате по этим местам работы попадает в документ Зарплата к выплате по данной организации без указания конкретного структурного подразделения, без учета налогов
  3. "Отражение заработной платы в регл учете" заполняется по правилам:
    • отражение происходит в одном документе целиком по всей организации. При этом в его строках для отдельных сотрудников указываются их структурные подразделения;
    • если сотрудник работает по совместительству в подразделении, выделенном на отдельный баланс (элемент справочника "Организации"), тогда на каждую организацию заполняется отдельный документ отражения зарплаты в регл учете
