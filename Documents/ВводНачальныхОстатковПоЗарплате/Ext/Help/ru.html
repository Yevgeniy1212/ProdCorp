Документ «Ввод начальных остатков по зарплате»

Документ предназначен для ввода всех сведений по взаиморасчетам, связанными с заработной платой, необходимых для дальнейшего ведения учета. В документ вносятся начальные данные для ведения учета.

Документ рекомендуется вносить последним днем месяца, за который рассчитаны данные. Для упрощения ввода желательно, чтобы уже были оформлены приемы на работу перед началом занесения остатков.

Заполнение табличной части документа

Подбор данных в табличную часть может быть выполнен несколькими способами, такими как ручной ввод строки, заполнение при помощи механизма подбора, автоматическое заполнение.

Для подбора данных в табличную часть используется кнопка Подбор. При ее нажатии открывается диалоговое окно со списком сотрудников, по которым оформлен прием на работу.

Существует возможность автоматически заполнить табличную часть сотрудниками, числящимися в определенном подразделении. Для этого следует нажать на кнопку Заполнить в командной панели табличной части. При выборе в выпадающем меню пункта По сотрудникам сотрудники, относящиеся к указанной организации и подразделению, будут автоматически добавлены в табличную часть. Если на момент заполнения в табличной части документа уже имеются некоторые строки, пользователю будет предложено выбрать: удалить существующие строки или же добавить к ним новые.

При выборе в выпадающем меню пункта По физлицам табличная часть будет заполнена на основании справочника физические лица.

Реквизиты шапки документа

  • Организация. Организация, в которой числятся сотрудники, указанные в табличной части.
  • Подразделение организации. Подразделение организации, в котором числятся сотрудники, указанные в табличной части.
  • Ответственный. Пользователь, ответственный за ввод информации в документ.
  • Комментарий. Текстовое примечание к документу.

Реквизиты табличной части "Зарплата и налоги"

  • Сотрудник. Сотрудник, для которого вводятся данные для учета.
  • Месяц. Определяет месяц предыдущего периода (до начала ведения учета в конфигурации), к которому относятся данные строки.
  • Задолженность по зарплате. Сумма задолженности перед сотрудником (вносится как положительное значение) или задолженности сотрудника перед организацией (вносится как отрицательное значение). После проведения документа данная сумма будет отражена как кредитовый остаток счета 3350 "Краткосрочная задолженность по оплате труда".
  • Задолженность по депонированным суммам.Сумма задолженности перед сотрудником (вносится как положительное значение) или задолженности сотрудника перед организацией (вносится как отрицательное значение). После проведения документа данная сумма будет отражена как кредитовый остаток счета 3384 "Задолженность по депонированной заработной плате".
  • Задолженность по ИПН (НУ). Сумма индивидуального подоходного налога, исчисленная в соответствующем месяце, которая еще не отражена в налоговой форме 201 в строке 201.00.006 "Сумма индивидуального подоходного налога, подлежащего перечислению" в связи с тем, что соответствующие доходы еще не были выплачены сотруднику. Следует обратить внимание, что данная сумма не отражается в бухгалтерском учете, поэтому остатки по счету 3120 следует ввести с помощью документа "Операция (бухгалтерский и налоговый учет)".

Реквизиты  табличной части "Взносы и отчисления"

  • Сотрудник. Сотрудник, для которого вводятся данные для учета.
  • Месяц. Определяет месяц предыдущего периода (до начала ведения учета в конфигурации), к которому относятся данные строки.
  • Вид платежа. Вид платежа, к которому относятся данные строки.
  • Задолженность по ОПВ. Сумма задолженности по неперечисленным (вносится как положительное значение) или излишне перечисленным (вносится как отрицательное значение) обязательным пенсионным взносам сотрудника. После проведения документа данная сумма будет отражена как кредитовый остаток счета 3220 "Обязательства по пенсионным отчислениям".
  • В т.ч подлежит перечислению (НУ). Сумма задолженность по неперечисленным (вносится как положительное значение) или излишне перечисленным (вносится как отрицательное значение) обязательным пенсионным взносам сотрудника, которая уже отражена в строке 201.00.008 налоговой формы 201 в связи с тем, что сотруднику выплачены соответствующие доходы. Данная сумма не отражается в бухгалтерском учете.
  • Задолженность по СО. Сумма задолженности по неперечисленным (вносится как положительное значение) или излишне перечисленным (вносится как отрицательное значение) обязательным социальным отчислениям на сотрудника. После проведения документа данная сумма будет отражена как кредитовый остаток счета 3210 "Обязательства по социальному страхованию";
  • Задолженность по ОППВ - сумма задолженности по не перечисленным (вносится как положительное значение) или излишне перечисленным (вносится как отрицательное значение) обязательным профессиональным пенсионным взносам сотрудников. После проведения документа данная сумма будет отражена как кредитовый остаток счета 3220 «Обязательства по пенсионным отчислениям»
  • Задолженность по взносам ОСМС - сумма задолженности по не перечисленным (вносится как положительное значение) или излишне перечисленным (вносится как отрицательное значение) взносам на обязательное социальное медицинское страхование сотрудников. После проведения документа данная сумма будет отражена как кредитовый остаток счета 3212 «Обязательства по взносам на социальное медицинское страхование»;
  • Задолженность по отчислениям ОСМС - сумма задолженности по не перечисленным (вносится как положительное значение) или излишне перечисленным (вносится как отрицательное значение) отчислениям на обязательное социальное медицинское страхование сотрудников. После проведения документа данная сумма будет отражена как кредитовый остаток счета 3213 «Обязательства по отчислениям на социальное медицинское страхование».
