Документ «Заявление на предоставление вычета по ИПН»

Документ предназначен для регистрации права предоставления налоговых вычетов физическому лицу по индивидуальному подоходному налогу в соответствии с пп. 4)-5) п. 1 ст. 166 Налогового Кодекса.

В документе указывается:

  • Физическое лицо;
  • Вид вычета ИПН;
  • Период предоставления вычета (в соответствии с графиком платежей по выплатам, относимым на вычет по ИПН);
  • Реквизиты документа, на основании которого предоставляется вычет:
    • Контрагент, с которым заключен договор накопительного страхования либо жилищный строительный сберегательный банк, в котором получен займ на проведение мероприятий по улучшению жилищных условий;
    • Номер и дата предоставленного подтверждающего документа - договора;
  •  В случае, если в условиях договора, являющегося основанием для предоставления вычета, произошли изменения, то в поле "Документ-основание" указывается первичный документ "Заявление на предоставление вычета по ИПН", информация которого будет обновлена в соответствии с данным, указанными в текущем документа-заявлении;
  • Табличная часть "График платежей" содержит сведения о датах платежей в соответствии с графиком платежей по заключенному договору накопительного страхования или жилищного займа, а также о суммах предоставляемых вычетов (размеры страховых премий или вознаграждений по жилищным займам).  Указанные вычеты будут учтены при расчете ИПН в месяце осуществления платежа.

Для удобства заполнения табличной части графиком платежей предназначен пункт "Заполнить суммой за период договора" в меню кнопки "Заполнить" на командной панели над табличным полем. При выборе этого пункта выдается запрос на указание суммы вычета, после чего табличная часть будет заполнена за все месяцы периода действия договора введенной суммой.
