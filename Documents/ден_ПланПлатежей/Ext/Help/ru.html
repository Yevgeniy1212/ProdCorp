Документ «План платежей»

Документ предназначен для утверждения суммы расхода по документам "Заявка на расходование средств". В состав документа входят:

  • Организация
  • Ответственный
  • Комментарий
  • Табличная часть "Платежи" - содержит список заявок. Состав:
    • Заявка на расходование средств - документ "Заявка на расходование средств"
    • Вид оплаты
    • Счет/Касса - счет или касса с которых будет произведена оплата
    • Валюта
    • Дата платежа по заявке - дата платежа, заданная в документе "Заявка на расходование средств" 
    • Сумма по заявке - сумма, заданная в документе "Заявка на расходование средств"
    • Дата платежа - утвержденная дата платежа
    • Сумма - утвержденная сумма расхода по заявке

По утвержденным данным, в документе можно сформировать платежные документы.
