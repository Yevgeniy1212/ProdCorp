Документ "Поступление зерна"

Документ «Поступление зерна» служит для отражения операций по поступлению зерна. С помощью этого документа можно отразить операцию покупки зерна.  На основании этого документа могут быть введены следующие виды документов: Расходный кассовый ордер, Счет-фактура полученный, Возврат зерна поставщику , Поступление услуг (доп. расходы), Платежное поручение (исходящее), Установка цен номенклатуры, Платежный ордер (списание денежных средств).

 

В состав документа входят:

  • Организация
  • Источник финансирования - источник финансирования покупки зерна
  • Тип операции - тип закупа зерна
  • Склад - склад, на котором будут храниться приобретенные ТМЗ
  • Контрагент - поставщик ТМЗ и ОС
  • Договор контрагента - договор, по которому осуществляется поставка
  • Валюта документа - валюта, в которой оформляется поступление ТМЗ и ОС. Всегда совпадает с валютой договора с контрагентом
  • Курс взаиморасчетов - курс валюты взаиморасчетов
  • Кратность взаиморасчетов - кратность валюты взаиморасчетов
  • Тип цен - тип цены, по которой производится поступление
  • Регистрировать цены поставщиков - документ может выполнять регистрацию цен номенклатуры для того типа цен, который указан в шапке документа. В этом случае необходимо взвести данный флаг
  • Учитывать НДС
  • СуммаВключает НДС
  • Сумма документа - общая сумма документа по итогам всех табличных частей
  • Счет учета расчетов с контрагентом - счет, на который будут отражены проводки по учету взаиморасчетов
  • Счет учета расчетов по авансам - в случае, если данный реквизит заполнен, будет проводится зачет авансов полученных организацией организацией
  • Сделка - расчетный документ для учета взаиморасчетов
  • Ответственный
  • Комментарий
  • Табличная часть "Зерно" - служит для отражения операций по приобретению зерна.
  •  Состав: 
    • Счет учета (БУ) - счет учета ресурсов, на который будет отнесена себестоимость приобретаемой номенклатуры
    • Номенклатура - зерно приобретаемое данным документом
    • Единица измерения - единица измерения номенклатуры
    • Коэффициент - коэффициент пересчета количества номенклатуры в базовые единицы
    • Количество
    • Цена
    • Сумма
    • Ставка НДС
    • Сумма НДС
    • Счет учета НДС (БУ) - счет учета для отражения суммы НДС по приобретению для бухгалтерских проводок
    • Вид оборота по НДС - аналитика для регистра "НДС к возмещению"
    • Вид поступления для НДС - аналитика для регистра "НДС к возмещению"
  • Печатные формы:
    • Приходная накладная
    • Типовая форма ТМЗ-3 (приходный ордер)

 

Бухгалтерский учет

Документ формирует проводки по бухгалтерскому учету. При проведении формируются проводки по оприходованию себестоимости зерна на счета учета, указанные в табличной части "Зерно",  формируется кредиторская задолженность на счетах учета расчетов с контрагентами и счетах учета расчетов по авансам, кроме того формируются проводки по отражению НДС по приобретению.

Одновременно с проводками по бухгалтерскому учету НДС производятся записи в регистре накопления "НДС к возмещению" на суммы НДС, указанные в документе.
