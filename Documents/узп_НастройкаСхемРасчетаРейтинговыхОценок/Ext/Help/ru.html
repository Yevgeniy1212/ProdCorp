Документ "Настройка схем расчета рейтинговых оценок"

Настройка схемы производится в документе "Настройка схем расчета рейтинговых оценок".

При создании документа в поле "Схема расчета" указывается схема расчета из справочника "Схемы расчета рейтинговых оценок поставщиков", для которой производится настройка. В том случае, если документ "Настройка схем расчета рейтинговых оценок поставщиков" создается при помощи кнопки "Настроить схему" из элемента справочника, то данное поле будет заполнено по умолчанию.

Табличную часть документа "Настройка схем рейтинговых оценок поставщиков"  можно заполнить автоматически по команде "Заполнить". При этом система произведет заполнение табличной части всеми имеющимися в системе критериями оценки (из плана видов характеристик "Критерии оценки поставщиков").

Меню "Распределить приоритеты" позволяет автоматически распределить вес критериев по определенным шаблонам: как в прошлом,  равными долями, лесенкой вверх, лесенкой вниз.

При этом сумма всех приоритетов критериев должна составлять 100%.

В случае, если при изменении доли одного из приоритетов необходимо автоматически пересчитывать остальные позиции в равных долях, можно воспользоваться кнопкой "Автоперерасчет".

Следует обратить внимание, что проведению подлежит только документ с состоянием "Утвержден".
