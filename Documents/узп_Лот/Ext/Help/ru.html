Документ "Лот"

Документ "Лот" служит для объявления необходимых для заказа позиций, указания схемы расчета рейтинговых оценок поставщиков и типа цен, используемых при расчетах.

Документ может вводиться одним из способов: вручную или на основании плана закупок. При создании лота вручную необходимо заполнить поле "Схема расчета" и табличную часть на закладке "Позиции" тем составом и количеством номенклатурных позиций, которые необходимо приобрести.

В случае создания документа "Лот" на основании плана закупок необходимо указать лишь выбранную схему расчета, а табличная часть на закладке "Позиции" будет заполнена по данным из плана закупок. В графе "Заявка МТС" для каждой номенклатурной позиции будет указана та заявка МТС, которая является основанием для включения данной позиции в план закупок.

Для того, чтобы на основании лота произвести рейтинговую оценку и выбор поставщика, необходимо в рамках указанного лота создать документы: "Оценка поставщиков" и "Регистрация цен поставщиков".

Данные документы будут отображаться на закладке "Документы поставщиков". Из данного списка также возможно открыть, либо создать недостающие для расчета документы.

Для каждого лота необходимо производить как оценку поставщика, так и регистрацию актуальных для него цен.

Следует обратить внимание, что проведению подлежит только документ с состоянием "Утвержден".
