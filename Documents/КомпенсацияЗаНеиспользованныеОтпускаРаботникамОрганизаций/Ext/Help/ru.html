Документ «Компенсация за неиспользованные отпуска сотрудникам организаций»

Документом «Компенсация за неиспользованные отпуска сотрудникам организаций» погашаются неиспользованные дни трудовых отпусков путем предоставления компенсации в денежной форме.

В документе указывается:

  • список сотрудников;
  • даты предоставления компенсации;
  • количество компенсируемых дней отпуска;
  • в случае компенсации части неиспользованных дней отпуска, указывается рабочий год, по который включительно будет произведена компенсация.

Автоматический расчет. По списку сотрудников, введенному в табличном поле Сотрудники, может быть произведен расчет неиспользованных дней отпуска на основании данных регистра сведений «Предоставляемые отпуска организаций»; для этого предназначена кнопка «Рассчитать» в панели инструментов формы документа.

По документу можно получить две печатные формы – на каждого сотрудника или на список сотрудников из документа.

Внимание! Для сотрудников, работающих в организации в том числе и по внутреннему совместительству, компенсация отпуска регистрируется отдельно для каждого места работы.
