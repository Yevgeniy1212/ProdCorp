Документ «Счет-фактура полученный»

Документ «Счет-фактура полученный» может вводится как на основании другого документа («Возврат товаров поставщику», «Поступление доп.расходов», «Поступление из переработки», «Поступление НМА», «Поступление товаров и услуг», «Сторнирование»), в котором уже есть все основные данные: о поставщике, о товарах, работах или услугах, так и непосредственно из списка полученных счетов-фактур, путем ввода нового документа. Также документ «Счет-фактура полученный» может вводиться напрямую из документа «Авансовый отчет».

Информация в документе «Счет-фактура полученный» заполняется в соответствии с информацией, введенной в документе-основании.

Для ввода документа «Счет-фактура полученный» в документах, на основании которых он вводится, предусмотрена кнопка «Ввести счет-фактуру».

В состав документа входят:

  • Организация - организация, выписавшая счет-фактуру. Сторона сделки по договору приобретения.
  • Покупатель - организация, указываемая в качестве покупателя в счете-фактуре (печатной форме и налоговой отчетности). В общем случае равна реквизиту "Организация", но в случае, если стороной сделки является филиал в качестве покупателя с 01.01.2009 года может быть указано головное предприятие.
  • Контрагент – покупатель ТМЗ, ОС, НМА и услуг.Сторона сделки по договору приобретения.
  • Поставщик - организация, указываемая в качестве поставщика в счете-фактуре(печатной форме и налоговой отчетности). В общем случае равна реквизиту "Контрагент", но в случае, если стороной сделки является филиал в качестве поставщика  с 01.01.2009 года может быть указано головное предприятие.
  • Договор контрагента - договор, по которому выписывается документ «Счет-фактура».
  • Валюта документа - валюта, в которой оформляется документ. Всегда совпадает с валютой договора с контрагентом
  • Курс взаиморасчетов - курс валюты взаиморасчетов
  • Кратность взаиморасчетов - кратность валюты взаиморасчетов
  • Тип цен - тип цены, по которому происходит оприходование товара.
  • Учитывать НДС
  • Сумма включает НДС
  • Сумма документа - общая сумма документа по итогам всех табличных частей (ОС, Товары, НМА, Услуги)
  • Дополнительный – признак дополнительного счета-фактуры, например, необходим в случае возврата поставщику.
  • Основной счет-фактура - указывается основной счет-фактура, к которому составлен данный счет-фактура. Обязателен для заполнения в случае если установлен признак «Дополнительный».
  • Подтвержден документами отгрузки – признак, что данные документа полностью подтверждены документами отгрузки. По умолчанию устанавливается, когда документ был введен на основании документов поступления, перечисленных выше. При установленном признаке данные документа не редактируются.
  • Дата входящего документа
  • Номер входящего документа
  • Дата оборота в табличной части - если в шапке не установлен данный признак, колонки "Дата оборота" в табличных частях не отображаются. Если его установить, то реквизит шапки "Дата оборота" становится недоступным (но не очищается), а колонки "Дата оборота" в табличных частях ТМЗ, Услуги, ОС, НМА становятся видимыми. В существующих строках пустые значения заполняются датой документа. Колонки доступны для изменения. При снятии флажка колонки перестают отображаться, но не очищаются.
  • Ответственный
  • Комментарий
  • Табличная часть "ОС" - служит для отражения данных по приобретаемым ОС:
    • Основное средство
    • Сумма
    • Ставка НДС
    • Сумма НДС
  • Табличная часть "ТМЗ" - служит для отражения сведений по приобретаемым ТМЗ. Состав:
    • Номенклатура - товары и материалы
    • Единица измерения - единица измерения номенклатуры
    • Коэффициент - коэффициент пересчета количества номенклатуры в базовые единицы
    • Количество
    • Цена
    • Сумма
    • Ставка НДС
    • Сумма НДС
    • Источник происхождения - источник происхождения товара. Заполняется, если в конфигурации используется учет товаров по источникам происхождения
  • Табличная часть "НМА" - служит для отражения сведений о нематериальных активах:
    • Нематериальный актив
    • Сумма
    • Ставка НДС
    • Сумма НДС
  • Табличная часть "Услуги" - служит для отражения сведений по приобретаемым услугам. Состав:
    • Номенклатура - услуга
    • Содержание - содержание услуги
    • Количество
    • Цена
    • Сумма
    • Ставка НДС  
    • Сумма НДС
  • Табличная часть "Документы-основания" - для указания документов поступления. Состав:
    • Документ-основание
       

При вводе счет-фактуры  на основании документов отгрузки  по умолчанию заполняются все табличные части и устанавливается признак Подтвержден документами отгрузки, и в данном случае данные документа не редактируются. Если необходимо внести изменения в счет-фактуру, то нужно снять признак Подтвержден документами отгрузки. Заполнение счет-фактуры происходит по совпадению табличных частей из документа-основания, исключение составляют документы:

  •  Документ "Поступление доп.расходов" - данные с табличной частью "Товары" документа попадут в табличную часть "Товары" счет-фактуры лишь в том случае, если указана сумма доп.расходов. А сумма расхода, расположенная в шапке документа "Поступление доп.расходов" попадет в табличную часть "Услуги" документа "Счет-фактура полученный". 

  • Документ "Авансовый отчет" - на основании данного документа можно ввести несколько счет-фактур, так как в документе "Авансовый отчет" счет-фактуры можно вводить для строк табличных частей "Товары" и "Прочее". В строках  указанных табличных частей документа "Авансовый отчет", для которых нужна счет-фактура необходима установить признак Предъявлен счет-фактура, заполнить номер, дату счет-фактуры и поставщика товаров или услуг. Группировка данных для счет-фактуры идет по номеру, дате счет-фактуры и поставщику, указанных в строках данных табличных частей документа (т.е. для строк с одинаковыми номерами СФ и поставщиками будет создан один документ "Счет-фактура полученный"). Документ "Счет-фактура полученный" создается автоматически при записи документа "Авансовый отчет". Строки с табличной части "Прочее" документа отгрузки попадут в табличную часть "Услуги" документа "Счет-фактура полученный".

Когда уже сформированы счета-фактуры для строк документа "Авансовый отчет", и были откорректированы данные: номер счет-фактуры, дата счет-фактуры или поставщик, тогда:

    • Если в счет-фактуре  список документов-оснований состоит только из текущего документа и не используется для других строк авансового отчета, то счет-фактура помечается на удаление.
    • Если в счет-фактуре  список документов-оснований состоит только из текущего документа, но он используется для других строк авансового отчета, то в этом случае, если у счет-фактуры установлен признак Подтвержден документами отгрузки, данные счет-фактуры перезаполняется, т.е. удалятся строки, в которых были изменены сведения о счет-фактуре.
    • если количество документов-оснований больше одного и не используется для других строк авансового отчета, тогда данный документ-отгрузки удаляется из списка документов-оснований счет-фактуры,  и  если у счет-фактуры установлен признак Подтвержден документами отгрузки, то данные счет-фактуры перезаполняются.
    • если количество документов-оснований больше одного, но в счет-фактуре содержатся данные из других строк авансового отчета, тогда если у счет-фактуры установлен признак Подтвержден документами отгрузки, то данные счет-фактуры перезаполняются.

При добавлении документов-оснований в счет-фактуру, табличные части автоматически не заполняются данными из документа отгрузки. Для заполнения счет-фактуры данными из документов-оснований можно воспользоваться действиями подменю Автозаполнение, которое расположено в верхней командной панели:

1) Перезаполнить по документам основания  - синхронизация данных счет-фактуры и документов-оснований.

2) Добавить из документа основания – добавление к уже имеющемся данным в счет-фактуре, данных по выбранному документу-основанию. В случае когда, документов-оснований несколько, то пользователю сначала предоставляется список документов, из которого он должен выбрать один документ-основание.

Если меняется документ отгрузки, у которого уже имеется сопутствующая счет-фактура и у нее установлен признак Подтвержден документами отгрузки, то  при проведения документа отгрузки данные счет-фактуры по умолчанию синхронизируются с данными документа-основания.

В случае, когда  документ отгрузки помечается на удаление, и для данного документа заведена счет-фактура тогда,

  • если список документов-оснований состоит только из текущего документа, то счет-фактура помечается на удаление.
  • если количество документов-оснований больше одного, в этом случае счет-фактура не помечается на удаление, а из списка документов-оснований удаляется данный документ-отгрузки, и  если у счет-фактуры установлен признак Подтвержден документами отгрузки, то данные счет-фактуры перезаполняются.

На основании счет-фактуры можно ввести следующие документы отгрузки: Возврат товаров поставщику», «Поступление доп.расходов», «Поступление из переработки», «Поступление НМА», «Поступление товаров и услуг», причем осуществляется контроль по уже введенным документам отгрузки, например: Товара А  было 30 штук, вы ввели документ отгрузки на 15 штук, то  в следующий раз уже при вводе на основании счета-фактуры документа отгрузки  заполнится 15 штук, а не 30 (поиск уже использованной номенклатуры  для ТМЗ  осуществляется по: номенклатуре, цене, ед.измерения, ставки НДС;  для Услуг  по :номенклатуре, содержанию, ставке НДС; для ОС по: основному средству, ставку НДС; для НМА по : нематериальному активу, ставке НДС).

При вводе документа  "Поступление доп.расходов" на основании счет-фактуры заполняются данные только с табличной части "Услуги", т.е. в документе "Поступление доп.расходов" должны заполниться только реквизиты шапки,  причем если в документе "Счет-фактура полученный" несколько строк, то сумма расхода берется суммарная по всем услугам, с учетом остатков и ставка НДС заполняется лишь только в том случае, если она одинакова по всем строкам оставшихся услуг.

Важно: При вводе документа "Поступление доп.расходов" на основании счет-фактуры данные с табличной части "Товары" счет-фактуры  не будут заполняться в табличную часть "Товары" документа "Поступление доп.расходов"!.
