Документ "Требование-накладная"

 

Документ предназначен для оформления операции выдачи ТМЗ со склада в производство. Стоимость передаваемых ТМЗ списывается на затраты производства.

В одном документе можно отразить следующие виды затрат:

  • Производственные
  • Общепроизводственные (ОПР)
 
В бухгалтерском учете эти виды затрат соответствуют списанию ТМЗ на счета:
  • 8110 "Основное производство", 8310 "Вспомогательные производства",  8210 "Полуфабрикаты собственного производства" - производственные;
  • 8410 "Накладные расходы" - общепроизводственные.

 

Общие реквизиты документа:

  • Склад - склад, с которого произведена выдача ТМЗ в производство;
  • Номенклатурная группа - аналитика затрат. Указывает, под выпуск какой номенклатурной группы нужно произвести списание ТМЗ на затраты;
  • Подразделение организации - подразделение организации, на затраты которого следует отнести списание ТМЗ. Аналитика по подразделениям организаций указана для всех счетов учета производственных затрат;
  • Документ-основание - документ, на основании которого будет заполнена табличная часть "Материалы";
  • Номенклатура - готовая продукция, по спецификации которой будет заполнена табличная часть "Материалы". После выбора продукции запрашивается ее количество, исходя из которого будет рассчитано количество необходимых материалов.

     

    Закладка "Материалы"

    На закладке перечисляются передаваемые в производство материалы и полуфабрикаты.

  • Номенклатура - материал (полуфабрикат), передаваемый в производство; 
  • Количество - количество материала (полуфабриката);
  • Счет учета (БУ) - счет бухгалтерского учета материала (полуфабриката);
  • Счет учета (НУ) - счет учета материала (полуфабриката) по налоговому учету.

     

  • Закладка "Материалы заказчика"

    На закладке перечисляются передаваемые в производство материалы заказчика (давальческое сырье).

  • Номенклатура - материал, передаваемый в производство; 
  • Количество - количество материала;
  • Счет учета (БУ) - счет бухгалтерского учета материала;
  • Счет передачи (БУ) - счет передачи.

  • Табличная часть "Источники происхождения"

    Служит для указания источников происхождения у материалов. Заполняется, если в конфигурации используется учет товаров по источникам происхождения.

  • Номенклатура - материал (полуфабрикат) из табличной части "Материалы"; 
  • Источник происхождения - источник происхождения материала (полуфабриката);
  • Количество - количество материала (полуфабриката).
