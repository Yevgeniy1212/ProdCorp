Документ «План-график учета рабочего времени»

Документ «План-график учета рабочего времени» предназначен для учета рабочего времени работников, рабочее время которых не поддается регламентированию с помощью графиков рабочего времени и рассчитывается вручную. К таким работникам могут быть отнесены работающие по сменным графикам с суммированным учетом рабочего времени.

В документе указывается:

  • список работников;
  • график работы, на основании которого производится первоначальное заполнение рабочего времени;
  • период работы;
  • объем рабочих часов по дням;
  • объем ночных часов по дням.

Автоматическое заполнение. Документ может быть автоматически заполнен данными, внесенными в сменные графики работы с признаком «Суммированный учет рабочего времени»; для этого предназначена кнопка «Заполнить» документа.
При проведении документов производится формирование «индивидуального» графика работы для каждого работника.

По документу можно получить печатную форму графика работы для внесенных в табличную часть работников с разбивкой рабочего времени по дням.

Внимание! В отличие от работающих по «стандартным» графикам работы, для начисления зарплаты работникам по сменным графикам с суммированным учетом рабочего времени, является обязательным заполнение документа «План-график учета рабочего времени» на соответствющий расчетный период, либо работники должны быть переведены на другой график работы (не имеющий признак «Суммированный учет рабочего времени») с помощью документа «Кадровое перемещение организаций».

Для определения нормы рабочего времени по сменным графикам с суммированным учетом рабочего времени используются сведения «Баланса рабочего времени».
