Документ "Возврат ТМЗ поставщику"

Документ «Возврат ТМЗ поставщику» служит для отражения операций по возврату товаров поставщику. Формирует такие же операции, как и документ Поступление ТМЗ (услуг), но с "обратным знаком". На основании этого документа могут быть введены следующие виды документов: Приходный кассовый ордер, Счет-фактура полученный, Платежное поручение (входящее), Платежный ордер (поступление денежных средств)

 

В состав документа входят:

  • Организация
  • Склад - склад, с которого будут списываться возвращаемые ТМЗ
  • Контрагент - поставщик ТМЗ
  • Договор контрагента - договор, по которому осуществляется возврат
  • Валюта документа - валюта, в которой оформляется возврат ТМЗ. Всегда совпадает с валютой договора с контрагентом
  • Курс взаиморасчетов - курс валюты взаиморасчетов
  • Кратность взаиморасчетов - кратность валюты взаиморасчетов
  • Тип цен - тип цены, по которой производится возврат
  • Учитывать НДС
  • СуммаВключает НДС
  • Сумма документа - общая сумма документа по итогам всех табличных частей
  • Счет учета расчетов с контрагентом - счет, на который будут отражены проводки по учету взаиморасчетов
  • Счет учета расчетов по возвратам - счет с которым будут корреспондировать проводки по возврату себестоимости и НДС по возвращаемым ТМЗ. После этого будут выполнены зачеты неоплаченной части возврата на счет учета расчетов с контрагентами. В случае, если счет не заполнен проводки по возврату себестоимости и НДС будут корреспондировать со счетом учета расчетов с контрагентом 
  • Сделка - расчетный документ для учета взаиморасчетов. В случае, если в качестве значения этого реквизита выбран документ Поступление товаров (услуг) проводки по списанию себестоимости и НДС будут формироваться с учетом проводок сделки.
  • Счет списания разницы в себестоимости (БУ) - при возврате товаров поставщику нередко возникает ситуация, когда себестоимость товара в табличной части (СуммаБезНДС) не совпадает с себестоимостью этого же товара по учетным данным организации. В этом случае доход/убыток от такой разницы (СуммаБезНДС-СебестоимостьПоУчетнымДанным) будет отнесен на счет списания себестоимости.
  • Статья списания разницы (БУ) - аналитика счета списания разницы
  • Ответственный
  • Комментарий
  • Табличная часть "ТМЗ" - служит для отражения операций по возврату ТМЗ. Состав:
    • Номенклатура - товары и материалы, возвращаемые данным документом
    • Единица измерения - единица измерения номенклатуры
    • Коэффициент - коэффициент пересчета количества номенклатуры в базовые единицы
    • Количество
    • Цена
    • Сумма
    • Ставка НДС
    • Сумма НДС
    • Счет учета (БУ) - счет учета запасов, с которого будет списана себестоимость возвращаемой номенклатуры
    • Счет учета НДС (БУ) - счет учета для отражения суммы НДС по возврату для бухгалтерских проводок
    • Вид оборота по НДС - аналитика счета НДС для регистра "Корректировка НДС к возмещению"
    • Вид поступления для НДС - аналитика счета НДС для регистра "Корректировка НДС к возмещению"
    • Источник происхождения - источник происхождения товара. Заполняется, если в конфигурации используется учет товаров по источникам происхождения

 

Бухгалтерский учет

Документ формирует проводки по бухгалтерскому учету. При проведении формируются проводки по списанию себестоимости ТМЗ со счета учета, указанные в табличной части "ТМЗ", формируется кредиторская задолженность на счетах учета расчетов с контрагентами и счетах учета расчетов по возвратам,  а также формируются проводки по отражению дохода/убытка от разницы в себестоимости.

Принятие НДС к зачету

При "стандартной" схеме, т.е. когда налоговый счет-фактура получен вместе с первичными документами.
При проведении документа будут сформированы бухгалтерские проводки по дебету счета 1421 "Налог на добавленную стоимость" и движения по регистру накопления "НДС к возмещению".

Отражение "нестандартной" схемы отнесения НДС в зачет.Ситуация по отложенному принятию НДС к зачету возникает в случае, когда налоговый счет-фактура получен позднее первичных документов. В этом случае в документе устанавливается признак "Отложить принятие НДС к зачету" в форме "Цена и валюта" по ссылке "Тип цен".
При проведении будет сформирована бухгалтерская проводка по дебету счета 1422 "Налог на добавленную стоимость (отложенное принятие к зачету)". Движения по регистру накопления "НДС к возмещению" будут отсутствовать.
