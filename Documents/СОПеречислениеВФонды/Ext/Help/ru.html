Документ «Социальное страхование: перечисление в фонды»

Документ предназначен для подготовки к перечислению исчисленных обязательных социальных отчислений, взносов и отчислений на обязательное медицинское социальное страхование в ГФСС.

В шапке документа указываются реквизиты "Организация", "Месяц исчисления", "Вид платежа" и "Способ заполнения документа".

Документы могут быть заполнены следующими способами:

  • По задолженности за месяц - при выборе данного способа сумма к перечислению рассчитывается по задолженности за месяц, указанный в документе.
  • По остаткам задолженности на конец месяца - при выборе данного способа сумма к перечислению рассчитывается по остаткам задолженности на конец месяца заполнения.

Для перечисления взносов на ОСМС возможен расчет сумм к перечислению по удельному весу. Если в настройках учетной политики налогового учета для организации признак "Регламентированные удержания принимать к учету как удержанные" установен в значение "при выплате доходов", то способы заполнения документа дополняются вариантами:

  • По задолженности за месяц, удержанной по НУ - при выборе данного способа сумма к перечислению рассчитывается по задолженности, рассчитанной по налоговому учету, за месяц, указанный в документе.
  • По остаткам задолженности на конец месяца, удержанной по НУ - при выборе данного способа сумма к перечислению рассчитывается по остаткам задолженности, рассчитанной по налоговому учету, на конец месяца заполнения.

Табличную часть документа можно заполнить автоматически (с помощью кнопки "Автозаполнение" командной панели формы документа). Если требуется рассчитать суммы, подлежащие перечислению, по предварительно введенному списку работников табличной части документа, то следует воспользоваться кнопкой "Рассчитать".

В правой части формы документа расположена контекстно-зависимая справка, в которой приводятся дополнительные разъяснения параметров заполнения документа.

Для регистрации факта перечисления взносов на основании документа "Социальное страхование: перечисление в фонды" необходимо ввести документ "Платежное поручение исходящее" или "Авансовый отчет".
