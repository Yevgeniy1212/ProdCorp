Документ «ГЗ: Объявление о проведение конкурса по государственным закупкам»

Конфигурация "1С-Рейтинг: Управление государственными закупками"

Документ предназначен для регистрации объявлений о проведении конкурсов по государственным закупкам. В документе предусмотрен сервис заполнения данными на основании утвержденных заказов подразделения, указанных в табличной части «Документы основания».

Для заполнения списка товаров, работи услуг данными из нескольких текущих заявок используется вкладка "Заказы подразделений". Для заполнения табличных частей используется меню "Автозаполнение - Заполнить из заказов подразделений".

Загрузить из Excel – по нажатию кнопки открывается обработка Загрузка данных из табличного документа, предназначенная для загрузки данных в табличную часть Товары, работы и услуги.

После того как заявка сформирована документ необходимо записать и провести.

Для ввода ценовых предложений используется кнопка "Ввести ценовые предложения" на вкладке "Ценовые предложения". В табличную часть ценовых предложений попадают товары и услуги, закупаемые способом "Запрос ценовых предложений". Позиции с другими способами закупок в ценовое предложение не попадают. После заполнения ценового предложения ставится признак "Проверено" и ценовое предложение записывается.

Для формирования итогов по заявке используется кнопка "Сформировать итоги государственных закупок" на вкладке "Итоги". При формировании итогов происходит анализ введенных ценовых предложений в разрезе даты предоставления и предложенных цен и определяются оптимальные поставщики по данным заказа. При этом в документ итогов попадают товары, работы и услуги, которые в данный момент находятся в состоянии "Конкурс". После формирования документ итогов открывается и проводится.
