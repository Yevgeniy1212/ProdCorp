Документ "Заявка МТС"

Документ "Заявка МТС" (заявка материально-технического снабжения) используется для планирования снабжения предприятия.

В заявке МТС в поле "Источник заявки" с помощью переключателя указывается источник заявки: подразделение или прочие источники (несколько подразделений). В том случае, если выбран источник "Подразделение", то в поле "Подразделение-заявитель" необходимо указать управленческое подразделение, которое может соответствовать одному или нескольким управленческим подразделениям, складу и т.д.  Если ведется учет затрат на ремонт основных средств (данная возможность включается в настройках параметров учета на закладке "Управление закупками") в поле "Объект ремонта" можно указать основное средство, на ремонт которого требуются указанные в заявке товары и услуги. В таком случае на каждый объект ремонта создается отдельная заявка. От одной единицы снабжения может быть подано любое количество заявок.

В поле "Параметры планирования" указываются сценарий и период, на который ведется планирование. Сценарий определяет количество и длительность периодов (месяц, квартал и т.д.), по которым ведется планирование в одном документе.

Документ "Заявка МТС" представляет собой планирование потребности единицы снабжения в товарах и услугах, и служит основной для последующего составления плана закупок. Номенклатурный состав заявки указывается на закладке "Спецификация".

Если используются характеристики номенклатуры (данная возможность включается в настройках параметров учета на закладке "Управление закупками"), то для любой позиции может быть указана характеристика, если она была ранее установлена для данной номенклатуры в справочнике "Характеристики номенклатурных позиций".

Если ведется учет по проектам (данная возможность включается в настройках параметров учета на закладке "Управление закупками"), в табличной части спецификации в поле "Проект" можно указать проект, к которому относится данная номенклатурная позиция.

Период планирования отображается в табличной части заявки в соответствии с установленными настройками пользователя. Период будет отображаться в графе "Период планирования", либо в матричном виде.

Также в табличной части для каждой номенклатурной позиции можно указать приоритет. Для этого используется справочник "Приоритеты позиций плана закупок".

Цена позиций рассчитывается по типу прогнозных цен, указанному в настройках параметров учета.

Заявки МТС могут служить основанием, как для плана закупок, так и для бухгалтерских документов перемещения и списания товаров. В строках таких бухгалтерских документов указывается конкретная заявка МТС, под исполнение которой эта операция выполняется.
