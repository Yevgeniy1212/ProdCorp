Документы, удостоверяющие личность

Для каждого физического лица можно указать реквизиты документа, удостоверяющего его личность. Для указания вида документа и используется этот справочник.

Справочник может быть заполнен при помощи классификатора, для этого в форме списка справочника предназначена кнопка «Подбор».

Реквизит «Код вида документа в налоговой отчетности» необходимо заполнить в случае, если документ удостоверяет личность лица, не являющегося гражданином Республики Казахстан. Значение данного реквизита используется при заполнении графы G «Код вида документа, удостоверяющего личность» приложения 210.02 регламентированного отчета «Форма 210.00. Декларация по индивидуальному подоходному налогу и социальному налогу».
