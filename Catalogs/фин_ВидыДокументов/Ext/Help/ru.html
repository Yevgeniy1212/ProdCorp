Справочник "Виды документов"

Справочник позволяет связать создаваемые в информационной базе настройки с определенным видом документов информационной базы. В поле «Программный идентификатор» выбирается имеющийся в информационной базе вид документов, представляемый данным справочником.

Флажок «Регистрировать первичные данные по бюджетированию» определяет, будет ли вообще проводиться регистрация фактических данных при проведении данных документов.

В поле «Применяемый набор шаблонов отражения» указывается набор шаблонов (схема), либо из выпадающего списка выбирается вариант «По настройкам для регистров» (для отражения факта по подчиненным регистрам, а не по документу), а в поле «Дата ввода в действие» указывается дата ввода в действие указанной схемы. Просмотреть историю с указанием дат, на которые были указаны используемые схемы, можно при помощи кнопки «История». При проведении документов определенного вида к ним применяется та схема отражения, которая задана на дату документа.

Применяемый набор шаблонов отражения выбирается из справочника «Наборы шаблонов отражения первичных данных в бюджетах» (меню «Учет фактических данных» - «Наборы шаблонов отражения первичных данных в бюджетах»).

Использование дополнительных реквизитов документа настраивается в  в группе полей «Дополнительные реквизиты для учета первичных данных в бюджетах». Эта группа полей доступна только  если документ регистрирует фактические данные.

Флажок «Использовать дополнительные реквизиты» включает возможность назначения документу дополнительных реквизитов. В списке добавляются элементы списка субконто бюджетирования, которые будут представлять дополнительные реквизиты документа, с указанием признака обязательности его заполнения пользователем. Если пользователь не заполнит реквизит, помеченный как обязательный, система не разрешит проведение документа.

Список субконто бюджетирования ведется в плане видов характеристик «Виды субконто бюджетирования».
