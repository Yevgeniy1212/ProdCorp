Справочник "Шаблоны отражения фактических данных по бюджетам"

Шаблоны, входящие в состав схемы отражения (набора шаблонов) хранятся в справочнике «Шаблоны отражения фактических данных по бюджетам» (меню «Учет фактических данных» - «Шаблоны отражения фактических данных по бюджетам»).
Шаблон представляет собой схему формирования проводок по бюджетам, созданную для конкретного документа. Шаблон просто определяет поля документа, значения которых будут использоваться для заполнения аналитических разрезов и показателей формируемых оборотов по бюджетам. Проводки по счетам  будут сформированы автоматически при формировании оборотов, на основании схем проводок, настроенных для статей оборотов.
 
Для шаблона, в поле «Вид документа», указывается вид документа, на основании структуры которого пользователь будет конструировать схему формирования проводок. В списке «Правила отражения по шаблону» приводится перечень правил, применяемых к документу.

Каждое правило редактируется в панели «Настройки правила отражения», расположенной справа от списка правил. Для каждого правила указывается источник данных:

·         По шапке документа – по данному правилу будет сформирована одна проводка по оборотам бюджетов на основании данных шапки;

·         По табличной части – по данному правилу будет сформировано столько же проводок по оборотам бюджетов, сколько будет строк в табличной части, указанной в поле «Табличная часть / набор движений», на основании данных шапки документа и указанной табличной части;

·         По движениям в регистрах – по данному правилу будет сформировано столько же проводок по оборотам бюджетов, сколько будет проводок в движениях документа по регистру, указанному в поле «Табличная часть / набор движений», на основании данных шапки документа и указанного регистра.

Кнопка «Заполнить» подбирает подходящие по типу реквизиты табличной части/набора движений/шапки/дополнительны реквизиты бюджетирования и заполняет правило на их основании.

Само правило представляет список полей записей регистра оборотов по бюджетам (то есть измерения и показатели оборота по бюджету) – финансовый показатель, ЦФО и т.д., с указанием источников их заполнения.

В колонке «Бюджетные данные» находится наименование заполняемого поля проводки. В колонке «Тип источника» указывается тип источника для заполнения указанного реквизита оборотов:

·         Шапка документа – в колонке «Источник заполнения» будет указан реквизит шапки, служащий для заполнения соответствующего реквизита;

·         Табличная часть/ движения – в колонке «Источник заполнения» будет указан реквизит выбранной табличной части/ набора движений, служащий для заполнения соответствующего реквизита;

·         Соответствие – в колонке «Источник заполнения» будет указана настройка использования вариантов получения соответствий первичным данным (опирающаяся на данные шапки указанного вида документов, либо выбранной табличной части/ набора движений), служащая для заполнения соответствующего реквизита. Для числовых реквизитов может использоваться вариант получения соответствий, базирующийся на формуле;

·         Дополнительные реквизиты бюджетирования – в колонке «Источник заполнения» будет указан дополнительный реквизит бюджетирования, назначенный для документа, значение которого служит для заполнения соответствующего реквизита;

·         Фиксированное значение – в колонке «Источник заполнения» будет указано фиксированное значение, служащее для заполнения соответствующего реквизита.

В поле «Условие применения правила» можно указать условие, при выполнении которого правило будет применено. Это удобно, например, если необходимо оформить обороты по бюджетам не по всем строкам табличной части, а только по удовлетворяющим условиям. Если условие не указано, правило будет применяться всегда. Перечень таких условий ведется в справочнике Условия выполнения операций по строкам документа (меню Учет фактических данных - выполнения операций по строкам документа). Настройка данного справочника аналогична справочнику Условия выполнения операций по документу, за исключением того, что в данной настройке можно указать источник данных (шапка, табличная часть, набор движений), на реквизиты которых может налагаться условие. Выполнение условия при формировании проводок проверяется для каждой строки табличной части или набора движений отдельно.

При регистрации оборотов по правилу, в регистре «Обороты бюджетов» в колонке «Шаблон отражения фактических данных» указывается ссылка на шаблон, содержащий правило отражения, а в колонке «Комментарий строки шаблона» указывается комментарий, настроенный в поле «Комментарий строки шаблона» для строки списка правил отражения.
