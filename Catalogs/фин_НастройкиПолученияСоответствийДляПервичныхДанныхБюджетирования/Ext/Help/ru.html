Справочник "Настройки использования вариантов получения соответствий первичным данным"

Используется в подсистеме "Бюджетирование" для определения связи реквизитов конкретного вида документов с абстрактными показателями варианта получения соответствий.

В поле «Разрез бюджетирования» указывается аналитический разрез, значение которого необходимо получать при помощи данной настройки.
В поле «Вид документов» указывается вид документов, для структуры которого настраивается получение соответствия.
В поле «Источник данных» выбирается одно из значений:

  • «По шапке документа» - привязка варианта осуществляется к структуре шапки документа;
  • «По табличной части» - привязка варианта осуществляется к структуре табличной части документа, указанной в поле «Табличная часть/ набор движений»;
  • «По движениям в регистрах» - привязка варианта осуществляется к структуре движений документа по регистру, указанному в поле «Табличная часть/ набор движений».

В поле «Вариант» выбирается элемент справочника «Варианты получения соответствий первичным данным».
Список «Способ заполнения показателей варианта» автоматически заполняется показателями варианта расчета, в соответствие которым выставляются реквизиты документа/табличной части/ набора движений, передаваемые в качестве значений параметров, либо, если для строки установлен признак «Фиксированное значение», можно выбрать определенное значение из любого справочника или другого источника нормативно-справочной информации. Если для показателя варианта установлено ограничение по типу (справочником, либо числовым типом), тогда для его заполнения могут выбираться только поля/фиксированные значения определенного типа.
При выборе в данном списке полей табличной части/ набора движений перед названием поля указывается название табличной части /набора движений.

Внимание!!! Если в данных присутствуют поля с набором субконто, и заранее не известно, под каким номером субконто будет находиться нужное значение (например, статья затрат), тогда в качестве поля источника можно выбрать поле с названием группы реквизитов - субконто, предваренное словом «Автовыбор По Типу». В таком случае система выберет из набора субконто значение нужного типа, но для этого у соответствующего показателя варианта необходимо установить ограничение значения определенным справочником. Если для показателя не назначен справочник, тогда для него не доступен выбор полей автоопределения по типу.

Предопределенные варианты получения соответствий первичным данным

В справочнике «Варианты получения соответствий первичным данным» предусмотрено два предопределенных элемента, использующих системные настройки получения соответствий:

  • Получение подразделения (ЦФО) по подразделению организации – определяет управленческое подразделение на основании подразделения организации по соответствию, задаваемому в списке справочника «Подразделения (ЦФО)» и хранимому в регистре сведений «Соответствие подразделений и подразделений организаций». Для использования данного варианта в настройках использования соответствия необходимо заполнить два показателя – подразделение организации и организацию, для которых определяется ЦФО;
  • По настройке соответствий для измерений бюджетирования – определяет соответствие для любого разреза бюджетирования на основании регистра сведений «Настройка соответствий для измерений бюджетирования». Для использования данного варианта в настройках использования соответствия необходимо заполнить показатель «Исходное Значение» - значение, для которого ищется соответствие по нужному разрезу бюджетирования.
