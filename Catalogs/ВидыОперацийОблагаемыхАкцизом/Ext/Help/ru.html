Справочник «Операции по движению подакцизных ТМЗ»

Данный справочник предназначен для ведения списка операций по движению подакцизных товаров и материалов, в соответствии с классификацией операций в Декларации по акцизу.

Элементами справочника являются конкретные виды операций по виду подакцизных ТМЗ, например, «Оптовая реализация бензина собственного производства», «Спирт, используемый для производства алкогольной продукции» и т. д.
