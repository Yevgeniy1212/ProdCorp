Справочник "Производственные модели"

Производственные модели настраиваются в одноименном справочнике («Производственные модели»). Для каждой модели указывается источник спецификаций номенклатуры:

·         Плановые спецификации – спецификации для расчета будут взяты из справочника «Плановые спецификации номенклатуры» на основании данных регистра сведений «Основные плановые спецификации номенклатуры»;

·         Бухгалтерские спецификации – спецификации для расчета будут взяты из справочника «Спецификации номенклатуры» (если таковой имеется в конфигурации) на основании данных регистра сведений «Основные спецификации номенклатуры»;

·         Произвольный источник расчета – в этом случае надо будет указать источник данных из справочника «Источники данных для расчетов бюджетирования», в котором должен быть определен показатель «Количество» (количество списываемых полуфабрикатов на единицу производимой продукции), измерения «Номенклатура» (производимая) и «Кор.: номенклатура» (списываемая).

Следует учесть, что при использовании как данных источника, так и готовых спецификаций система учитывает только поля, определяющие списываемую номенклатуру и ее количество на производимую продукцию. В прикладных решениях со спецификациями, имеющими более расширенную настройку, дополнительные возможности спецификаций не будут учтены.

На закладке «Область действия модели» настраивается фильтр на значения разрезов исходного финансового показателя, к которым модель будет применяться.

В поле «Ограничение области применения производственной модели по номенклатуре» указывается одно из следующих значений:

·         Не ограничивать – модель будет применяться без ограничений по номенклатуре финансового показателя;

·         По списку – модель будет ограничена фиксированным списком номенклатуры;

·         По наличию настройки спецификации – модель будет ограничена только теми номенклатурными позициями, для которых имеется настройка спецификации согласно данной модели.

В поле «Ограничение области действия по прочим полям» можно настроить отбор по прочим разрезам планирования.

На закладке «Отражение результатов» перечисляются строки, определяющие отражение результатов расчета по спецификации. Для каждой строки указывается вид комплектующей, списываемой по спецификации в производство:

·         Покупное сырье – комплектующая, для которой не обнаружено спецификации и она считается покупным сырьем;

·         Производимое сырье – комплектующая, для которой обнаружена спецификация, и она считается полуфабрикатом собственного производства;

·         Услуга – комплектующая, для которой установлен признак услуги;

·         Брак при выпуске – учет брака по нормативам при выпуске (количество, определяемое от значения основного финансового показателя, умноженное на норматив брака, если он задан в периодическом регистре сведений «Параметры планирования производства»);

·         Порча при списании – порча списываемых материалов, рассчитанная по нормативам (если задано в регистре сведений «Нормы порчи при списании» в разрезе производимых номенклатур), сверх списываемого по спецификации количества;

Для каждого вида комплектующей может быть создано несколько строк – например, для производимого сырья следует создать одну строку для учета списания, а другую – для учета воспроизводства, указав финансовый показатель, для которого также задана производственная модель. Таким образом, можно организовать расчет производства по произвольному количеству переделов.

В следующих колонках указывается способ заполнения финансовых показателей и прочих разрезов планирования в результатах расчета:

·         Фиксированное значение;

·         По производимой позиции – если разрез следует заполнить из аналогичного разреза исходного показателя производства;

·         По списываемой позиции – из результирующих разрезов. Следует использовать только если необходимо указывать списываемую номенклатуру в одном из разрезов;

·         Производимая номенклатура – если необходимо в одном из разрезов указать производимую номенклатуру, в  производство которой производится списание;

·         Параметры планирования производства – разрез заполняется из периодического регистра сведений «Параметры планирования производства», где для каждой номенклатуры можно указать соответствующие ей параметры, а также процент брака при выпуске;

·        По общему правилу автоматического расчета - для указания необходимости рассчитывать разрез по механизму глобальных правил автоматического расчета;

·        Номенклатурная группа - разрез должен быть заполнен номенклатурной группой той номенклатуры, которая будет определена для данной строки расчета;

·        Номенклатурная группа производимой позиции - разрез должен быть заполнен номенклатурной группой той номенклатуры, производство которой рассчитывается по данной модели;

Если установлен признак «Формировать проводки по управленческому плану счетов», тогда в группе колонок «Шаблон проводок» можно указать шаблон проводок по управленческому плану счетов для строки (если требуется отражение проводок), а также источник аналитики для субконто по дебету и по кредиту. Этот признак можно использовать, если требуется формирование проводок по управленческому плану счетов с различными значениями аналитики по дебету и кредиту.

На закладке «Расчет прямых трудовых затрат» можно настроить формирование планов по прямым трудозатратам на производство с использованием норм, заданных в регистре сведений «Плановые затраты труда на выпуск продукции». На этой закладке необходимо указать финансовый показатель для отражения результатов, а также требуемые разрезы расчета. Список «Разрезы, рассчитываемые по общим правилам автоматического расчета» служит для указания тех разрезов в получаемом расчете трудозатрат, которые следует рассчитывать по механизму глобальных правил автоматического расчета разрезов.

На странице «Дополнительные расчеты» можно указать дополнительные обороты, которые следует формировать не от списываемых позиций, а от производимой. Например, можно использовать для планирования накладных расходов.
