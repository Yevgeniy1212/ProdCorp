Справочник "Шаблоны отражения данных регистров по бюджетам"

Справочник «Классификатор регистров» хранит ссылки на регистры системы, для которых можно настроить отражение проводок по фактическим данным бюджетов. Такие настройки хранятся в данном справочнике. Настройка шаблонов для них идентична настройке шаблонов для документов. Если задан шаблон отражения по регистру, система формирует фактические данные по нему при проведении любых документов, за исключением тех, у которых в позиции справочника «Виды документов» установлен шаблон отражения отличный от предопределенного варианта «По настройкам для регистров». Если для документа настроек в справочнике нет, отражение по его движениям будет выполнено.
