Справочник "Сценарии планирования"

Справочник Сценарии планирования (меню Баланс - Сценарии планирования) хранит перечень используемых сценариев. Сценарий представляет собой возможный вариант развития событий, используемый при планировании. Все бюджеты формируются в разрезе сценариев.

Сценарии предназначены для:

·         параллельного формирования бюджетов различного охвата срока действия;

·         параллельного формирования бюджетов с различными вариантами развития событий;

·         сценарного анализа данных бюджетирования.

Для каждого сценария настраивается структура бюджетного периода и горизонт планирования, а также детализация планирования по сценарию.

На закладке «Основная», в группе «Периоды планирования», настраивается структура бюджетного периода.

·         Периодичность – определяет длительность интервалов планирования (месяц, квартал и т.д.), на которые будет разбит бюджетный период, а следовательно, задает степень детализации планировании;

·         Горизонт планирования - определяет количество интервалов планирования, заданных в поле «Периодичность», которые будут входить в бюджетный период (например, если необходимо формировать бюджеты на квартал с помесячной детализацией, тогда периодичностью будет месяц, а горизонтом планирования - количество месяцев в бюджетном периоде, то есть 3);

·         Использовать точки планирования – позволяет зафиксировать выбираемые в первичных документах по сценарию бюджетные периоды конкретными временными рамками. Если признак установлен, тогда становится доступной закладка «Точки планирования», на которой задаются даты начала бюджетных периодов. Данный механизм используется в том случае, когда необходимо ограничить выбираемые пользователем периоды планирования заранее известным списком. Например, если по сценарию планирование ведется на квартал помесячно и необходимо запретить пользователю указать в качестве бюджетного периода «февраль-апрель», и оставить возможность выбирать только периоды, соответствующие кварталам года. В таком случае в перечень будет записан в список точек планирования, например «январь-март», «апрель-июнь» и т.д.

В группе элементов «Валюта планирования» настраивается работа с валютой при планировании по сценарию.

·         Валюта планирования – валюта, в которой автоматически будет вестись параллельный учет всех планируемых показателей. В данной валюте можно будет просматривать отчеты по плановым показателям в целом по сценарию;

·         Использовать курсы сценария – установка этого признака делает доступной закладку «Курсы валют» и означает, что при планировании по данному сценарию для валютных пересчетов будут использованы не те курсы, которые заданы в регламентированном учете, а собственные курсы данного сценария, установленные на закладке «Курсы валют». Таким образом, система позволяет моделировать различные варианты развития событий при различном поведения курса валюты. При планировании оборотов в различных валютах бывает необходимо подготовить несколько сценариев развития ситуации в зависимости от прогнозируемой динамики изменения курсов валют операций относительно валюты сценария. Например, в ряде случаев планирование осуществляется по курсу, установленному не ЦБ РК, а по внутреннему курсу предприятия, который отличается от официального курса. Если выбирается использование линейки курсов валют, то необходимо будет указывать курсы не только для валюты сценария, но и для валюты управленческого учета и для всех валют, в которых планируется оформлять бюджетные операции. То есть в этом случае курсы валют, указанные в справочнике валют системы, вообще не будут использоваться.

Важно!!! Курс любой валюты в линейке курсов задается по одному значению на период согласно периодичности планирования.

Например, при периодичности планирования по сценарию равной месяцу курсы валют устанавливаются на первое число каждого месяца. Этот курс будет действовать в течение одного периода планирования. Если для какого-то периода курс валюты не будет указан, будет использоваться курс предыдущего периода. Если для сценария не используется своя линейка курсов, данные о курсах валют берутся из общего справочника.

В группе «Прочее» указываются следующие параметры:

·         Детализация планирования. Если для планирования необходимо вести детализацию по номенклатуре, то в сценарии можно указать точность ведения такой детализации — будут ли обороты вестись с точностью до элементов номенклатуры или укрупненно, с точностью до номенклатурных групп. Необходимость ведения планирования по номенклатуре будет указываться отдельно для каждой статьи оборотов.
