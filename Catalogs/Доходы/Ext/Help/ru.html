Справочник "Доходы"

Справочник  предназначен для хранения перечня видов дохода организации.

Для каждого элемента присутствует возможность указания вида деятельности. с которым связано получение этого вида доходов.

Для отражения вида дохода в регламентированной налоговой отчетности используются следующие данные:

  • для Декларации по корпоративному подоходному налогу (форма 100.00) - классификатор кодов строк декларации, соответственно, доходы должны быть детализированы не менее, чем в налоговом учете.
  • для Упрощенной декларации для субъектов малого бизнеса (форма 910.00) - признак "Является объектом налогообложения". Если признак установлен, то доход включается в подсчет общей суммы налогооблагаемого дохода, полученного за налоговый период.
