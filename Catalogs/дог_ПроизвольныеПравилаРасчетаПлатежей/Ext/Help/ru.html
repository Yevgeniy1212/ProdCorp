Справочник "Произвольные правила расчета платежей"

В данном справочнике можно хранить запросы, используемые для расчета. В табличной части "Параметры запроса" указываются значения параметров запроса (либо фиксированные значения, либо поля из контекста расчета). Чтобы передать поле из контекста расчета, нужно установить флажок "Р - Параметр контекста расчета" и в колонке "Значение параметра" можно будет выбрать предопределенное поле. Значения из контекста будут подставляться в момент расчета, т.е. правило может быть общее для всех договоров. Числовое поле запроса с результатом расчета суммы должно иметь имя "СуммаСобытия".
В документе "Регистрация договора" с видами оплаты "По графику" или "Регулярная" можно установить флажок "Автоматическое создание событий для расчетов по графику" и заполнить поле "Правило расчета платежей по событиям" созданным элементом справочника
