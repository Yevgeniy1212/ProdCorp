Справочник «Расчетные показатели»

Список расчетных показателей для использования в источниках данных хранится в справочнике «Расчетные показатели» (меню «Расчеты и зависимости» - «Расчетные показатели»).

У расчетного показателя определяется способ ввода:

·         по мере изменения – значения показателя будут вводиться так часто, как нужно пользователю. До тех пока не будет установлено новое значение, будет использоваться предыдущее, независимо от срока давности;

·         в каждом периоде – значения показателя будут вводиться для каждого планового периода. Если значение в периоде не введено, значение предыдущего периода использовано не будет.

В поле «Разрезы учета» указываются разрезы планирования, по которым будет детализироваться значение показателя при вводе.

Ввод числовых значений расчетных показателей осуществляется в документе «Ввод значений расчетных показателей».
