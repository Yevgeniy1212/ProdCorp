Справочник "Резервы"

Предназначен для хранения информации о начисляемых в бухгалтерском учете резервов.

Реквизит "Вид резерва" предназначен для указания того, под какие затраты в будущем создается резерв.

Для резервов по вознаграждениям работникам в табличной части "Базовые виды расчета" необходимо задать список видов начислений заработной платы, которые будут выступать в качестве базы для расчета ежемесячных отчислений в резерв.
