Справочник Агрегаты статей оборотов по бюджетированию

Элемент данного справочника представляет собой некоторое объединение нескольких финансовых показателей, обладающих некоторыми общими учетными признаками и имеющих сходные зависимости. В группе полей «Признаки учета» настраиваются признаки и разрезы учета по показателю. В него можно будет добавить только те финансовые показатели, которые имеют указанные признаки/разрезы.

Справочник используется для групповой установки зависимостей между статьями оборотов.
