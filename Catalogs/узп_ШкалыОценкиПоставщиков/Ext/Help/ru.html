Справочник "Шкалы оценки поставщиков"

Справочник "Шкалы оценки поставщиков" содержит перечень шкал оценки поставщиков, используемых при расчете критериев оценки поставщиков со способом расчета оценки "Расчет по шкале".

Для шкалы имеется возможность указать одно из четырех типов значений шкалы: число, булево, дата, значения критериев оценки.

Заполнить шкалу можно непосредственно из элемента справочника "Шкалы оценки поставщиков", для этого необходимо нажать кнопку "Открыть регистр". Заполнение шкалы производится в регистре сведений "Значение шкал оценки поставщиков".
