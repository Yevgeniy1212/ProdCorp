Справочник "Правила автоматического расчета разрезов бюджетирования"

Справочник позволяет настроить правила расчета зависимых разрезов бюджетирования, которые могут быть определены на основании значения исходного разреза. Например, номенклатурная группа на основании указанной номенклатуры, контрагент на основании указанного договора и т.д.

На созданную настройку можно будет ссылаться в различных расчетных механизмах системы, таким образом сокращается объем локальных настроек в отдельных механизмах системы. Для того, чтобы правило использовалось в качестве "глобального", необходимо указать его основным для разреза, создав настройку в регистре сведений "Применение правил автоматического расчета разрезов".

В правиле указывается:

  • Рассчитываемый разрез - разрез, который должен рассчитываться на основании другого;
  • Исходный разрез - разрез, данные которого будут использоваться для расчета;
  • Вариант расчета - правило расчета результирующего значения:
    • Копирование значения - значение рассчитываемого разреза должно быть равно исходному;
    • Владелец - если исходный разрез подчинен рассчитываемому, то определяется владелец указанного в разрезе элемента (например, контрагент, договор которого указан);
    • Номенклатурная группа - для настройки связи между разрезом "Номенклатура" и разрезом, хранящим номенклатурные группы;
    • Указанное поле - рассчитываемый разрез берется из определенного реквизита исходного разреза. Этот реквизит указывается в поле "Поле исходного разреза";
    • Соответствие - можно будет заполнить список значений исходного разреза и соответствующих ему значений результирующего.
