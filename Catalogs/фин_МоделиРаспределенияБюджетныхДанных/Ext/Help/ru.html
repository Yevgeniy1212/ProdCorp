Справочник "Модели распределения бюджетных данных"

Справочник позволяет настроить правила расчета финансовых показателей, значения которых должны являться результатом распределения исходных показателей по выбранной базе распределения. Например, сюда могут относиться результаты распределения накладных расходов на основные затраты и т.д.

Показатели, которые будут подлежать распределению, настраиваются в регистре сведений «Расчет моделей распределения».

На закладке «Область действия модели» настраивается фильтр на значения разрезов исходного финансового показателя, к которым модель будет применяться.

База распределения

На закладке «База распределения» определяются правила получения данных, которые будут выступать в качестве базы распределения. Здесь указывается список финансовых показателей, входящих в базу распределения. В колонке «Данные показателя» указывается, будет ли в качестве базы использоваться сумма или количество, введенные по базовому показателю.

Настройка «Вариант отбора аналитики базы распределения» определяет порядок фильтрации данных по указанным финансовым показателям для их использования в качестве базы распределения:

  • Без отбора - в базу распределения попадают все данные по финансовым показателям, независимо от значений разрезов планирования;
  • Соответствует распределяемым данным - в базу распределения для каждой распределяемой суммы по исходному финансовому показателю попадают только те данные по финансовым показателям, комбинация плановых разрезов которых соответствует комбинации плановых разрезов исходной суммы;
  • Индивидуальная настройка - в этом случае становятся доступными колонки с разрезами и для каждого разреза можно будет настроить индивидуальное правило фильтрации;

Результаты распределения

На закладке «Отражение результатов» перечисляются строки, определяющие отражение результатов расчета по модели. В том случае, если результат распределения очередной суммы по исходному финансовому показателю необходимо отражать целиком по одному результирующему показателю, в этом списке будет одна строка. Если необходимо результат распределения по базе, в свою очередь, распределить на несколько финансовых показателей по фиксированным коэффициентам, строк будет несколько. База распределения будет определяться значением колонки «Коэффициент».

Настройка «Источник аналитики для результирующих строк» определяет порядок заполнения значений плановых разрезов в результирующих строках:

  • Из распределяемых данных - все плановые разрезы будут заполняться на основании значений разрезов распределяемых данных;
  • Из базы распределения - все плановые разрезы будут заполняться на основании значений разрезов из базы распределения;
  • Индивидуальная настройка - в таблице будут отображены колонки с плановыми разрезами, в которых для каждого разреза можно будет настроить индивидуальное правило:
    • Из распределяемых данных;
    • Из базы распределения;
    • Фиксированное значение - значение, указанное пользователем;
    • По общему правилу автоматического расчета - если для разреза настроено общее правило расчета на основании данных влияющего разреза;

Если установлен признак «Формировать проводки по управленческому плану счетов», тогда в группе колонок «Шаблон проводок» можно указать шаблон проводок по управленческому плану счетов для строки (если требуется отражение проводок), а также источник аналитики для субконто по дебету и по кредиту. Этот признак можно использовать, если требуется формирование проводок по управленческому плану счетов с различными значениями аналитики по дебету и кредиту.
