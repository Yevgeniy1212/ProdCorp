Справочник "Национальности народов мира"

Справочник содержит перечень национальностей.  Используется для заполнения значения реквизита "Национальность" справочника «Физические лица», а также возможно использование при формирование вопросов анкетирования («Вопросы для анкетирования») .

Справочник может быть заполнен по данным из общего классификатора национальностей народов мира. Для этого в форме списка имеется кнопка «Подбор». При этом открывается список классификатора, в котором можно найти, выбрать и занести в справочник нужную национальность.
