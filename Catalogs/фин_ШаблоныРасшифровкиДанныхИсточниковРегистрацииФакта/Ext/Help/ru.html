Справочник "Шаблоны расшифровки данных источников регистрации факта"

Справочник используется для настройки расшифровки данных источников информационной базы в отчете "Контрольная ведомость отражения фактических данных по бюджетам". Позволяет настроить состав полей, выводимых в расшифровку, и их расположение. Для каждого источника применяется только одна настройка.

После выбора источника можно заполнить настройку всеми его полями автоматически, при нажатии на кнопку "Автозаполнение структуры".

В списке "Группы колонок" настраивается перечень колонок, расположение подчиненных колонок по вертикали/горизонтали, и, при необходимости, максимальная ширина. В списке "Колонки" настраивается список полей источника, выводимых в колонку, выбранную в данный момент в списке "Группы колонок".
