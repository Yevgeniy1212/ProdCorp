Справочник "Классификатор видов расходов периода"

Конфигурация   "1С-Рейтинг: Налоговый мониторинг".

 

Справочник содержит перечень видов расходов периода, необходимых для формирования отчета "Расходы периода" настройки "1С-Рейтинг: Налоговый мониторинг", которые заполняются значениями по умолчанию при помощи кнопки "Заполнить". Элементы справочника устанавливаются в соответствие элементам статей затрат при помощи регистра сведений "Соответствия статей затрат". Справочник вызывается для ввода и редактирования информации из пункта меню «Мониторинг» / «Классификаторы мониторинга» / «Классификатор видов расходов периода».

В данный справочник заносится информация о виде расходов периода, его расшифровке в виде строки.
