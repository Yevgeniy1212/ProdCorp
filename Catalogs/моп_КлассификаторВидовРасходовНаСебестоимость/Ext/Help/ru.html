Справочник "Классификатор видов расходов на себестоимость"

Настройка "1С-Рейтинг: Налоговый мониторинг".

Справочник содержит перечень видов расходов на себестоимость, необходимых для формирования ряда отчета "Форма 1.6. Себестоимость произведенной продукции (работ, услуг) (ХС)" настройки "1С-Рейтинг: Налоговый мониторинг", которые заполняются значениями по умолчанию при помощи кнопки "Заполнить". Элементы справочника устанавливаются в соответствие элементам статей затрат при помощи регистра сведений "Соответствия статей затрат". Справочник вызывается для ввода и редактирования информации из пункта меню «Мониторинг» / «Классификаторы мониторинга» / «Классификатор видов расходов на себестоимость».

В данный справочник заносится информация о коде вида расходов на себестоимость, его расшифровке в виде строки.
