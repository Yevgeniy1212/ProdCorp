Карточка перехода

Карточка перехода позволяет редактировать его свойства. Если результат редактирования необходимо сохранить, то по окончании внесения изменений необходимо нажать кнопку «ОК».

Указываются следующие параметры:

·         Идентификатор – программное имя этапа. Создается без пробелов. Если заполнить сначала представление, то идентификатор будет сформирован автоматически;

·         Представление – текстовое представление перехода в карте маршрута;

·         Список «Предшествующие этапы» хранит список этапов, от которых производится переход на текущий. Здесь могут быть указаны только этапы и переходы с меньшим уровнем. Если предшествующий этап – также переход, то в колонке «Условие» указывается одно из условий перехода, при котором предшествующий переход переключает маршрут на данный переход. Если условие не указано, то переход будет считаться безусловным (т.е. если не выполнено ни одно из условий перехода);

·         Условные переходы – список ветвлений перехода. В колонке «Условие» указывается условие, созданное для текущего вида документа, а в колонке «Этап» указывается этап или другой переход, на который переключается маршрут при выполнении условия. Можно выбирать только этапы и переходы с уровнем больше текущего. В данном списке должен обязательно присутствовать безусловный переход, т.е. строка без указания условия, которая определяет ветку маршрута при невыполнении ни одного из указанных условий.
