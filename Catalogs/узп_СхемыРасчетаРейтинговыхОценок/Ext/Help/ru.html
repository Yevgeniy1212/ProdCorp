Справочник "Схемы расчета рейтинговых оценок поставщиков"

Справочник "Схемы расчета рейтинговых оценок поставщиков" хранит наименования используемых схем расчета.

Схема расчета - это набор каких-либо критериев для оценки с установкой веса (приоритета) для каждого критерия. То есть некоторые критерии имеют больший вес в данной схеме расчета, чем другие.

Настройка схем расчета представляет собой настройку двух объектов конфигурации: справочник "Схемы расчета рейтинговых оценок поставщиков" и документ "Настройка схем расчета рейтинговых оценок поставщиков".

Для того, чтобы произвести настройку схемы необходимо перейти к заполнению схемы по кнопке "Настроить схему" формы элемента справочника "Схемы расчета рейтинговых оценок поставщиков".
