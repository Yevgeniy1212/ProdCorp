Справочник "Наборы шаблонов отражения первичных данных в бюджетах"

В элементе данного справочника указывается вид документов, к которому он может применяться. В списке «Состав шаблонов набора» приводится список отдельных шаблонов интерактивного отражения, входящих в состав набора шаблонов (схемы отражения). Настройка и назначение отдельных шаблонов рассматривается далее. Для каждого отдельного шаблона указывается способ его использования в колонке «Способ использования»:

  • Всегда – шаблон из данной строки будет использоваться в любом случае;
  • Всегда, если удовлетворяет условию - шаблон из данной строки будет использоваться только в том случае, если условие, указанное в поле «Условие применения шаблона», выполняется для конкретного документа. Прочие параметры не влияют на применение шаблона из данной строки. Задание условия обязательно;
  • Если удовлетворяет условию - шаблон из данной строки будет использоваться только в том случае, если условие, указанное в поле «Условие применения шаблона», выполняется для конкретного документа, и к документу не применен ни один из шаблонов с таким же способом использования. Таким образом, если для документа удовлетворяет условию один из шаблонов с таким же способом использования, то применение прочих шаблонов с аналогичным способом использования не проверяется и не осуществляется. То есть из шаблонов с таким способом использования выбирается первый по порядку, который удовлетворяет условию. Задание условия обязательно;
  • Если не найден удовлетворяющий условиям шаблон – применяется только если не найден ни один подходящий шаблон из шаблонов со способом использования «Если удовлетворяет условию». Если шаблонов со способом использования «Если удовлетворяет условию» не задано, то такой шаблон будет применяться всегда.
