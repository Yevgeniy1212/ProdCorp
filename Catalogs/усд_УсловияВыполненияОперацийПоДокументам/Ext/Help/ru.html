Справочник "Условия выполнения операций по документу"

Условие выполнения операций по документам представляет собой правила проверки значений  реквизитов шапки выбранного документа, при соблюдении которых условие считается выполненным.

Перечень условий по документам, применяемых для настройки шаблонов, ведется в справочнике Условия выполнения операций по документу (меню Учет фактических данных - Условия выполнения операций по документу).

·         Вид документов – вид документов из одноименного справочника, к которому будет применяться условие;

·         Все составные части условия обязательны – если признак установлен, тогда для истинности условия потребуется выполнение всех его составных частей, иначе потребуется выполнение хотя бы одной из составных частей условия;

·         Составные части условия – список условий на данные документа, входящих в состав основного условия. Каждая строка списка редактируется в отдельном окне:

В окне редактирования условия заполняются следующие реквизиты:

·         Реквизит – реквизит документа, значение которого сравнивается с требуемым значением;

·         Режим сравнения – определяет способ получения значений для сравнения с реквизитом документа:

o    Сравнение с полем регистра сведений – для сравнения со значением используется ресурс регистра сведений, получаемый по соответствию измерений регистра определенным реквизитам документа;

o    Сравнение с произвольным значением - для сравнения со значением используется фиксированное введенное значение;

o    Сравнение с полем произвольного источника - для сравнения со значением используется поле запроса, полученного на основании произвольного текста и данных документа;

·         Вид сравнения – определяет вид сравнения реквизита документа с полученным значением – равно, не равно, больше и т.д. Для сравнения с произвольным значением доступны режимы сравнения «в списке», «не в списке», «в списке по иерархии» и «не в списке по иерархии».

В зависимости от режима сравнения ниже пользователь видит различные по составу наборы полей.

Для сравнения с полем регистра сведений:

·         Источник данных – выбирается регистр сведений из списка регистров;

·         Поле источника для сравнения – ресурс, выступающий как значение для сравнения с реквизитом документа;

·         Поле источника (верхняя граница) – ресурс, выступающий как верхнее значение для сравнения с реквизитом документа (доступен для видов сравнения, работающих с диапазоном значений);

·         Связь документа и регистра – соответствие реквизитов документа и измерений регистра сведений для определения ресурса. Если регистр периодический, то значение берется на дату документа. Если связь между документом и регистром задана неоднозначно, и по указанному соединению может быть найдено несколько записей, то будет использовано первое из полученных значений ресурсов;

Для сравнения с произвольным значением:

·         Значение – фиксированное значение, выступающее как значение для сравнения с реквизитом документа;

·         Верхняя граница – фиксированное значение, выступающее как верхнее значение для сравнения с реквизитом документа (доступен для видов сравнения, работающих с диапазоном значений);

·         Список значений для сравнения – список фиксированных значений, используемых для сравнения с реквизитом документа в случае применения одного из списочных видов сравнения.

Для сравнения с полем произвольного источника:

С помощью конструктора строится запрос. При помощи кнопки Проверить запрос предусмотрена возможность определить корректность построенного запроса. После нажатия этой кнопки становятся доступными для выбора в реквизитах «Поле источника» и «Поле источника (верхняя граница)» поля созданного запроса. При составлении запроса можно в качестве параметров использовать имена реквизитов и свойств документа, так как все реквизиты и свойства будут переданы в выполняемые запросы при проверке применительно к конкретному документу из проверяемого документа.

После редактирования части условия необходимо нажать кнопку ОК.

В форме элемента присутствует кнопка Проверка исполнения. При нажатии на данную кнопку открывается обработка проверки выполнимости условий по документам (помощник настройки условий).
