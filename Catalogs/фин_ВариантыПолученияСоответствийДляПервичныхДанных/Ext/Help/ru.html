Справочник "Варианты получения соответствий первичным данным"

Данный справочник используется в подсистеме бюджетирования.

Используется непосредственно для описания способа вычисления соответствия (конкретного значения аналитического разреза бюджетирования) на основании абстрактного набора показателей.

Соответствие первичным данным представляет собой правило автоматического расчета значения выбранного аналитического разреза бюджетирования на основании первичных значений.
Данные правила используются при интерактивном отражении фактических данных по бюджетам, для того, чтобы получить недостающую информацию об отражении документа в бюджетах.
Расчет соответствия производится на основании определенного набора входящих показателей. Для связи же первичных данных с таким расчетом используется промежуточное соответствие реквизитов документа и входящих показателей для расчета.
Такая двухуровневая настройка обусловлена тем, что определенное правило расчета может быть общим для всех документов, но реквизиты, из которых берется значение для расчета, могут по-разному называться в различных документах.

Настройка

Для каждого варианта получения соответствий указывается аналитический разрез бюджетирования (в поле «Разрез»), значение которого будет определяться при помощи данного варианта.

В поле «Способ поиска» указывается способ расчета соответствия по данному варианту.

На закладке «Входящие показатели» приводится перечень входящих показателей расчета (для некоторых способов поиска можно задать только один показатель). В колонке «Показатель» вводится текстовое представление показателя, без пробелов. В колонке «Тип» указывается, должен ли входящий показатель быть числового типа, логического, либо представлять собой элемент справочника или другого объекта системы. Если показатель ссылочного типа, то в колонке «Справочник/перечисление» выбирается конкретный объект информационной базы, значения которого могут передаваться в данный показатель. Для расчета должен быть задан хотя бы один показатель.

Способы поиска, задаваемые в поле «Способ поиска», уточняются на закладке «Настройки расчета», и могут быть следующими:

·         «По списку» - для входящего показателя задается фиксированный список возможных значений показателя и соответствующие им значения определяемого аналитического разреза.

Такой список соответствий задается на закладке «Настройки расчета». Для отсутствующих в списке значений входящего показателя результат поиска будет пустым;

·         «По шкале» - для первого (порядок важен, остальные показатели не используются) показателя из списка показателей (который должен быть обязательно числовым) задается шакала интервалов возможных значений показателя и соответствующие им значения определяемого аналитического разреза.

Шкала соответствий задается на закладке «Настройки расчета». Для значений входящего показателя, не охваченных интервалами, результат поиска будет пустым;

·         «По формуле» - для списка показателей (которые должны быть обязательно числовыми или булево) задается формула. Предназначен только для расчета реквизитов бюджетирования «Количество» и «Сумма».

Формула  расчета задается на закладке «Настройки расчета». В формуле могут быть использованы показатели варианта. Формула редактируется при нажатии на кнопку «Формула» справа от поля текста формулы.

В окне ввода формулы, в нижней части приведен перечень показателей варианта, которые могут быть добавлены в текущую позицию курсора в поле ввода формулы при двойном щелчке по строке показателя. Функции добавляются в текущую позицию курсора в поле ввода формулы при нажатии на соответствующую кнопку. Для проверки условий используется команда ?(<условие>, <значение если условие выполнено>, <значение если условие не выполнено>). Аргументы функций калькулятора описаны во всплывающих подсказках к соответствующим кнопкам;

·         «По регистру соответствий» - позволяет обратиться к любому регистру сведений системы, который может использоваться как список соответствий исходным показателям.

Настройка получения соответствий задается на закладке «Настройки расчета». В поле «Регистр соответствий» выбирается регистр сведений (не подчиненный регистратору). В реквизите «Поле регистра - результат» - выбирается ресурс регистра сведений, тип которого соответствует типу вычисляемого разреза аналитики бюджетирования. Найденное значение данного ресурса будет передано как значение искомого разреза аналитики. Поля регистра, тип которых не соответствует типу разреза аналитики, недоступны в списке выбора.

В списке «Настройка связи с регистром» указывается связь измерений регистра и входящих показателей варианта, устанавливающих отбор по соответствующим им измерениям. Если для показателя настроено ограничение справочником, тогда для него возможно выбрать только измерение регистра с соответствующим типом. Если регистр сведений периодический, то значение ищется на дату проводимого документа. Если соответствие не найдено, возвращается пустое значение.

Вариант получения соответствий не зависит от структуры определенного документа и является универсальным.
