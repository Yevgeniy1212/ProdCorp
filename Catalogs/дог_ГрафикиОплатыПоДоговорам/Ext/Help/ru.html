Справочник "Графики оплаты по договорам"

Справочник предназначен для формирования графиков оплаты по договорам. Для каждого элемента заполняются Наименование и табличная часть, содержащая расписание оплаты по договорам. В табличной части заполняются:

  • Расчет даты оплаты:
    • Порядок расчета даты платежа - устанавливается порядок расчета даты платежа (фиксированная дата/ от даты события по договору)
    • Дата оплаты/Событие - дата оплаты, либо событие по которому будет произведена оплата
    • Отсрочка, дней - количество дней отсрочки от момента наступления события(только при порядке расчета платежа - "От даты события по договору")
  • Этап оплаты - выбирается из справочника "Этапы оплаты"(в одном графике не должно быть повторяющихся этапов оплаты)
  • Расчет суммы платежа:
    • Способ расчета суммы платежа - устанавливается способ расчета суммы платежа:
      • Фиксированной суммой:
        • Сумма - сумма платежа
      • Задолженность на дату платежа:
        • Процент от суммы - процент от суммы задолженности, который будет погашен при оплате
        • Минимальная сумма платежа - устанавливается минимальная сумма платежа
      • По задолженности на начало периода:
        • Периодичность - выбирается периодичность, на начало которой вычисляется сумма задолженности
        • Процент от суммы - процент от суммы задолженности, который будет погашен при оплате
        • Минимальная сумма платежа - устанавливается минимальная сумма платежа
      • Плановая сумма по бюджетам:
        • Сценарий - сценарий планирования, по которому будет расчитана сумма платежа
        • Дата сценария - дата начала периода планирования сценария,
        • Финансовый показатель для расчета - финансовый показатель, по которому будет расчитана сумма платежа
        • Процент от суммы - процент от суммы задолженности, который будет погашен при оплате
        • Минимальная сумма платежа - устанавливается минимальная сумма платежа
  • Финансовый показатель - финансовый показатель, устанавливаемый в формируемые документы по умолчанию
