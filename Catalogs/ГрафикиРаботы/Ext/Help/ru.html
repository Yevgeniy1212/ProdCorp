Справочник «Графики работы»

Справочник содержит перечень всех графиков работы предприятия и предназначен для целей учета рабочего времени по принципу отклонений.

График хранит в себе информацию об объемах рабочего времени по дням и/или сменам. Графики настраиваются и заполняются в форме элемента справочника.

График может быть одного из следующих типов: пятидневный, шестидневный и сменный.

Настройка графика осуществляется через Помощник заполнения, который позволяет выбрать способ заполнения графика: "Заполнить по шаблону" либо "Настроить вручную".

  1. Способ заполнения "Заполнить по шаблону". Используя данный способ нужно выбрать готовый шаблон для заполнения графика: Пятидневка (40-час.раб.неделя), Шестидневка (40-час.раб.неделя), Сутки через двое (40-час.раб.неделя), Сутки через трое (40-час.раб.неделя). При нажатии на кнопку "Заполнить" произойдет заполнение графика в соответствии с выбранным шаблоном.  
    • Пятидневка (40-час.раб.неделя) - стандартная 40-часовая пятидневная рабочая неделя, 8- часовой рабочий день, выходной: суббота, воскресенье.
    • Шестидневка (40-час.раб.неделя) - 40-часовая шестидневная рабочая неделя, 7-часовой рабочий день (пон. - пт.), 5-часовой рабочий день (сб.), выходной: воскресенье.
    • Сутки через двое (40-час.раб.неделя) - 40-часовая рабочая неделя, суммированный учет рабочего времени.
    • Сутки через трое (40-час.раб.неделя) - 40-часовая рабочая неделя, суммированный учет рабочего времени.
  2. Для шаблонов "Сутки через двое " и "Сутки через трое" имеется возможность вести учет ночных часов. При установке признака "Вести учет ночных часов" начало дневных часов будет считаться с 06-00ч., начало ночных часов с 22-00ч.
  3. Также для шаблонов "Сутки через двое " и "Сутки через трое" необходимо установить "Дату начала отсчета периодичности сменного графика". По умолчанию дата устанавливается на 01 января.
  4. Способ заполнения "Настроить вручную". Данный способ используется для просмотра настроек шаблонных графиков, а также для ручной настройки необходимых параметров для заполнения графика.

Настройка графика:

Описание признаков "Суммированный учет рабочего времени", "Используется совместно с план-графиком", "Вести учет ночных часов", "Учитывать праздничные дни", "Неполное или сокращенное рабочее время":

  • Признак "Суммированный учет рабочего времени". Суммированный учет рабочего времени применяется в непрерывно действующих производствах, цехах, участках и на некоторых видах работ, где по условиям производства (работы) не может быть соблюдена установленная для данной категории работников ежедневная или еженедельная продолжительность рабочего времени.
  • Признак "Используется совместно с план-графиком". Используется в ситуациях замещения одного работника другими за счет смещений производственных смен, когда должен был работать работник. В этом случае удобно указать график работы как используемый совместно с план-графиком (установить соответствующий флажок). При этом заполнять такой график следует наиболее распространенной схемой рабочего времени. Этот график будет рассматриваться как основной график работника, при необходимости ему можно будет вести индивидуальный график за месяц с помощью документа «План-график учета рабочего времени», данные которого будут использоваться при расчетах регламентированной зарплаты.
  • Признак "Вести учет ночных часов". Устанавливается в случае, если график предусматривает работу в ночное время, и часы работы в ночное время оплачиваются работникам в повышенном размере.
  • Признак "Учитывать праздничные дни". Используется в случаях, когда необходимо, чтобы при заполнении календаря графика учитывались праздничные дни по производственному календарю: смена, выпавшая на праздничный день в этом случае будет нерабочей.
  • Признак "Неполное или сокращенное рабочее время". Признак устанавливается, если график предусматривает работу в режиме неполного или сокращенного рабочего времени.  Графиком может быть предусмотрено сокращение продолжительности ежедневной работы на определенное количество рабочих часов во все дни недели, либо сокращение рабочих дней в рабочей неделе. При использовании неполного рабочего времени можно указать график для расчета нормы времени: график соответствующего ему полного рабочего времени. В таком случае при расчете начислений по тарифным ставкам отработанное время будет рассчитано по назначенному сотруднику графику неполного рабочего времени, а норма времени – по тому графику, который будет задан как график полного рабочего времени. Таким образом, назначенная сотруднику тарифная ставка будет соответствующим образом уменьшена при расчете.
