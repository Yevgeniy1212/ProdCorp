Справочник "Классификатор кодов ТН ВЭД"

Конфигурация   "1С-Рейтинг: Налоговый мониторинг".

 

Справочник содержит перечень кодов ТН ВЭД, необходимых для формирования ряда отчетов конфигурации "1С-Рейтинг: Налоговый мониторинг". Элементы справочника устанавливаются в соответствие элементам номенклатуры, нематериальных активов и основных средств в регистре сведений Объекты мониторинга. Справочник доступен из пункта меню «Мониторинг» / «Классификаторы мониторинга» / «Классификатор кодов ТН ВЭД».

Справочник может быть заполнен автоматически, с помощью кнопки подменю "Заполнить", в котором доступны следующие пункты:

  • Заполнить по классификатору ТН ВЭД таможенного союза (www.tsouz.ru) - справочник будет заполнен по классификатору, утвержденному комитетом таможенного союза и опубликованному на портале http://www.tsouz.ru/.
  • Заполнить по классификатору ТН ВЭД ИС МОП НП и ИС СОНО - справочник будет заполнен по классификатору, поставляемому с ИС МОП и ИС СОНО.
