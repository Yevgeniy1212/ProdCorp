Справочник "Фонды выплат организаций"

Справочник содержит список фондов, из которых выплачивается заработная плата работникам организаций. Признак "Фонд заработной платы" служит для указания тех фондов, которые входят в фонд заработной платы организаций в соответствии с Инструкцией по заполнению статистической отчетности по труду, утвержденной Приказов Председателя Агентства Республики Казахстан по статистике от 12 июля 2002 года N 38-Г Об утверждении Инструкции по заполнению статистической отчетности по труду.
