Справочник "Виды трудовых отпусков"

Справочник содержит список видов трудовых отпусков, предоставляемых работникам организаций.

Реквизиты справочника:

  • Вид начисления - вид расчета из основных начислений организаций, которым вводится оплата данного вида отпуска;
  • Очередность предоставления трудового отпуска работникам - число, задающее порядок предоставления данного вида отпуска в пределах отпусков за один рабочий год. Первыми предоставляются виды отпуска, имеющие меньшее значение очередности.

В табличной части "Исключаемые из стажа состояния" указываются состояния работников, время нахождения в которых исключается из общего трудового стажа, дающего права на данный вид отпуска. Например, если за время нахождения в отпуске по уходу за ребенком не предоставляется данный вид трудового отпуска, то в табличную часть "Исключаемые из стажа состояния" необходимо добавить состояние "Отпуск по уходу за ребенком".

Указание размера предоставляемых отпусков для различных видов отпуска производится в регистре сведений "Предоставляемые отпуска организаций".
