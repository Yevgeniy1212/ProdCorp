Регистр "Налог на добавленную стоимость"

Регистр предназначен для учета налога на добавленную стоимость, начисленного при реализации товаров (работ, услуг)

    Данные записи регистра:

  • Налоговый агент - организация, либо подразделение организации, в случае, если оно является самостоятельным плательщиком НДС.

  • Вид реализации - вид операции по реализации в Декларации по НДС

  • Вид налога НДС - тип налога на добавленную стоимость - НДС, НДС по импорту, НДС с измененным сроком уплаты, НДС методом зачета.

  • Ставка НДС - ставка налога на добавленную стоимость

  • Контрагент - покупатель ТМЗ, работ, услуг

  • ТМЗ - наименование реализуемого товара, работы, услуги

     

    На основании данных регистра заполняются Декларация по НДС, форма 300.00, отчет по реализации

  • Сумма реализации без НДС

  • Оборот по реализации

  • Сумма НДС
