Регистр накопления "ВОСМС подлежит перечислению в фонды"

Регистр предназначен для учета взносов на обязательное медицинское страхование, подлежащих перечислению в пенсионные фонды. Используется для отражения в регламентированной налоговой отчетности суммы ВОСМС, подлежащей перечислению, а также для расчета пени за несвоевременное перечисление ВОСМС.

 

Измерения регистра:

  • Системное поле Период - определяет дату регистрации взаиморасчетов;
  • Физическое лицо - физическое лицо, к которому относится запись регистра;
  • Структурная единиц - обособленное подразделение или организация, определяющая структурную единицу, по которой будут отражены взаиморасчеты в налоговых отчетах;

  • Месяц налогового периода - месяц, за который исчислялись ВОСМС;

  • Месяц выплаты доходов - месяц, в котором были выплачены доходы, с которых исчислены ВОСМС. Используется для определения сроков перечисления при расчете пени за несвоевременное перечисление ВОСМС;

  • Организация - организация, в которой осуществляются расчеты с работником;
  • Налогоплательщик - организация - налоговый агент, предоставляющая налоговую отчетность по ВОСМС.

 

Ресурсы регистра:

  • Взнос - сумма ВОСМС.

Реквизиты регистра:

  • Вид строки - характеристика записи регистра, определяющая в результате какой операции (удержания, перечисления, возврата) введена текущая строка регистра.

 

Приход по регистру осуществляется при удержании (отнесении ВОСМС к суммам подлежащим перечислению), а также возврате ошибочно перечисленных взносов. Расход по регистру производится при перечислении ВОСМС.
