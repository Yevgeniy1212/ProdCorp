Регистр "Акциз"

Регистр предназначен для учета акциза, начисленного при осуществлении операций, облагаемых данным видом налога

    Данные записи регистра:

  • Налоговый агент - организация, либо подразделение организации, в случае, если оно является самостоятельным плательщиком НДС.

  • Вид операции - вид операции, облагаемой акцизом,  в соответствии с Декларацией по акцизу

  • Ставка акциза - ставка акциза, на единицу измерения налоговой базы

  • ТМЗ - наименование подакцизного товара

На основании данных регистра заполняются Декларация по акцизу, форма 400.00

  • Количество

  • Сумма акциза
