Регистр "Взаиморасчеты с бюджетом по НДС с измененным сроком уплаты"

Регистр предназначен для учета взаиморасчетов с бюджетом по налогу на добавленную стоимость, срок уплаты которого при импорте товаров был изменен в соответствии со ст. 249 НК РК

Данные записи регистра:

  • Налоговый агент - организация, либо подразделение организации, в случае, если оно является самостоятельным плательщиком НДС.

  • Вид  НДС - тип налога на добавленную стоимость - НДС, НДС по импорту, НДС с измененным сроком уплаты, НДС методом зачета.

  • Вид поступления - вид операции поступления в Декларации по НДС

  • Вид налога НДС - тип налога на добавленную стоимость - НДС, НДС по импорту, НДС с измененным сроком уплаты, НДС методом зачета.

  • Контрагент - поставщик ТМЗ, работ, услуг

  • ГТД - грузовая таможенная декларация

  • Срок платежа - срок (измененный), установленный для погашения налога
  • Сумма НДС - сумма налога на добавленную стоимость

    На основании данных регистра заполняется приложение 300.07 к Декларации по НДС и дополнительные формы 300.07.001 и 300.07.002
