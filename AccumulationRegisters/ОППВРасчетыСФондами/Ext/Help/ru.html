Регистр накопления "ОППВ расчеты с фондами"

Регистр предназначен для учета взаиморасчетов по обязательным профессиональным пенсионным взносам. Данные регистра детализируют информацию на счете 3220 бухгалтерского учета.

Измерения регистра:

  • Системное поле Период - определяет дату регистрации взаиморасчетов;
  • Физическое лицо - физическое лицо, к которому относится запись регистра;
  • Структурная единиц - обособленное подразделение или организация, определяющая структурную единицу, по которой будут отражены взаиморасчеты в налоговых отчетах;

  • Месяц налогового периода - месяц, за который исчислялись обязательные профессиональные пенсионные взносы;

  • Вид платежа - вид платежа по пенсионным взносам (непосредственно взносы, пени, штрафы и т.д.);

  • Организация - организация, в которой осуществляются расчеты с работником;
  • Налогоплательщик - организация - налоговый агент, предоставляющая налоговую отчетность по обязательным профессиональным пенсионным взносам.

Ресурсы регистра:

  • Взнос- сумма профессиональных пенсионных взносов.

Реквизиты регистра:

  • Вид строки - характеристика записи регистра, определяющая в результате какой операции (исчисления, перечисления, возврата) введена текущая строка регистра;
  • Дата начала и Дата окончания - используются при расчете пени за несвоевременное перечисление профессиональных пенсионных взносов, определяют период, за который рассчитывается пени.

Приход по регистру осуществляется при исчислении и возврате ошибочно перечисленный сумм пенсионных взносов. Расход по регистру производится при перечислении взносов в фонды.

Для анализа взаиморасчетов по обязательным профессиональным пенсионным взносам предназначен отчет "Структура задолженности организаций по ОППВ".
