Регистр накопления "Мониторинг экспорта"

Настройка "1С-Рейтинг: Налоговый мониторинг".

Регистр содержит данные, необходимые для формирования следующих форм отчетов:

 

Мониторинг крупных предприятий, хозяйствующих субъектов:

   - Экспортный валютный контроль (форма 113);

 

Мониторинг крупных предприятий, субъектов рынка нефтепродуктов:

   - Экспортный валютный контроль (форма 800);

 

Заполняется при помощи документа "Движения мониторинга".
