Регистр накопления "Взаиморасчеты организаций с получателями по ИЛ"

Регистр предназначен для учета расчетов с получателями удержаний по исполнительным листам. Данные регистра детализируют информацию на счете 3385 в бухгалтерском учете.

 

Измерения регистра:

  • Получатель - контрагент - получатель удержаний по исполнительным листам;
  • Структурная единиц - обособленное подразделение или организация, определяющая структурную единицу, по которой будет отражен доход в налоговых отчетах;
  • Документ-основание - документ "Исполнительный лист", определяющий по какому исполнительному листу осуществляются расчеты с получателем.

  • Период взаиморасчетов - месяц, в котором были произведены удержания в пользу получателя.

  • Организация - организация, в которой осуществляются расчеты с получателем удержаний;
  • Налогоплательщик - организация - налоговый агент, предоставляющая налоговую отчетность по индивидуальному подоходному налогу;

 

Ресурсы регистра:

  • Сумма взаиморасчетов - сумма удержания, направляемая получателю;
  • Сумма сборов - сумма почтовых/комиссионных сборов, удерживаемая с работника для компенсации затрат на почтовые/банковские переводы.

 

Приход по регистру осуществляется при удержаний суммы с работника в пользу получателя документом "Расчет ИПН, ОПВ и удержаний", расход по регистру производится при перечислении (выплате) суммы получателю документами "Расходный кассовый ордер", "Платежное поручение исходящее" или "Авансовый отчет".

Для анализа взаиморасчетов с получателями удержаний по исполнительным листам предназначен отчет "Структура задолженности организаций перед получателями ИЛ".
