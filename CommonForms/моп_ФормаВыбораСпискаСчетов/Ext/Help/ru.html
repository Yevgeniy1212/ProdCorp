Форма выбора списка счетов

Конфигурация   "1С-Рейтинг: Налоговый мониторинг".

 

Форма предназначена для выбора списка счетов из плана счетов Типовой.

Кнопка "Подбор" предназначена для группового добавления счетов. При её нажатии открывается форма списка плана счетов, из которой можно добавить сразу несколько счетов без необходимости повторного её открытия.

Кнопка "Восстановить по умолчанию" устанавливает перечень счетов, который был задан по умолчанию.

Для выбора могут быть доступны не все счета. Доступность тех или иных счетов определяется тем для каких целей они выбираются. Например, при выборе счетов для заполнения отчета Форма 1.7. Расшифровка  дебиторской и кредиторской задолженности будут доступны только те счета, которые имеют субконто "Контрагенты" и для дебиторской задолженности будут доступны только активные счета, а для кредиторской  - пассивные и активно-пассивные.
