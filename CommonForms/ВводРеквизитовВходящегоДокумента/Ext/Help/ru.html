Ввод реквизитов входящего документа

Дополнительная форма "Ввод реквизитов входящего документа" предназначена для заполнения либо изменения реквизитов документа "Вид входящего документа", "Дата входящего документа", "Номер входящего документа". Данная форма открывается в том случае, если при записи нового документа не заполнено хотя бы одно из вышеперечисленных полей. При открытии формы данные поля заполняются значениями по умолчанию:
- вид входящего документа -  "Накладная/Чек",
- дата входящего документа - дата документа, из которого была вызвана данная форма
- номер входящего документа заполняется вручную.

При нажатии кнопки "ОК" введённые данные записываются в соответствующие реквизиты документа. При нажатии кнопки "Закрыть", форма закрывается без внесения каких либо изменений в документ
